常见问题

怎么制作表情包？

选择内置模板或上传自己的图片（支持JPG/PNG），在顶部和底部输入文字，调整字体大小、颜色后点击「生成表情包」即可。支持拖拽调整文字位置，完成一键下载。

支持哪些图片格式？

支持 JPG、PNG、GIF 等常见图片格式上传。建议使用 800×800 像素以上的图片以获得最佳效果。生成的图片为 PNG 格式，支持透明背景。

我的图片会被上传到服务器吗？

不会。所有图片处理和文字叠加完全在您的浏览器中通过 Canvas API 完成。图片不会上传到任何服务器，您可以放心使用，包括商用图片。
