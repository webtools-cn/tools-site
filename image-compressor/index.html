📖 使用教程

点击页面中央的「选择文件」按钮或直接拖拽图片到上传区域，即可将图片添加到工具中。支持 JPG、PNG、WebP、GIF、BMP 等多种格式，单次可上传多张图片进行批量处理。上传后，每张图片会显示原图尺寸和大小。

在压缩设置区域，可以调整输出格式和质量参数。质量 slider 范围从 1 到 100，数值越低压缩率越高、文件越小，但图片质量会相应下降。建议根据使用场景选择：社交媒体分享推荐 60-80%，网页展示推荐 70-85%，需要高保真打印的场景建议 90% 以上。输出格式可选择保持原格式或统一转换为 JPEG / PNG / WebP。

设置完成后，点击「开始压缩」按钮，工具会在浏览器本地完成压缩处理。压缩完成后，每张图片会显示压缩前后的尺寸、大小对比和压缩率。点击「下载」按钮可保存单张图片，或点击「打包下载」将所有压缩后的图片打包为 ZIP 文件一次性下载。所有处理均在本地完成，图片不会上传到任何服务器。

🎯 应用场景

图片压缩工具在网站开发和日常工作中非常实用。前端开发者在优化网页性能时，经常需要将设计稿或素材图压缩到合适的大小，以减少页面加载时间和带宽消耗。一张原本 2MB 的高清图片，经过适当压缩后可能只有 200KB，加载速度提升 10 倍。

💡 扩展知识

图片压缩主要分为有损压缩和无损压缩两种方式。有损压缩（如 JPEG）通过丢弃人眼不太敏感的高频信息来减小文件体积，压缩率较高但会损失部分细节。无损压缩（如 PNG）则保留所有原始信息，通过更高效的编码方式减小体积，适合包含文字、线条或需要精确色彩还原的图片。

WebP 是 Google 开发的新一代图片格式，同时支持有损和无损压缩，在相同质量下通常比 JPEG 小 25-35%，比 PNG 小 26%。目前主流浏览器都已支持 WebP 格式。需要注意的是，过度压缩会导致图片出现明显的压缩伪影（Artifacts），特别是在包含渐变色或文字的区域，建议根据实际用途选择合适的压缩参数。

❓ 常见问题

图片压缩工具能做什么？

本工具可以在浏览器本地对图片进行压缩、格式转换和尺寸调整。支持 JPEG、PNG、WebP 互转，通过调节质量参数控制压缩比，还能批量处理多张图片，大幅提升网页加载速度和节省存储空间。

JPEG、PNG、WebP 怎么选？

JPEG 适合照片类图片，压缩率高但有损；PNG 适合图标、截图等需要透明背景的图片，无损但体积大；WebP 是新一代格式，压缩率比 JPEG 高 25-35% 且支持透明，兼容性也很好，推荐优先使用。

压缩后画质会变差吗？

JPEG 和 WebP 是有损压缩，质量设置越低画质越差。建议将质量保持在 70-85% 之间，这是画质与体积的最佳平衡点。PNG 为无损压缩，画质不变但体积压缩有限。您可以通过对比滑块实时查看效果。

批量压缩怎么用？

上传时一次选择多张图片，或在拖拽区域一次性拖入多个文件。设置好统一的输出格式、质量和尺寸后，点击「批量压缩全部」即可并行处理。处理完成后可以逐个下载，也可以点击「打包下载全部」一次性获取 ZIP 文件。

图片会发送到服务器吗？

绝对不会。本工具采用纯前端 Canvas API 在浏览器本地完成所有处理，图片数据仅在您的设备内存中短暂存在，处理完毕后立即释放。无需注册、无需联网验证，甚至可以断网使用。

支持的最大图片尺寸是多少？

理论上受限于浏览器内存，一般 50MB 以内的图片都能正常处理。建议单次批量不超过 50 张。如果图片尺寸过大，可以设置最大宽度/高度来限制输出尺寸，避免因超大 Canvas 导致浏览器卡顿。
