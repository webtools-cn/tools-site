这款免费的Base64转文件解码器将Base64编码的字符串还原为原始二进制文件。无论您需要从CSS data URI中提取图片、解码Base64编码的PDF，还是转换任何Base64数据，此工具都能在浏览器本地完成全部处理。

常见问题

原始Base64和data URI有什么区别？

data URI包含前缀（如data:image/png;base64,）后跟Base64字符串。原始Base64只是编码数据没有前缀。此工具支持两种格式——自动检测data URI前缀并提取MIME类型，也可以手动为原始Base64字符串指定文件类型。

能同时解码多个Base64字符串吗？

目前工具一次处理一个Base64字符串。如需处理多个文件，可以逐个粘贴并解码，下载结果后再处理下一个。

我的数据安全吗？

安全。所有解码在浏览器本地使用内置的atob()函数和Blob API完成。您的Base64数据永远不会传输到任何服务器。这使得工具适合解码API密钥、证书或私人文档等敏感数据。

为什么解码后的文件看起来损坏了？

通常是因为Base64字符串不完整或包含无效字符。请确保复制完整的Base64字符串，没有截断。另请验证MIME类型是否正确——PNG被解码为JPEG会显示为损坏。

最大Base64字符串大小是多少？

取决于浏览器和设备内存。大多数浏览器可处理最大50MB的Base64字符串（解码后约37MB）。超大文件可能导致浏览器变慢或内存不足错误。
