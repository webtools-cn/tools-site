❓ 常见问题 (FAQ)

什么是摩斯密码？
摩斯密码（Morse Code）是一种信号编码方式，由萨缪尔·摩斯在1836年发明。它使用点（.）和划（-）两种基本信号的不同组合来表示字母、数字和标点符号。摩斯密码可以通过声音、灯光、电信号等多种媒介传输。
摩斯密码中的点和划的时长规则是什么？
摩斯密码的计时单位是"点"（dit）的时长：1划（dah）= 3个点时长。字符内点划间隔 = 1个点时长。字符间间隔 = 3个点时长。单词间间隔 = 7个点时长。本工具可播放符合标准时长的摩斯码音频。
什么是SOS信号？
SOS是国际通用的摩斯密码遇险信号，由三个短、三个长、三个短组成（···---···）。注意SOS不是缩略词，而是因为它的摩斯码最简单易辨。SOS信号在航海、航空遇险时仍被广泛使用。
摩斯密码在现代还有用吗？
是的，摩斯密码在现代仍有实际用途：1）业余无线电通信；2）航空导航信标（如NDB台站标识）；3）紧急情况下的备用通信手段；4）辅助残障人士交流；5）军事通信训练。美国联邦通信委员会（FCC）仍颁发摩斯密码操作员执照。
如何快速学习摩斯密码？
建议学习方法：1）先学习基本字母（E·、T-、A·-、N-·等高频字母优先）；2）使用凯斯韦尔法或法恩斯沃思法进行听力训练；3）每天练习15-20分钟；4）使用本工具的音频播放功能辅助学习；5）在lcwo.net等在线平台进行系统训练。
