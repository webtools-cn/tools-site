使用方法

按照以下简单步骤测量图片中的像素距离。所有处理都在你的浏览器中本地完成。

  1. 上传一张需要测量的截图或设计稿图片。
  2. 选择"距离"或"角度"测量模式。
  3. 在图片上点击两个点，测量结果实时显示。
  4. 使用"清除标注"重新开始，或"导出截图"保存标注结果。

常见问题

像素尺是什么？有什么用？

像素尺是一种在图片上测量距离的工具。设计师可以用它测量UI元素的间距和尺寸，前端开发者可以用它验证设计稿的像素精度，图片编辑者可以用它测量图片中物体的实际像素大小。

测量结果准确吗？

测量精度受图片缩放比例影响。工具会考虑图片的实际像素尺寸和显示尺寸之间的比例，给出原始图片像素级的准确测量结果。建议使用原始尺寸图片获得最精确的测量。

支持哪些测量功能？

支持距离测量（两点间像素距离）和角度测量（两条线之间的角度）。测量结果实时显示在图片上和下方的信息面板中，包括距离、宽度差、高度差和角度值。

我的图片会上传吗？

绝对不会。本工具纯前端运行，你的图片只在浏览器内存中处理，不会发送到任何服务器。你可以断网使用。

可以测量屏幕上的元素吗？

本工具基于上传的图片进行测量。你可以截图后上传到工具中进行测量。对于浏览器外的屏幕元素，建议使用系统自带的截图工具截图后再用本工具测量。

技术说明

本工具使用HTML5 Canvas API实现图片渲染和测量标注。所有测量计算在浏览器本地完成，考虑了图片实际像素尺寸和展示缩放比例，确保测量结果准确。数据永远不会发送到任何服务器。
