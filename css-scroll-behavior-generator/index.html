📖 CSS scroll-behavior 详解

scroll-behavior 控制元素滚动时的过渡动画。设置为 smooth 后，所有内部锚点跳转和 scrollIntoView() 调用都会带有平滑动画。

配合属性

  • scroll-padding — 定义滚动容器的内边距，让目标元素不会紧贴容器边缘
  • scroll-margin — 定义单个元素的外边距，用于锚点定位时的额外偏移
  • scroll-snap-type / scroll-snap-align — 滚动捕捉，做轮播/分页效果
