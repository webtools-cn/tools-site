📖 使用说明

JSON转Properties工具将JSON对象转换为Java Properties格式的key=value文本。

使用方法

  1. 在输入框中粘贴JSON文本
  2. 点击"转换"按钮
  3. 嵌套对象会自动展开为扁平键（如 server.port=8080）
  4. 点击"复制"获取Properties文本

转换规则

  • 嵌套对象 — 使用点号分隔（a.b.c=value）
  • 数组 — 使用索引作为键（items.0=value）
  • 特殊字符 — 自动转义等号、冒号、换行

嵌套的JSON对象如何处理？

嵌套对象会被展开为扁平键。例如 {"server":{"port":8080}} 会变成 server.port=8080。数组使用索引作为键。

使用时数据会上传到服务器吗？

不会。所有转换都在你的浏览器中完成，数据不会上传到任何服务器。

特殊字符需要转义吗？

是的，工具会自动处理Properties格式的特殊字符转义。例如等号(=)和冒号(:)会被转义。

支持哪些JSON格式？

支持标准JSON格式，包括对象、数组、字符串、数字、布尔值和null。
