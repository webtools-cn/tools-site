JSON差异对比工具能做什么？

JSON差异对比工具是一款纯前端在线工具，用于精确计算两个JSON对象之间的差异。输入原始JSON和修改后JSON，工具自动检测新增、删除和修改的字段，生成符合RFC 6902规范的JSON Patch，并可视化展示每个变化的具体路径和值。支持深度递归对比和浅层对比两种模式，数组支持按索引和按值匹配两种策略。所有数据在浏览器本地处理，绝不上传服务器。

核心功能

  • 增删改检测：精确识别新增字段（add）、删除字段（remove）和值变更（replace），每个变化标注完整JSON Path
  • JSON Patch生成：输出符合RFC 6902标准的JSON Patch，可直接用于API请求实现部分更新
  • 深度/浅层对比：深度模式递归比较所有嵌套层级，浅层模式仅比较顶层键值
  • 数组对比策略：按索引逐项对比或按值智能匹配，适应不同业务场景
  • 可视化差异展示：用颜色区分增删改，直观展示每个变化的路径和值

使用教程

步骤1：在左侧输入框粘贴原始JSON（A），右侧输入框粘贴修改后JSON（B），或拖拽.json文件上传。

步骤2：选择对比模式——深度对比（递归所有层级）或浅层对比（仅顶层），以及数组对比策略。

步骤3：点击"对比差异"按钮或使用Ctrl+Enter快捷键，工具自动计算差异。

步骤4：查看差异详情和JSON Patch，点击"复制Patch"或"下载Patch"保存结果，Ctrl+Shift+C一键复制。

扩展知识

JSON Patch（RFC 6902）是一种描述JSON文档变化的标准格式，包含6种操作：add（添加）、remove（删除）、replace（替换）、move（移动）、copy（复制）、test（测试）。JSON Pointer（RFC 6901）用于定位JSON文档中的特定值，如/foo/0/bar表示根对象foo数组第0项的bar字段。与JSON Merge Patch（RFC 7396）不同，JSON Patch更精确，能区分null值删除和字段不存在。
