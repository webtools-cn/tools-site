功能说明

按行去重

将输入文本按换行符分割，逐行检查是否重复。首次出现的行保留，后续重复行自动去除。适合处理列表、日志记录、数据导出等场景。

按词去重

将输入文本按指定分隔符（逗号、空格、分号等）分割为词，对每个词去重。适合处理关键词列表、标签列表、CSV字段等。

大小写敏感

开启时 Hello 和 hello 视为不同文本，均保留。关闭时视为重复，仅保留首次出现的。处理英文内容时根据需求选择。

去除空白行/空项

自动移除仅包含空格、制表符或完全空白的行/词。整理从 Excel 或数据库导出的数据时非常有用。

结果排序

去重后可对结果按字母顺序排序，方便对比和查找。默认保留原始顺序。

使用教程

本工具支持按行和按词两种去重模式，可灵活控制大小写敏感、空白行处理和排序选项，是处理文本列表、关键词去重、数据清洗的实用工具。以下是详细使用步骤：

第一步：输入待去重文本。在输入文本框中粘贴需要去重的内容。按行去重模式下，每一行作为一个去重单元；按词去重模式下，文本会按指定分隔符拆分为词进行去重。输入区下方会实时显示当前文本的行数和字符数。

第二步：配置去重选项。根据需要调整以下选项：选择「按行去重」或「按词去重」模式；按词去重时选择分隔符（逗号、空格、分号或自定义）；开启或关闭「大小写敏感」；勾选「去除首尾空格」自动清理每行/每词前后的空白字符；勾选「去除空白行/空项」过滤空行和空词；如需排序结果则勾选「结果排序」。

第三步：执行去重。点击「🧹 立即去重」按钮，工具会立即处理并显示去重后的结果。结果区下方会显示原始行数、去重后行数和重复行数三项统计。如检测到重复项，下方会自动展示被去除的重复项列表，每项标注重复次数，方便您核对去重结果。

第四步：复制或下载结果。点击「📋 复制结果」将去重后的文本复制到剪贴板，或点击「💾 下载为TXT」将结果保存为文本文件。如需对结果再次处理，可点击「↑ 结果填入输入」将结果回传至输入区继续操作。

应用场景

关键词列表去重：SEO运营人员和广告投放专员经常需要合并多个关键词列表，去除重复关键词后进行批量分析。将多个列表粘贴到输入框中，选择按行去重模式，一键即可得到去重后的干净列表，配合排序功能还能快速生成按字母排序的关键词清单。

数据清洗与预处理：从 Excel、CSV、数据库导出的原始数据经常包含重复记录。使用本工具按行去重，可以快速清理重复行、空行和格式不一致的记录，为后续数据分析或导入提供干净的数据源。去除首尾空格功能还能规范化因格式不一致导致的「假重复」问题。

日志分析与去重：运维和开发人员在排查问题时，经常面对大量重复的日志条目。将日志粘贴到工具中按行去重，可以快速过滤掉重复的报错信息，聚焦于唯一的异常记录，大幅提升排查效率。配合大小写不敏感模式，还能合并仅大小写不同的同类日志。

邮件/通讯录去重：整理客户邮件列表或联系人信息时，可能因为多次收集导致同一邮箱重复出现。使用本工具按行去重，一键去除重复邮箱，确保每个联系人只出现一次，避免重复发送邮件造成的困扰。

扩展知识

去重算法原理：本工具采用基于哈希表（Hash Set）的去重算法。将每个文本项作为键存入 Set 数据结构，利用 Set 的唯一性特性自动去重。JavaScript 的 Set 基于 SameValueZero 算法判断相等性，对于字符串比较与 === 运算符行为一致（区别在于 NaN 等于 NaN）。时间复杂度为 O(n)，即处理速度与输入项数成线性关系，即使处理数万行也能在毫秒级完成。

顺序保留的重要性：去重时保留首次出现的顺序（而非排序后去重）在许多场景中至关重要。例如处理有先后逻辑顺序的日志、按时间排列的记录、或按优先级排序的列表时，打乱顺序可能导致信息丢失或理解偏差。本工具默认使用 LinkedHashMap 思想实现：用 Set 记录已出现项，用数组维护输出顺序，确保结果与原文顺序一致。

大小写敏感与数据规范化：在处理英文数据时，大小写不一致是导致「假重复」的常见原因。例如 Apple、APPLE、apple 在大小写不敏感模式下会被视为同一项。建议先使用「去除首尾空格」清理格式，再根据数据特性决定是否开启大小写敏感。对于编程变量名、邮箱地址等大小写有意义的场景应保持敏感；对于一般文本描述则可关闭以获得更好的去重效果。

常见问题

文本去重工具是怎么工作的？

工具将输入文本按换行符分割成行（或按指定分隔符分割成词），然后逐行检查是否已出现过。首次出现的行保留，后续重复的行自动去除。默认保留首次出现的位置顺序，因此结果中行的排列与原文一致。

大小写敏感是什么意思？

开启大小写敏感时，Hello 和 hello 被视为两行不同的文本，都会保留。关闭大小写敏感时，Hello 和 hello 被视为重复，仅保留首次出现的 Hello。处理英文关键词列表时建议根据需求选择。

可以去除空白行吗？

可以。勾选「去除空白行」选项后，工具会自动移除仅包含空格、制表符或完全空白的行。这在整理从 Excel、数据库或日志文件导出的数据时非常有用。

按行去重和按词去重有什么区别？

按行去重以换行符为分隔，每一整行作为一个单元进行去重，适合处理列表、日志、数据记录。按词去重以空格、逗号或自定义分隔符分割文本，对每个词进行去重，适合处理关键词列表、标签列表等。

我的数据会上传到服务器吗？

不会。本工具为纯前端实现，所有去重操作均在您的浏览器本地完成，数据不会发送到任何服务器。即使断网也能正常使用，完全保护您的数据隐私。

去重后可以查看哪些被删除了吗？

可以。去重完成后，工具会自动统计原始行数、去重后行数和重复行数，并在结果区下方展示被去除的重复行列表，方便您核对去重结果是否正确。
