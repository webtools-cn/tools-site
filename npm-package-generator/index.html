常见问题

package.json 有哪些必须字段？

最核心的字段是 name 和 version，这是 NPM 包唯一标识。推荐包含 description、main、scripts、keywords、author、license 等字段。如果发布到 NPM，还需要设置 repository、bugs、homepage 等。

如何选择开源许可证？

常见选择：MIT（最宽松，允许商用）、Apache-2.0（类似 MIT 但含专利授权）、GPL-3.0（传染性要求必须开源）、ISC（类似 MIT，更简短）、BSD-2-Clause/BSD-3-Clause。不确定时推荐 MIT。

package.json 中的版本号格式？

采用语义化版本（SemVer）MAJOR.MINOR.PATCH。版本号前可加符号：^（兼容主版本内更新）、~（兼容补丁版本更新）、无符号（锁定版本）、*（任意版本）。建议首次发布使用 1.0.0。
