使用说明

使用Web Crypto API在浏览器本地计算HMAC哈希值。适用于API签名、JWT认证、webhook验证、消息完整性校验等场景。

主要功能：

  • 🔐 支持 HMAC-SHA256 / SHA384 / SHA512
  • 🔣 支持 Hex 和 Base64 两种编码输出
  • ⚡ 使用浏览器原生 Web Crypto API
  • 📋 一键复制计算结果
  • 🔒 纯本地运算，密钥和数据不上传
