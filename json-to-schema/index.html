常见问题 (FAQ)

什么是JSON Schema？

JSON Schema是描述JSON数据结构的标准规范（RFC 8927）。它定义字段类型、必填项、取值范围等约束，广泛用于API文档（Swagger/OpenAPI）、数据验证、IDE自动补全等场景。

生成的Schema遵循哪个版本？

生成Draft-7版本（$schema: http://json-schema.org/draft-07/schema#）。这是目前兼容性最好、使用最广泛的版本，支持所有主流JSON Schema验证器。

如何判断必填字段？

工具假设所有出现的字段都是必填的。如果JSON中有null值，该字段会被标记为可选（type数组中包含null）。建议用完整的JSON示例来获得准确的Schema。

数组元素如何推断？

如果数组非空，会根据第一个元素推断items类型。如果数组包含不同类型的元素，会使用anyOf列出所有可能的类型。空数组默认items为空对象{}。
