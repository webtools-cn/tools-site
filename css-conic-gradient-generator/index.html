❓ 常见问题

Conic Gradient 和 Linear/Radial Gradient 有什么区别？

线性渐变(Linear)沿直线过渡，径向渐变(Radial)从中心向外扩散，而圆锥渐变(Conic)则围绕中心点按角度旋转过渡，类似扇形或饼图效果。Conic Gradient 非常适合制作饼图、色环、日出日落效果等。

重复圆锥渐变有什么用？

重复圆锥渐变(repeating-conic-gradient)会在 360° 之后继续重复色标模式，适合创建齿轮效果、万花筒纹理、雷达扫描图等需要周期性重复的场景。

浏览器兼容性如何？

CSS conic-gradient() 在现代浏览器中拥有良好支持，包括 Chrome 69+、Firefox 83+、Safari 12.1+、Edge 79+。建议为不支持的老旧浏览器提供回退背景色。
