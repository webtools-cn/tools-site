📖 关于JSON转TypeScript

功能说明

将JSON数据自动转换为TypeScript的interface或type类型定义。支持嵌套对象、数组类型推断、联合类型、可选字段等特性。

使用场景

  • 从API响应快速生成TypeScript类型定义
  • 为已有JSON数据创建类型安全的接口
  • 加速TypeScript项目中的类型编写
  • 将配置文件转换为类型定义

支持的JSON类型映射

  • string → string
  • number → number
  • boolean → boolean
  • null → null
  • array → Type[]
  • object → 嵌套interface
