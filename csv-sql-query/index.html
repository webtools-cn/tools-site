常见问题

Q1: 支持哪些SQL语法？

基于AlaSQL引擎，支持SELECT、WHERE、ORDER BY、GROUP BY、HAVING、JOIN、UNION、DISTINCT、LIMIT、OFFSET、聚合函数（COUNT/SUM/AVG/MIN/MAX）、字符串函数、日期函数等标准SQL语法。表名统一为csvdata。

Q2: 数据会上传到服务器吗？

完全不会！所有数据处理都在浏览器本地完成，使用AlaSQL JavaScript引擎在内存中执行SQL查询，数据绝不上传到任何服务器，安全可靠。

Q3: 支持多大的CSV文件？

建议CSV文件不超过10MB（约10万行），超过此大小可能导致浏览器内存不足或查询速度变慢。对于大数据量，建议先过滤再查询，或使用LIMIT限制结果行数。
