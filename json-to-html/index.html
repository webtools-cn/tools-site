嵌套JSON如何处理？

嵌套对象会被自动展开为多列，使用点号分隔列名。例如 {"address":{"city":"北京"}} 会展开为 address.city 列。嵌套数组会被转换为JSON字符串显示在单元格中。如果勾选"展开嵌套对象"选项，所有层级的对象都会被递归展开。

生成的HTML代码可以直接用在项目中吗？

可以。生成的HTML代码包含完整的table标签和内联CSS样式，可以直接复制粘贴到任何HTML页面中使用。代码是自包含的，不依赖外部CSS或JS文件。你也可以选择不同的样式主题来匹配你的项目风格。
