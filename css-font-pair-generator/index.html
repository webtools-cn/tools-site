💡 配对建议

经典组合: 衬线 + 无衬线（如 Playfair Display + Source Sans Pro）
现代风格: 无衬线 + 无衬线（如 Inter + Roboto）
优雅风格: 手写体 + 衬线（如 Cormorant Garamond + Lora）
技术风格: 等宽 + 无衬线（如 Fira Code + Inter）
规则: 同一种分类的字体通常不太适合搭配，除非有明显的大小/粗细对比。

❓ 常见问题

如何选择合适的字体搭配？

一个稳妥的原则是：选择一款标题字体和一款正文字体，确保它们有足够的对比度（如衬线+无衬线）。避免使用两款过于相似的字体。建议先确定品牌调性，再选择匹配的字体风格：正式场合选衬线，现代简约选无衬线，创意设计可选手写体或展示字体。

Google Fonts 加载速度如何优化？

使用 @import 或 加载 Google Fonts 时，建议仅加载需要的字重和字符集。例如：?family=Roboto:wght@400;700&subset=latin。同时可以使用 font-display: swap 确保文字在字体加载期间仍然可见，避免 FOIT（Flash of Invisible Text）。

中文字体如何搭配？

中文字体文件较大，建议仅加载常用字重（如 400/700）。中英文混排时，可为中文和英文分别指定字体：font-family: 'Noto Sans SC', 'Inter', sans-serif。推荐的中文字体搭配：Noto Sans SC + Noto Serif SC，或思源黑体 + 思源宋体。
