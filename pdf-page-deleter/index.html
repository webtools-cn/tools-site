📖使用教程

本工具帮助您快速删除PDF文档中不需要的页面，无需安装任何软件，在浏览器中即可完成操作。所有处理都在本地进行，您的文件不会上传到服务器。

操作步骤

  1. 上传PDF：点击上传区域选择PDF文件，或将PDF文件拖拽到虚线框内释放。
  2. 预览页面：文件上传后，工具会自动提取并渲染所有页面的缩略图，您可以清晰看到每一页的内容。
  3. 选择删除：点击页面缩略图即可选中要删除的页面。被选中的页面会显示红色标记边框。您可以多选不连续的页面。
  4. 批量操作：使用"全选"按钮快速选中所有页面，"反选"按钮翻转当前选择，"清空选择"取消所有选择。
  5. 生成新文件：确认选择无误后，点击"生成新PDF"按钮。工具会从原始文件中移除被选中的页面，保留其余页面并生成新的PDF文件。
  6. 下载保存：新生成的PDF会自动触发浏览器下载。建议在下载完成后核对页面是否正确。

注意事项

  • 处理前建议备份原始PDF，以防误删重要页面。
  • 受密码保护的PDF需要先解密才能使用本工具。
  • 超大文件可能需要较长时间渲染预览，请耐心等待。
  • 生成的PDF会保留剩余页面的原始格式和字体。

🔍扩展知识

PDF页面结构

PDF（Portable Document Format）是一种跨平台的文档格式，其内部采用对象树结构存储页面。每一页都是一个独立的对象，包含内容流、资源字典和页面属性。删除页面操作本质上是修改文档的页面树节点，移除指定页面对象的引用，而不会影响其他页面的数据完整性。

纯前端PDF处理技术

本工具基于 pdf-lib.js 库实现，该库完全在浏览器环境中运行，利用 JavaScript 的 ArrayBuffer 和 Uint8Array 直接操作 PDF 二进制数据。与服务器端处理相比，纯前端方案具有零延迟、零上传、高隐私的优势，特别适合处理敏感文档。

PDF页面旋转与删除的区别

页面旋转仅修改页面的旋转属性（Rotate 字段），不改变文档结构和页数。而页面删除则是永久移除页面对象，会改变总页数和页面编号。删除操作不可逆，执行前务必确认。

❓常见问题

Q1: 删除PDF页面后文件会上传到服务器吗？

A: 不会。本工具完全在浏览器本地运行，所有PDF处理都在您的设备上完成，不会上传到任何服务器。删除操作和文件下载均在本地进行，确保您的文档隐私安全。

Q2: 支持删除多个页面吗？

A: 支持。您可以勾选任意数量的页面进行删除，包括不连续的页面。删除后会生成一个新的PDF文件，仅保留未被勾选的页面，且页面顺序保持不变。

Q3: 删除页面后PDF的格式会改变吗？

A: 不会。工具使用pdf-lib库进行处理，能够完整保留原始PDF中剩余页面的内容、字体、图片、链接等元素。仅移除被删除的页面，其余页面格式完全保留。

Q4: PDF文件大小有限制吗？

A: 单次处理建议文件大小不超过200MB。由于纯前端处理，文件大小受限于浏览器内存。超大文件可能导致浏览器卡顿，建议分多次处理或使用桌面版PDF工具。

Q5: 可以撤销删除操作吗？

A: 在点击生成新PDF之前，您可以随时取消勾选来撤销删除选择。但一旦生成并下载了新PDF，就无法自动撤销。建议先备份原始PDF文件，以防误删重要页面。

Q6: 支持密码保护的PDF吗？

A: 不支持。受密码保护的PDF文件需要先移除密码才能使用本工具。您可以使用其他PDF解密工具处理后再进行页面删除操作。

Q7: 删除页面后书签和目录会保留吗？

A: 部分保留。删除后的PDF会保留原始文件的大部分元数据和书签，但指向已删除页面的书签可能失效。复杂的多层书签结构建议在删除后手动检查。

Q8: 手机浏览器可以使用吗？

A: 可以。本工具采用响应式设计，在手机浏览器上同样可用。但由于手机屏幕较小，预览缩略图可能显示较小，建议在电脑端操作以获得最佳体验。
