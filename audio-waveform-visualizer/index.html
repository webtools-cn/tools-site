💡 说明

什么是音频波形？

音频波形是声音信号的图形化表示，横轴表示时间，纵轴表示振幅（音量）。波形的起伏反映了声音的强弱变化。

如何处理？

本工具使用 Web Audio API 解码音频文件并提取波形数据，所有处理在浏览器本地进行，文件不上传到服务器。

支持哪些格式？

支持浏览器支持的所有音频格式，包括 MP3、WAV、OGG、M4A (AAC)、FLAC 等。推荐使用 MP3 或 WAV 格式。
