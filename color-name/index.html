常见问题

这个工具能做什么？

输入任意颜色的Hex值（如#FF0000）、RGB值（如rgb(255,0,0)）或HSL值，工具会立即显示该颜色的预览色块、CSS标准颜色名称（如red、coral、skyblue）以及最接近的相近颜色名称。支持150多种CSS标准颜色名称。

支持哪些输入格式？

支持三种输入格式：1) Hex格式：#FF0000、#f00、FF0000（带或不带#号）；2) RGB格式：rgb(255,0,0)、255,0,0；3) HSL格式：hsl(0,100%,50%)。工具自动检测输入格式并转换，无需手动选择。

如何找到颜色的官方名称？

本工具内置了150+ CSS标准颜色名称数据库（包括CSS Color Module Level 4标准中的所有颜色）。输入颜色值后，工具通过计算欧几里得距离在RGB色彩空间中查找最接近的颜色名称，并显示颜色匹配度（0-100%）。

为什么有些颜色找不到准确名称？

CSS标准颜色名称只有150多种，而人眼可以分辨约1000万种颜色。因此大多数颜色没有准确的官方名称。对于这些颜色，工具会显示最接近的CSS颜色名称及匹配度百分比。匹配度越高，表示越接近该标准色。

颜色名称在网页设计中有什么作用？

使用颜色名称（而非Hex值）在CSS中定义颜色可以提高代码可读性。如 color: steelblue 比 color: #4682B4 更直观。同时，颜色名称的语义化有助于团队协作和后期维护。但注意颜色名称在不同浏览器中的渲染可能有轻微差异。
