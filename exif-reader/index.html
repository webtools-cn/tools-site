什么是EXIF数据？

EXIF（Exchangeable Image File Format）是嵌入在照片文件中的元数据标准。它记录了拍摄时的各种参数，包括相机型号、镜头信息、光圈、快门速度、ISO感光度、焦距、白平衡、GPS地理位置、拍摄日期和时间等。

照片会传到服务器吗？

不会。所有EXIF数据解析在浏览器本地使用JavaScript完成，照片文件不上传任何服务器。你可以验证：打开DevTools → Network标签，看不到任何上传请求。

支持哪些照片格式？

主要支持JPEG/JPG格式（EXIF数据通常嵌入在JPEG文件中）。部分PNG和WebP文件也可能包含元数据，但不保证完全解析。

为什么有些照片读取不到EXIF数据？

可能原因：1) 图片格式不是JPEG；2) 照片经过重压缩或社交平台上传（微信/微博等会清空EXIF）；3) 拍照时未启用位置记录；4) 隐私保护工具已清除了EXIF数据。

需要联网使用吗？

纯前端应用，加载后断开网络也能正常使用。所有解析在浏览器本地完成。

EXIF数据包含哪些信息？

📸 相机信息：相机型号、镜头信息、软件版本

⚙️ 拍摄参数：光圈（F值）、快门速度、ISO感光度、焦距、曝光补偿、闪光灯、白平衡、测光模式

📍 GPS信息：纬度、经度、海拔高度（如果拍摄时开启了位置记录）

📅 日期时间：拍摄日期、数字化日期、修改日期

📁 文件信息：文件大小、图片尺寸（宽×高）、分辨率

常见问题

Q：EXIF数据怎么查看？为什么我看不到GPS信息？

EXIF数据是照片文件的一部分，由相机或手机自动写入。GPS信息需要拍摄时启用了位置记录功能。部分设备和App默认关闭GPS标记，或在分享时自动清除了位置信息。

Q：如何清除照片的EXIF数据？

可以使用本工具查看EXIF后，用专业EXIF清理工具移除敏感信息（如GPS位置）。常见方法：1) Windows右键→属性→详细信息→删除属性和个人信息；2) 使用批量EXIF清理软件；3) 社交平台分享时大多会自动清除EXIF。

Q：哪些设备写入EXIF数据？

几乎所有数码相机、DSLR单反、微单相机、智能手机都写入EXIF数据。不同设备写入的信息量不同：专业相机写入完整的拍摄参数，手机通常包含GPS信息，老款手机可能只记录基本参数。

Q：EXIF数据会被修改吗？

可以修改。很多图片编辑软件和专用EXIF编辑器可以修改EXIF标签。但部分司法鉴定场景下，修改后的EXIF可以被检测出来。在社交平台上传照片后，EXIF数据通常会被压缩或清除。
