💡 使用说明

  • 将 CSS 代码粘贴到左侧输入框
  • 选择压缩模式（标准/极限）
  • 点击「压缩 CSS」即可获得压缩结果
  • 点击「复制」一键复制压缩后的代码
  • 点击「下载」保存为 .min.css 文件

⚙️ 压缩策略

  • 移除所有注释（/* ... */）
  • 移除多余空格和换行
  • 缩短十六进制颜色值（#ff0000 → #f00）
  • 合并相同选择器的规则
  • 移除不必要的分号和引号
  • 极限模式额外移除最后一个分号
