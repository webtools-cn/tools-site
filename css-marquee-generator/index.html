常见问题

CSS跑马灯效果有哪些实现方式？

主要有两种方式：1）HTML 标签 - 简单但已废弃，不支持所有浏览器；2）纯CSS animation + @keyframes - 现代推荐方式，兼容性好且可自定义更多效果。本工具支持两种方式生成。

纯CSS跑马灯相比HTML marquee有什么优势？

纯CSS实现优势包括：更好的浏览器兼容性（包括移动端）、支持暂停/继续、更流畅的动画效果、可自定义缓动函数、可组合其他CSS效果（如渐变文字、阴影）、不依赖废弃标签。

跑马灯效果适合哪些场景使用？

常见应用场景：网站公告栏/通知栏、新闻头条滚动、社交媒体信息流、股价/数据实时显示、广告横幅、音乐播放器歌词滚动、活动倒计时滚动提示等。

这个工具会收集我生成的代码吗？

完全不会。所有CSS代码生成都在您的浏览器本地完成，不会上传到任何服务器。您可以断开网络离线使用。

生成的代码兼容哪些浏览器？

纯CSS方式兼容所有现代浏览器（Chrome/Firefox/Safari/Edge），移动端也完全支持。HTML marquee方式兼容Chrome和Firefox，Safari部分支持。
