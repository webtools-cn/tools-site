❓ 常见问题

什么是排版比例（Modular Scale）？

排版比例是一种基于固定比值逐级递增的字号系统。例如使用纯四度 1.333 比例，如果正文 16px，则下一级标题为 16×1.333≈21px，再下一级为 21×1.333≈28px，以此类推。这样生成的字体大小形成和谐的视觉层级，比随意取整更美观。

为什么推荐使用 clamp()？

clamp(MIN, PREFERRED, MAX) 是 CSS 的流体排版函数，让字体随视口变化平滑缩放。它结合了小屏最小值、大屏最大值和中间值，避免了在固定断点处字体大小突变。生成的 clamp() 值可直接用于生产环境。

哪种比例最适合我？

纯四度(1.333)和纯五度(1.500)是最常用的网页排版比例，层级区分明显。黄金比例(1.618)适合强调视觉对比的设计。小比例(1.200-1.250)适合紧凑型界面如仪表盘。建议从 1.333 开始尝试。
