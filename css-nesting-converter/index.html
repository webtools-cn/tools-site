什么是CSS嵌套?

CSS嵌套是CSS原生支持的语法，允许将子选择器写在父选择器内部，类似SCSS的嵌套功能。例如 .card { h2 { color: red; } } 等同于 .card h2 { color: red; }。CSS嵌套于2023年成为标准，现代浏览器已广泛支持。

CSS嵌套的浏览器支持情况?

CSS嵌套已被Chrome 120+、Firefox 117+、Safari 17.2+支持。对于需要兼容旧浏览器的项目，可以使用本工具将嵌套CSS展开为扁平格式。

嵌套CSS和SCSS嵌套有什么区别?

原生CSS嵌套不需要预处理器，直接在浏览器运行。语法上，CSS嵌套使用&引用父选择器(与SCSS类似)，但某些情况下&是必需的(如伪元素、伪类)。SCSS需要编译步骤，而原生CSS嵌套不需要。

数据会上传到服务器吗?

不会。所有CSS转换都在浏览器本地完成，不发送任何数据到服务器。

转换会丢失CSS规则吗?

不会。转换器会保留所有CSS规则和属性，只是改变选择器的组织方式。从扁平转嵌套时，会智能识别共享前缀的选择器进行分组。
