常见问题

什么是JSONPath？

JSONPath是一种用于查询JSON数据的表达式语言，类似于XPath对XML的作用。它允许你通过路径表达式从JSON文档中提取特定的值或节点，例如 $.store.book[0].title 可以获取第一本书的标题。

支持哪些JSONPath语法？

本工具支持标准JSONPath语法，包括：$（根节点）、.（子节点）、[]（数组索引）、[*]（所有元素）、..（递归下降）、[?()]（过滤表达式）、[start:end]（数组切片）等常用操作符。

查询结果可以导出吗？

可以。查询结果支持一键复制为JSON格式，也可以下载为.json文件。匹配结果会在原始JSON中高亮显示，方便定位数据位置。

工具会保存我的JSON数据吗？

不会。本工具完全在浏览器本地运行，所有数据处理都在你的设备上完成，不会上传到任何服务器。你可以放心测试包含敏感信息的JSON数据。

JSONPath和jq有什么区别？

JSONPath是一种查询语言标准，专注于数据提取；jq是命令行工具，除了查询还支持转换和操作。本工具专注于JSONPath查询测试，适合快速验证表达式和提取数据。

过滤表达式支持哪些运算符？

过滤表达式支持比较运算符（==、!=、<、<=、>、>=）、逻辑运算符（&&、||）和正则匹配（=~）。例如 $[?(@.price < 10)] 筛选价格小于10的项。
