JSON Schema对比工具能做什么？

JSON Schema对比工具是一款免费在线工具，能够深度比较两个JSON Schema对比工具是一款免费在线工具，能够深度比较两个JSON Schema的差异。自动检测新增属性、删除属性、类型变更、约束条件修改、required字段变化等，并以可视化方式展示差异。支持递归对比嵌套对象和数组，路径用点号表示。所有对比在浏览器本地完成，数据绝不上传服务器。

核心功能

  • 深度递归对比：递归比较嵌套的properties和items，完整展示深层差异
  • 多维度差异检测：类型变更、格式变更、约束变更（min/max/length等）、required变更、枚举值变更、描述变更
  • 可视化差异展示：绿色=新增、红色=删除、黄色=修改，直观展示变更
  • 差异过滤：按变更类型（新增/删除/修改）过滤查看
  • 多格式导出：复制摘要、下载JSON差异报告、下载Markdown变更日志

使用教程

1. 在左侧输入框粘贴旧版JSON Schema。

2. 在右侧输入框粘贴新版JSON Schema。

3. 选择对比选项（深度递归、显示未变更项、过滤类型）。

4. 点击"对比差异"按钮，查看可视化差异结果。

5. 导出差异报告或变更日志。

扩展知识

JSON Schema是描述JSON数据结构的标准规范，当前最新版本为Draft 2020-12。在API开发中，Schema的变更管理至关重要——新增字段通常是向后兼容的，但删除字段或修改类型可能导致客户端崩溃。OpenAPI/Swagger规范使用JSON Schema定义请求和响应格式。良好的Schema版本管理实践包括：使用$schema声明版本、新增字段不设required、删除字段先标记deprecated、类型变更通过oneOf过渡。本工具帮助开发者快速识别Schema变更的影响范围。

❓ 常见问题

能检测哪些差异？

新增/删除属性、类型变更、约束变更（min/max/length/pattern等）、required变更、枚举值变更、描述变更、嵌套对象递归差异。

数据会上传到服务器吗？

不会。纯前端技术，所有对比在浏览器本地完成，数据绝不离开您的设备。

支持Draft几的Schema？

支持Draft-04到Draft 2020-12各版本。对比基于属性结构，不依赖特定版本语法。

如何处理嵌套对象？

递归对比嵌套的properties和items，路径用点号表示（如user.address.city）。

可以导出差异吗？

可以。支持复制摘要、下载JSON差异报告、下载Markdown变更日志。

支持拖拽文件吗？

支持。将.json文件拖拽到输入框即可自动读取内容。
