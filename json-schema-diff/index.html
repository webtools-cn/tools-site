JSON SchemaеҜ№жҜ”е·Ҙе…·иғҪеҒҡд»Җд№Ҳпјҹ

JSON SchemaеҜ№жҜ”е·Ҙе…·жҳҜдёҖж¬ҫе…Қиҙ№еңЁзәҝе·Ҙе…·пјҢиғҪеӨҹж·ұеәҰжҜ”иҫғдёӨдёӘJSON SchemaеҜ№жҜ”е·Ҙе…·жҳҜдёҖж¬ҫе…Қиҙ№еңЁзәҝе·Ҙе…·пјҢиғҪеӨҹж·ұеәҰжҜ”иҫғдёӨдёӘJSON Schemaзҡ„е·®ејӮгҖӮиҮӘеҠЁжЈҖжөӢж–°еўһеұһжҖ§гҖҒеҲ йҷӨеұһжҖ§гҖҒзұ»еһӢеҸҳжӣҙгҖҒзәҰжқҹжқЎд»¶дҝ®ж”№гҖҒrequiredеӯ—ж®өеҸҳеҢ–зӯүпјҢе№¶д»ҘеҸҜи§ҶеҢ–ж–№ејҸеұ•зӨәе·®ејӮгҖӮж”ҜжҢҒйҖ’еҪ’еҜ№жҜ”еөҢеҘ—еҜ№иұЎе’Ңж•°з»„пјҢи·Ҝеҫ„з”ЁзӮ№еҸ·иЎЁзӨәгҖӮжүҖжңүеҜ№жҜ”еңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј жңҚеҠЎеҷЁгҖӮ

ж ёеҝғеҠҹиғҪ

  • ж·ұеәҰйҖ’еҪ’еҜ№жҜ”пјҡйҖ’еҪ’жҜ”иҫғеөҢеҘ—зҡ„propertiesе’ҢitemsпјҢе®Ңж•ҙеұ•зӨәж·ұеұӮе·®ејӮ
  • еӨҡз»ҙеәҰе·®ејӮжЈҖжөӢпјҡзұ»еһӢеҸҳжӣҙгҖҒж јејҸеҸҳжӣҙгҖҒзәҰжқҹеҸҳжӣҙпјҲmin/max/lengthзӯүпјүгҖҒrequiredеҸҳжӣҙгҖҒжһҡдёҫеҖјеҸҳжӣҙгҖҒжҸҸиҝ°еҸҳжӣҙ
  • еҸҜи§ҶеҢ–е·®ејӮеұ•зӨәпјҡз»ҝиүІ=ж–°еўһгҖҒзәўиүІ=еҲ йҷӨгҖҒй»„иүІ=дҝ®ж”№пјҢзӣҙи§Ӯеұ•зӨәеҸҳжӣҙ
  • е·®ејӮиҝҮж»ӨпјҡжҢүеҸҳжӣҙзұ»еһӢпјҲж–°еўһ/еҲ йҷӨ/дҝ®ж”№пјүиҝҮж»ӨжҹҘзңӢ
  • еӨҡж јејҸеҜјеҮәпјҡеӨҚеҲ¶ж‘ҳиҰҒгҖҒдёӢиҪҪJSONе·®ејӮжҠҘе‘ҠгҖҒдёӢиҪҪMarkdownеҸҳжӣҙж—Ҙеҝ—

дҪҝз”Ёж•ҷзЁӢ

1. еңЁе·Ұдҫ§иҫ“е…ҘжЎҶзІҳиҙҙж—§зүҲJSON SchemaгҖӮ

2. еңЁеҸідҫ§иҫ“е…ҘжЎҶзІҳиҙҙж–°зүҲJSON SchemaгҖӮ

3. йҖүжӢ©еҜ№жҜ”йҖүйЎ№пјҲж·ұеәҰйҖ’еҪ’гҖҒжҳҫзӨәжңӘеҸҳжӣҙйЎ№гҖҒиҝҮж»Өзұ»еһӢпјүгҖӮ

4. зӮ№еҮ»"еҜ№жҜ”е·®ејӮ"жҢүй’®пјҢжҹҘзңӢеҸҜи§ҶеҢ–е·®ејӮз»“жһңгҖӮ

5. еҜјеҮәе·®ејӮжҠҘе‘ҠжҲ–еҸҳжӣҙж—Ҙеҝ—гҖӮ
