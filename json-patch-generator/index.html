JSON PointerжҳҜд»Җд№Ҳпјҹ

RFC 6901е®ҡд№үзҡ„и·Ҝеҫ„иЎЁиҫҫејҸпјҢз”Ё/еҲҶйҡ”й”®еҗҚпјҢеҰӮ/user/nameгҖӮж•°з»„з”Ёж•°еӯ—зҙўеј•гҖӮ

е’ҢJSON Merge Patchзҡ„еҢәеҲ«пјҹ

ж•°жҚ®дјҡдёҠдј еҲ°жңҚеҠЎеҷЁеҗ—пјҹ

дёҚдјҡгҖӮжүҖжңүеҜ№жҜ”е’ҢиЎҘдёҒз”ҹжҲҗеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗгҖӮ

JSON Patchз”ҹжҲҗеҷЁиғҪеҒҡд»Җд№Ҳпјҹ

JSON Patchз”ҹжҲҗеҷЁжҳҜдёҖж¬ҫе…Қиҙ№еңЁзәҝејҖеҸ‘иҖ…е·Ҙе…·пјҢйҖҡиҝҮеҜ№жҜ”дёӨдёӘJSONж–ҮжЎЈиҮӘеҠЁз”ҹжҲҗRFC 6902ж ҮеҮҶзҡ„JSON PatchиЎҘдёҒгҖӮж”ҜжҢҒaddгҖҒremoveгҖҒreplaceгҖҒmoveгҖҒcopyгҖҒtestе…ӯз§Қж“ҚдҪңпјҢеҸҜеә”з”ЁиЎҘдёҒйӘҢиҜҒз»“жһңжҳҜеҗҰдёҺзӣ®ж ҮдёҖиҮҙгҖӮйҖӮз”ЁдәҺAPIйғЁеҲҶжӣҙж–°гҖҒй…ҚзҪ®еҸҳжӣҙиҝҪиёӘгҖҒзүҲжң¬е·®ејӮеҗҢжӯҘзӯүеңәжҷҜгҖӮжүҖжңүи®Ўз®—еңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј жңҚеҠЎеҷЁгҖӮ

ж ёеҝғеҠҹиғҪ

  • иҮӘеҠЁеҜ№жҜ”пјҡйҖ’еҪ’еҜ№жҜ”дёӨдёӘJSONпјҢиҮӘеҠЁз”ҹжҲҗжңҖдјҳиЎҘдёҒж“ҚдҪң
  • 6з§Қж“ҚдҪңпјҡadd/remove/replace/move/copy/testе…ЁиҰҶзӣ–
  • еә”з”ЁйӘҢиҜҒпјҡе°ҶиЎҘдёҒеә”з”ЁеҲ°еҺҹе§ӢJSONпјҢйӘҢиҜҒз»“жһңжҳҜеҗҰжӯЈзЎ®
  • еҸҜи§ҶеҢ–еҲ—иЎЁпјҡжҜҸдёӘж“ҚдҪңз”ЁйўңиүІж ҮзӯҫеҢәеҲҶпјҢзӣҙи§Ӯжҳ“иҜ»
  • JSON PointerпјҡиҮӘеҠЁз”ҹжҲҗRFC 6901ж ҮеҮҶи·Ҝеҫ„

дҪҝз”Ёж•ҷзЁӢ

1. еңЁ"еҺҹе§ӢJSON"иҫ“е…ҘжЎҶзІҳиҙҙжәҗJSONж•°жҚ®гҖӮ

2. еңЁ"зӣ®ж ҮJSON"иҫ“е…ҘжЎҶзІҳиҙҙдҝ®ж”№еҗҺзҡ„JSONж•°жҚ®гҖӮ

3. зӮ№еҮ»"з”ҹжҲҗPatch"жҹҘзңӢе·®ејӮж“ҚдҪңеҲ—иЎЁе’ҢиЎҘдёҒд»Јз ҒгҖӮ

4. зӮ№еҮ»"еә”з”ЁPatchйӘҢиҜҒ"зЎ®и®ӨиЎҘдёҒжӯЈзЎ®жҖ§гҖӮ

еә”з”ЁеңәжҷҜ

еңәжҷҜ1пјҡAPIйғЁеҲҶжӣҙж–°

еҗҺз«ҜејҖеҸ‘иҖ…дҪҝз”ЁJSON Patchе®һзҺ°APIзҡ„PATCHж–№жі•пјҢеҸӘдј иҫ“еҸҳжӣҙйғЁеҲҶпјҢеҮҸе°‘зҪ‘з»ңејҖй”ҖгҖӮ

еңәжҷҜ2пјҡй…ҚзҪ®еҸҳжӣҙиҝҪиёӘ

иҝҗз»ҙдәәе‘ҳз”ЁJSON Patchи®°еҪ•й…ҚзҪ®ж–Үд»¶зҡ„жҜҸж¬ЎеҸҳжӣҙпјҢдҫҝдәҺе®Ўи®Ўе’Ңеӣһж»ҡгҖӮ

еңәжҷҜ3пјҡеҚҸеҗҢзј–иҫ‘

еӨҡдәәеҚҸдҪңж—¶пјҢз”ЁJSON PatchеҗҢжӯҘеҗ„з«Ҝдҝ®ж”№пјҢи§ЈеҶіеҶІзӘҒгҖӮ

JSON PatchпјҲRFC 6902пјүжҳҜдёҖз§ҚжҸҸиҝ°JSONж–ҮжЎЈеҸҳжӣҙзҡ„ж ҮеҮҶж јејҸпјҢз”ұдёҖз»„жңүеәҸж“ҚдҪңз»„жҲҗгҖӮжҜҸдёӘж“ҚдҪңеҢ…еҗ«opпјҲж“ҚдҪңзұ»еһӢпјүгҖҒpathпјҲJSON Pointerи·Ҝеҫ„пјүе’ҢvalueпјҲеҖјпјҢйғЁеҲҶж“ҚдҪңйңҖиҰҒпјүгҖӮJSON PointerпјҲRFC 6901пјүз”Ё/еҲҶйҡ”и·Ҝеҫ„пјҢзү№ж®Ҡеӯ—з¬Ұ~е’Ң/йңҖиҪ¬д№үдёә~0е’Ң~1гҖӮJSON PatchжҳҜеҺҹеӯҗж“ҚдҪңвҖ”вҖ”иҰҒд№Ҳе…ЁйғЁжҲҗеҠҹпјҢиҰҒд№Ҳе…ЁйғЁеӨұиҙҘгҖӮtestж“ҚдҪңеҸҜз”ЁдәҺжқЎд»¶еҲӨж–ӯпјҢеҖјдёҚеҢ№й…Қж—¶ж•ҙдёӘPatchеӨұиҙҘгҖӮдёҺJSON Merge PatchпјҲRFC 7396пјүзӣёжҜ”пјҢJSON PatchжӣҙејәеӨ§пјҢиғҪеӨ„зҗҶж•°з»„ж“ҚдҪңпјҢдҪҶиҜӯжі•жӣҙеӨҚжқӮгҖӮ
