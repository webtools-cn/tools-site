❓ 常见问题

什么是渐变文字效果？

渐变文字效果是通过CSS background-clip和text-fill-color属性，将渐变背景应用到文字上，使文字呈现从一种颜色到另一种颜色的过渡效果。

渐变文字在所有浏览器都支持吗？

现代浏览器（Chrome、Firefox、Safari、Edge）均支持。需要使用-webkit-background-clip和-webkit-text-fill-color前缀确保兼容性。本工具生成的代码已包含兼容性处理。

支持哪些渐变类型？

支持线性渐变（linear-gradient）和径向渐变（radial-gradient），可自定义渐变方向、颜色和过渡点。

如何将渐变文字应用到网页？

复制生成的CSS代码，应用到目标元素的class或style属性即可。代码包含所有必要的前缀属性。

需要安装任何依赖吗？

不需要。生成的是纯CSS代码，无需任何框架或库。
