📖 关于小数转分数

将小数转换为分数是一个常见的数学需求。本工具支持两种类型：

  • 有限小数 — 如 0.75 = 75/100 = 3/4，将小数乘以10的幂后约分
  • 无限循环小数 — 如 0.333... = 1/3，通过解方程 x = 0.333..., 10x = 3.333... 得 x = 3/9 = 1/3

所有结果都会自动化简到最简分数，并显示换算过程。

❓ 常见问题

循环小数怎么输入？

用括号标注循环节。例如：0.3(3)表示0.333...，0.1(428571)表示0.142857142857...，1.2(7)表示1.2777...。纯循环小数和混循环小数都支持。

检查输入格式

输入应为合法数字，可以以+或-开头，可以有小数点。循环节括号必须完整匹配。例如：0.25（有限小数）、0.(142857)（纯循环小数）、0.16(6)（混循环小数）。

为什么分数能化简？

工具使用欧几里得算法（辗转相除法）计算分子分母的最大公约数(GCD)，然后将分子分母同时除以GCD，得到最简分数。例如 42/56 的GCD是14，化简后为 3/4。

带分数怎么用？

当分数值大于1时，工具会同时显示假分数（如7/4）和带分数（如1¾）。带分数格式中，整数部分和分数部分之间没有运算符，直接拼写表示加法（1 + 3/4 = 1.75）。

结果精确吗？

完全精确。本工具不使用浮点数近似，而是用JavaScript的大整数(BigInt)进行精确的有理数运算。循环小数通过代数方程精确求解，所有结果都是数学上精确的分数。
