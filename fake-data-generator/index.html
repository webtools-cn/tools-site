📖 使用教程

假数据生成器是一款专为开发者和测试人员设计的在线工具，能够在几秒钟内生成大量结构化的虚拟数据。无论您是在搭建数据库原型、编写单元测试，还是进行UI演示，这款工具都能帮您快速获取所需数据。

🚀 快速开始

1. 选择字段：在左侧面板勾选需要生成的数据字段，包括姓名、邮箱、手机号、地址等15种类型。

2. 设置数量：输入需要生成的记录数（1-10000条），建议初次测试时使用10-50条。

3. 生成数据：点击"生成数据"按钮，结果会实时显示在右侧表格中。

4. 导出使用：支持CSV、JSON和纯文本三种格式导出，可直接导入数据库或用于API测试。

🎯 适用场景

  • 数据库测试：在开发阶段填充测试数据库，验证表结构和查询语句的正确性。
  • API原型开发：快速生成Mock数据用于前端开发，无需等待后端接口就绪。
  • UI/UX设计展示：为设计稿提供真实感数据，让客户更好地理解最终产品效果。
  • 自动化测试：生成边界值、异常值等测试数据，提升测试覆盖率。
  • 教学演示：在编程课程中为学生提供结构化的示例数据。

🔒 隐私与安全

本工具完全在浏览器本地运行，所有数据生成和导出过程均不会上传任何信息到服务器。生成的姓名、邮箱、手机号等数据均为算法随机组合，与任何真实个人无关，可放心使用。

💡 扩展知识

假数据（Mock Data）是软件开发中不可或缺的环节。合理使用假数据可以显著提升开发效率和软件质量。

为什么需要假数据？

在真实生产环境中，直接操作真实用户数据存在隐私合规风险（如GDPR、个人信息保护法）。使用假数据可以在保护隐私的同时，获得与真实数据高度相似的测试效果。

数据生成策略

本工具采用"结构化随机"策略：姓氏从中国百家姓中随机选取，名字从常用字库中组合；手机号遵循中国大陆号段规则；地址包含省市区和街道信息的合理组合。这种策略确保生成的数据"看起来像真的"，同时避免与真实数据冲突。

CSV与JSON格式对比

CSV格式兼容性强，可直接用Excel打开，适合数据分析师和非技术人员。但CSV不支持嵌套结构和复杂数据类型。JSON格式天然支持层级结构，是前后端数据交换的标准格式，尤其适合API开发和NoSQL数据库。根据使用场景选择即可。

❓ 常见问题

生成的数据是真实的吗？

不是。本工具生成的所有数据均为算法随机生成的虚拟数据，与任何真实个人无关。姓名由常见姓氏和名字组合随机生成，邮箱和手机号均非真实存在。

数据会保存到服务器吗？

不会。所有数据生成和导出均在您的浏览器本地完成，不会上传任何数据到服务器，确保您的隐私安全。

支持哪些导出格式？

支持 CSV（逗号分隔值）、JSON（键值对格式）和纯文本三种导出格式。CSV 可直接在 Excel 中打开，JSON 适合开发者直接用于 API 测试。

一次最多能生成多少条数据？

单次最多支持生成 10,000 条数据。对于大多数测试场景，建议一次生成 100-500 条即可满足需求。

生成中文姓名还是英文姓名？

当前版本主要生成中文姓名。如需英文姓名，请在导出后自行替换或使用英文版工具。

身份证号格式正确吗？

是的。生成的身份证号符合中国大陆身份证编码规则（GB 11643-1999），包含正确的地址码、出生日期码和校验码，但均为虚拟地址码，不对应真实行政区划。

可以生成特定格式的数据吗？

当前版本提供预定义字段格式。如需自定义格式（如特定正则表达式），可使用本站 正则表达式生成器 配合生成。

生成大量数据会卡顿吗？

不会。得益于现代浏览器的 JavaScript 引擎，生成 10000 条数据通常在 1 秒内完成。如遇到卡顿，建议减少单次生成数量或关闭不必要的字段。
