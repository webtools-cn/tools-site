е№ҝе‘ҠдҪҚ - дёӯйғЁ (728Г—90)

рҹ“– дҪҝз”Ёж•ҷзЁӢ

еҒҮж•°жҚ®з”ҹжҲҗеҷЁжҳҜдёҖж¬ҫдё“дёәејҖеҸ‘иҖ…е’ҢжөӢиҜ•дәәе‘ҳи®ҫи®Ўзҡ„еңЁзәҝе·Ҙе…·пјҢиғҪеӨҹеңЁеҮ з§’й’ҹеҶ…з”ҹжҲҗеӨ§йҮҸз»“жһ„еҢ–зҡ„иҷҡжӢҹж•°жҚ®гҖӮж— и®әжӮЁжҳҜеңЁжҗӯе»әж•°жҚ®еә“еҺҹеһӢгҖҒзј–еҶҷеҚ•е…ғжөӢиҜ•пјҢиҝҳжҳҜиҝӣиЎҢUIжј”зӨәпјҢиҝҷж¬ҫе·Ҙе…·йғҪиғҪеё®жӮЁеҝ«йҖҹиҺ·еҸ–жүҖйңҖж•°жҚ®гҖӮ

рҹҡҖ еҝ«йҖҹејҖе§Ӣ

1. йҖүжӢ©еӯ—ж®өпјҡеңЁе·Ұдҫ§йқўжқҝеӢҫйҖүйңҖиҰҒз”ҹжҲҗзҡ„ж•°жҚ®еӯ—ж®өпјҢеҢ…жӢ¬е§“еҗҚгҖҒйӮ®з®ұгҖҒжүӢжңәеҸ·гҖҒең°еқҖзӯү15з§Қзұ»еһӢгҖӮ

2. и®ҫзҪ®ж•°йҮҸпјҡиҫ“е…ҘйңҖиҰҒз”ҹжҲҗзҡ„и®°еҪ•ж•°пјҲ1-10000жқЎпјүпјҢе»әи®®еҲқж¬ЎжөӢиҜ•ж—¶дҪҝз”Ё10-50жқЎгҖӮ

3. з”ҹжҲҗж•°жҚ®пјҡзӮ№еҮ»"з”ҹжҲҗж•°жҚ®"жҢүй’®пјҢз»“жһңдјҡе®һж—¶жҳҫзӨәеңЁеҸідҫ§иЎЁж јдёӯгҖӮ

4. еҜјеҮәдҪҝз”Ёпјҡж”ҜжҢҒCSVгҖҒJSONе’ҢзәҜж–Үжң¬дёүз§Қж јејҸеҜјеҮәпјҢеҸҜзӣҙжҺҘеҜје…Ҙж•°жҚ®еә“жҲ–з”ЁдәҺAPIжөӢиҜ•гҖӮ

рҹҺҜ йҖӮз”ЁеңәжҷҜ

  • ж•°жҚ®еә“жөӢиҜ•пјҡеңЁејҖеҸ‘йҳ¶ж®өеЎ«е……жөӢиҜ•ж•°жҚ®еә“пјҢйӘҢиҜҒиЎЁз»“жһ„е’ҢжҹҘиҜўиҜӯеҸҘзҡ„жӯЈзЎ®жҖ§гҖӮ
  • APIеҺҹеһӢејҖеҸ‘пјҡеҝ«йҖҹз”ҹжҲҗMockж•°жҚ®з”ЁдәҺеүҚз«ҜејҖеҸ‘пјҢж— йңҖзӯүеҫ…еҗҺз«ҜжҺҘеҸЈе°ұз»ӘгҖӮ
  • UI/UXи®ҫи®Ўеұ•зӨәпјҡдёәи®ҫи®ЎзЁҝжҸҗдҫӣзңҹе®һж„ҹж•°жҚ®пјҢи®©е®ўжҲ·жӣҙеҘҪең°зҗҶи§ЈжңҖз»Ҳдә§е“Ғж•ҲжһңгҖӮ
  • иҮӘеҠЁеҢ–жөӢиҜ•пјҡз”ҹжҲҗиҫ№з•ҢеҖјгҖҒејӮеёёеҖјзӯүжөӢиҜ•ж•°жҚ®пјҢжҸҗеҚҮжөӢиҜ•иҰҶзӣ–зҺҮгҖӮ
  • ж•ҷеӯҰжј”зӨәпјҡеңЁзј–зЁӢиҜҫзЁӢдёӯдёәеӯҰз”ҹжҸҗдҫӣз»“жһ„еҢ–зҡ„зӨәдҫӢж•°жҚ®гҖӮ

рҹ”’ йҡҗз§ҒдёҺе®үе…Ё

жң¬е·Ҙе…·е®Ңе…ЁеңЁжөҸи§ҲеҷЁжң¬ең°иҝҗиЎҢпјҢжүҖжңүж•°жҚ®з”ҹжҲҗе’ҢеҜјеҮәиҝҮзЁӢеқҮдёҚдјҡдёҠдј д»»дҪ•дҝЎжҒҜеҲ°жңҚеҠЎеҷЁгҖӮз”ҹжҲҗзҡ„е§“еҗҚгҖҒйӮ®з®ұгҖҒжүӢжңәеҸ·зӯүж•°жҚ®еқҮдёәз®—жі•йҡҸжңәз»„еҗҲпјҢдёҺд»»дҪ•зңҹе®һдёӘдәәж— е…іпјҢеҸҜж”ҫеҝғдҪҝз”ЁгҖӮ

рҹ’Ў жү©еұ•зҹҘиҜҶ

еҒҮж•°жҚ®пјҲMock DataпјүжҳҜиҪҜд»¶ејҖеҸ‘дёӯдёҚеҸҜжҲ–зјәзҡ„зҺҜиҠӮгҖӮеҗҲзҗҶдҪҝз”ЁеҒҮж•°жҚ®еҸҜд»Ҙжҳҫи‘—жҸҗеҚҮејҖеҸ‘ж•ҲзҺҮе’ҢиҪҜд»¶иҙЁйҮҸгҖӮ

дёәд»Җд№ҲйңҖиҰҒеҒҮж•°жҚ®пјҹ

еңЁзңҹе®һз”ҹдә§зҺҜеўғдёӯпјҢзӣҙжҺҘж“ҚдҪңзңҹе®һз”ЁжҲ·ж•°жҚ®еӯҳеңЁйҡҗз§ҒеҗҲи§„йЈҺйҷ©пјҲеҰӮGDPRгҖҒдёӘдәәдҝЎжҒҜдҝқжҠӨжі•пјүгҖӮдҪҝз”ЁеҒҮж•°жҚ®еҸҜд»ҘеңЁдҝқжҠӨйҡҗз§Ғзҡ„еҗҢж—¶пјҢиҺ·еҫ—дёҺзңҹе®һж•°жҚ®й«ҳеәҰзӣёдјјзҡ„жөӢиҜ•ж•ҲжһңгҖӮ

ж•°жҚ®з”ҹжҲҗзӯ–з•Ҙ

жң¬е·Ҙе…·йҮҮз”Ё"з»“жһ„еҢ–йҡҸжңә"зӯ–з•Ҙпјҡе§“ж°Ҹд»ҺдёӯеӣҪзҷҫе®¶е§“дёӯйҡҸжңәйҖүеҸ–пјҢеҗҚеӯ—д»Һеёёз”Ёеӯ—еә“дёӯз»„еҗҲпјӣжүӢжңәеҸ·йҒөеҫӘдёӯеӣҪеӨ§йҷҶеҸ·ж®өи§„еҲҷпјӣең°еқҖеҢ…еҗ«зңҒеёӮеҢәе’ҢиЎ—йҒ“дҝЎжҒҜзҡ„еҗҲзҗҶз»„еҗҲгҖӮиҝҷз§Қзӯ–з•ҘзЎ®дҝқз”ҹжҲҗзҡ„ж•°жҚ®"зңӢиө·жқҘеғҸзңҹзҡ„"пјҢеҗҢж—¶йҒҝе…ҚдёҺзңҹе®һж•°жҚ®еҶІзӘҒгҖӮ

CSVдёҺJSONж јејҸеҜ№жҜ”

CSVж јејҸе…је®№жҖ§ејәпјҢеҸҜзӣҙжҺҘз”ЁExcelжү“ејҖпјҢйҖӮеҗҲж•°жҚ®еҲҶжһҗеёҲе’ҢйқһжҠҖжңҜдәәе‘ҳгҖӮдҪҶCSVдёҚж”ҜжҢҒеөҢеҘ—з»“жһ„е’ҢеӨҚжқӮж•°жҚ®зұ»еһӢгҖӮJSONж јејҸеӨ©з„¶ж”ҜжҢҒеұӮзә§з»“жһ„пјҢжҳҜеүҚеҗҺз«Ҝж•°жҚ®дәӨжҚўзҡ„ж ҮеҮҶж јејҸпјҢе°Өе…¶йҖӮеҗҲAPIејҖеҸ‘е’ҢNoSQLж•°жҚ®еә“гҖӮж №жҚ®дҪҝз”ЁеңәжҷҜйҖүжӢ©еҚіеҸҜгҖӮ

вқ“ еёёи§Ғй—®йўҳ

з”ҹжҲҗзҡ„ж•°жҚ®жҳҜзңҹе®һзҡ„еҗ—пјҹ

дёҚжҳҜгҖӮжң¬е·Ҙе…·з”ҹжҲҗзҡ„жүҖжңүж•°жҚ®еқҮдёәз®—жі•йҡҸжңәз”ҹжҲҗзҡ„иҷҡжӢҹж•°жҚ®пјҢдёҺд»»дҪ•зңҹе®һдёӘдәәж— е…ігҖӮе§“еҗҚз”ұеёёи§Ғе§“ж°Ҹе’ҢеҗҚеӯ—з»„еҗҲйҡҸжңәз”ҹжҲҗпјҢйӮ®з®ұе’ҢжүӢжңәеҸ·еқҮйқһзңҹе®һеӯҳеңЁгҖӮ

ж•°жҚ®дјҡдҝқеӯҳеҲ°жңҚеҠЎеҷЁеҗ—пјҹ

дёҚдјҡгҖӮжүҖжңүж•°жҚ®з”ҹжҲҗе’ҢеҜјеҮәеқҮеңЁжӮЁзҡ„жөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢдёҚдјҡдёҠдј д»»дҪ•ж•°жҚ®еҲ°жңҚеҠЎеҷЁпјҢзЎ®дҝқжӮЁзҡ„йҡҗз§Ғе®үе…ЁгҖӮ

ж”ҜжҢҒе“ӘдәӣеҜјеҮәж јејҸпјҹ

ж”ҜжҢҒ CSVпјҲйҖ—еҸ·еҲҶйҡ”еҖјпјүгҖҒJSONпјҲй”®еҖјеҜ№ж јејҸпјүе’ҢзәҜж–Үжң¬дёүз§ҚеҜјеҮәж јејҸгҖӮCSV еҸҜзӣҙжҺҘеңЁ Excel дёӯжү“ејҖпјҢJSON йҖӮеҗҲејҖеҸ‘иҖ…зӣҙжҺҘз”ЁдәҺ API жөӢиҜ•гҖӮ

дёҖж¬ЎжңҖеӨҡиғҪз”ҹжҲҗеӨҡе°‘жқЎж•°жҚ®пјҹ

еҚ•ж¬ЎжңҖеӨҡж”ҜжҢҒз”ҹжҲҗ 10,000 жқЎж•°жҚ®гҖӮеҜ№дәҺеӨ§еӨҡж•°жөӢиҜ•еңәжҷҜпјҢе»әи®®дёҖж¬Ўз”ҹжҲҗ 100-500 жқЎеҚіеҸҜж»Ўи¶ійңҖжұӮгҖӮ

з”ҹжҲҗдёӯж–Үе§“еҗҚиҝҳжҳҜиӢұж–Үе§“еҗҚпјҹ

еҪ“еүҚзүҲжң¬дё»иҰҒз”ҹжҲҗдёӯж–Үе§“еҗҚгҖӮеҰӮйңҖиӢұж–Үе§“еҗҚпјҢиҜ·еңЁеҜјеҮәеҗҺиҮӘиЎҢжӣҝжҚўжҲ–дҪҝз”ЁиӢұж–ҮзүҲе·Ҙе…·гҖӮ

иә«д»ҪиҜҒеҸ·ж јејҸжӯЈзЎ®еҗ—пјҹ

жҳҜзҡ„гҖӮз”ҹжҲҗзҡ„иә«д»ҪиҜҒеҸ·з¬ҰеҗҲдёӯеӣҪеӨ§йҷҶиә«д»ҪиҜҒзј–з Ғи§„еҲҷпјҲGB 11643-1999пјүпјҢеҢ…еҗ«жӯЈзЎ®зҡ„ең°еқҖз ҒгҖҒеҮәз”ҹж—Ҙжңҹз Ғе’Ңж ЎйӘҢз ҒпјҢдҪҶеқҮдёәиҷҡжӢҹең°еқҖз ҒпјҢдёҚеҜ№еә”зңҹе®һиЎҢж”ҝеҢәеҲ’гҖӮ

еҸҜд»Ҙз”ҹжҲҗзү№е®ҡж јејҸзҡ„ж•°жҚ®еҗ—пјҹ

еҪ“еүҚзүҲжң¬жҸҗдҫӣйў„е®ҡд№үеӯ—ж®өж јејҸгҖӮеҰӮйңҖиҮӘе®ҡд№үж јејҸпјҲеҰӮзү№е®ҡжӯЈеҲҷиЎЁиҫҫејҸпјүпјҢеҸҜдҪҝз”Ёжң¬з«ҷ жӯЈеҲҷиЎЁиҫҫејҸз”ҹжҲҗеҷЁ й…ҚеҗҲз”ҹжҲҗгҖӮ

з”ҹжҲҗеӨ§йҮҸж•°жҚ®дјҡеҚЎйЎҝеҗ—пјҹ

дёҚдјҡгҖӮеҫ—зӣҠдәҺзҺ°д»ЈжөҸи§ҲеҷЁзҡ„ JavaScript еј•ж“ҺпјҢз”ҹжҲҗ 10000 жқЎж•°жҚ®йҖҡеёёеңЁ 1 з§’еҶ…е®ҢжҲҗгҖӮеҰӮйҒҮеҲ°еҚЎйЎҝпјҢе»әи®®еҮҸе°‘еҚ•ж¬Ўз”ҹжҲҗж•°йҮҸжҲ–е…ій—ӯдёҚеҝ…иҰҒзҡ„еӯ—ж®өгҖӮ
