❓ 常见问题

        

            
什么是Web Component？

            
Web Component是一组W3C标准技术，允许开发者创建可复用的自定义HTML元素。它由三个核心技术组成：Custom Elements（自定义元素）、Shadow DOM（影子DOM）和HTML Templates（HTML模板），实现封装和复用。

        

        

            
什么是Shadow DOM？

            
Shadow DOM是一种DOM封装技术，允许将组件的内部结构和样式与页面其他部分隔离。Shadow DOM内的CSS不会影响外部，外部CSS也不会影响内部，实现真正的样式封装。

        

        

            
组件名称为什么必须包含连字符？

            
根据HTML规范，自定义元素名称必须包含至少一个连字符（-），以区分标准HTML元素和自定义元素。这确保了命名空间不会冲突，例如 my-button、custom-card 都是合法的。

        

        

            
Web Component支持哪些生命周期回调？

            
Web Component支持以下生命周期回调：connectedCallback（元素插入DOM时调用）、disconnectedCallback（元素从DOM移除时调用）、adoptedCallback（元素移动到新文档时调用）、attributeChangedCallback（观察的属性变化时调用）。

完全安全！所有代码生成在浏览器本地完成，不发送任何数据到服务器，不收集任何个人信息。
