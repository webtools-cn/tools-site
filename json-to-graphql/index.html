常见问题

JSON如何转换为GraphQL Schema？

本工具分析JSON数据的结构和类型，自动推断字段类型（String、Int、Float、Boolean等），生成对应的GraphQL Type Definition。嵌套对象会生成独立的type，数组会自动识别为列表类型。

支持哪些GraphQL输出格式？

支持三种输出：1) GraphQL Schema（类型定义）；2) GraphQL Query（查询语句）；3) GraphQL Mutation（变更语句）。可一键切换。

如何处理嵌套JSON？

嵌套的JSON对象会自动拆分为独立的GraphQL type，并使用引用关系连接。数组中的对象也会被识别并生成对应的type。

如何使用JSON转GraphQL

使用JSON转GraphQL进行转换只需三个步骤：

  1. 输入要转换的值
    在输入区域输入或粘贴要转换的数据。JSON转GraphQL支持多种输入格式——输入数据后工具会自动识别。
  2. 选择源格式和目标格式
    从可用选项中选择源格式和目标格式。工具支持双向转换（如适用）。设置会即时生效。
  3. 获取转换结果
    转换结果实时显示。点击"复制"复制到剪贴板，或点击"下载"保存。所有转换都在您的设备本地完成。
