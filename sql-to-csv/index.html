使用说明

将SQL INSERT语句粘贴到输入框，点击转换即可得到CSV格式数据。支持单行INSERT和多行批量INSERT格式，所有处理在浏览器本地完成。

❓ 常见问题

这个工具免费吗？

完全免费，无需注册，无需下载。所有功能在浏览器本地运行，数据不会上传到任何服务器。

数据会经过服务器吗？

不会。本工具是纯前端应用，所有操作都在您的浏览器中本地完成，数据绝不离开您的设备。

SQL转CSV支持哪些SQL语句？

主要支持INSERT INTO语句的解析，支持格式包括：INSERT INTO table (col1, col2) VALUES (val1, val2); 和多行批量INSERT INTO table VALUES (v1,v2),(v3,v4);。同时也支持不带列名的INSERT语句。目前不支持SELECT、UPDATE、DELETE等语句。

如何处理包含特殊字符的字段？

工具会自动处理SQL中的转义字符：单引号内的字段内容会被正确解析，数值类型保持原样，NULL值转换为空字符串。如果字段包含逗号或换行符，生成的CSV会自动用双引号包裹该字段。日期、时间戳等格式也会被保留。

SQL转CSV能处理多大数据的SQL？

处理能力取决于你的浏览器和设备性能。对于包含大量INSERT语句的SQL文件（比如数据库导出），建议分批处理。大多数现代浏览器可以流畅处理数千行INSERT语句，超出浏览器内存限制时建议分批转换。
