常见问题 (FAQ)

❓ JSON转Python字典支持哪些数据类型映射？

支持string→str、number→int/float、boolean→bool、null→None、array→list、object→dict的自动映射。JSON字符串中的ISO日期格式会自动转换为datetime对象，数字字符串会保持为str类型。嵌套对象会递归生成完整的dict结构。

❓ 生成的Python代码可以直接使用吗？

可以。生成的代码是标准的Python dict字面量语法，可直接复制到Python项目中使用。支持Python 3.6+语法（包含f-string和类型提示的版本会有所不同）。工具默认生成纯dict格式，兼容Python 2.7+，无需额外依赖。

❓ 如何处理JSON中的null值？

JSON中的null值会转换为Python的None关键字。如果选择'带类型注释'模式，null字段会被标记为 Optional[Any] 类型。工具也支持将null替换为默认空值（空字符串、空列表、空字典等），通过选项可配置。

❓ 支持嵌套JSON对象吗？

完全支持。嵌套对象会递归转换为Python嵌套dict结构，保持原始数据的层级关系和缩进格式。数组中的对象也会被正确处理，生成包含dict元素的list结构。工具能处理任意深度的嵌套。

❓ 我的数据安全吗？

数据安全完全保障。本工具100%纯前端运行，你的JSON数据仅在浏览器本地内存中处理，不会上传到任何服务器。处理完成后数据即时清除，无需担心数据泄露。支持离线使用，断网也可正常操作。
