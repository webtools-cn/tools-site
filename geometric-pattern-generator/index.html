常见问题 (FAQ)

支持哪些几何形状？

支持6种基础几何形状：三角形（等边/等腰）、正方形、正六边形、圆形、菱形、星形（五角星/六角星）。每种形状都可以独立调整大小、旋转角度和间距。选择「随机混合」模式可以在一张图案中组合多种形状。

图案的排列方式有哪些？

支持3种排列方式：① 网格排列（Grid）：规则的水平和垂直排列；② 交错排列（Offset）：类似砖墙的交错排列，适合六边形和菱形；③ 随机排列（Random）：位置和大小随机生成，适合有机风格的图案。每种排列方式都有独立的间距和密度控制。

生成的图案可以导出吗？支持哪些用途？

图案可以导出为 PNG 图片（支持透明背景）或复制为 DataURL。生成的图案适合用作：网页背景、包装设计纹理、壁纸图案、织物印花设计等。建议尺寸为 512×512 像素可无缝平铺，也可生成更大尺寸的单图。

如何创建无缝平铺图案？

当选择网格排列或交错排列时，默认生成可无缝平铺的图案。您只需将生成的 PNG 图案设置为 CSS background-image，设置 `background-repeat: repeat` 即可实现无缝平铺。建议在导出时选择 512×512 像素的标准尺寸，这是 CSS 背景图的最佳实践。
