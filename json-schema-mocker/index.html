JSON Schema Mock数据生成器能做什么？

JSON Schema Mock数据生成器是一款面向开发者的在线工具，能够根据输入的JSON Schema定义自动生成符合类型、约束和格式要求的模拟数据。无论是开发API原型、前端联调、单元测试还是数据库填充，都能快速获得逼真的测试数据，无需手动编写。

核心功能

  • 智能类型推断：根据type/string/number/integer/boolean/array/object自动生成对应类型数据，支持嵌套对象和数组
  • 约束遵守：严格遵守minimum/maximum/minLength/maxLength/pattern/enum/const/format等约束条件
  • 格式化生成：支持email/uri/date/date-time/hostname/ipv4/ipv6/uuid等format类型，生成逼真数据
  • 批量生成：1-100条数据批量生成，支持JSON数组和CSV两种导出格式
  • 可复现性：支持随机种子，相同Schema+种子生成相同数据，便于测试复现

使用教程

1. 在编辑器中输入JSON Schema定义，或点击"加载示例"快速填充一个用户信息Schema。Schema必须符合JSON Schema Draft-07规范。

2. 配置生成参数：设置生成数量（1-100）、是否使用默认值和枚举值、可选设置随机种子保证可复现。

3. 点击"生成Mock数据"按钮，结果区域即时显示生成的JSON数据。支持一键复制或下载为JSON/CSV文件。

4. 修改Schema后重新生成，历史记录自动保存最近5次操作，可随时回溯。

应用场景

场景1：前端开发联调

前端开发者根据后端提供的JSON Schema快速生成Mock数据，在后端API未就绪时即可开始开发和调试界面。

场景2：API原型开发

后端开发者定义好API的Schema后，用本工具生成示例响应数据，用于API文档和接口测试。

场景3：单元测试数据

测试工程师用Schema生成大量边界值和随机数据，覆盖各种输入场景，提高测试覆盖率。

场景4：数据库填充

开发阶段需要填充测试数据库，用Schema生成符合表结构的模拟数据，导出CSV后批量导入。

扩展知识

JSON Schema是JSON数据结构的声明式验证语言，当前主流版本为Draft-07（2019）和Draft 2020-12。Mock数据生成遵循Schema约束但填充随机值，与Faker.js等库的区别在于：Mock生成器严格遵循Schema定义的类型和约束，而Faker侧重生成特定语义的假数据（如姓名、地址）。两者结合使用效果最佳——Schema保证结构正确，Faker保证语义逼真。本工具内置了常见format的语义生成器，如email生成真实格式的邮箱地址。

❓ 支持哪些JSON Schema特性？

支持type/string/number/integer/boolean/array/object/enum/const/format(minimum/maximum/minLength/maxLength/pattern/email/uri/date等)、嵌套对象、$ref引用、items等核心特性。

❓ 可以生成多条数据吗？

可以，通过数量滑块设置1-100条，支持批量生成并导出为JSON数组或CSV。

❓ 生成的数据是随机的吗？

是的，数据基于Schema约束随机生成，但严格遵守类型、范围、格式等约束。设置随机种子可保证可复现。

❓ 数据会上传到服务器吗？

不会，所有Schema解析和数据生成均在浏览器本地完成，数据绝不上传。

❓ 支持哪些format类型？

支持email/uri/url/date/date-time/hostname/ipv4/ipv6/uuid/regex等常见format，生成符合格式的逼真数据。

❓ Schema语法错误怎么办？

工具会实时检测JSON语法和Schema结构，在编辑器下方显示具体错误信息，修正后重新生成即可。

❓ 可以自定义字段值吗？

可以，使用enum/const/default指定固定值，使用examples提供示例值，工具会优先使用这些值。
