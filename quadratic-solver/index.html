使用说明

  1. 输入系数
    输入二次方程 ax² + bx + c = 0 的系数 a、b、c。a 不能为 0。
  2. 求解方程
    点击「求解」按钮计算根。工具会显示判别式、根（实根或复数根）、顶点坐标和开口方向。
  3. 查看详细解法
    获取完整的解题步骤，包括求根公式应用、因式分解（整数根时）和抛物线属性。

常见问题

什么是求根公式？

求根公式 x = [-b ± √(b² - 4ac)] / (2a) 可求解任意一元二次方程 ax² + bx + c = 0。判别式 Δ = b² - 4ac 决定根的类型。

判别式告诉我们什么？

Δ > 0：两个不同的实根。Δ = 0：一个二重实根。Δ < 0：一对共轭复数根。

什么是抛物线的顶点？

顶点是抛物线的转折点，位于 x = -b/(2a)。它代表二次函数的最小值（a>0）或最大值（a<0）。

可以因式分解二次方程吗？

可以！当二次方程有整数根和整数系数时，求解器会自动显示因式分解形式。

什么是复数根？

当判别式为负时，根涉及虚数单位 i (√-1)。它们以共轭对形式出现：p + qi 和 p - qi。
