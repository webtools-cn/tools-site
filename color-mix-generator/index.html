常见问题 (FAQ)

CSS color-mix()是什么？

color-mix()是CSS Color Level 5函数，允许在CSS中直接混合两种颜色，无需预处理器或JavaScript。

不同色彩空间有什么区别？

sRGB混合符合直觉，OKLCH/OKLAB感知均匀混合更自然，HSL按色相/饱和度/亮度混合适合色相渐变。

浏览器兼容性如何？

Chrome 111+、Firefox 113+、Safari 16.2+支持，不支持的浏览器可用CSS变量和预计算值降级。
