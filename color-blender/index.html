常见问题 (FAQ)

颜色混合的原理是什么？支持哪些混合模式？

本工具支持两种颜色混合算法：① RGB 线性插值：在 RGB 颜色空间中对两个颜色的 R、G、B 通道分别按比例线性插值；② Lab 颜色插值：将颜色转换到 Lab 色彩空间后插值，结果更接近人眼感知。默认使用 RGB 混合模式，您可以在设置中切换到 Lab 模式以获得更自然的混合效果。

支持哪些颜色输入格式？

支持3种颜色输入方式：① 颜色选择器（可视化选色）；② HEX 格式（如 #FF5500）；③ RGB 格式（如 rgb(255,85,0)）。输入框支持实时解析，输入任意合法颜色值后自动更新预览。

混合结果可以输出哪些格式？

混合结果的颜色值可以复制为 HEX 格式、RGB 格式或 CSS 变量格式。同时，混合过程中产生的中间色阶（步进值）会以列表形式展示，方便您复制整组色值用于 CSS 渐变或设计系统。

颜色混合在实际设计中有哪些应用场景？

常见应用场景包括：① 品牌配色设计：混合主色和辅色生成完整的色彩体系；② 渐变色设计：通过多步混合创建平滑渐变；③ 设计系统 Token 生成：使用混合算法生成 light/dark 变体色；④ 色彩教学：直观展示颜色混合原理。
