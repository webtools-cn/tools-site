рҹ”’жүҖжңүSQLиҪ¬жҚўеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј 
рҹ“Ҡ Ent Schemaд»Јз Ғ

SQLиҪ¬Go Ent ORMе·Ҙе…·иғҪеҒҡд»Җд№Ҳпјҹ

SQLиҪ¬Go Ent ORMе·Ҙе…·жҳҜдёҖж¬ҫзәҜеүҚз«ҜеңЁзәҝе·Ҙе…·пјҢз”ЁдәҺе°ҶSQL CREATE TABLEиҜӯеҸҘиҮӘеҠЁиҪ¬жҚўдёәGo EntжЎҶжһ¶зҡ„Schemaе®ҡд№үд»Јз ҒгҖӮиҫ“е…ҘSQLе»әиЎЁиҜӯеҸҘпјҢе·Ҙе…·иҮӘеҠЁжҳ е°„еӯ—ж®өзұ»еһӢгҖҒз”ҹжҲҗеӯ—ж®өе®ҡд№үгҖҒзҙўеј•гҖҒеӨ–й”®е…ізі»пјҲEdgeпјүгҖҒжһҡдёҫзұ»еһӢзӯүпјҢиҫ“еҮәз¬ҰеҗҲEntи§„иҢғзҡ„Goд»Јз ҒгҖӮж”ҜжҢҒеӨҡиЎЁжү№йҮҸиҪ¬жҚўпјҢиҮӘеҠЁиҜҶеҲ«иЎЁй—ҙе…ізі»гҖӮжүҖжңүеӨ„зҗҶеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗгҖӮ

ж ёеҝғеҠҹиғҪ

  • жҷәиғҪзұ»еһӢжҳ е°„пјҡиҮӘеҠЁе°ҶSQLзұ»еһӢжҳ е°„дёәEntеӯ—ж®өзұ»еһӢпјҢеҰӮVARCHARвҶ’StringгҖҒINTвҶ’IntгҖҒTIMESTAMPвҶ’Time
  • е…ізі»з”ҹжҲҗпјҡиҜҶеҲ«FOREIGN KEYзәҰжқҹпјҢиҮӘеҠЁз”ҹжҲҗEnt Edgeе…ізі»е®ҡд№ү
  • зҙўеј•з”ҹжҲҗпјҡиҮӘеҠЁиҜҶеҲ«UNIQUEгҖҒINDEXзәҰжқҹпјҢз”ҹжҲҗEnt Indexе®ҡд№ү
  • жһҡдёҫж”ҜжҢҒпјҡSQL ENUMзұ»еһӢиҮӘеҠЁиҪ¬жҚўдёәEnt enumеӯ—ж®өе’ҢGoеёёйҮҸ
  • й»ҳи®ӨеҖјжҳ е°„пјҡSQL DEFAULTеҖјиҮӘеҠЁиҪ¬жҚўдёәEnt DefaultеҮҪж•°

дҪҝз”Ёж•ҷзЁӢ

жӯҘйӘӨ1пјҡеңЁиҫ“е…ҘжЎҶзІҳиҙҙSQL CREATE TABLEиҜӯеҸҘгҖӮ

жӯҘйӘӨ2пјҡйҖүжӢ©з”ҹжҲҗйҖүйЎ№пјҲEdgeе…ізі»гҖҒзҙўеј•гҖҒйӘҢиҜҒеҷЁгҖҒеҢ…еҗҚпјүгҖӮ

жӯҘйӘӨ3пјҡзӮ№еҮ»"иҪ¬жҚўдёәEnt"жҲ–Ctrl+EnterпјҢжҹҘзңӢз”ҹжҲҗзҡ„Goд»Јз ҒгҖӮ

жӯҘйӘӨ4пјҡзӮ№еҮ»"еӨҚеҲ¶д»Јз Ғ"жҲ–"дёӢиҪҪ"еҜјеҮәз»“жһңгҖӮ

еә”з”ЁеңәжҷҜ

ж•°жҚ®еә“иҝҒз§»

д»ҺзҺ°жңүж•°жҚ®еә“Schemaз”ҹжҲҗEntд»Јз ҒпјҢеҝ«йҖҹе°Ҷдј з»ҹSQLйЎ№зӣ®иҝҒз§»еҲ°Ent ORMжЎҶжһ¶гҖӮ

ж–°йЎ№зӣ®еҲқе§ӢеҢ–

е…Ҳз”ЁSQLи®ҫи®Ўж•°жҚ®жЁЎеһӢпјҢеҶҚдёҖй”®з”ҹжҲҗEnt SchemaпјҢеҠ йҖҹGoйЎ№зӣ®ејҖеҸ‘еҗҜеҠЁгҖӮ

еӣўйҳҹеҚҸдҪң

DBAз”ЁSQLи®ҫи®ЎиЎЁз»“жһ„пјҢејҖеҸ‘иҖ…дёҖй”®иҪ¬Entд»Јз ҒпјҢеҮҸе°‘жүӢеҠЁиҪ¬жҚўй”ҷиҜҜгҖӮ

ж–ҮжЎЈз”ҹжҲҗ

е°ҶSQL DDLиҪ¬дёәEnt Schemaд»Јз ҒпјҢдҪңдёәйЎ№зӣ®ж–ҮжЎЈе’Ңд»Јз ҒеҸӮиҖғгҖӮ

жү©еұ•зҹҘиҜҶ

EntжҳҜFacebookејҖжәҗзҡ„GoиҜӯиЁҖORMжЎҶжһ¶пјҢйҮҮз”Ёд»Јз Ғз”ҹжҲҗж–№ејҸпјҢйҖҡиҝҮSchemaе®ҡд№үж•°жҚ®жЁЎеһӢпјҢиҮӘеҠЁз”ҹжҲҗCRUDд»Јз ҒгҖӮEnt SchemaдҪҝз”ЁжөҒз•…зҡ„APIе®ҡд№үеӯ—ж®өе’Ңе…ізі»пјҢеҰӮfield.String("name").Default("")гҖӮдёҺGORMзӯүеҸҚе°„ејҸORMдёҚеҗҢпјҢEntеңЁзј–иҜ‘жңҹз”ҹжҲҗзұ»еһӢе®үе…Ёд»Јз ҒпјҢжҖ§иғҪжӣҙдјҳгҖӮSQLеҲ°Entзҡ„жҳ е°„ж ёеҝғжҳҜзұ»еһӢиҪ¬жҚўпјҡSQLзҡ„VARCHAR/CHARвҶ’Ent StringпјҢINT/INTEGERвҶ’Ent IntпјҢBIGINTвҶ’Ent Int64пјҢBOOLEAN/BOOLвҶ’Ent BoolпјҢTIMESTAMP/DATETIMEвҶ’Ent TimeпјҢDECIMALвҶ’Ent Float64зӯүгҖӮ

еёёи§Ғй—®йўҳ (FAQ)

д»Җд№ҲжҳҜSQLиҪ¬Entпјҹ

SQLиҪ¬EntжҳҜе°ҶSQL CREATE TABLEиҜӯеҸҘиҮӘеҠЁиҪ¬жҚўдёәGo Ent ORMзҡ„Schemaе®ҡд№үд»Јз ҒпјҢеҢ…жӢ¬еӯ—ж®өзұ»еһӢжҳ е°„гҖҒй»ҳи®ӨеҖјгҖҒзҙўеј•гҖҒеӨ–й”®е…ізі»зӯүгҖӮ

ж”ҜжҢҒе“ӘдәӣSQLзұ»еһӢпјҹ

ж”ҜжҢҒINT/BIGINT/VARCHAR/TEXT/BOOLEAN/TIMESTAMP/DATE/DECIMAL/FLOAT/JSON/ENUMзӯүеёёи§ҒSQLзұ»еһӢзҡ„иҮӘеҠЁжҳ е°„гҖӮ

иғҪеӨ„зҗҶеӨ–й”®е…ізі»еҗ—пјҹ

еҸҜд»ҘгҖӮе·Ҙе…·иҮӘеҠЁиҜҶеҲ«FOREIGN KEYзәҰжқҹпјҢз”ҹжҲҗEntзҡ„Edgeе…ізі»е®ҡд№үпјҲO2O/O2M/M2O/M2MпјүгҖӮ

ж”ҜжҢҒжһҡдёҫзұ»еһӢеҗ—пјҹ

ж”ҜжҢҒгҖӮSQL ENUMзұ»еһӢдјҡиҮӘеҠЁиҪ¬жҚўдёәEntзҡ„enumеӯ—ж®өе®ҡд№үе’ҢGoеёёйҮҸгҖӮ

ж•°жҚ®дјҡдёҠдј жңҚеҠЎеҷЁеҗ—пјҹ

дёҚдјҡгҖӮжүҖжңүSQLиҪ¬Entж“ҚдҪңеқҮеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј жңҚеҠЎеҷЁгҖӮ

з”ҹжҲҗзҡ„д»Јз ҒеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёеҗ—пјҹ

з”ҹжҲҗзҡ„д»Јз ҒжҳҜж ҮеҮҶзҡ„Ent Schemaж јејҸпјҢеҸҜзӣҙжҺҘж”ҫе…ҘEntйЎ№зӣ®дёӯдҪҝз”ЁгҖӮе»әи®®ж №жҚ®дёҡеҠЎйңҖжұӮи°ғж•ҙеӯ—ж®өйӘҢиҜҒе’Ңй’©еӯҗгҖӮ
