关于URL编码

什么是URL编码？

URL编码（Percent Encoding）是一种将非ASCII字符和特殊字符转换为可在URL中安全传输的格式。%20 代表空格，%23 代表 #。

常见编码对照

%20 → 空格    %21 → !    %23 → #
%26 → &    %2F → /   %3F → ?
