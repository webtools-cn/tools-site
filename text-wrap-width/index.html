文本定宽换行工具能做什么？

文本定宽换行工具是一款免费在线文字处理工具，能够将长文本按指定的字符宽度自动插入换行符，使每行不超过设定的字符数。支持自定义行宽、首行缩进、后续行缩进和多种断词模式。适用于邮件排版、终端显示、代码注释格式化等场景。所有处理在浏览器本地完成，数据绝不上传服务器。

核心功能

  • 自定义行宽：设置10-200字符数（10-200），精确控制每行宽度
  • 缩进控制：分别设置首行缩进和后续行缩进
  • 断词模式：不断词/断词/断词+连字符三种模式
  • 实时预览：参数变更即时更新结果
  • 统计信息：显示行数、字符数等统计

使用教程

1. 在输入框中粘贴或输入需要换行的文本。

2. 设置行宽、缩进和断词模式参数。

3. 点击"换行"或使用Ctrl+Enter，查看结果。

4. 点击"复制结果"或下载为TXT文件。

应用场景

场景1：邮件排版

将长段落文本按78字符宽度换行，符合RFC 5322邮件格式标准。

场景2：终端显示

将输出文本按终端宽度换行，确保在80列终端中正确显示。

场景3：代码注释

将长注释按指定宽度换行，保持代码风格一致。

扩展知识

文本换行（Word Wrap）是文本处理的基本操作。RFC 5322规定电子邮件正文每行不超过78字符（推荐）和998字符（硬限制）。在终端环境中，标准终端宽度为80列。断词（Word Break）处理涉及连字符规则：英语中通常在音节间断词并添加连字符，而中文则可以在任意字符间断行。本工具支持三种断词模式以适应不同需求。

❓ 常见问题

什么是定宽换行？

定宽换行是将文本按指定字符宽度自动插入换行符，使每行不超过设定字符数。

支持中文吗？

支持。工具正确处理中文字符，每个中文字符计为1个宽度单位。

断词模式是什么？

断词模式决定单词超宽时的处理：不断词保持完整、断词在行宽处截断、断词+连字符在截断处加连字符。

数据会上传到服务器吗？

不会。所有处理在浏览器本地完成。

可以设置缩进吗？

可以。支持设置首行缩进和后续行缩进。
