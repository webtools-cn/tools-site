使用说明

  1. 选择转换方向：分数→小数 或 小数→分数
  2. 分数→小数：输入分子和分母
  3. 小数→分数：输入小数（如 0.75）
  4. 点击"计算"立即获得结果

分数与小数转换

分数转小数：分子 ÷ 分母 = 小数。例如 3/4 = 3÷4 = 0.75

小数转分数：将小数转换为最简分数。例如 0.75 = 75/100 = 3/4

本工具支持真分数、假分数、带分数（使用加号连接，如 1+1/2 = 1.5）。

常见问题

Q：分数转小数的精度如何？

对于有理数（有限小数），结果精确。对于无理数（无限不循环小数），保留10位小数。对于循环小数，会标记循环节。

Q：如何转换带分数？

带分数如 1½，可以在分子输入3（1×2+1），分母输入2。或者选择小数→分数模式，输入1.5。也可使用加号格式：在分子输入"1 1/2"格式。

Q：支持分数化简吗？

支持。小数转分数时，工具会自动将结果化为最简分数（如 0.5 → 1/2 而非 2/4），并显示计算步骤。

Q：分母可以为0吗？

不可以。分母为0的分数没有定义。工具会提示"分母不能为0"的错误信息。

Q：能处理百分数吗？

百分数即分母为100的分数。例如 75% = 75/100 = 3/4。输入百分数时建议先转换为小数或分数形式。

Q：需要联网使用吗？

纯前端应用，加载后断开网络也能正常使用。所有计算在浏览器本地完成。
