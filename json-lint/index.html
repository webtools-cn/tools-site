❓ 常见问题

JSON校验工具能检测哪些错误？

能检测常见JSON语法错误：缺少引号、多余的逗号（尾部逗号）、括号不匹配（{ } [ ]）、键名未加引号、使用单引号、包含注释、值类型错误等。会精确显示错误行号、列号和具体原因，方便快速定位修复。

支持多大的JSON文件？

纯前端实现，受浏览器内存限制。一般支持10MB以内的JSON文件。处理超大JSON时浏览器可能短暂卡顿。对于超大JSON文件建议使用专门的命令行工具如jq进行处理。

数据会上传到服务器吗？

不会。所有JSON解析和校验都在浏览器本地完成，您的数据不会离开设备。即使包含API密钥、用户数据等敏感信息的JSON配置，也可以放心使用。完全隐私安全。
