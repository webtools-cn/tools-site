常见问题

tsconfig.json 中必须配置哪些选项？

最核心的是 compilerOptions 中的 target（目标 JS 版本，如 ES2020/ESNext）和 module（模块系统，如 ESNext/CommonJS）。强烈推荐启用 strict 模式（包含所有严格类型检查）。outDir 指定输出目录，rootDir 指定源码目录。

strict 模式包含哪些检查？

strict: true 会启用一系列严格检查：strictNullChecks（严格的 null 检查）、noImplicitAny（禁止隐式 any）、strictFunctionTypes（严格函数类型）、strictBindCallApply（严格 bind/call/apply）、noImplicitThis（禁止不明确的 this）、alwaysStrict（始终使用严格模式）。

target 和 module 应该怎么选择？

target 决定了输出的 JS 语法版本：ES5 兼容旧浏览器但代码更长；ES2020/ESNext 支持最新语法。module 决定了模块系统：Node.js 推荐 CommonJS，前端项目推荐 ESNext，库作者推荐 ESNext+esModuleInterop。
