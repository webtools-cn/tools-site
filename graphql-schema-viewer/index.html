рҹ“Ө еҜјеҮә

д»Җд№ҲжҳҜGraphQL Schemaпјҹ

GraphQL SchemaжҳҜAPIзҡ„зұ»еһӢе®ҡд№үпјҢдҪҝз”ЁSDLжҸҸиҝ°ж•°жҚ®зұ»еһӢгҖҒжҹҘиҜўе…ҘеҸЈгҖҒеҸҳжӣҙж“ҚдҪңе’Ңи®ўйҳ…пјҢжҳҜе®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«Ҝзҡ„еҘ‘зәҰгҖӮ

д»Җд№ҲжҳҜSDLж јејҸпјҹ

SDLжҳҜGraphQLзҡ„зұ»еһӢе®ҡд№үиҜӯиЁҖпјҢз”Ёз®ҖжҙҒж–Үжң¬иҜӯжі•е®ҡд№үзұ»еһӢгҖҒеӯ—ж®өгҖҒеҸӮж•°е’Ңе…ізі»пјҢеҰӮtype User { id: ID! name: String! }гҖӮ

ж”ҜжҢҒе“ӘдәӣGraphQLзұ»еһӢпјҹ

ж”ҜжҢҒжүҖжңүж ҮеҮҶзұ»еһӢпјҡObject TypeгҖҒInterfaceгҖҒUnionгҖҒEnumгҖҒInput TypeгҖҒScalarпјҢд»ҘеҸҠQuery/Mutation/Subscriptionж №зұ»еһӢгҖӮ

ж•°жҚ®е®үе…Ёеҗ—пјҹ

е®Ңе…Ёе®үе…ЁгҖӮжүҖжңүSchemaи§Јжһҗе’Ңеұ•зӨәеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј гҖӮ

еҰӮдҪ•жҹҘзңӢзұ»еһӢиҜҰжғ…пјҹ

зӮ№еҮ»зұ»еһӢеҚЎзүҮеұ•ејҖиҜҰжғ…пјҢжҹҘзңӢеӯ—ж®өеҲ—иЎЁгҖҒеҸӮж•°гҖҒиҝ”еӣһзұ»еһӢе’ҢжҸҸиҝ°гҖӮ

еҸҜд»Ҙжҗңзҙўзұ»еһӢеҗ—пјҹ

еҸҜд»ҘгҖӮеңЁжҗңзҙўжЎҶдёӯиҫ“е…Ҙзұ»еһӢеҗҚжҲ–еӯ—ж®өеҗҚпјҢе®һж—¶иҝҮж»ӨеҢ№й…Қз»“жһңгҖӮ

ж”ҜжҢҒжӢ–жӢҪеҜје…Ҙж–Үд»¶еҗ—пјҹ

ж”ҜжҢҒгҖӮзӣҙжҺҘжӢ–жӢҪ.graphqlж–Үд»¶еҲ°зј–иҫ‘еҢәеҹҹеҚіеҸҜиҮӘеҠЁи§ЈжһҗгҖӮ

GraphQL SchemaжҹҘзңӢеҷЁиғҪеҒҡд»Җд№Ҳпјҹ

GraphQL SchemaжҹҘзңӢеҷЁжҳҜдёҖж¬ҫе…Қиҙ№еңЁзәҝејҖеҸ‘иҖ…е·Ҙе…·пјҢз”ЁдәҺи§Јжһҗе’ҢеҸҜи§ҶеҢ–еұ•зӨәGraphQL Schemaе®ҡд№үгҖӮж”ҜжҢҒSDLпјҲSchema Definition Languageпјүж јејҸиҫ“е…ҘпјҢиҮӘеҠЁиҜҶеҲ«жүҖжңүзұ»еһӢе®ҡд№үпјҢеҢ…жӢ¬Object TypeгҖҒQueryгҖҒMutationгҖҒSubscriptionгҖҒEnumгҖҒInput TypeгҖҒInterfaceе’ҢScalarгҖӮи§ЈжһҗеҗҺд»Ҙзӣҙи§Ӯзҡ„еҚЎзүҮеҪўејҸеұ•зӨәзұ»еһӢз»“жһ„е’Ңеӯ—ж®өдҝЎжҒҜпјҢж”ҜжҢҒжҗңзҙўзӯӣйҖүе’ҢеҲҶзұ»жөҸи§ҲгҖӮ

ж ёеҝғеҠҹиғҪ

  • SDLи§ЈжһҗпјҡиҮӘеҠЁи§ЈжһҗGraphQL Schema Definition LanguageпјҢиҜҶеҲ«жүҖжңүзұ»еһӢе®ҡд№үе’Ңеӯ—ж®ө
  • зұ»еһӢеҸҜи§ҶеҢ–пјҡжҢүзұ»еҲ«еұ•зӨәжүҖжңүзұ»еһӢпјҢеҢ…жӢ¬еӯ—ж®өеҗҚгҖҒзұ»еһӢгҖҒеҸӮж•°е’ҢжҸҸиҝ°
  • еҲҶзұ»зӯӣйҖүпјҡжҢүObject/Query/Mutation/Subscription/Enum/Input/InterfaceеҲҶзұ»жөҸи§Ҳ
  • жҗңзҙўеҠҹиғҪпјҡе®һж—¶жҗңзҙўзұ»еһӢеҗҚжҲ–еӯ—ж®өеҗҚпјҢеҝ«йҖҹе®ҡдҪҚзӣ®ж Үе®ҡд№ү
  • иҜҰжғ…еұ•ејҖпјҡзӮ№еҮ»зұ»еһӢеҚЎзүҮеұ•ејҖе®Ңж•ҙеӯ—ж®өеҲ—иЎЁе’ҢеҸӮж•°дҝЎжҒҜ
  • жӢ–жӢҪеҜје…ҘпјҡзӣҙжҺҘжӢ–жӢҪ.graphqlж–Үд»¶еҲ°зј–иҫ‘еҢәеҹҹиҮӘеҠЁи§Јжһҗ

дҪҝз”Ёж•ҷзЁӢ

1. еңЁж–Үжң¬жЎҶдёӯзІҳиҙҙGraphQL Schema SDLеҶ…е®№пјҢжҲ–жӢ–жӢҪ.graphqlж–Үд»¶еҜје…ҘгҖӮ

2. зӮ№еҮ»"и§Јжһҗ"жҢүй’®пјҢе·Ҙе…·иҮӘеҠЁиҜҶеҲ«жүҖжңүзұ»еһӢе®ҡд№үгҖӮ

3. еңЁз»ҹи®ЎеҢәжҹҘзңӢеҗ„зұ»еһӢж•°йҮҸпјҢдҪҝз”ЁеҲҶзұ»ж ҮзӯҫзӯӣйҖүзү№е®ҡзұ»еһӢгҖӮ

4. зӮ№еҮ»зұ»еһӢеҚЎзүҮеұ•ејҖжҹҘзңӢе®Ңж•ҙеӯ—ж®өеҲ—иЎЁгҖҒеҸӮж•°е’ҢжҸҸиҝ°гҖӮ

5. дҪҝз”ЁжҗңзҙўжЎҶеҝ«йҖҹжҹҘжүҫзұ»еһӢжҲ–еӯ—ж®өгҖӮCtrl+Enterеҝ«йҖҹи§ЈжһҗпјҢCtrl+Shift+CеӨҚеҲ¶з»“жһңгҖӮ

еә”з”ЁеңәжҷҜ

еңәжҷҜ1пјҡAPIж–ҮжЎЈе®Ўйҳ…

еүҚз«ҜејҖеҸ‘иҖ…йӣҶжҲҗGraphQL APIеүҚпјҢдҪҝз”Ёжң¬е·Ҙе…·еҝ«йҖҹжөҸи§ҲSchemaе®ҡд№үпјҢдәҶи§ЈеҸҜз”ЁжҹҘиҜўе’Ңзұ»еһӢз»“жһ„гҖӮ

еңәжҷҜ2пјҡSchemaи®ҫи®ЎиҜ„е®Ў

еҗҺз«Ҝеӣўйҳҹи®ҫи®ЎGraphQL Schemaж—¶пјҢйҖҡиҝҮеҸҜи§ҶеҢ–жҹҘзңӢзұ»еһӢе…ізі»е’Ңеӯ—ж®өе®ҡд№үпјҢзЎ®дҝқи®ҫи®ЎеҗҲзҗҶгҖӮ

еңәжҷҜ3пјҡеӯҰд№ GraphQL

еҲқеӯҰиҖ…йҖҡиҝҮи§ЈжһҗзӨәдҫӢSchemaпјҢзӣҙи§ӮзҗҶи§ЈGraphQLзҡ„зұ»еһӢзі»з»ҹгҖҒжҹҘиҜўе…ҘеҸЈе’Ңж•°жҚ®е…ізі»гҖӮ

еңәжҷҜ4пјҡSchemaиҝҒз§»

ејҖеҸ‘иҖ…еҜ№жҜ”ж–°ж—§Schemaж—¶пјҢеҝ«йҖҹжҹҘзңӢзұ»еһӢеҸҳжӣҙпјҢзЎ®и®Өеӯ—ж®өеўһеҲ е’Ңзұ»еһӢдҝ®ж”№гҖӮ

жү©еұ•зҹҘиҜҶ

GraphQL SchemaжҳҜGraphQL APIзҡ„ж ёеҝғпјҢе®ҡд№үдәҶж•°жҚ®зҡ„зұ»еһӢзі»з»ҹе’Ңж“ҚдҪңе…ҘеҸЈгҖӮSDLжҳҜGraphQLзҡ„зұ»еһӢе®ҡд№үиҜӯиЁҖпјҢдҪҝз”Ёз®ҖжҙҒзҡ„ж–Үжң¬иҜӯжі•жҸҸиҝ°зұ»еһӢе’Ңе…ізі»гҖӮSchemaеҢ…еҗ«дёүз§Қж №ж“ҚдҪңзұ»еһӢпјҡQueryпјҲиҜ»еҸ–ж•°жҚ®пјүгҖҒMutationпјҲдҝ®ж”№ж•°жҚ®пјүе’ҢSubscriptionпјҲе®һж—¶и®ўйҳ…пјүгҖӮзұ»еһӢзі»з»ҹж”ҜжҢҒObject TypeпјҲеҜ№иұЎзұ»еһӢпјүгҖҒInterfaceпјҲжҺҘеҸЈпјүгҖҒUnionпјҲиҒ”еҗҲзұ»еһӢпјүгҖҒEnumпјҲжһҡдёҫпјүгҖҒInput TypeпјҲиҫ“е…Ҙзұ»еһӢпјүе’ҢScalarпјҲж ҮйҮҸпјүгҖӮеӯ—ж®өеҸҜд»ҘеёҰеҸӮж•°пјҲargumentsпјүпјҢдҪҝз”Ё!ж Үи®°йқһз©әзұ»еһӢпјҢдҪҝз”Ё[Type]иЎЁзӨәеҲ—иЎЁзұ»еһӢгҖӮIntrospectionжҹҘиҜўе…Ғи®ёе®ўжҲ·з«ҜеңЁиҝҗиЎҢж—¶иҺ·еҸ–SchemaдҝЎжҒҜгҖӮ
