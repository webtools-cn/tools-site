е№ҝе‘ҠдҪҚ - дёӯйғЁ (728Г—90)

дҪҝз”Ёж•ҷзЁӢ

жң¬JSONж ЎйӘҢе·Ҙе…·ж“ҚдҪңз®ҖеҚ•дё”еҠҹиғҪе®Ңе–„пјҢд»ҘдёӢжҳҜиҜҰз»Ҷзҡ„дҪҝз”ЁжҢҮеҚ—пјҡ

зІҳиҙҙJSONпјҡеңЁиҫ“е…ҘжЎҶдёӯзІҳиҙҙйңҖиҰҒж ЎйӘҢзҡ„JSONж–Үжң¬гҖӮе·Ҙе…·ж”ҜжҢҒд»»ж„ҸеҗҲжі•зҡ„JSONж јејҸпјҢеҢ…жӢ¬еөҢеҘ—еҜ№иұЎгҖҒж•°з»„гҖҒеӯ—з¬ҰдёІгҖҒж•°еӯ—гҖҒеёғе°”еҖје’ҢnullгҖӮиҫ“е…Ҙж—¶дёӢж–№дјҡе®һж—¶жҳҫзӨәеӯ—з¬Ұж•°е’ҢиЎҢж•°з»ҹи®ЎгҖӮ

зӮ№еҮ»ж ЎйӘҢпјҡзӮ№еҮ»гҖҢж ЎйӘҢгҖҚжҢүй’®пјҢе·Ҙе…·дјҡи§ЈжһҗJSONе№¶з»ҷеҮәиҜҰз»Ҷз»“жһңгҖӮеҰӮжһңJSONж јејҸжӯЈзЎ®пјҢдјҡжҳҫзӨәз»ҝиүІзҡ„гҖҢвң… жңүж•ҲJSONгҖҚзҠ¶жҖҒпјҢе№¶еңЁз»“жһңеҢәеҹҹеұ•зӨәз»“жһ„еҢ–зҡ„иҜӯжі•й«ҳдә®и§ҶеӣҫгҖӮеҰӮжһңж јејҸжңүиҜҜпјҢдјҡжҳҫзӨәзәўиүІзҡ„й”ҷиҜҜдҝЎжҒҜпјҢеҢ…жӢ¬й”ҷиҜҜзұ»еһӢгҖҒжүҖеңЁиЎҢеҸ·е’ҢеҲ—еҸ·пјҢеё®еҠ©дҪ еҝ«йҖҹе®ҡдҪҚй—®йўҳгҖӮ

ж јејҸеҢ–дёҺеҺӢзј©пјҡгҖҢж јејҸеҢ–гҖҚжҢүй’®дјҡе°ҶJSONжҢүеұӮзә§зј©иҝӣпјҢж–№дҫҝйҳ…иҜ»е’ҢеҲҶжһҗеөҢеҘ—з»“жһ„гҖӮгҖҢеҺӢзј©гҖҚжҢүй’®дјҡз§»йҷӨжүҖжңүз©әзҷҪеӯ—з¬ҰпјҢе°ҶJSONеҺӢзј©дёәдёҖиЎҢпјҢйҖӮеҗҲз”ЁдәҺзҪ‘з»ңдј иҫ“е’ҢеӯҳеӮЁдјҳеҢ–гҖӮж јејҸеҢ–еҗҺзҡ„з»“жһңеҗҢж ·ж”ҜжҢҒиҜӯжі•й«ҳдә®жҳҫзӨәгҖӮ

еҠ иҪҪзӨәдҫӢпјҡзӮ№еҮ»гҖҢеҠ иҪҪзӨәдҫӢгҖҚеҸҜеҝ«йҖҹеЎ«е……дёҖж®өж ҮеҮҶJSONпјҢж–№дҫҝж–°з”ЁжҲ·дәҶи§Је·Ҙе…·дҪҝз”Ёж–№жі•гҖӮзӨәдҫӢеҢ…еҗ«дәҶеҜ№иұЎгҖҒж•°з»„гҖҒеӯ—з¬ҰдёІгҖҒж•°еӯ—гҖҒеёғе°”еҖјзӯүеӨҡз§Қж•°жҚ®зұ»еһӢгҖӮ

еә”з”ЁеңәжҷҜ

JSONж ЎйӘҢе·Ҙе…·еңЁWebејҖеҸ‘е’Ңж•°жҚ®дәӨжҚўдёӯеә”з”Ёе№ҝжіӣпјҢд»ҘдёӢжҳҜеҮ дёӘе…ёеһӢзҡ„дҪҝз”ЁеңәжҷҜпјҡ

APIжҺҘеҸЈи°ғиҜ•пјҡејҖеҸ‘RESTful APIж—¶пјҢеүҚеҗҺз«ҜйҖҡиҝҮJSONдј иҫ“ж•°жҚ®гҖӮеҪ“жҺҘеҸЈиҝ”еӣһејӮеёёж—¶пјҢејҖеҸ‘иҖ…йңҖиҰҒе°Ҷиҝ”еӣһеҶ…е®№зІҳиҙҙеҲ°ж ЎйӘҢе·Ҙе…·дёӯпјҢеҝ«йҖҹзЎ®и®ӨжҳҜJSONж јејҸй—®йўҳиҝҳжҳҜдёҡеҠЎйҖ»иҫ‘й—®йўҳгҖӮзү№еҲ«жҳҜйҒҮеҲ°500й”ҷиҜҜж—¶пјҢжңҚеҠЎз«ҜеҸҜиғҪиҝ”еӣһдәҶйқһж ҮеҮҶJSONпјҲеҰӮHTMLй”ҷиҜҜйЎөйқўпјүпјҢжӯӨж—¶ж ЎйӘҢе·Ҙе…·еҸҜд»Ҙз«ӢеҚіиҜҶеҲ«гҖӮ

й…ҚзҪ®ж–Үд»¶жЈҖжҹҘпјҡзҺ°д»Јеә”з”ЁеӨ§йҮҸдҪҝз”ЁJSONдҪңдёәй…ҚзҪ®ж–Үд»¶ж јејҸпјҲеҰӮpackage.jsonгҖҒtsconfig.jsonгҖҒmanifest.jsonзӯүпјүгҖӮеңЁдҝ®ж”№иҝҷдәӣж–Үд»¶еҗҺпјҢдҪҝз”Ёж ЎйӘҢе·Ҙе…·еҸҜд»ҘзЎ®дҝқиҜӯжі•жӯЈзЎ®пјҢйҒҝе…Қеӣ дёҖдёӘеӨҡдҪҷзҡ„йҖ—еҸ·еҜјиҮҙж•ҙдёӘеә”з”Ёж— жі•еҗҜеҠЁгҖӮ

ж•°жҚ®дәӨжҚўйӘҢиҜҒпјҡеңЁеҫ®жңҚеҠЎжһ¶жһ„дёӯпјҢжңҚеҠЎд№Ӣй—ҙйҖҡиҝҮJSONж¶ҲжҒҜйҳҹеҲ—жҲ–HTTPиҜ·жұӮдәӨжҚўж•°жҚ®гҖӮжҺҘж”¶ж–№еңЁи§ЈжһҗеүҚе…Ҳз”Ёж ЎйӘҢе·Ҙе…·йӘҢиҜҒJSONж јејҸпјҢеҸҜд»ҘжҸҗеүҚеҸ‘зҺ°ж•°жҚ®иҙЁйҮҸй—®йўҳпјҢйҒҝе…Қи§ЈжһҗејӮеёёеҜјиҮҙзҡ„жңҚеҠЎж•…йҡңгҖӮ

жү©еұ•зҹҘиҜҶ

JSONи§„иҢғжј”иҝӣпјҡJSONпјҲJavaScript Object Notationпјүз”ұDouglas CrockfordдәҺ2001е№ҙжҸҗеҮәпјҢ2006е№ҙиў«RFC 4627ж ҮеҮҶеҢ–пјҢ2017е№ҙжӣҙж–°дёәRFC 8259гҖӮе®ғжәҗдәҺJavaScriptзҡ„еҜ№иұЎеӯ—йқўйҮҸиҜӯжі•пјҢдҪҶе·Із»ҸжҲҗдёәзӢ¬з«ӢдәҺиҜӯиЁҖзҡ„ж•°жҚ®ж јејҸпјҢиў«еҮ д№ҺжүҖжңүзј–зЁӢиҜӯиЁҖеҺҹз”ҹж”ҜжҢҒгҖӮ

еёёи§ҒJSONй”ҷиҜҜпјҡзјәе°‘йҖ—еҸ·пјҲеҰӮ {"a":1 "b":2}пјүжҳҜжңҖеёёи§Ғзҡ„й”ҷиҜҜпјӣеҚ•еј•еҸ·д»ЈжӣҝеҸҢеј•еҸ·пјҲеҰӮ {'a':1}пјүеңЁJavaScriptеҜ№иұЎдёӯеҗҲжі•дҪҶеңЁJSONдёӯдёҚеҗҲжі•пјӣе°ҫйғЁйҖ—еҸ·пјҲеҰӮ [1,2,]пјүеңЁES2019еҗҺзҡ„JavaScriptдёӯеҗҲжі•дҪҶJSONдёҚе…Ғи®ёпјӣжіЁйҮҠпјҲеҰӮ // commentпјүеңЁJSONдёӯе®Ңе…ЁдёҚиў«ж”ҜжҢҒпјҢиҝҷд№ҹжҳҜJSONдёҺJSONC/JSON5зҡ„дё»иҰҒеҢәеҲ«гҖӮ

JSONжӣҝд»Јж–№жЎҲпјҡеҜ№дәҺйңҖиҰҒжіЁйҮҠзҡ„еңәжҷҜпјҢеҸҜд»ҘдҪҝз”ЁJSON5жҲ–JSONCпјҲJSON with CommentsпјүпјӣеҜ№дәҺйңҖиҰҒжӣҙдёҘж јзұ»еһӢзі»з»ҹзҡ„еңәжҷҜпјҢеҸҜд»ҘдҪҝз”ЁProtobufжҲ–AvroпјӣеҜ№дәҺй…ҚзҪ®ж–Үд»¶пјҢYAMLеӣ ж”ҜжҢҒжіЁйҮҠе’Ңжӣҙз®ҖжҙҒзҡ„иҜӯжі•иҖҢе№ҝеҸ—ж¬ўиҝҺпјҢдҪҶYAMLзҡ„зј©иҝӣж•Ҹж„ҹжҖ§д№ҹеҜјиҮҙе®ғжӣҙе®№жҳ“еҮәй”ҷгҖӮ

еёёи§Ғй—®йўҳ FAQ

JSONж ЎйӘҢе·Ҙе…·иғҪжЈҖжөӢе“Әдәӣй”ҷиҜҜпјҹ

еҸҜд»ҘжЈҖжөӢиҜӯжі•й”ҷиҜҜпјҲеҰӮзјәе°‘йҖ—еҸ·гҖҒжӢ¬еҸ·дёҚеҢ№й…ҚпјүгҖҒйқһжі•еӯ—з¬ҰгҖҒJSONж јејҸиҝқи§„пјҲеҰӮеҚ•еј•еҸ·д»ЈжӣҝеҸҢеј•еҸ·гҖҒе°ҫйғЁйҖ—еҸ·пјүгҖҒж•°жҚ®зұ»еһӢй”ҷиҜҜзӯүпјҢе№¶зІҫзЎ®е®ҡдҪҚеҲ°е…·дҪ“иЎҢеҸ·е’ҢеҲ—еҸ·гҖӮ

JSONе’ҢJavaScriptеҜ№иұЎжңүд»Җд№ҲеҢәеҲ«пјҹ

JSONжҳҜж•°жҚ®дәӨжҚўж јејҸпјҢJavaScriptеҜ№иұЎжҳҜзј–зЁӢиҜӯиЁҖдёӯзҡ„ж•°жҚ®з»“жһ„гҖӮе…ій”®еҢәеҲ«пјҡJSONй”®еҝ…йЎ»еҠ еҸҢеј•еҸ·пјӣJSONдёҚж”ҜжҢҒеҮҪж•°гҖҒundefinedгҖҒжіЁйҮҠпјӣJSONеӯ—з¬ҰдёІеҝ…йЎ»з”ЁеҸҢеј•еҸ·гҖӮ

дёәд»Җд№ҲJSONдёҚиғҪжңүе°ҫйҡҸйҖ—еҸ·пјҹ

JSONи§„иҢғпјҲRFC 8259пјүдёҚе…Ғи®ёеҜ№иұЎжҲ–ж•°з»„жңҖеҗҺдёҖдёӘе…ғзҙ еҗҺжңүйҖ—еҸ·гҖӮиҝҷжҳҜдёәдәҶз®ҖеҢ–и§ЈжһҗеҷЁе®һзҺ°е’ҢжҸҗй«ҳи·ЁиҜӯиЁҖе…је®№жҖ§гҖӮJavaScriptеҜ№иұЎе…Ғи®ёе°ҫйҡҸйҖ—еҸ·пјҢдҪҶJSONдёҚиЎҢгҖӮ

ж ЎйӘҢз»“жһңдёӯзҡ„иҜӯжі•й«ҳдә®жҳҜд»Җд№ҲйўңиүІпјҹ

й”®еҗҚжҳҫзӨәдёәи“қиүІпјҲkeyпјүпјҢеӯ—з¬ҰдёІжҳҫзӨәдёәз»ҝиүІпјҲ"value"пјүпјҢж•°еӯ—жҳҫзӨәдёәй»„иүІпјҲ123пјүпјҢеёғе°”еҖјжҳҫзӨәдёәзІүиүІпјҲtrueпјүпјҢnullжҳҫзӨәдёәзҒ°иүІпјҲnullпјүгҖӮ

е·Ҙе…·ж”ҜжҢҒи¶…еӨ§JSONж–Үд»¶еҗ—пјҹ

е·Ҙе…·еңЁжөҸи§ҲеҷЁз«ҜиҝҗиЎҢпјҢе»әи®®еӨ„зҗҶдёҚи¶…иҝҮ10MBзҡ„JSONж–Үд»¶гҖӮеҜ№дәҺи¶…еӨ§ж–Үд»¶пјҢе»әи®®дҪҝз”Ёе‘Ҫд»ӨиЎҢе·Ҙе…·еҰӮ jq жҲ–дё“й—Ёзҡ„JSONеӨ„зҗҶиҪҜд»¶гҖӮ

ж•°жҚ®дјҡдёҠдј еҲ°жңҚеҠЎеҷЁеҗ—пјҹ

дёҚдјҡгҖӮжүҖжңүеӨ„зҗҶйғҪеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј жңҚеҠЎеҷЁпјҢдҝқйҡңдҪ зҡ„ж•°жҚ®йҡҗз§Ғе®үе…ЁгҖӮ

дёәд»Җд№Ҳж јејҸеҢ–еҗҺзҡ„JSONжңүзј©иҝӣпјҹ

ж јејҸеҢ–еҠҹиғҪдҪҝз”Ё2дёӘз©әж јдҪңдёәзј©иҝӣеҚ•дҪҚпјҢиҝҷжҳҜдёҡз•Ңе№ҝжіӣи®ӨеҸҜзҡ„и§„иҢғпјҲеҰӮPrettierй»ҳи®Өй…ҚзҪ®пјүгҖӮзј©иҝӣеұӮзә§дёҺJSONзҡ„еөҢеҘ—ж·ұеәҰдёҖиҮҙпјҢдҫҝдәҺйҳ…иҜ»е’Ңи°ғиҜ•гҖӮ

еҸҜд»Ҙж ЎйӘҢJSON Schemaеҗ—пјҹ

еҪ“еүҚзүҲжң¬д»…ж ЎйӘҢJSONиҜӯжі•жӯЈзЎ®жҖ§пјҢжҡӮдёҚж”ҜжҢҒJSON SchemaйӘҢиҜҒгҖӮеҰӮйңҖSchemaйӘҢиҜҒпјҢеҸҜдҪҝз”ЁJSON Schema Validatorзӯүдё“й—Ёе·Ҙе…·гҖӮ

JSONйӘҢиҜҒжЈҖжҹҘе“Әдәӣи§„еҲҷпјҹ

JSONйӘҢиҜҒдҫқжҚ® RFC 8259 ж ҮеҮҶжЈҖжҹҘпјҡеӯ—з¬ҰдёІеҝ…йЎ»дҪҝз”ЁеҸҢеј•еҸ·гҖҒдёҚиғҪжңүе°ҫйҡҸйҖ—еҸ·гҖҒй”®еҗҚеҝ…йЎ»дёәеӯ—з¬ҰдёІгҖҒж•°еҖјдёҚиғҪжңүеүҚеҜјйӣ¶гҖҒUnicodeеӯ—з¬Ұеҝ…йЎ»жӯЈзЎ®иҪ¬д№үзӯүгҖӮ
