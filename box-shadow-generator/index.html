常见问题

CSS盒子阴影生成器能做什么？
这是一个可视化CSS box-shadow代码生成工具。您可以通过滑块直观调节阴影的X/Y偏移、模糊半径、扩散半径、颜色和透明度，选择外阴影或内阴影（inset），添加多层阴影叠加，实时预览效果，最后一键复制生成的CSS代码到您的项目中。
什么是box-shadow的各个参数？
CSS box-shadow语法：box-shadow: offset-x offset-y blur-radius spread-radius color inset。offset-x/y控制阴影水平/垂直偏移；blur-radius控制模糊程度；spread-radius控制阴影扩散大小；color设置阴影颜色；inset关键字将外阴影变为内阴影。
可以添加多层阴影吗？
可以！点击「添加阴影层」按钮即可添加新的阴影层。CSS支持用逗号分隔多个box-shadow值来实现多层阴影叠加效果。每个阴影层可独立调节所有参数，实现复杂的阴影设计。
