🎯 应用场景

单位换算器在日常生活和专业工作中都非常实用。出国旅行时，可以将英里转换为公里、华氏度转换为摄氏度；烹饪时，可以将盎司转换为克、杯转换为毫升。工程技术人员在阅读和转换国外技术文档时，经常需要进行各种英制和公制单位的转换。

在 IT 领域，数据存储单位的换算尤为重要。硬盘厂商通常使用十进制（1GB = 10^9 字节）标注容量，而操作系统使用二进制（1GiB = 2^30 字节）显示，导致用户常感到容量「缩水」。本工具同时支持两种标准，帮助用户理解差异。

💡 扩展知识

国际单位制（SI）是目前世界上最广泛采用的计量系统，以米、千克、秒、安培、开尔文、摩尔和坎德拉七个基本单位为基础。大多数国家（除美国、缅甸、利比里亚等少数国家外）已在官方场合全面采用公制单位。

在数据存储领域，十进制和二进制的混淆由来已久。IEC 于 1998 年制定了二进制前缀标准：KiB（kibibyte）、MiB（mebibyte）、GiB（gibibyte）等，以区别于 KB、MB、GB 等十进制前缀。然而在实际使用中，厂商和用户的习惯仍未统一，导致理解上的混乱。

❓ 常见问题解答
这个单位换算器支持哪些单位类别？
支持6大类单位：长度（米、千米、英寸、英尺、英里、海里等）、重量（千克、克、磅、盎司、斤、两等）、温度（摄氏度、华氏度、开尔文、兰氏度）、速度（米/秒、千米/小时、英里/小时、节、马赫等）、面积（平方米、平方千米、公顷、英亩、平方英尺等）、数据存储（字节、KB、MB、GB、TB、PB及二进制单位）。
单位换算是如何计算的？
所有换算均在浏览器本地通过JavaScript实时计算。对于温度，使用各温标间的数学公式进行转换；对于其他物理量，以国际标准单位（SI单位）为基准，通过换算系数进行比例转换。
批量换算模式怎么用？
在批量换算模式下，您可以输入多行数据，每行格式为"数值 单位"（例如"100 米"），系统会自动将每行数据换算为当前类别的所有单位结果，并以表格形式展示。适合需要一次性换算多个数值的场景。
温度换算为什么和其他单位不一样？
温度是非线性单位，不同温标间的换算需要加减常数，而不是简单的比例关系。例如：摄氏度转华氏度是 F = C × 9/5 + 32；摄氏度转开尔文是 K = C + 273.15。其他物理量（长度、重量等）都是线性关系，可以直接通过比例系数换算。
数据存储中的KB和KiB有什么区别？
KB（千字节）使用十进制，1 KB = 1000 字节；KiB（Kibibyte）使用二进制，1 KiB = 1024 字节。同理，MB = 1,000,000 字节，MiB = 1,048,576 字节。操作系统通常显示的是二进制单位（如Windows显示GB但实际上是GiB），而硬盘厂商通常使用十进制单位。
这个工具的数据会上传吗？
绝对不会。所有计算都在浏览器本地完成，没有任何数据上传到服务器。您可以断网使用此工具，也可以放心输入任何数值，您的数据完全在本地处理。

国际单位制（SI）是什么？

根据 国际计量局 (BIPM)：SI单位制是全球通用的度量衡标准，包括7个基本单位（米、千克、秒、安培、开尔文、摩尔、坎德拉）。所有其他单位都可由基本单位导出。本工具使用SI标准的精确换算系数。
