❓ 帮助说明

什么是Cookie？

Cookie（小型文本文件）是网站存储在用户浏览器中的小块数据，用于保持登录状态、记录偏好设置、追踪用户行为等。每个Cookie包含名称、值和可选的属性（过期时间、域名、路径、安全标志等）。

如何查看当前网站的Cookie？

打开工具的「Cookie管理」标签页，工具会自动读取当前页面域名下的所有Cookie，并以表格形式展示。点击「刷新Cookie」可重新读取。注意：出于安全考虑，HttpOnly标记的Cookie无法通过JavaScript读取。

Cookie属性说明

  • Domain — 指定Cookie可以发送到的域名
  • Path — 指定Cookie在域名下的路径范围
  • Expires/Max-Age — Cookie的过期时间，过期后浏览器会自动删除
  • Secure — 只在HTTPS连接下传输
  • HttpOnly — 禁止JavaScript访问，防止XSS攻击
  • SameSite — 控制跨站请求时是否发送Cookie（Strict/Lax/None）
