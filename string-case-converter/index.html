字符串大小写转换器能做什么？

字符串大小写转换器是一款免费在线开发者工具，支持将字符串在camelCase、PascalCase、snake_case、kebab-case、CONSTANT_CASE、dot.case、path/case和Sentence case等8种命名格式之间互转。支持批量处理，自动识别输入格式，适用于代码重构、API命名规范统一、数据库字段映射等场景。所有处理在浏览器本地完成。

核心功能

  • 8种格式互转：camelCase、PascalCase、snake_case、kebab-case、CONSTANT_CASE、dot.case、path/case、Sentence case
  • 自动识别：智能检测输入字符串的命名格式
  • 批量处理：每行一个字符串，批量转换
  • 一键复制：点击任意结果行即可复制
  • 实时预览：输入即转换，无需点击按钮

使用教程

1. 在输入框输入要转换的字符串，支持多行批量输入。

2. 工具自动识别格式并实时显示所有8种转换结果。

3. 点击结果行右侧的复制按钮复制单个结果。

4. 点击"复制全部"获取所有格式的转换结果。

应用场景

场景1：代码重构

将JavaScript的camelCase变量名批量转换为Python的snake_case，或反之。

场景2：API设计

统一API字段命名规范，在JSON的camelCase和数据库的snake_case之间转换。

场景3：CSS类名

在BEM的kebab-case和CSS Modules的camelCase之间切换。

扩展知识

命名规范因语言而异：JavaScript/Java偏好camelCase，Python/Ruby用snake_case，HTML/CSS用kebab-case，C用PascalCase，环境变量用CONSTANT_CASE。转换原理是先分词（按大写字母、下划线、连字符、点号、斜杠分割），再按目标格式重组。连续大写字母（如HTMLParser）作为整体处理，避免错误分割。

❓ 常见问题

支持哪些命名格式？

支持camelCase、PascalCase、snake_case、kebab-case、CONSTANT_CASE、dot.case、path/case和Sentence case共8种。

如何识别输入格式？

自动检测分隔符（下划线、连字符、大写字母、点号、斜杠）来判断格式。

支持批量转换吗？

支持。每行一个字符串，批量转换后逐行显示结果。

连续大写字母怎么处理？

连续大写字母（如HTMLParser中的HTML）作为整体保留，不会错误分割。

数据会上传到服务器吗？

不会。所有转换在浏览器本地完成。

camelCase和PascalCase的区别？

camelCase首字母小写（myVariable），PascalCase首字母大写（MyVariable）。
