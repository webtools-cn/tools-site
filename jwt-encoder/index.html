❓ 常见问题

JWT编码器是什么？

JWT编码器是一种在线工具，用于创建和签名JSON Web Token。您可以自定义Header（算法和类型）和Payload（声明claims），然后使用密钥对token进行签名，生成完整的JWT字符串。常用于API认证、单点登录等场景。

本工具支持哪些签名算法？

本工具支持HMAC系列对称签名算法：HS256（HMAC-SHA256）、HS384（HMAC-SHA384）和HS512（HMAC-SHA512）。这些算法使用共享密钥进行签名，适合大多数应用场景。RSA和ECDSA等非对称算法需要更复杂的密钥管理，暂不支持。

我的密钥安全吗？

完全安全。所有签名操作都在浏览器本地完成，使用Web Crypto API进行加密运算，密钥和token数据绝不会上传到任何服务器。使用完毕后建议关闭页面以清除敏感数据。

JWT和JWS有什么区别？

JWT（JSON Web Token）是令牌格式规范，JWS（JSON Web Signature）是签名机制。我们通常说的JWT实际上就是JWS——即经过签名的JWT。还有JWE（加密的JWT），本工具生成的是JWS格式的JWT。
