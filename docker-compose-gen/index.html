🔧 配置Docker Compose服务

📄 生成的 docker-compose.yml

📖 使用教程

Docker Compose生成器帮助您快速创建docker-compose.yml文件。点击上方的服务模板按钮即可添加常用服务（MySQL、Redis、Nginx等），或使用"自定义服务"手动添加任意容器。每个服务都可以配置镜像、端口映射、卷挂载、环境变量和重启策略。

配置完成后点击"生成docker-compose.yml"即可获得标准格式的YAML文件，支持一键复制或下载。所有配置在浏览器本地完成，无需上传数据。

💼 应用场景

Docker Compose生成器适用于多种开发场景：本地开发环境快速搭建（MySQL+Redis+App一键生成）；微服务架构的容器编排配置；CI/CD流水线中的测试环境定义；团队新成员快速获取开发环境配置；教学演示中的多容器应用示例。
