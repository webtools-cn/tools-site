Docker Run转Compose工具能做什么？

手动将docker run命令改写为docker-compose.yml文件容易遗漏参数且格式繁琐。本工具将docker run命令一键转换为标准docker-compose.yml配置，自动处理端口映射、环境变量、数据卷、网络、重启策略等参数。所有转换在浏览器本地完成。

核心功能

  • 完整参数支持：--name/-p/-e/-v/--network/--restart/-d等
  • 标准YAML输出：生成version 3.8格式的docker-compose.yml
  • 自动服务命名：从镜像名自动提取服务名
  • 网络配置：自动生成networks定义
  • 批量转换：支持多行docker run命令批量转换

使用教程

步骤1：粘贴docker run命令。可以从终端历史复制，也可以手动输入。

步骤2：工具自动解析参数并生成docker-compose.yml。

步骤3：复制YAML内容或下载为docker-compose.yml文件。快捷键Ctrl+Enter转换、Ctrl+Shift+C复制。
