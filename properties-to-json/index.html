📖 使用说明

Properties转JSON工具将Java Properties格式的key=value文本转换为JSON对象，支持点号分隔的嵌套键。

使用方法

  1. 在输入框中粘贴Properties文本
  2. 点击"转换"按钮
  3. 点号分隔的键会自动展开为嵌套对象
  4. 点击"复制"获取JSON文本

转换规则

  • 嵌套键 — server.port=8080 变为 {"server":{"port":8080}}
  • 注释行 — 以 # 或 ! 开头的行会被忽略
  • 类型推断 — 自动识别数字、布尔值和null

❓ 常见问题

Properties转JSON工具是免费的吗？

是的，完全免费使用，无需注册。

点号分隔的键如何处理？

点号分隔的键会自动展开为嵌套JSON对象。例如 server.port=8080 会变为 {"server":{"port":8080}}。

使用时数据会上传到服务器吗？

不会。所有转换都在你的浏览器中完成。

支持注释行吗？

支持。以 # 或 ! 开头的行会被自动忽略。

值类型如何识别？

工具会自动推断值类型：true/false转为布尔值，纯数字转为数字，null转为null，其他保持字符串。
