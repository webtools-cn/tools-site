💡 什么是 Base64？

Base64 是一种将二进制数据（如图片、文件）编码为 ASCII 字符串的方法，常用于在 HTML、CSS、JSON 中嵌入文件。

常见用途：

• 将小图标/图片直接嵌入 CSS，减少 HTTP 请求

• 在 HTML 中内联显示图片（<img src="data:image/png;base64,...">)

• 在 API JSON 响应中传输图片或文件

• 将字体文件转为 base64 在 CSS 中使用

注意：Base64 编码后体积比原文件大约 33%，因此仅适合小文件（建议 < 1MB）。
