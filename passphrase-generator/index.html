密码短语生成器能做什么？

密码短语生成器是一款纯前端运行的安全密码创建工具，它从大型词表中随机选取多个单词，组合成既安全又易记的密码短语。与传统随机字符密码相比，密码短语具有更高的熵值和更好的可记忆性，是现代密码安全推荐的方案。

核心功能

多单词组合：支持3-10个单词自由组合，单词越多安全性越高，4个单词即可达到44位熵值。

多种分隔符：支持连字符、点号、下划线、空格、无分隔和自定义分隔符，满足不同网站密码规则。

首字母大写：可选将每个单词首字母大写，增加密码复杂度同时保持可读性。

随机数字：可在末尾添加随机数字，满足要求包含数字的密码策略。

中文词表：支持使用中文词表生成密码短语，适合中文用户记忆。

强度评估：实时计算密码熵值并可视化显示强度等级，帮助你选择足够安全的密码。

使用教程

1. 设置参数：在左侧面板中选择单词数量（推荐4-6个），选择分隔符类型，根据需要勾选首字母大写和添加数字选项。

2. 生成密码：点击"生成密码短语"按钮，工具将使用加密安全随机数从词表中选取单词并组合。可多次点击直到满意。

3. 查看强度：生成后查看强度评估条和熵值，绿色表示安全，黄色表示一般，红色表示较弱。

4. 复制使用：点击复制按钮将密码短语复制到剪贴板，或下载为文本文件保存。

应用场景

场景1：主密码管理

为密码管理器（如1Password、Bitwarden）设置主密码，使用6个以上单词的密码短语，既安全又不容易忘记。

场景2：加密货币钱包

加密货币钱包的助记词本质上就是密码短语。本工具可以帮助你理解密码短语的原理，选择足够安全的词数。

场景3：Wi-Fi密码

为家庭或办公室Wi-Fi设置密码，使用4-5个单词的密码短语，方便告知访客同时保持足够安全性。

场景4：账户恢复密语

为重要账户设置安全问题和恢复密语，使用密码短语代替简单答案，大幅提高安全性。

扩展知识

熵值计算：密码的安全性用熵值（entropy）衡量，单位为比特。每个从8192词表中随机选取的单词贡献约13位熵值（log2(8192)=13）。4个单词约52位，6个单词约78位。NIST建议至少80位熵值用于高安全场景。

Diceware 方法：Diceware由Arnold Reinhold于1995年提出，使用5个骰子从7776词表中选词。本工具使用Web Crypto API的crypto.getRandomValues()替代骰子，提供同等安全性的真随机性。

最佳实践：①不要在多个重要账户使用同一密码短语；②将密码短语写在纸上存放在安全位置比存在电脑上更安全；③定期更换高价值账户的密码短语；④使用密码管理器存储和管理密码。

常见问题

密码短语生成器是什么？

密码短语生成器是一种创建由多个随机单词组成的密码的工具。与传统随机字符密码相比，密码短语更易记忆且同样安全，例如 horse-battery-staple-correct 比 Xq#9kL!2 更容易记住。

密码短语比传统密码更安全吗？

是的。4个随机单词的密码短语约有52位熵值，相当于8位完全随机字符的密码。而6个单词则约有78位熵值，远超大多数安全要求。密码短语的优势在于高熵值加易记忆。

什么是 Diceware 词表？

Diceware 是一种使用骰子从预定义词表中随机选择单词来生成密码短语的方法。本工具使用加密安全的随机数生成器替代骰子，但原理相同，确保每个单词的选择都是真正随机的。

数据会上传到服务器吗？

不会。本工具纯前端运行，所有密码生成操作都在浏览器本地完成，使用 Web Crypto API 生成随机数，数据绝不上传服务器，保障隐私安全。

推荐使用几个单词？

一般用途建议4-5个单词，高安全需求（加密货币钱包、主密码）建议6个以上。单词越多，密码强度指数级增长，每增加一个单词约增加13位熵值。

分隔符对安全性有影响吗？

分隔符本身对安全性影响很小，但使用特殊字符作为分隔符（如 - . _ !）可以增加少量熵值，同时满足某些网站的密码复杂度要求。建议选择你能轻松输入的分隔符。
