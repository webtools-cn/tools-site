什么是JSONPath？
JSONPath是用于查询JSON数据的表达式语言，类似于XPath对XML的作用。它使用点号和方括号语法来定位JSON中的特定节点，如$.store.book[0].title。
JSONPath常用语法有哪些？
$表示根节点，.表示子节点，[]表示数组索引或条件，*表示通配符，..表示递归下降，[?()]表示过滤表达式。
如何提取JSON中的特定数据？
粘贴JSON数据，输入JSONPath表达式，工具会实时显示匹配结果。也可以点击JSON树中的节点自动生成路径。
数据会上传到服务器吗？
不会。所有查询都在浏览器本地完成，不会上传任何数据。

JSON路径查找器

JSONPath是查询JSON数据的标准表达式语言，类似于XPath对XML的作用。本工具让你输入JSONPath表达式，实时查看匹配结果，快速定位和提取JSON中的特定数据。所有查询在浏览器本地完成，数据不会上传。

JSONPath语法速查

  • $ — 根节点
  • . 或 [] — 子节点
  • [] — 数组索引（从0开始）
  • * — 通配符，匹配所有元素
  • .. — 递归下降
  • [start:end] — 数组切片
  • [?()] — 过滤表达式
  • [(@.length-1)] — 数组最后一个元素
