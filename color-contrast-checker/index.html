使用教程

颜色对比度检查器是一个帮助设计师和开发者确保网页可访问性的实用工具。以下是详细的使用指南：

选择颜色：在工具顶部，你可以看到两个颜色选择区域——「前景色（文字）」和「背景色」。点击颜色选择器可以直接从调色板中选取颜色，或在旁边的输入框中输入颜色代码（支持HEX、RGB、HSL格式）。选择颜色后，工具会实时计算对比度比率并更新WCAG合规性检测结果。

解读对比度比率：对比度比率越大，表示两种颜色之间的明暗差异越大，可读性越好。例如黑色文字配白色背景对比度为21:1，而浅灰色文字配白色背景可能只有2:1。WCAG标准规定：AA级小号文本需要≥4.5:1，AAA级小号文本需要≥7:1。工具会在结果区域用✅/❌直观显示是否达标。

使用预设方案：如果你不确定选择什么颜色，可以点击下方的「预设配色方案」区域，这里提供了多种经过验证的安全配色组合。点击任意方案即可自动应用到工具中。你也可以使用「随机颜色」按钮快速生成配色灵感，或使用「交换颜色」和「反转颜色」功能快速测试不同组合。

应用场景

颜色对比度检查器在网页设计、UI设计和无障碍开发中有广泛应用：

网页无障碍检测：在开发网站时，确保文字与背景之间有足够的对比度是无障碍设计的基本要求。使用本工具可以实时检测你的设计是否符合WCAG AA或AAA标准，帮助色弱、色盲和老年用户也能清晰阅读内容。许多国家的法律法规（如美国ADA、欧盟EN 301 549）都要求网站满足WCAG标准。

UI设计审核：在设计系统或组件库时，需要确保所有文本在不同背景色上的可读性。设计师可以使用本工具批量检查按钮、卡片、弹窗等组件的颜色组合，提前发现对比度不足的问题，避免设计返工。结合设计工具（如Figma、Sketch）的插件使用效果更佳。

品牌配色优化：企业在制定品牌色时，往往需要确保主色调与各种背景色的对比度达标。使用本工具可以验证品牌色在不同场景下的可用性，确保从宣传物料到网站设计的视觉一致性。例如，验证LOGO在深色和浅色背景上的显示效果，或检查产品界面中的信息层级。

扩展知识

对比度计算公式：WCAG对比度比率基于颜色的相对亮度（Relative Luminance）计算。首先将sRGB值（范围0-255）归一化，然后计算每个通道的线性值，最后组合得到相对亮度L。公式为：L = 0.2126 * R + 0.7152 * G + 0.0722 * B。对比度比率 = (L1 + 0.05) / (L2 + 0.05)，其中L1为较亮的颜色，L2为较暗的颜色。

色觉缺陷模拟：约8%的男性和0.5%的女性存在某种形式的色觉缺陷（色盲）。最常见的类型是红绿色盲（Deuteranopia和Protanopia），其次是蓝黄色盲（Tritanopia）。高对比度不仅帮助色觉缺陷用户，还能在强光环境、低亮度屏幕或投影演示等场景下提升可读性。

相关标准与法规：除了WCAG，还有许多与颜色对比度相关的标准和指南。例如ISO 9241-307（办公设备的图像质量标准）、Section 508（美国联邦政府IT无障碍标准）以及WAI-ARIA规范。了解这些标准有助于构建真正包容的Web体验。

WCAG AA 和 AAA 标准有什么区别？

WCAG AA是最低推荐标准，要求小号文本对比度至少4.5:1，大号文本至少3:1。WCAG AAA是更高标准，要求小号文本对比度至少7:1，大号文本至少4.5:1。大部分网站只需要满足AA标准即可。

对比度比率是怎么计算的？

对比度比率基于颜色的相对亮度计算。首先将颜色的RGB值转换为sRGB，再计算相对亮度（L），最后用公式 (L1 + 0.05) / (L2 + 0.05) 计算对比度比率。比值越大，对比度越高。

什么是对比度？

对比度是指两种颜色在视觉上的明暗差异程度。高对比度意味着颜色之间的差异明显，更容易区分。在网页设计中，足够的对比度能确保色弱用户和老年人也能清晰阅读内容。

支持哪些颜色格式？

本工具支持HEX（如#FF5733）、RGB（如rgb(255,87,51)）和HSL（如hsl(11,100%,60%)）三种格式。可以直接输入颜色代码，或使用颜色选择器快速选择。

对比度多少算合格？

根据WCAG标准，小号文本（18px以下或14px以下加粗）需要至少4.5:1的对比度（AA级）或7:1（AAA级）。大号文本（18px以上或14px以上加粗）需要至少3:1（AA级）或4.5:1（AAA级）。

这个工具需要联网吗？

不需要。本工具纯前端实现，所有计算都在浏览器本地完成，数据不会上传到服务器。你可以放心使用，无需担心颜色数据泄露。

如何判断我的网站是否无障碍？

除了使用本工具检查颜色对比度，还可以使用浏览器的开发者工具（Chrome DevTools的Contrast面板）、Lighthouse无障碍审计、WAVE工具等进行全面检测。建议结合自动化工具和人工审核。
