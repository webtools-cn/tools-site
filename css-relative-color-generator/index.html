📖 什么是CSS相对颜色语法？

CSS相对颜色语法（Relative Color Syntax）是CSS Color Level 4的新特性，允许你基于一个现有颜色的通道值来创建新颜色。

语法格式

color-space(from <color> channel1 channel2 channel3 / alpha)

示例

oklch(from #3b82f6 calc(l + 0.2) c h) - 基于蓝色创建更亮的变体

hsl(from #3b82f6 h calc(s * 0.5) l) - 降低饱和度

rgb(from #3b82f6 r g b / 0.5) - 半透明版本

浏览器支持

✅ Chrome 119+ | ✅ Edge 119+ | ✅ Safari 16.4+ | 🔄 Firefox (开发中)
