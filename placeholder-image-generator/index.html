使用教程

选择尺寸：您可以从上方的预设尺寸按钮中快速选择常用规格，包括广告位标准尺寸（如300×250、728×90等）、屏幕分辨率（如1920×1080、1280×720）以及通用测试尺寸。点击任意预设按钮，宽度和高度输入框会自动填充对应数值。您也可以直接在宽度和高度输入框中输入任意自定义数值（范围1-4096像素）。

应用场景

占位图在Web开发和UI设计中有广泛的应用场景，以下是几个典型用例：

前端开发测试：在开发商品列表、相册、文章封面等页面时，后端API可能尚未返回真实图片数据。使用占位图可以确保前端布局和样式在开发阶段就保持完整，避免图片缺失导致的布局错乱。开发者可以根据设计稿中图片的实际尺寸生成对应比例的占位图。

UI/UX设计原型：设计师在制作高保真原型时，经常需要填充图片区域以展示完整的视觉效果。占位图可以帮助设计师快速填充内容区域，让客户或团队成员更直观地理解最终产品的视觉呈现，而无需等待真实素材到位。

广告位占位：在广告系统开发中，不同广告位有严格的尺寸要求（如Google AdSense的728×90横幅、300×250矩形等）。使用对应尺寸的占位图可以精确模拟广告加载效果，帮助开发者调试响应式布局和广告容器的自适应逻辑。

扩展知识

什么是Placeholder Image？Placeholder（占位符）图片是在正式内容尚未就绪时使用的临时图像。它的核心作用是维持页面布局结构稳定，确保开发和设计工作可以并行推进，不受素材准备进度的阻塞。专业的占位图应该具有明确的尺寸标识，方便团队成员快速识别其规格。

常见广告位尺寸标准：IAB（Interactive Advertising Bureau）定义了全球通用的广告位尺寸标准。最常用包括：300×250（中矩形，最常用）、728×90（页首横幅）、160×600（宽幅摩天楼）、320×50（移动横幅）、970×90（超大页首横幅）。了解这些标准尺寸有助于生成符合行业规范的占位图。

Canvas API 与图片生成：本工具基于HTML5 Canvas API在浏览器本地渲染图片。Canvas提供了像素级的绘图控制能力，包括填充颜色、绘制文字、绘制圆角矩形等。通过canvas.toDataURL('image/png')方法可以将画布内容导出为PNG格式的Data URL，再触发下载即可实现无后端服务的纯前端图片生成。

常见问题 (FAQ)

占位图生成器支持哪些格式？

目前支持PNG格式下载。PNG支持透明通道和高质量无损压缩，非常适合网页开发和设计原型使用。后续版本可能增加JPG和WebP格式支持。

生成的占位图可以商用吗？

可以。本工具生成的占位图为程序自动生成的纯色背景加文字，不包含任何受版权保护的素材，您可以自由用于商业项目、个人项目或任何场景。

最大支持多大的尺寸？

由于浏览器Canvas API的限制，建议生成尺寸不超过4096×4096像素。超过此尺寸可能导致画布无法创建或浏览器性能下降。大多数开发场景（如广告位、商品图、头像等）远低于此限制。

为什么我的文字显示不完整？

当图片尺寸较小时，默认字体大小可能超出画布显示范围。请调小字体大小，或减少文字内容。工具会自动计算文字是否在画布内，超出时会给出提示。

数据会上传到服务器吗？

不会。本工具完全在浏览器前端运行，使用HTML5 Canvas API在本地生成图片。所有输入的尺寸、颜色、文字等数据均不会离开您的设备，无需担心隐私问题。

可以批量生成多种尺寸的占位图吗？

目前需要逐一生成和下载。您可以依次点击不同预设尺寸按钮，分别下载需要的规格。后续版本可能会增加批量生成和打包下载功能。

生成的图片支持透明背景吗？

目前不支持透明背景。所有生成的占位图均为纯色背景。如果您需要透明背景的占位图，可以使用图像编辑软件（如Photoshop、GIMP）将下载的PNG背景删除，或者使用其他专门的透明图片生成工具。

中文字体能正常显示吗？

支持中文显示。Canvas会优先使用系统默认中文字体（如"PingFang SC"、"Microsoft YaHei"等）。如果设备上没有中文字体，中文可能会以默认字体回退显示，但仍可识别。
