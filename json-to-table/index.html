📖 使用说明

支持的JSON格式

1. 对象数组：[{...}, {...}] — 最常用格式，每个对象为一行
2. 单个对象：{...} — 自动包装为单行表格
3. 嵌套对象：{"user": {"name": "张三"}} — 自动展开为user.name列
4. 数组值：数组字段以逗号分隔的字符串显示

嵌套对象处理

当JSON中包含嵌套对象时，工具会自动将其展开。例如 {"address": {"city": "北京", "zip": "100000"}} 会展开为 address.city 和 address.zip 两列，让数据一目了然。
