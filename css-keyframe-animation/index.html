常见问题

什么是CSS关键帧动画？

CSS关键帧动画（@keyframes）是一种强大的CSS特性，允许您通过定义多个关键帧（不同时间点的样式状态）来创建复杂的动画效果。与过渡（transition）不同，关键帧动画可以循环播放、精确控制每个阶段的样式。本工具帮助您可视化地创建这类动画。

如何创建关键帧动画？

本工具提供直观的界面：1）选择动画类型（淡入、滑动、缩放、旋转、弹跳等）；2）调整持续时间和方向；3）设置动画曲线（缓动函数）。生成后可直接复制CSS代码使用。

可以自定义动画的定时和方向吗？

可以。您可以选择动画的持续时间（0.1-10秒）、延迟、播放次数（无限循环/指定次数）、方向（正向/反向/交替）和缓动函数（ease、ease-in、ease-out、linear、cubic-bezier等）。

生成的CSS代码兼容哪些浏览器？

生成的CSS代码使用标准@keyframes语法，兼容所有现代浏览器：Chrome 43+、Firefox 16+、Safari 9+、Edge 12+、Opera 30+。对于需要-webkit-前缀的旧版浏览器，代码已包含相关前缀。
