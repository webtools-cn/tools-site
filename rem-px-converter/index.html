常见问题

px 和 rem 有什么区别？

px（像素）是绝对单位，1px 等于屏幕上的一个物理像素点。rem（root em）是相对单位，相对于根元素（html）的字体大小。浏览器默认字体大小为 16px，因此 1rem = 16px。使用 rem 可以让页面根据用户浏览器设置自适应。

em 和 rem 有什么不同？

em 相对于当前元素的字体大小，而 rem 始终相对于根元素（html）的字体大小。使用 em 会导致嵌套元素的字号叠加效果（复合效应），而 rem 则始终保持一致。通常推荐使用 rem 进行全局布局，使用 em 控制特定组件的相对尺寸。

为什么基准字体大小是 16px？

16px 是所有主流浏览器的默认字体大小。用户可以在浏览器设置中调整默认字体大小，使用 rem 单位会自动响应用户的偏好设置，提供更好的可访问性。如果项目使用了 CSS reset 或设置了 html { font-size: 62.5% }（即 10px），则基准会变为 10px。

如何将设计稿中的 px 转换为 rem？

以 16px 为基准，转换公式为：rem = px / 16。例如设计稿中 32px → 32/16 = 2rem。如果设计稿宽度为 375px，使用 375/16 = 23.4375rem。为了简化计算，也可以设置 html { font-size: 62.5% }（10px），这样 1rem = 10px，计算更直观。

pt（磅）是什么单位？

pt（point，磅）是印刷行业常用的单位，1pt = 1/72 英寸。在 CSS 中，1pt ≈ 1.333px。pt 通常用于打印样式表中，不太推荐在屏幕设计中使用，因为不同屏幕的 dpi 不同，pt 的显示效果可能会有差异。
