📖 使用说明

文本转YAML工具支持将JSON或key=value格式的文本转换为YAML格式，自动处理缩进、引号和特殊字符。

使用方法

  1. 在输入框中粘贴JSON或key=value文本
  2. 点击"转换"按钮
  3. 工具会自动识别输入格式并转换为YAML
  4. 点击"复制"获取YAML文本

转换规则

  • 嵌套对象 — 使用缩进表示层级
  • 数组 — 使用 - 前缀表示列表项
  • 特殊值 — true/false/null/数字自动识别
  • 引号 — 需要时自动添加双引号
