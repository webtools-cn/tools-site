📖 使用说明

文本转YAML工具支持将JSON或key=value格式的文本转换为YAML格式，自动处理缩进、引号和特殊字符。

使用方法

  1. 在输入框中粘贴JSON或key=value文本
  2. 点击"转换"按钮
  3. 工具会自动识别输入格式并转换为YAML
  4. 点击"复制"获取YAML文本

转换规则

  • 嵌套对象 — 使用缩进表示层级
  • 数组 — 使用 - 前缀表示列表项
  • 特殊值 — true/false/null/数字自动识别
  • 引号 — 需要时自动添加双引号

❓ 常见问题

文本转YAML工具是免费的吗？

是的，完全免费使用，无需注册。

支持哪些输入格式？

支持JSON格式和key=value（Properties）格式。工具会自动识别输入格式并转换。

使用时数据会上传到服务器吗？

不会。所有转换都在浏览器中完成。

YAML中的特殊字符如何处理？

工具会自动判断是否需要引号。包含特殊字符（如冒号、井号）的值会被自动加上双引号。

支持数组吗？

支持。JSON数组会被转换为YAML的列表格式（使用-前缀）。
