рҹ“„ Avro Schema

JSONиҪ¬Avro Schemaе·Ҙе…·иғҪеҒҡд»Җд№Ҳпјҹ

JSONиҪ¬Avro Schemaе·Ҙе…·жҳҜдёҖж¬ҫе…Қиҙ№еңЁзәҝејҖеҸ‘иҖ…е·Ҙе…·пјҢиғҪеӨҹе°ҶJSONж•°жҚ®дёҖй”®иҪ¬жҚўдёәApache Avro Schemaе®ҡд№үгҖӮиҮӘеҠЁиҜҶеҲ«JSONеӯ—ж®өзұ»еһӢе№¶жҳ е°„дёәAvroзұ»еһӢпјҢж”ҜжҢҒеөҢеҘ—еҜ№иұЎз”ҹжҲҗrecordзұ»еһӢгҖҒж•°з»„з”ҹжҲҗarrayзұ»еһӢгҖҒnullеҖјз”ҹжҲҗunionзұ»еһӢзӯүеҠҹиғҪгҖӮйҖӮз”ЁдәҺеӨ§ж•°жҚ®ејҖеҸ‘иҖ…еҝ«йҖҹд»ҺJSONзӨәдҫӢз”ҹжҲҗAvro SchemaпјҢз”ЁдәҺKafkaгҖҒHadoopзӯүзі»з»ҹзҡ„ж•°жҚ®еәҸеҲ—еҢ–гҖӮжүҖжңүиҪ¬жҚўеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј жңҚеҠЎеҷЁгҖӮ

ж ёеҝғеҠҹиғҪ

  • жҷәиғҪзұ»еһӢжҳ е°„пјҡиҮӘеҠЁе°ҶstringвҶ’stringгҖҒnumberвҶ’long/doubleгҖҒbooleanвҶ’booleanгҖҒarrayвҶ’arrayгҖҒnullвҶ’unionзұ»еһӢжҳ е°„
  • еөҢеҘ—и®°еҪ•пјҡJSONеөҢеҘ—еҜ№иұЎиҮӘеҠЁз”ҹжҲҗAvro recordзұ»еһӢпјҢж”ҜжҢҒеӨҡеұӮеөҢеҘ—
  • Unionзұ»еһӢпјҡnullеҖјиҮӘеҠЁз”ҹжҲҗ[\"null\",\"е®һйҷ…зұ»еһӢ\"]зҡ„unionзұ»еһӢпјҢзЎ®дҝқSchemaе…је®№еҸҜйҖүеӯ—ж®ө
  • е‘ҪеҗҚз©әй—ҙж”ҜжҢҒпјҡеҸҜиҮӘе®ҡд№үAvro Schemaзҡ„namespaceпјҢж»Ўи¶ідёҚеҗҢйЎ№зӣ®зҡ„еҢ…е‘ҪеҗҚи§„иҢғ
  • еҺҶеҸІи®°еҪ•пјҡиҮӘеҠЁдҝқеӯҳжңҖиҝ‘5ж¬ЎиҪ¬жҚўи®°еҪ•

дҪҝз”Ёж•ҷзЁӢ

1. еңЁJSONиҫ“е…ҘжЎҶдёӯзІҳиҙҙж•°жҚ®пјҢжҲ–зӮ№еҮ»"еҠ иҪҪзӨәдҫӢ"еҝ«йҖҹдҪ“йӘҢгҖӮ

2. и®ҫзҪ®и®°еҪ•еҗҚз§°гҖҒе‘ҪеҗҚз©әй—ҙе’ҢиҪ¬жҚўйҖүйЎ№гҖӮ

3. зӮ№еҮ»"иҪ¬жҚў"жҲ–дҪҝз”ЁCtrl+Enterжү§иЎҢгҖӮ

4. еңЁдёӢж–№жҹҘзңӢз”ҹжҲҗзҡ„Avro SchemaгҖӮ

5. зӮ№еҮ»"еӨҚеҲ¶"жҲ–дёӢиҪҪдёә.avscж–Үд»¶гҖӮ

еә”з”ЁеңәжҷҜ

еңәжҷҜ1пјҡKafkaж•°жҚ®з®ЎйҒ“

еңЁKafkaж¶ҲжҒҜдј йҖ’дёӯдҪҝз”ЁAvro SchemaиҝӣиЎҢж•°жҚ®еәҸеҲ—еҢ–пјҢз”ЁжӯӨе·Ҙе…·еҝ«йҖҹд»ҺJSONзӨәдҫӢз”ҹжҲҗSchemaе®ҡд№үгҖӮ

еңәжҷҜ2пјҡHadoopж•°жҚ®еӨ„зҗҶ

е°ҶJSONж јејҸзҡ„дёҡеҠЎж•°жҚ®иҪ¬жҚўдёәAvro SchemaпјҢз”ЁдәҺHadoop MapReduceжҲ–Sparkж•°жҚ®еӨ„зҗҶгҖӮ

еңәжҷҜ3пјҡеҫ®жңҚеҠЎжҺҘеҸЈе®ҡд№ү

еҫ®жңҚеҠЎй—ҙдҪҝз”ЁAvroдҪңдёәж•°жҚ®дәӨжҚўж јејҸпјҢд»ҺJSON APIе“Қеә”еҝ«йҖҹз”ҹжҲҗAvro SchemaгҖӮ

жү©еұ•зҹҘиҜҶ

Apache AvroжҳҜдёҖз§Қж•°жҚ®еәҸеҲ—еҢ–зі»з»ҹпјҢжҸҗдҫӣзҙ§еҮ‘зҡ„дәҢиҝӣеҲ¶ж јејҸе’ҢеҹәдәҺJSONзҡ„Schemaе®ҡд№үиҜӯиЁҖгҖӮAvro Schemaж”ҜжҢҒеҹәжң¬зұ»еһӢпјҲnullгҖҒbooleanгҖҒintгҖҒlongгҖҒfloatгҖҒdoubleгҖҒbytesгҖҒstringпјүе’ҢеӨҚеҗҲзұ»еһӢпјҲrecordгҖҒenumгҖҒarrayгҖҒmapгҖҒunionгҖҒfixedпјүгҖӮдёҺProtobufзӣёжҜ”пјҢAvroдёҚйңҖиҰҒд»Јз Ғз”ҹжҲҗжӯҘйӘӨпјҢSchemaеҸҜд»ҘеҠЁжҖҒжј”еҢ–гҖӮAvroе№ҝжіӣз”ЁдәҺApache Kafkaзҡ„Schema RegistryгҖҒHadoopж•°жҚ®еӯҳеӮЁе’ҢApache Beamж•°жҚ®еӨ„зҗҶз®ЎйҒ“дёӯгҖӮ

вқ“ еёёи§Ғй—®йўҳ

д»Җд№ҲжҳҜApache Avroпјҹ

Apache AvroжҳҜдёҖз§Қж•°жҚ®еәҸеҲ—еҢ–зі»з»ҹпјҢж”ҜжҢҒдё°еҜҢзҡ„ж•°жҚ®з»“жһ„пјҢжҸҗдҫӣзҙ§еҮ‘зҡ„дәҢиҝӣеҲ¶ж јејҸе’ҢJSONж јејҸзҡ„Schemaе®ҡд№үгҖӮеёёз”ЁдәҺеӨ§ж•°жҚ®еӨ„зҗҶдёӯзҡ„ж•°жҚ®дәӨжҚўгҖӮ

ж”ҜжҢҒеөҢеҘ—JSONеҗ—пјҹ

ж”ҜжҢҒгҖӮеөҢеҘ—еҜ№иұЎиҮӘеҠЁиҪ¬жҚўдёәAvro recordзұ»еһӢпјҢж•°з»„иҪ¬жҚўдёәarrayзұ»еһӢгҖӮ

еҰӮдҪ•еӨ„зҗҶnullеҖјпјҹ

nullеҖјз”ҹжҲҗAvro unionзұ»еһӢ[\"null\",\"е®һйҷ…зұ»еһӢ\"]пјҢзЎ®дҝқSchemaе…је®№еҸҜйҖүеӯ—ж®өгҖӮ

ж•°жҚ®дјҡдёҠдј еҲ°жңҚеҠЎеҷЁеҗ—пјҹ

дёҚдјҡгҖӮжүҖжңүиҪ¬жҚўеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚзҰ»ејҖжӮЁзҡ„и®ҫеӨҮгҖӮ

ж”ҜжҢҒж–Үд»¶еҜје…Ҙеҗ—пјҹ

ж”ҜжҢҒгҖӮеҸҜд»ҘжӢ–жӢҪJSONж–Үд»¶еҲ°иҫ“е…ҘеҢәеҹҹпјҢжҲ–зӮ№еҮ»еҠ иҪҪзӨәдҫӢгҖӮ

еҸҜд»ҘиҮӘе®ҡд№үе‘ҪеҗҚз©әй—ҙеҗ—пјҹ

еҸҜд»ҘгҖӮйҖҡиҝҮйҖүйЎ№и®ҫзҪ®Avro Schemaзҡ„namespaceеӯ—ж®өгҖӮ
