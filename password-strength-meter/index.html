📖 使用教程

本密码强度检测器可以实时分析您密码的安全性，以下是详细的使用指南：

1. 输入密码

在输入框中输入或粘贴要测试的密码。工具会实时分析密码强度，无需点击任何按钮。

2. 查看强度评分

强度条和文字会实时显示密码的安全等级：非常弱（红色）、弱（橙色）、一般（黄色）、强（绿色）、非常强（青色）。

3. 分析详情

工具会显示密码的献值（熵）、长度、字符类型数量，以及在线和离线暴力破解的估算时间。

4. 改进建议

根据分析结果，工具会提供具体的改进建议，如增加长度、添加特殊字符等，帮助您创建更安全的密码。

❓ 常见问题

什么样的密码才算强密码？

强密码通常包含：12位以上长度、同时包含大写字母和小写字母、至少一个数字、至少一个特殊符号（如 !@#$%^&*）。不应包含常见单词、日期、姓名或键盘序列（如 qwerty）。使用密码管理器生成的随机密码最为安全。

密码献值是什么意思？

密码献值是衡量密码随机性和不可预测性的指标，单位是比特（bit）。献值越高，密码越安全。通常献值≥50 bit的密码可以抵抗在线攻击，≥80 bit可以抵抗离线暴力破解。每增加一位字符大约增加6-7 bit献值。

这个工具会上传我的密码吗？

绝对不会。所有密码分析完全在您的浏览器本地进行，密码不会被发送到任何服务器。您甚至可以断开网络后离线使用本工具。这是纯前端应用，零后端请求。

密码被破解需要的时间是怎么计算的？

破解时间基于密码献值和当前硬件算力估算。在线攻击按 1000 次/秒计算（速率限制），离线攻击按 10亿 次/秒计算（GPU暴力破解）。实际时间取决于攻击者的算力，估算值仅供参考。
