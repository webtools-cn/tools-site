常见问题

什么是CSS锚点定位？

CSS锚点定位是CSS新特性，允许一个元素相对于另一个元素进行定位，无需JavaScript。常用于tooltip、popover、dropdown等组件。

anchor-name和position-anchor有什么区别？

anchor-name定义在锚点元素上，给锚点起名字；position-anchor定义在定位元素上，引用锚点名字建立定位关系。

浏览器兼容性如何？

Chrome 125+和Edge 125+支持。Firefox和Safari尚在开发中。建议使用@supports做渐进增强。
