💡 关于 GraphQL 格式化

GraphQL 查询格式化工具帮助您美化（pretty-print）或压缩 GraphQL 查询语句。正确缩进的 GraphQL 代码更易于阅读、调试和维护。

格式化规则

✅ 2空格缩进标准格式

✅ 选择（selection set）自动换行缩进

✅ 参数（arguments）智能排列

✅ 指令（directive）格式化

✅ 片段定义（fragment）分段展示

使用场景

• API 调试时格式化从浏览器/工具中复制的查询

❓ 常见问题

支持复杂的 GraphQL 语法吗？

支持标准 GraphQL 语法：query、mutation、subscription、fragment、inline fragments、variables definition、directives、interfaces、unions 等。

格式化后语法变化吗？

格式化仅改变代码的排版（缩进和换行），不会改变代码的语法结构。格式化后的查询可以直接运行。

数据会上传服务器吗？

不会。所有处理在您的浏览器本地完成，数据不会上传到任何服务器，确保您的查询内容安全。

如何使用在线GraphQL格式化工具

使用在线GraphQL格式化工具非常简单，只需三个步骤：

  1. 输入代码或数据
    在输入区域输入或粘贴要格式化的内容。支持直接文本输入、文件上传和拖放。所有处理均在浏览器本地完成，数据不会上传到任何服务器。
  2. 配置格式化选项
    选择首选的格式化选项，如缩进大小、排序偏好或压缩模式。点击"格式化"、"美化"或"压缩"按钮，内容即刻处理完成。
  3. 复制或下载结果
    格式化结果会立即显示在输出区域。点击"复制"按钮复制到剪贴板，或点击"下载"保存为文件。原始内容不会上传到任何服务器。
