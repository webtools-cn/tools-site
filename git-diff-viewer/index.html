❓ 常见问题

Git Diff查看器是免费的吗？

是的，完全免费，无需注册，无使用次数限制。所有处理在浏览器本地完成。

支持哪些Diff格式？

支持标准Git diff格式（unified diff直接粘贴）和两段文本对比模式。自动识别输入格式。

统一视图和并排视图有什么区别？

统一视图将增删行按顺序显示在一个面板中，添加行绿色、删除行红色。并排视图将原始文本和修改文本左右对照显示，方便逐行对比。

可以对比两段不同的文本吗？

可以。切换到'文本对比'模式，分别在左右输入框输入原始文本和修改文本，点击对比即可查看差异。

数据安全吗？

绝对安全。所有处理都在浏览器本地完成，代码和文本数据不会上传到任何服务器。
