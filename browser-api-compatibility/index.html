рҹ“– жөҸи§ҲеҷЁе…је®№жҖ§иҜҰи§Ј

дёәд»Җд№ҲйңҖиҰҒе…іжіЁжөҸи§ҲеҷЁе…је®№жҖ§

Web APIеңЁдёҚеҗҢжөҸи§ҲеҷЁдёӯзҡ„ж”ҜжҢҒзЁӢеәҰдёҚеҗҢгҖӮдҪҝз”ЁдёҚе…је®№зҡ„APIдјҡеҜјиҮҙеҠҹиғҪеңЁйғЁеҲҶз”ЁжҲ·жөҸи§ҲеҷЁдёӯж— жі•жӯЈеёёе·ҘдҪңгҖӮдәҶи§Је…је®№жҖ§ж•°жҚ®еҸҜд»Ҙеё®еҠ©дҪ еҒҡеҮәжҠҖжңҜйҖүеһӢеҶізӯ–пјҢйҖүжӢ©еҗҲйҖӮзҡ„polyfillзӯ–з•ҘгҖӮ

е…је®№жҖ§зӯүзә§иҜҙжҳҺ

  • вң… ж”ҜжҢҒ вҖ” иҜҘжөҸи§ҲеҷЁзүҲжң¬е®Ңе…Ёж”ҜжҢҒжӯӨAPI
  • вҡ пёҸ йғЁеҲҶж”ҜжҢҒ вҖ” ж”ҜжҢҒдҪҶеҸҜиғҪйңҖиҰҒеүҚзјҖжҲ–еӯҳеңЁйҷҗеҲ¶
  • вқҢ дёҚж”ҜжҢҒ вҖ” иҜҘжөҸи§ҲеҷЁзүҲжң¬дёҚж”ҜжҢҒжӯӨAPI

жңҖдҪіе®һи·ө

  1. дҪҝз”Ёзү№жҖ§жЈҖжөӢиҖҢйқһжөҸи§ҲеҷЁжЈҖжөӢ
  2. дёәе…ій”®еҠҹиғҪжҸҗдҫӣеӣһйҖҖж–№жЎҲ
  3. дҪҝз”Ё@supportsжЈҖжөӢCSSзү№жҖ§
  4. е…іжіЁMDNе’Ңcaniuse.comзҡ„жңҖж–°ж•°жҚ®
  5. дҪҝз”ЁAutoprefixerиҮӘеҠЁж·»еҠ жөҸи§ҲеҷЁеүҚзјҖ

рҹ“– еёёи§Ғй—®йўҳ

еҰӮдҪ•еҲӨж–ӯдёҖдёӘWeb APIжҳҜеҗҰеҸҜд»Ҙе®үе…ЁдҪҝз”Ё?

еҲӨж–ӯж ҮеҮҶпјҡ1)жҹҘзңӢеҗ„жөҸи§ҲеҷЁж”ҜжҢҒзүҲжң¬вҖ”вҖ”йҖҡеёёйңҖиҰҒдё»жөҒжөҸи§ҲеҷЁжңҖиҝ‘2дёӘзүҲжң¬йғҪж”ҜжҢҒпјӣ2)иҖғиҷ‘зӣ®ж Үз”ЁжҲ·зҫӨдҪ“вҖ”вҖ”еҰӮжһңз”ЁжҲ·дё»иҰҒжҳҜжҠҖжңҜз”ЁжҲ·пјҢеҸҜдҪҝз”Ёиҫғж–°APIпјӣ3)дҪҝз”Ё@supportsжҲ–зү№жҖ§жЈҖжөӢвҖ”вҖ”еҜ№дёҚж”ҜжҢҒзҡ„жөҸи§ҲеҷЁжҸҗдҫӣеӣһйҖҖж–№жЎҲпјӣ4)еҸӮиҖғMDNе’Ңcaniuse.comзҡ„е…је®№жҖ§ж•°жҚ®гҖӮдёҖиҲ¬и§„еҲҷпјҡе…Ёзҗғз”ЁжҲ·дә§е“ҒйңҖ95%+ж”ҜжҢҒзҺҮгҖӮ

д»Җд№ҲжҳҜжёҗиҝӣеўһејәе’Ңдјҳйӣ…йҷҚзә§?

жёҗиҝӣеўһејә(Progressive Enhancement)пјҡе…ҲзЎ®дҝқеҹәжң¬еҠҹиғҪеңЁжүҖжңүжөҸи§ҲеҷЁеҸҜз”ЁпјҢеҶҚдёәж”ҜжҢҒж–°APIзҡ„жөҸи§ҲеҷЁж·»еҠ еўһејәеҠҹиғҪгҖӮдјҳйӣ…йҷҚзә§(Graceful Degradation)пјҡе…Ҳе®һзҺ°е®Ңж•ҙеҠҹиғҪпјҢеҶҚдёәдёҚж”ҜжҢҒзҡ„жөҸи§ҲеҷЁжҸҗдҫӣз®ҖеҢ–зүҲжң¬гҖӮзҺ°д»ЈеүҚз«ҜжҺЁиҚҗжёҗиҝӣеўһејәзӯ–з•ҘпјҢдҪҝз”Ё@supportsжЈҖжөӢе’ҢpolyfillгҖӮ

еҰӮдҪ•еӨ„зҗҶжөҸи§ҲеҷЁе…је®№жҖ§й—®йўҳ?

еёёз”Ёж–№жі•пјҡ1)зү№жҖ§жЈҖжөӢвҖ”вҖ”дҪҝз”Ёif('serviceWorker' in navigator)жЈҖжөӢAPIжҳҜеҗҰеӯҳеңЁпјӣ2)PolyfillвҖ”вҖ”дёәдёҚж”ҜжҢҒзҡ„жөҸи§ҲеҷЁж·»еҠ APIе®һзҺ°пјӣ3)CSS @supportsвҖ”вҖ”жЈҖжөӢCSSзү№жҖ§ж”ҜжҢҒпјӣ4)Babel/PostCSSвҖ”вҖ”иҮӘеҠЁиҪ¬иҜ‘ж–°иҜӯжі•дёәе…је®№д»Јз Ғпјӣ5)жөҸи§ҲеҷЁеүҚзјҖвҖ”вҖ”-webkit-гҖҒ-moz-зӯүеүҚзјҖзЎ®дҝқе…је®№пјӣ6)жқЎд»¶еҠ иҪҪвҖ”вҖ”жҢүжөҸи§ҲеҷЁиғҪеҠӣеҠ иҪҪдёҚеҗҢд»Јз ҒгҖӮ

е“ӘдәӣWeb APIжңҖеҖјеҫ—еӯҰд№ ?

2026е№ҙжңҖеҖјеҫ—еӯҰд№ зҡ„Web APIпјҡ1)View Transitions APIвҖ”вҖ”йЎөйқўеҲҮжҚўеҠЁз”»пјӣ2)CSS Anchor PositioningвҖ”вҖ”е…ғзҙ е®ҡдҪҚпјӣ3)File System Access APIвҖ”вҖ”ж–Үд»¶ж“ҚдҪңпјӣ4)WebGPUвҖ”вҖ”GPUи®Ўз®—е’ҢжёІжҹ“пјӣ5)Container QueriesвҖ”вҖ”з»„д»¶зә§е“Қеә”ејҸпјӣ6)Popover APIвҖ”вҖ”еј№еҮәеұӮпјӣ7)Dialogе…ғзҙ вҖ”вҖ”жЁЎжҖҒжЎҶпјӣ8)CSS :has()вҖ”вҖ”зҲ¶йҖүжӢ©еҷЁгҖӮиҝҷдәӣAPIе·ІиҺ·дё»жөҒжөҸи§ҲеҷЁж”ҜжҢҒгҖӮ

ж•°жҚ®дјҡдёҠдј еҲ°жңҚеҠЎеҷЁеҗ—?

дёҚдјҡгҖӮжүҖжңүе…је®№жҖ§жҹҘиҜўйғҪеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢдёҚеҸ‘йҖҒд»»дҪ•ж•°жҚ®еҲ°жңҚеҠЎеҷЁгҖӮAPIе…је®№жҖ§ж•°жҚ®дёәеҶ…зҪ®йқҷжҖҒж•°жҚ®пјҢж— йңҖиҒ”зҪ‘гҖӮ
