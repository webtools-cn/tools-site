📖 CSS链接样式详解

CSS链接样式通过四个伪类来定义链接的不同交互状态：:link、:visited、:hover、:active。

LoVe/HAte 规则

正确的声明顺序是 :link → :visited → :hover → :active。助记：LoVe/HAte（爱/恨）。如果顺序不对，样式可能不生效。

text-decoration 详解

text-decoration 属性可以组合使用：text-decoration: underline dotted #ff0000 可同时控制线型、样式和颜色。现代浏览器（Chrome 76+、Firefox 70+、Safari 12.1+）支持彩色下划线。

过渡动画

使用 transition 属性让链接状态切换更平滑。建议 150-300ms 的过渡时长，让颜色和样式变化自然流畅。
