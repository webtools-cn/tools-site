❓ 常见问题

支持嵌套JSON吗？

支持。嵌套对象会自动生成为独立的数据类，每个类都有对应的类型名称。

如何处理null值？

开启"可空类型"后，null字段生成为可空类型（如String?），关闭则生成非空类型带默认值。

支持数组类型吗？

支持。JSON数组转换为Kotlin的List<T>类型，T根据元素类型自动推断。

数据会上传到服务器吗？

不会。所有转换在浏览器本地完成，数据绝不离开您的设备。

支持哪些注解？

支持Gson @SerializedName、Moshi @Json、Kotlinx Serialization @SerialName三种注解。

生成的代码可以直接使用吗？

可以。生成的代码符合Kotlin规范，可直接复制到.kt文件中使用。

JSON转Kotlin数据类工具能做什么？

JSON转Kotlin数据类工具是一款免费在线开发者工具，能够将JSON数据一键转换为Kotlin语言的data class定义代码。自动识别JSON字段类型并映射为Kotlin类型，支持嵌套对象生成独立数据类、Gson/Moshi/Kotlinx Serialization注解、可空类型处理等功能。适用于Android和Kotlin后端开发者快速从API响应生成数据类定义，大幅减少手动编码工作量。所有转换在浏览器本地完成，数据绝不上传服务器。

核心功能

  • 智能类型映射：自动将string→String、number→Int/Double、boolean→Boolean、array→List、null→可空类型
  • 嵌套数据类：JSON中的嵌套对象自动生成为独立的数据类定义，保持类型层次清晰
  • 序列化注解：支持Gson @SerializedName、Moshi @Json、Kotlinx @SerialName三种注解
  • 可空类型：null值字段自动生成为可空类型（String?），确保Kotlin类型安全
  • 实时预览：输入JSON后实时生成Kotlin代码，无需手动点击

使用教程

1. 在JSON输入框中粘贴您的JSON数据，或点击"加载示例"快速体验。

2. 设置类名和转换选项：注解类型、是否使用可空类型、整数类型等。

3. 点击"转换"按钮，或使用快捷键Ctrl+Enter执行转换。

4. 在下方结果区域查看生成的Kotlin数据类代码。

5. 点击"复制代码"将结果复制到剪贴板，或下载为.kt文件。

扩展知识

Kotlin的data class是专门用于持有数据的类，自动生成equals()、hashCode()、toString()、copy()等方法，比Java的POJO更简洁安全。在Android开发中，Gson是最常用的JSON序列化库，通过@SerializedName注解映射JSON字段名到Kotlin属性。Moshi是Square推出的现代JSON库，使用@Json注解，对Kotlin支持更好。Kotlinx Serialization是JetBrains官方方案，编译时生成序列化代码，性能更优，使用@SerialName注解。可空类型（Type?）是Kotlin的重要特性，精确区分null和非null值，避免NPE。
