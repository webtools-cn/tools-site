扩展知识

-webkit-line-clamp是CSS中实现多行文本截断的核心属性，需要配合display:-webkit-box和-webkit-box-orient:vertical以及overflow:hidden使用。虽然带有-webkit前缀，但已被所有主流浏览器支持（Chrome 6+、Firefox 68+、Safari 5+、Edge 79+），全球覆盖率超过97%。CSS Overflow Level 4规范中的line-clamp属性是标准化版本，目前仍在开发中。对于需要精确控制截断位置的场景，可以使用JavaScript计算文本高度来实现。

常见问题

CSS line-clamp如何实现多行文本截断？

使用-webkit-line-clamp属性配合display:-webkit-box和-webkit-box-orient:vertical，再加上overflow:hidden即可实现多行文本截断并显示省略号。这是目前最简洁的纯CSS方案。

line-clamp浏览器兼容性如何？

-webkit-line-clamp已被所有主流浏览器支持，包括Chrome、Firefox、Safari、Edge。虽然带有-webkit前缀，但Firefox从68版本开始也支持了。覆盖率超过97%。

如何实现点击展开全文？

纯CSS方案可使用checkbox hack实现，通过隐藏的checkbox控制展开状态。JS方案更灵活，点击时移除line-clamp属性即可展开，再次点击恢复截断。本工具两种方案都支持。

单行省略和多行省略有什么区别？

单行省略使用text-overflow:ellipsis配合white-space:nowrap和overflow:hidden。多行省略使用-webkit-line-clamp:N指定行数。单行方案兼容性更好，多行方案更灵活。
