使用教程

在线图片滤镜工具是一款功能强大且操作简单的纯前端图片处理工具，无需注册即可使用。以下是详细的使用指南：

上传图片：点击上传区域或直接将图片拖拽到指定区域即可上传。支持 JPG、PNG、GIF、WebP 等常见图片格式。上传后图片会显示在预览区域，你可以开始应用滤镜效果。

预设滤镜：工具提供了8种预设滤镜，包括灰度、复古、模糊、亮度、对比度、饱和度、色相、反转。点击预设按钮即可一键应用对应效果。选择「原图」可以恢复到原始状态。

自定义调节：每种滤镜都配有独立的参数滑块，你可以自由调节滤镜的强度。例如，灰度滤镜可以从0%（完全彩色）调节到100%（完全黑白），让你精确控制效果。

对比原图：点击「对比原图」按钮可以临时查看原始图片，再次点击即可回到滤镜效果。这有助于你比较滤镜前后的差异。

导出图片：完成滤镜调整后，点击「导出图片」按钮即可将处理后的图片下载为PNG格式。导出的图片保留了原始图片的分辨率和质量。

应用场景

图片滤镜工具在多个领域都有广泛的应用场景：

社交媒体：在发布图片到社交媒体之前，使用滤镜可以快速美化图片，增加视觉吸引力。复古滤镜可以让照片更有质感，饱和度调节可以让颜色更加鲜艳。

电商产品图：电商平台的产品图片需要统一风格。使用亮度、对比度调节可以让产品图片更加清晰，灰度滤镜可以制作产品图的黑白版本用于特定设计需求。

设计原型：设计师在制作原型时，经常需要调整图片的色调和风格。色相旋转可以快速改变图片的整体色调，对比度调节可以突出图片的层次感。

文档配图：在文档和演示文稿中，有时需要特定风格的图片。使用模糊滤镜可以制作背景虚化效果，反转滤镜可以制作特殊的视觉效果。

扩展知识

CSS滤镜原理：本工具基于 CSS filter 属性实现图片滤镜效果。CSS filter 是 W3C 标准属性，所有现代浏览器都支持。它使用 GPU 加速渲染，性能优异，即使处理大图片也能保持流畅。

滤镜组合效果：多种滤镜可以叠加使用，产生更丰富的效果。例如，同时应用灰度和对比度滤镜可以制作出高对比度的黑白照片效果。同时应用复古和模糊滤镜可以制作出老照片的朦胧感。

导出质量：导出功能使用 Canvas 的 toDataURL() 方法将处理后的图片转换为 PNG 格式。导出时保留了原始图片的分辨率，但请注意，如果原始图片格式为 JPG，导出后文件大小可能会增加（PNG 是无损压缩）。

常见问题

图片滤镜工具支持哪些滤镜效果？

支持8种滤镜效果：灰度、复古、模糊、亮度、对比度、饱和度、色相、反转。每种滤镜都可以通过滑块调节强度，满足不同的图片处理需求。

滤镜处理后的图片可以导出吗？

可以。点击「导出图片」按钮即可将滤镜处理后的图片下载为PNG格式。导出的图片保留了原始图片的分辨率和质量。

数据会上传到服务器吗？

不会。本工具是纯前端实现，所有处理都在浏览器本地完成，数据不会上传到任何服务器。

支持哪些图片格式？

支持 JPG、PNG、GIF、WebP 等所有浏览器支持的图片格式。上传的图片会直接显示在预览区域，无需转换格式。

多种滤镜可以叠加使用吗？

可以。多种滤镜可以叠加使用，产生更丰富的效果。例如，同时应用灰度和对比度滤镜可以制作出高对比度的黑白照片效果。

导出后的图片质量会下降吗？

不会。导出功能使用 Canvas 的 toDataURL() 方法，保留了原始图片的分辨率。但请注意，如果原始图片格式为 JPG，导出后文件大小可能会增加。

可以批量处理多张图片吗？

目前不支持批量处理。每次只能上传和处理一张图片。如需批量处理，建议多次上传并分别导出。

滤镜效果可以保存为预设吗？

目前不支持保存自定义预设。你可以记录下滑块的参数值，下次使用时手动调节到相同位置。
