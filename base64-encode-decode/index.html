📖 什么是Base64？

Base64是一种基于64个可打印字符（A-Z、a-z、0-9、+、/）来表示二进制数据的编码方式。它常用于在需要处理文本数据的场合中传输或存储二进制数据。

Base64编码原理

  • 将每3个字节（24位）分为4组，每组6位
  • 每组6位映射到一个可打印字符
  • 不足3字节时用 = 填充
  • 编码后数据量增加约33%

常见应用场景

  • Data URL：将图片内嵌到HTML/CSS中，如 data:image/png;base64,iVBOR...
  • HTTP Basic Auth：用户名:密码编码后放入Authorization头
  • JWT令牌：Header和Payload部分使用Base64URL编码
  • 邮件附件：MIME标准中用于编码二进制附件
  • API传输：在JSON等文本格式中传输二进制数据

Base64 vs Base64URL

  • Base64：使用 + 和 /，用 = 填充
  • Base64URL：使用 - 和 _，无填充，适合URL和文件名
