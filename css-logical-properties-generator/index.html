❓ 常见问题

什么是CSS逻辑属性？

CSS逻辑属性是基于书写方向而非物理方向的属性。例如用margin-block-start代替margin-top，使布局自动适应不同书写模式（如从右到左的阿拉伯语）。

为什么要使用逻辑属性？

逻辑属性让CSS自动适应不同的书写方向和语言环境，无需为RTL语言单独编写样式，提高代码的国际化和可维护性。

逻辑属性和物理属性有什么对应关系？

margin-top → margin-block-start, margin-left → margin-inline-start, width → inline-size, height → block-size等。本工具提供完整的转换对照。

所有浏览器都支持逻辑属性吗？

现代浏览器（Chrome 69+、Firefox 66+、Safari 12.1+、Edge 79+）均已支持。对于旧浏览器，本工具可同时生成物理属性作为回退。

如何开始使用逻辑属性？

将现有CSS中的物理属性替换为对应的逻辑属性即可。本工具可自动完成转换。
