CSS调色板提取器介绍

CSS调色板提取器可以从任何CSS代码中自动提取所有颜色值，生成可视化调色板和CSS变量定义。帮助设计师和开发者快速了解项目的颜色体系。

核心功能

  • 多格式支持：HEX、RGB、RGBA、HSL、HSLA
  • 频率排序：按使用频率排列颜色
  • 一键复制：点击色块复制颜色值
  • CSS变量生成：自动生成:root变量定义
  • 对比度检测：自动判断浅色/深色背景

使用场景

设计系统构建、主题提取、代码审计、颜色一致性检查等。

常见问题

能提取哪些颜色格式？

支持提取HEX（#fff, #ffffff, #ffffffff）、RGB/RGBA、HSL/HSLA等所有CSS颜色格式。

颜色按什么排序？

按在CSS中出现的频率排序，出现次数最多的颜色排在前面。

生成的CSS变量有什么用？

CSS变量（Custom Properties）可以统一管理项目中的颜色，方便主题切换和全局修改。

支持CSS预处理器语法吗？

主要支持标准CSS语法。Sass/Less变量中的颜色如果以标准格式出现也能提取。

数据安全吗？

所有处理在浏览器本地完成，CSS代码不会上传到任何服务器。
