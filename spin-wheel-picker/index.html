扩展知识

转盘算法设计：采用"先随机结果，后计算动画"的设计。先使用密码学安全随机数确定最终结果，再计算动画需要经过的角度和圈数，确保结果真正随机。公平性：每个选项的中奖概率与占用转盘面积成正比，均等大小下概率均等。

常见问题 FAQ

转盘结果是真随机吗？

是的。转盘使用 Crypto.getRandomValues() API 确定最终选中结果，旋转动画根据结果动态计算角度和圈数。既保证随机性，又提供流畅的视觉体验。

可以添加多少个选项？

最多200个。超过20个时建议使用较短的选项名称，以保证转盘文字清晰可读。

可以保存配置吗？

当前版本支持浏览器本地保存（localStorage）。可通过截图分享结果。如需分享配置，可手动记录选项列表。

配色可以自定义吗？

可以。提供多种预设配色方案。颜色设置会保存在浏览器本地。

适合什么场景？

课堂教学点名、年会抽奖、朋友聚会决定、运动分组、日常决策、视频直播互动等。
