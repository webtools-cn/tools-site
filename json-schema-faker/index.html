📚 JSON Schema模拟数据生成能做什么？

JSON Schema模拟数据生成工具根据JSON Schema定义自动生成符合规范的模拟数据，适用于API开发、单元测试、前端开发等场景。支持所有基本类型、嵌套结构、枚举和格式约束。

核心功能

• 支持所有JSON Schema类型：string/number/integer/boolean/array/object/null

• 格式约束：email/date/date-time/uri/uuid/ipv4/ipv6等

• 范围约束：minimum/maximum/minLength/maxLength/minItems/maxItems

• 枚举值：enum关键字生成指定值之一

• 批量生成：1-100条模拟数据

• 递归处理嵌套对象和数组

使用教程

1. 输入或粘贴JSON Schema定义

2. 设置生成数量和选项

3. 点击"生成数据"查看结果

4. 复制或下载生成的模拟数据

应用场景

• API开发：生成Mock数据用于接口开发和测试

• 前端开发：生成模拟数据填充UI组件

• 单元测试：生成测试数据集

• 数据库填充：生成测试数据导入数据库

扩展知识

JSON Schema是IETF标准（draft-07/latest），用于描述JSON数据的结构和约束。它定义了类型、格式、范围、模式等验证规则。Mock数据生成是API开发的重要环节，常见的Mock工具还有Faker.js、json-server等。JSON Schema Faker的优势是可以根据已有的Schema定义精确生成符合规范的数据。

❓ 常见问题

什么是JSON Schema？

JSON Schema是描述JSON数据结构的规范，定义了字段类型、格式、约束条件等。通过JSON Schema可以验证数据是否符合规范，也可以根据Schema生成符合规范的模拟数据。

支持哪些数据类型？

支持string、number、integer、boolean、array、object、null等所有JSON Schema基本类型，以及enum枚举、format格式（email/date/uri等）、minLength/maxLength、minimum/maximum等约束。

可以生成多条数据吗？

可以。设置生成数量（1-100条），工具会根据同一个Schema生成多条不同的模拟数据，适合批量生成测试数据集。

生成的数据是随机的吗？

是的。生成的数据是随机模拟数据，但严格遵守Schema定义的类型和约束。字符串会生成随机内容，数字在约束范围内随机。

支持嵌套对象和数组吗？

支持。工具递归处理嵌套的object和array类型，包括items、properties、additionalProperties等Schema关键字，生成完整的嵌套数据结构。

生成的数据可以导出吗？

可以。支持一键复制到剪贴板和下载为JSON文件。批量生成的数据会以JSON数组格式导出。
