❓ 常见问题

Cron表达式是什么？

Cron表达式是Unix/Linux系统中用于配置定时任务的时间规则字符串，由5个字段组成（分 时 日 月 周），广泛用于Linux crontab、Kubernetes CronJob、Spring Boot @Scheduled等场景。

0 0 * * * 和 @daily 有什么区别？

0 0 * * * 等同于 @daily，都表示每天零点执行。@daily等是部分Cron实现的快捷方式，本工具生成标准5字段表达式，兼容性更好。

如何在Kubernetes中使用Cron表达式？

在Kubernetes CronJob的spec.schedule字段中使用本工具生成的表达式，格式相同。K8s也支持6字段格式（含秒），需在前面加0，如0 0 * * *变为0 0 0 * * *。

表达式中的?和*有什么区别？

*表示匹配任意值，?表示不指定值（仅用于日和周字段）。当日和周字段都需要匹配时，其中一个必须用?，因为不能同时指定日和周。
