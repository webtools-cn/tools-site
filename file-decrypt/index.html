📖 常见问题

能解密什么文件？

可解密本站文件加密工具生成的.encrypted文件，兼容AES-256-GCM + PBKDF2(SHA-256,100K迭代)+salt(16B)+IV(12B)格式。

密码错误会怎样？

解密会失败。AES-GCM有认证功能，密码错误时认证标签验证不通过，确保不会产生损坏文件。

解密后是原始格式吗？

是的。会还原为原始文件名和格式，内容与加密前完全一致。

忘记密码还能解密吗？

不能。AES-256没有后门。密码是唯一解密方式，请务必保管好。
