❓ 常见问题

Git速查表包含哪些命令？

包含仓库操作、暂存提交、分支管理、合并变基、远程操作、查看历史、撤销操作、标签管理、储藏操作等9大分类共50+条常用Git命令。每条命令都有中文说明。

如何搜索Git命令？

在搜索框中输入关键词即可实时过滤。支持按命令名搜索（如commit），也支持按中文描述（如"撤销"）或分类关键词搜索。

这些命令可以直接复制使用吗？

可以。每条命令旁边都有复制按钮，点击即可复制到剪贴板。示例中的<占位符>需要替换为实际名称。建议先在测试仓库中验证命令效果。

数据安全吗？

完全安全。本工具是纯前端应用，所有操作都在浏览器本地完成，不会收集或上传任何数据。无需注册即可使用。
