📖 什么是NPV（净现值）

NPV（Net Present Value，净现值）是财务分析中最重要的投资评估指标之一。它将项目未来各期产生的现金流，按照一定的折现率折算为当前价值，再减去初始投资成本，得出项目的净价值。

计算公式

NPV = -C₀ + Σ(Cₜ / (1+r)^t)

其中：C₀ = 初始投资，Cₜ = 第t期现金流，r = 折现率，t = 期数

判断标准

  • NPV > 0：投资项目可行，能创造超额价值
  • NPV = 0：投资恰好保本，回报等于折现率
  • NPV < 0：投资不可行，回报低于折现率

应用场景

  • 企业项目投资决策
  • 房地产投资分析
  • 设备采购评估
  • 创业项目估值
  • 债券定价分析

❓ 常见问题

什么是NPV（净现值）？

NPV（Net Present Value，净现值）是将未来各期现金流按折现率折算为现值后减去初始投资的差额。NPV大于0表示投资可行，NPV小于0表示投资不可行，NPV等于0表示投资恰好保本。

NPV计算公式是什么？

NPV = -C₀ + Σ(Cₜ / (1+r)^t)，其中C₀为初始投资，Cₜ为第t期现金流，r为折现率，t为期数。折现率反映资金的时间价值和风险溢价。

折现率如何选择？

折现率通常选择：1) 企业的加权平均资本成本(WACC)；2) 行业平均回报率；3) 银行贷款利率加风险溢价；4) 投资者要求的最低回报率。一般项目取8%-15%，高风险项目取15%以上。

NPV和IRR有什么区别？

NPV给出投资创造的绝对价值金额，IRR给出投资的百分比回报率。NPV更适合比较不同规模的投资项目，IRR更适合判断单一项目的回报水平。两者通常配合使用。
