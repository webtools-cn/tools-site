About Case Conversion

Case Conversion Guide:

• UPPERCASE — All letters capital, useful for emphasis

• lowercase — All letters lowercase, default text

• Title Case — First letter of each word capitalized

• aLtErNaTiNg — Alternating case letters

• camelCase — First word lowercase, rest capitalized (JavaScript variables)

• snake_case — Words separated by underscores (Python variables)

• kebab-case — Words separated by hyphens (CSS classes, URLs)
