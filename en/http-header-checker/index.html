❓ FAQ

What can this HTTP header checker do?

It shows complete HTTP response headers including Server, Content-Type, Cache-Control, security headers (CSP/HSTS/X-Frame-Options etc.), Cookie info, and redirect chains. Also provides a security configuration score.

Can it follow redirects?

Yes! The tool detects HTTP 301/302/307/308 redirects and shows the final destination URL.

Is my query logged or stored?

Due to browser CORS restrictions, this tool uses public CORS proxies. Queries are anonymous and no data is stored. You can also configure your own proxy address.

Can I check localhost/internal URLs?

No, browser security prevents access to localhost or intranet addresses. But you can check any publicly accessible URL via HTTP or HTTPS.
