📖 How to Use

1. Select Images: Click or drag & drop JPG/PNG/WebP images. Batch selection supported.

2. Adjust Quality: Drag the quality slider (1-100%) to set compression level. Preview in real-time.

3. Download Results: Download individually or use 'Batch Download' for all compressed images.

❓ FAQ

Will compression reduce image quality?

Yes, some quality loss occurs, but you can adjust the quality slider (1-100%) to find the sweet spot. At 80% quality, the difference is nearly invisible while file size can shrink by 60-80%.

What formats and file sizes are supported?

JPG, PNG, and WebP are supported. Keep files under 50MB for best performance. All compression uses Canvas locally — your images never leave the browser.

Can I compress multiple images at once?

Yes! Select multiple images, set quality individually or globally, then download one by one or batch download.

Will the output format change?

By default, output keeps the original format. You can also choose a unified output format. PNG→JPG loses transparency but makes smaller files; JPG→WebP can further reduce size with similar quality.
