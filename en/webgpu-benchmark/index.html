What is WebGPU Benchmark?

WebGPU基准测试是一个免费在线工具，用于测试浏览器的WebGPU API性能。运行计算着色器、渲染管线和Memory Bandwidth测试，生成综合性能评pts。所有测试在浏览器本地GPU上运行，数据不上传服务器。

Key Features

  • Compute Shader Test：Test GPU Compute Shader throughput
  • Render Pipeline Test：Draw call and triangle render throughput test
  • Memory Bandwidth测试：GPU memory data transfer rate test
  • 综合评pts：基于三项测试的加权综合性能评pts
  • Compatibility Detection：Auto-detect browser WebGPU support
  • Device Info：显示GPU Device名称、驱动版本等信息

Use Cases

Use Case 1: Hardware Comparison

Compare WebGPU performance across different devices or browsers.

Use Case 2: Development Debugging

Verify WebGPU functionality and check for performance bottlenecks.

Use Case 3: Browser Compatibility

Detect whether browser supports WebGPU API and to what extent.

❓ 常见问题

What is WebGPU Benchmark?

用于测试浏览器WebGPU API性能的免费在线工具，运行计算、渲染和内存测试生成综合评pts。

What is WebGPU?

WebGPU is the next-gen web graphics and compute API, succeeding WebGL. Supported in Chrome 113+ and other modern browsers.

Does my browser support WebGPU?

工具会自动检测。目前Chrome 113+、Edge 113+和部ptsFirefox版本支持。

What metrics are tested?

Compute Performance、Render Performance和Memory Bandwidth三大指标。

评pts标准是什么？

综合评pts基于三项测试加权平均，满pts10000pts。

Is data uploaded to servers?

No. All tests run locally on your GPU.
