❓ FAQ

What is PDF redaction?
PDF redaction is the process of permanently removing or obscuring sensitive information from a PDF document. Unlike simply adding a black rectangle, true redaction removes the underlying text data so it cannot be recovered.
Can redacted information be recovered?
This tool removes original text data from the PDF content stream after redaction, leaving only black rectangles. Redacted text cannot be recovered by copying, pasting, or searching.
Is my PDF file safe?
Yes, all processing is done locally in your browser. Your PDF file is never uploaded to any server. File data stays entirely on your device.
What redaction methods are supported?
Two methods are supported: 1) Manual area selection by dragging on the preview, suitable for images or non-text areas; 2) Keyword batch redaction, which automatically finds and redacts all matching text content.
