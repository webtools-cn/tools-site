About SWIFT/BIC Code Validator

The SWIFT/BIC Code Validator helps you quickly verify whether a bank SWIFT code's format is correct. The tool checks code length, character composition, country code validity, and more, while parsing the meaning of each code component including bank code, country, location, and branch information.

A SWIFT code (Society for Worldwide Interbank Financial Telecommunication) is a bank identifier standard developed by the SWIFT organization, also known as BIC (Bank Identifier Code). Each SWIFT code uniquely identifies a bank or its branch, widely used in international wire transfers and financial transactions.

SWIFT Code Structure

A SWIFT code consists of 8 or 11 characters, divided into four parts:

Bank Code (4 letters): Identifies the specific bank institution, assigned by SWIFT. For example, CHAS represents JPMorgan Chase Bank.

Country Code (2 letters): Uses ISO 3166-1 standard country codes. For example, US for United States, GB for United Kingdom.

Location Code (2 letters or digits): Identifies the city or region where the bank is located. For example, 33 for New York.

Branch Code (3 letters or digits, optional): Identifies the specific branch. If omitted or XXX, it represents the bank's head office.

FAQ

What is a SWIFT/BIC code?

A SWIFT code (also called BIC code) is a bank identifier used to uniquely identify banks in international financial transactions. It consists of 8 or 11 characters including bank code, country code, location code, and branch code.

What is the SWIFT code format?

Format: 4-character bank code (letters) + 2-character country code + 2-character location code + 3-character branch code (optional). Example: CHASUS33 or CHASUS33XXX.

Does validation passing mean the code is valid?

This tool validates format conformance to ISO 9362 standards. Format validation passing does not guarantee the code corresponds to a real bank. Check the official SWIFT directory to confirm.

What is the difference between SWIFT and BIC codes?

They are the same thing. BIC is the official name in ISO 9362, while SWIFT code is the common name because the system is managed by SWIFT. They are completely equivalent.

What is the difference between 8 and 11 character codes?

An 8-character code identifies the bank's head office, while an 11-character code identifies a specific branch. If the last 3 characters of an 11-character code are XXX, it is equivalent to the 8-character code (head office).
