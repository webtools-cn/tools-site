❓ Frequently Asked Questions

Is this simulator safe? Are images uploaded?

Completely safe. All processing happens in your browser — images are never uploaded to any server.

Which types are supported?

Protanopia (red-blind), Deuteranopia (green-blind), Tritanopia (blue-blind), and Achromatopsia (total color blindness).

What algorithm is used?

The Brettel-Viénot-Mollon color vision model using LMS color space matrix transformations.

Why use this simulator?

Color blindness affects ~8% of males and ~0.5% of females. Designers use it to ensure accessibility for all users.

Can I download the result?

Yes. Download the simulated image as a PNG file for design reviews or reports.
