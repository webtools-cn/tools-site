What can HAR File Viewer do?

HAR（HTTP Archive）文件查看器是一款免费的在线工具，帮助开发者快速解析和分析浏览器导出的HAR文件。拖拽上传后即可查看所有HTTP请求的详细信息，包括请求URL、Method、Status、响应时间、请求/Response Headers、请求体Size等。所有处理在浏览器本地完成，HAR文件不会上传到任何服务器。

Key Features

- Request List：表格展示所有HTTP请求，支持按URL、Method、Status过滤
- Timeline：柱状图展示每 requests的Time，快速定位慢请求
- Status分布：饼图展示2xx/3xx/4xx/5xx的比例，一目了然
- Domain Stats：柱状图展示各域名的请求数量和Total Traffic
- Size Ranking：TOP20最大响应体排名，优化加载性能

How to Use

1. 在Chrome中按F12打开开发者工具，切换到Network标签
2. 刷新页面或执行操作，等待请求加载完成
3. 右键Request List → "Save all as HAR with content"导出HAR文件
4. 将HAR文件拖拽到本工具的上传区域
5. 自动解析后即可查看Request List和各维度统计图表

应用场景

场景1：Performance Optimization

前端开发者使用HAR查看器分析页面加载时的所有请求，定位Time最长的接口或资源，针对性优化。

场景2：API Debugging

后端开发者通过HAR文件检查API请求的完整信息：Request Headers、请求体、响应Status、响应内容，快速定位接口问题。

场景3：Security Audit

Security engineers review HAR files for sensitive information leaks, checking for plaintext tokens, passwords, and other sensitive data.

FAQ

Does HAR file contain sensitive information?

Yes, HAR files may contain sensitive data such as cookies, tokens, and passwords. Do not share HAR files with untrusted third parties. All processing is done locally in your browser — data is never uploaded.

What size HAR files are supported?

理论上支持任意Size，但超过50MB的HAR文件可能导致浏览器卡顿。建议在导出时勾选"Preserve log"而非保存所有响应内容。

Can encrypted HAR files be parsed?

No. HAR files are standard JSON format. If encrypted or compressed, they must be decrypted first.
