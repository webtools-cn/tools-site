FAQ

How to calculate sphere volume?

Sphere volume: V = 4/3 pi r^3. Example: sphere radius 3cm has volume approx 4/3 x pi x 27 approx 113.1 cm^3.

How is cylinder volume calculated?

Cylinder volume = base area x height = pi r^2 h.

What is the cone volume formula?

Cone volume = 1/3 x base area x height = 1/3 x pi r^2 h. One-third of a cylinder with same base and height.

How to calculate frustum volume?

Frustum volume = 1/3 pi h(R^2 + Rr + r^2), where R = bottom radius, r = top radius, h = height.

How is triangular prism volume calculated?

Triangular prism volume = base triangle area x prism length/height.
