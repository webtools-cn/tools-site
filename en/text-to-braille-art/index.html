How to Use

Convert text to braille dot patterns using Unicode braille encoding (U+2800-U+28FF). Each character is composed of 8 dots in a 2-column, 4-row grid. Supports bidirectional conversion between text and braille.

FAQ

What is Unicode Braille?

Unicode Braille is a character encoding in the Unicode standard (U+2800-U+28FF), with 256 characters covering all 8-dot braille combinations.

Does it support non-English braille?

Currently only Grade 1 English braille is supported. Other languages require more complex conversion rules.
