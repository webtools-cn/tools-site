Tutorial

This font preview tool is easy to use. Here is a detailed guide:

Search字体：在Search框中输入字体Name（如"Roboto"）或关键词（如"serif"、"中文"），工具会实时筛选匹配的字体。支持中英文Search。

Category筛选：通过Category下拉菜单，可以按字体类型筛选：Sans Serif（Sans Serif）、Serif（Serif）、Display（Display）、Handwriting（Handwriting）和Monospace（Monospace）。

自定义预览：在Preview Text框中输入你想测试的文字内容，调整Size滑块和Weight选择，所有字体卡片会实时更新显示效果。

Get Code: Click any font card to select it. CSS import code is auto-generated below, including @import and font-family declarations. Copy with one click.

Use Cases

Web Design: Compare different fonts before starting your design. Good font selection directly impacts user experience and brand identity.

品牌设计：为品牌选择主字体和辅助字体时，需要在不同文本内容、不同Size下预览效果。本工具支持自定义Preview Text，方便测试品牌Name、标语等关键文案。

前端开发：开发者可以直接获取字体的CSS代码，无需手动查找字体Name和引入方式。生成的代码包含完整的引入语句，直接粘贴到项目中即可使用。

Knowledge Base

Web Font Evolution: Early web pages could only use system fonts. After Google Fonts launched in 2010, web fonts became free and accessible. Now Google Fonts has 1500+ font families with billions of monthly views.

字体加载优化：使用Web字体时需注意性能优化。推荐使用font-display: swap确保文字始终可见；使用preconnect预连接字体CDN；只引入实际使用的Weight和子集，减少下载量。

字体搭配原则：经典搭配是Serif+Sans Serif组合，如标题用Serif（如Merriweather），正文用Sans Serif（如Open Sans）。同类型字体搭配时，选择特征差异明显的组合效果更好。

FAQ

字体预览器支持哪些字体？

支持Google FontsAll1500+字体，包括思源黑体、思源宋体、Roboto、Open Sans、Lato、Montserrat等Popular字体，以及各类Handwriting、Display和Monospace。

字体加载速度慢怎么办？

字体从Google CDN加载，首次使用需下载字体文件。建议按Category筛选缩小范围，或使用Search功能直接查找目标字体。已加载的字体会被浏览器缓存。

如何获取字体的CSS代码？

After selecting a font, the tool auto-generates complete CSS code including @import statement and font-family declaration. Click copy to use in your project.

支持中文字体吗？

支持。Google Fonts包含多款中文字体，如Noto Sans SC（思源黑体）、Noto Serif SC（思源宋体）、Noto Sans TC等。在Search框输入中文即可筛选。

数据会上传到服务器吗？

No. All preview and code generation runs locally in your browser. Only font files load from Google CDN, no data is uploaded.
