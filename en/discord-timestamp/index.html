📖 Format Reference

Discord Timestamp Syntax

Format: <t:UNIX_timestamp:style_letter>. For example, <t:1721664000:F> displays as "Wednesday, July 22, 2026 12:00 AM".

Does it adjust to viewer's timezone?

Yes! Discord automatically converts timestamps to each user's local timezone. Set a UTC time and everyone sees it in their local time.

Which Discord clients are supported?

All Discord clients: Desktop, Web, iOS, and Android. Timestamps display correctly on every platform.
