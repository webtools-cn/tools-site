What Can the Meta Tag Extractor Do?

The Meta Tag Extractor is a free online tool that extracts all meta tags from HTML code, including basic meta tags, Open Graph tags, Twitter Card tags, and link tags. It also provides SEO scoring analysis, checking title length, description quality, OG tag completeness, and other key SEO factors. Supports URL import, drag-and-drop files, and multi-format export. All processing runs locally in your browser — data is never uploaded to any server.

Core Features

  • Full Extraction: Extract all meta tags, title, link tags, and base tags, grouped by type
  • Open Graph Analysis: Extract og:title, og:description, og:image and check completeness
  • Twitter Card Analysis: Extract twitter:card, twitter:title and verify card type
  • SEO Scoring: Check title length, description length, viewport, canonical, and other key factors
  • Multi-format Export: Copy tags, download JSON report, or download CSV

How to Use

1. Paste HTML code in the input box, or enter a URL and click "Fetch from URL".

2. You can also drag and drop an HTML file onto the input box.

3. Click "Extract Meta Tags" to see results.

4. Review the SEO score and individual check results.

5. Copy or export the extracted tag data.

Technical Background

Meta tags are metadata elements in HTML's section, crucial for SEO and social sharing. The description tag influences search result snippets (recommended 50-160 characters). Open Graph protocol was created by Facebook and is now supported by all major social platforms; og:image should be 1200×630 pixels. Twitter Card has two types: summary and summary_large_image. The canonical tag specifies the preferred URL to avoid duplicate content issues. The viewport tag controls mobile display and is essential for responsive design. The robots tag controls search engine crawling behavior; noindex prevents page indexing.
