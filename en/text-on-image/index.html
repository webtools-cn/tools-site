Frequently Asked Questions

Is this text on image tool free?

Yes, it's completely free with no registration, login, or usage limits. Use it as many times as you need.

Does it upload my images?

No. All processing happens locally in your browser using the HTML5 Canvas API. Your images never leave your device.

What fonts are available?

We offer system fonts including Arial, Helvetica, Times New Roman, Georgia, Courier New, Verdana, Impact, and monospace. Size ranges from 12px to 200px with bold and italic options.

Can I add watermark text?

Yes — use the opacity slider to make text semi-transparent, which works perfectly for watermarking. We recommend placing watermarks in corners (use X: 10-20, Y: 80-90) with low opacity.

What image formats are supported?

JPG, PNG, and WebP images are supported. The output is always a PNG file with the text overlaid.
