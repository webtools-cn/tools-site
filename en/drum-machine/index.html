📖 What can the Online Drum Machine do?

The Online Drum Machine is a rhythm-making tool based on Web Audio API, providing 8 drum sounds and a 16-step sequencer for arranging various drum patterns. Supports BPM control, swing amount, and WAV export.

核心功能

  • 8 Sounds: Kick, Snare, Closed HH, Open HH, Hi Tom, Lo Tom, Clap, Shaker
  • 16-Step Sequencer: Classic 4/4 16th-note grid arrangement
  • BPM Control: 40-300 BPM tempo range
  • Swing: 0-80% swing feel for groove
  • WAV Export: Render sequence as WAV file download

How to Use

  1. Click sequencer grid to activate/deactivate sound steps
  2. Adjust BPM for tempo, swing for groove feel
  3. Click "Play" to preview the rhythm
  4. Use "Random" for inspiration, or arrange manually
  5. Click "Export WAV" to download when satisfied

Use Cases

Case 1: Music Production

Quickly create drum pattern prototypes, export and import into DAW for further editing.

Case 2: Practice Accompaniment

Generate beat accompaniment for instrument practice, adjust BPM to match practice tempo.

Case 3: Teaching Demo

Visually demonstrate rhythm arrangement principles, help students understand step sequencer workflow.

扩展知识

Step sequencers originated from analog synthesizers (like Roland TR-808), iconic tools in electronic music. 16 steps correspond to 16 sixteenth notes in 4/4 time, every 4 steps = 1 beat. Web Audio API enables browser-based audio synthesis without plugins or servers.

📖 FAQ

What is an online drum machine?

An online drum machine is a rhythm-making tool using a step sequencer to arrange drum patterns with multiple sounds and BPM control.

What is a step sequencer?

A step sequencer is a time-step grid where each column is a step and each row is a sound. Click to activate sounds at specific steps.

What does BPM mean?

BPM (Beats Per Minute) measures tempo. 120BPM is a common pop music tempo.

How to export drum patterns as audio?

Click export. The tool renders the sequence as WAV using OfflineAudioContext and downloads it.

What is Swing?

Swing controls rhythmic feel. 0% = precise timing. Increasing swing delays even steps for a swing feel.
