📖 How to Use

All processing is done locally in your browser. No data is uploaded.

FAQ

What statistics does the Text Statistics Counter provide?

This tool provides 8 real-time metrics: characters (with spaces), characters (without spaces), word count, sentence count, paragraph count, line count, non-Chinese character count, and digit count.

How is word count calculated for mixed Chinese-English text?

Chinese characters are counted individually (each character = 1 word), while English words are counted by space-separated tokens. For example, "Hello world" counts as 2 words, and Chinese text counts each character as one word.

Are statistics updated in real-time?

Yes, all statistics update in real-time as you type or modify text. Every keystroke triggers a recalculation with instant results.

Is my text data safe?

Completely safe. All text statistics are calculated locally in your browser. No text content is uploaded to any server. Data disappears when you close the page.

What text formats are supported?

Supports plain text, mixed Chinese-English text, text with punctuation and numbers, and multi-paragraph text. Rich text formats (RTF/DOC) are not supported—please convert to plain text first.
