Usage

The <iframe> tag embeds another HTML page within the current page.
• src — URL of the page to embed
• width/height — iframe dimensions
• loading — lazy defers loading for perf, eager loads immediately
• sandbox — security restrictions on forms, scripts, navigation
• allowfullscreen — allows fullscreen video playback
• Note: Some websites block iframe embedding via X-Frame-Options header.
