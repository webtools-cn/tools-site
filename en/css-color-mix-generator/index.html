Usage

CSS color-mix() mixes two colors in a specified color space with given ratios.
Syntax: color-mix(in <colorspace>, <color1> <percentage>, <color2> <percentage>)
Color spaces: sRGB (standard), OKLCH (perceptually uniform), HSL (hue saturation), Lab (perceptual) and more.
Note: Percentages don't need to sum to 100%; the browser normalizes automatically.
