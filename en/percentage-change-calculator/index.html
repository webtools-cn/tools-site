Usage

Percentage change measures how much a value has changed relative to its original value.
• Formula: (New - Old) ÷ Old × 100%
• Positive = percentage increase
• Negative = percentage decrease
• Use cases: stock returns, price changes, sales growth, grade changes, population stats
• Note: If original value is 0, percentage change is undefined (infinity).
