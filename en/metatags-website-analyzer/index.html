What are Meta tags?

Meta tags are HTML elements in the head section that provide metadata about a webpage to search engines and browsers. Common Meta tags include title, description, keywords, viewport, etc.

Impact of Meta Tags on SEO

  • Title tag: Important ranking factor, recommended 50-60 characters
  • Meta description: Affects click-through rate, recommended 120-160 characters
  • OG tags: Control display when shared on social media
  • Canonical: Avoid duplicate content, specify canonical URL
  • Robots: Control search engine crawling and indexing
