📖 What Can the Stair Calculator Do?

The stair calculator automatically determines the number of steps, actual riser height, tread depth, total run, angle, and stringer length based on the total floor height and desired step dimensions. It also checks design compliance using the Blondel formula (2R+T) and building code standards.

Key Features

  • Auto Step Distribution: Calculates optimal step count from total height and ideal riser
  • Multi-Type Support: Straight, L-shaped, and U-shaped stairs with landing
  • Code Compliance Check: Validates 2R+T rule and angle range
  • Visual Diagram: Auto-generates side-view stair diagram
  • Stringer Calculation: Computes stringer length via Pythagorean theorem

How to Use

  1. Enter Total Height: Measure the vertical height between floors (cm).
  2. Set Ideal Riser: Enter desired riser height, recommended 16-19cm, default 18cm.
  3. Set Tread Depth: Enter tread horizontal depth, recommended 25-30cm, default 28cm.
  4. Select Stair Type: Choose straight, L-shaped, or U-shaped based on space.
  5. View Results: Results show step count, riser, total run, angle, with compliance check.

Background Knowledge

The Blondel formula (2R+T=63±2cm) was proposed by French architect François Blondel in the 17th century and remains the golden rule for stair design. R is the riser height, T is the tread depth. The formula is based on the natural rhythm of human stride. The stringer length is calculated using the Pythagorean theorem: L=√(H²+R²), where H is total height and R is total horizontal run.

FAQ

What is the ideal riser height for stairs?

For residential stairs, the ideal riser height is typically 16-19cm (6.3-7.5 inches). Public buildings recommend 15-17cm. A comfortable riser height is about 17-18cm.

What should the tread depth be?

Residential stair tread depth should be at least 25cm (10 inches), public buildings at least 28cm. Deeper treads are more comfortable but increase total stair length.

What is the 2R+T rule?

The 2R+T rule (R=riser height, T=tread depth) is the classic stair comfort formula. The ideal value is 60-65cm. For example, riser 18cm + tread 28cm gives 2R+T=64cm, meeting the comfort standard.

What angle should stairs be?

A comfortable stair angle is 30-35 degrees. Residential stairs should not exceed 42 degrees, public buildings 38 degrees. Steeper angles are difficult and unsafe to climb.
