📖 How to Use

Press any key on your keyboard to see all its event properties in real time.

Displays keyCode, key value, code, which, modifier keys, and more.

Supports modifier key detection (Shift, Ctrl, Alt, Meta). All processing in your browser.

⚡ How It Works

This tool uses pure front-end technology. All data processing is done locally in your browser. No server support needed — your data never leaves your device, ensuring privacy and security.

❓ FAQ

❓ Is Keycode Finder free to use?
Yes, it is completely free to use. No registration or login required.
❓ Does it upload my data?
No. It uses pure front-end technology — all processing happens in your browser. No data is uploaded to any server.
❓ What key properties are displayed?
Full KeyboardEvent properties: key, code, keyCode, which, location, shiftKey, ctrlKey, altKey, metaKey, repeat, and more.
❓ Does it support modifier keys?
Yes. It detects Shift, Ctrl, Alt, Meta modifier keys in real time and shows the current combination.
❓ Can I use it on my phone?
Phone functionality is limited since a physical keyboard is needed. It works with external keyboards on mobile devices.
❓ Do I need to install software?
No. Just open the webpage in your browser. No downloads or installations needed.
