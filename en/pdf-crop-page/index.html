FAQ

Q1: Does cropping PDF pages delete content?

Cropping modifies the CropBox property (visible area) of PDF pages. Cropped content still exists in the file but becomes invisible. You can restore original content by resetting the CropBox.

Q2: What units are supported?

Three units are supported: points (pt), millimeters (mm), and inches (in). 1 inch = 72 points = 25.4 mm. You can switch units freely and values will auto-convert.

Q3: Is my PDF file secure?

Completely secure. All processing happens locally in your browser. Your PDF file is never uploaded to any server. File data is automatically cleared from memory after processing.
