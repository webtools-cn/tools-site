📖 FAQ

Is this secure?

Yes. AES-256-GCM is military-grade encryption. Keys are generated locally. Files never leave your device. Keep your password safe — no recovery.

What file types are supported?

Any file type: documents, images, videos, archives. Output is .encrypted file, only restorable with the same password.

What if I forget my password?

No recovery possible. AES-256 has no backdoor. Use a password manager.

Will the file get bigger?

Slightly. ~28 bytes added (IV+salt+metadata), typically under 1% increase.
