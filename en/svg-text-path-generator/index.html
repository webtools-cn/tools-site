Frequently Asked Questions

What is SVG text path?

SVG text path allows text to follow a curved or shaped path, creating flowing typography effects that aren't possible with standard text.

Can I use my own SVG path?

Yes. Select 'Custom Path' and enter your SVG path d-attribute (the path data string) in the input field.

How do I export the result?

Click the 'Export SVG' button to download the text path graphic as an SVG file.

Can I use this in commercial projects?

Yes. The exported SVG files are yours to use in any project, including commercial ones.

What does startOffset do?

Path offset shifts the starting position of text along the path. Negative values move it left, positive values move it right.
