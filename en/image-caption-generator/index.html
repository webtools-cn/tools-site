FAQ

How do I add a caption to an image?

Three steps: 1) Upload your image 2) Enter caption text and adjust font size, color, position, and other styles 3) Click download to save the captioned image. All operations are real-time WYSIWYG preview.

What image formats are supported?

Upload supports JPG, PNG, and WebP formats. Download supports PNG (with transparency) and JPG formats. PNG offers best quality, JPG produces smaller file sizes.

Does the caption text support Chinese?

Yes, it fully supports Chinese, Japanese, Korean, and all Unicode text. The tool automatically detects and uses the most suitable system font (prioritizing PingFang SC, Microsoft YaHei, Noto Sans SC).

Are my images uploaded to a server?

Absolutely not. All image processing happens entirely in your browser using the Canvas API. Your image data never leaves your device, safe for sensitive or private images.

Can I add multiple captions?

The current version supports a single caption text block. For multiple text overlays (e.g., top title + bottom description), generate the first caption, then upload the result as a new image for the second caption.
