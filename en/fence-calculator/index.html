📖 How to Use

  • Total fence length is the straight-line distance your fence will cover
  • Board width + gap determines each board's pitch, automatically computing board count
  • Post count = fence length ÷ post spacing + 1 (one at each end)
  • We recommend buying 5%-10% extra boards for waste and breakage
  • Toggle between metric and imperial — all units adapt automatically
