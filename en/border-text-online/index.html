Background

Border text uses Unicode Box Drawing characters (U+2500-U+257F) to create text frames. Originally designed for terminal and DOS text UI, now widely used for social media decoration and Markdown formatting.

FAQ

What is border text?

Border text surrounds regular text with special characters to make it stand out in plain text environments. Popular for social media, code comments, and terminal output.

What border styles are available?

Box, rounded, double-line, star, wave, dashed, fancy, and more.

Can I customize the border width?

Yes. Adjust left/right and top/bottom padding to control border width.

Can I copy the generated text?

Yes. Click the copy button to copy to clipboard instantly.

Does it support non-English text?

Fully supports all Unicode characters including Chinese, Japanese, Korean, and emojis.
