❓ FAQ

Will resizing a GIF lose the animation?

No. This tool parses and rebuilds all GIF frames using Canvas, resizing each frame individually before re-encoding. The original animation and frame order are fully preserved.

Will the file size increase or decrease after resizing?

Generally, reducing dimensions decreases file size, while enlarging increases it. However, quality settings affect the result. Find the right balance for your use case.

What file size limit is supported?

We recommend GIFs under 10MB with fewer than 200 frames. Larger files may cause browser memory issues.

Why is the preview static?
