💡 How to Use

  • Paste your CSS code into the input area
  • Choose compression mode (Standard/Aggressive)
  • Click "Minify CSS" to get the compressed result
  • Click "Copy" to copy the minified code
  • Click "Download" to save as .min.css file

⚙️ Compression Strategy

  • Remove all comments (/* ... */)
  • Remove unnecessary whitespace and newlines
  • Shorten hex colors (#ff0000 → #f00)
  • Merge same-selector rules
  • Remove unnecessary semicolons and quotes
  • Aggressive mode removes trailing semicolons
