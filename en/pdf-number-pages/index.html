📖 How to Use

Upload PDF: Click or drag & drop a PDF file. The tool will load and display file information.

Configure Page Numbers: Select position (bottom center/left/right), font size, starting number. Add optional prefix/suffix.

Preview: The preview area shows the first 5 pages with your settings.

Download: Click "Download Numbered PDF" to generate and download the result.

🎯 Use Cases

Document Standardization: Add page numbers to contracts, reports, and papers for easy reference.

eBook Creation: Improve reading experience by adding page numbers to PDF eBooks.

Print Preparation: Add page numbers before printing to maintain page order.

❓ Frequently Asked Questions

How much does page numbering increase file size?

Very little. Page numbers are added as text elements via the pdf-lib library. For a 100-page PDF, the size increase is typically under 10KB.

Can I customize font size and color?

Yes. You can choose page number position (bottom center, left, or right), font size (small/medium/large), and starting page number. Advanced options allow font and color customization.

Does my PDF get uploaded to any server?

No. All processing happens locally in your browser. Your file never leaves your device, ensuring complete privacy and data security.

Can I start numbering from a specific page?

Yes. Set the starting page number in the options. If the first 2 pages are a cover and table of contents, you can skip them and start numbering from page 3 as page 1.

Will page numbers overlap with existing content?

Page numbers are placed in the bottom margin area to avoid body content. If your PDF already has footers, choose a different position like bottom-left or bottom-right to avoid overlap.
