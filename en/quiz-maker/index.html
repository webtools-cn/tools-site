Quiz Maker - Free Online Quiz & Test Creator

Create quizzes and tests online for free. Support single/multiple choice and true/false questions with auto-grading. Export and share quizzes. All data saved locally, no registration required.

FAQ

What question types are supported?▼
Single choice, multiple choice, and true/false. Each question supports 2-6 options with specified correct answers.
Is data saved?▼
Yes. Auto-saved in localStorage. Export JSON for backup.
How to share?▼
Export as JSON file, send to others to import.
How to take a quiz?▼
Click Start Quiz, answer questions, submit for auto-grading.
Is data uploaded?▼
No. Everything runs locally. Nothing uploaded.
