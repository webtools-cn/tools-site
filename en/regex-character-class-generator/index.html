📖 FAQ

What are regex character classes?

正则表达式字符类是用方Brackets[...]enclosed character set, matching anyBracketsany one character in the set. For example[abc]matches a, b, orcany one in the set. [a-z] matches anyLowercase。[^0-9]匹配任意非Digits字符。Character classes are one of the most commonly used regex constructs。

How to use hyphens in character classes?

In character classes, hyphens represent ranges, like [a-z] for all characters from a to z. To match a literal hyphen, place it at the start or end like [-abc] or [abc-], or escape it with [\\-].

What does negation class [^...] mean?

在字符类的开头使用脱字符(^)means negation, matching any character NOT in theBracketsany character in the set. For example[^0-9]匹配任意非Digits字符，[^aeiou]matches any non-vowel. Note^must be at the start to mean negation; elsewhere it only matches^the character itself。

What predefined character classes are available?

常用预定义字符类包括：\d（Digits，equivalent to [0-9]), \D (non-Digits，equivalent to [^0-9]), \w (word characters，equivalent to [a-zA-Z0-9_]), \W (non-word characters）、\s（Whitespace）、\S（non-whitespace). These shorthands work inside or outside character classes。
