How It Works

The BIC/SWIFT Code Checker validates the format of Bank Identifier Codes and decodes each part.

BIC Structure: 4-letter bank code + 2-letter country code (ISO 3166) + 2-character location code + 3-character branch code (optional)

8-character BIC: Represents the bank's head office, e.g. DEUTDEFF

11-character BIC: Represents a specific branch, e.g. DEUTDEFF500

Note: A valid format does not guarantee the code exists. Confirm with your bank.

FAQ

What is the difference between BIC and SWIFT code?

BIC (Bank Identifier Code) and SWIFT code are different names for the same thing. SWIFT is the organization name, BIC is the standard code name. They are completely equivalent.

What does XXX mean in the branch code?

Branch code XXX indicates the bank's primary office (headquarters), equivalent to an 8-character BIC code.

Why is my code valid in format but doesn't exist?

Format validation only checks if the code follows ISO 9362 standards, not whether it's registered in the SWIFT network. Check with your bank or the SWIFT official directory.

Do I need a BIC code for international transfers?

Yes, international SWIFT transfers typically require the recipient bank's BIC/SWIFT code to ensure funds reach the correct bank.
