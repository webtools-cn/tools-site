FAQ

How deep should I lay gravel?

Driveways typically need 15-20cm, walkways and patios 5-10cm, and drainage layers 10-15cm. This tool supports custom depth for any project requirement.

Do different gravels have different densities?

Yes. Common gravel densities range from 1.4-1.8 t/m³. Decomposed granite is denser (~1.68), while pea gravel is lighter (~1.5). Selecting the correct type improves accuracy.

Does the result include waste?

No. This tool calculates the theoretical amount. We recommend adding 5%-10% extra for compaction, waste, and uneven spreading during actual construction.

Is my data safe?

Absolutely. This tool runs entirely in your browser. All calculations are performed locally — no data is uploaded to any server.

How to Use the Gravel Calculator

  2. Choose area shape — Rectangle, circle, or trapezoid
  3. Enter dimensions and depth — Input the area dimensions and gravel depth
  4. View results — Instantly see volume, weight, and bag count
