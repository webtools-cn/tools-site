❓ FAQ

支持哪些Chart Type？

支持Bar Chart、Line Chart、Pie Chart、Doughnut Chart、Radar Chart等常见Chart Type。使用Canvas原生绘制，无需任何第三方库。

Can I export charts?

Yes! Export to PNG format. Click the download button to save a high-resolution chart.

How to input data?

Use JSON format for data input：[{"label":"Jan","value":30},...]。支持任意数量的数据项。

❓ Frequently Asked Questions

Is Chart Maker free to use?

Yes, Chart Maker is completely free to use. No registration or login required — all features are available immediately.

Are there any usage limits for Chart Maker?

Chart Maker has no usage limits — it's free and unlimited. Since processing is done client-side, speed depends on your device. For very large files, we recommend processing in batches for the best experience.

Do I need to install any software?

No. Chart Maker is a web-based tool — just open it in your browser and start using it. No downloads, plugins, or extensions required.

Is my data safe when using Chart Maker?

Absolutely. All processing happens locally in your browser. No data is sent to any server, so there's zero risk of data leaks. Your information stays completely under your control.
