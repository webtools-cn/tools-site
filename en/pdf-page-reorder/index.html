📖 How to Use

Upload PDF: Click or drag & drop a PDF file. The tool will load and display page information.

Reorder Pages: Use the up/down buttons on each page card to adjust page order. Click a card to select it, then use global buttons.

Delete & Duplicate: Click "✕" on a page card to delete it. Select a page and click "Duplicate" to copy it. "Reset" restores original order.

Download: Click "Download Reordered PDF" to generate and download the new PDF file.

🎯 Use Cases

Document Organization: Fix scrambled page order from a scanner.

Post-Merge Adjustment: After merging multiple PDFs, reorder pages as needed.

Page Extraction: Delete unwanted pages to keep only the content you need.

❓ Frequently Asked Questions

Does page reordering affect file quality?

No. The tool uses the pdf-lib library to directly manipulate PDF page objects. Reordering only changes page sequence without re-encoding or compressing content. All original text, images, links, and formatting remain intact with zero quality loss.

Can I delete pages during reordering?

Yes. Each page card has a delete button to remove unwanted pages. Deletion is reversible — use the 'Reset' button to restore all deleted pages before downloading.

What is the maximum PDF file size supported?

Recommended file size is under 50MB. Since processing happens entirely in the browser, memory availability is the main constraint. For larger files, consider compressing them first.

Does my PDF get uploaded to any server?

No. All processing happens locally in your browser. Your file never leaves your device, ensuring complete privacy and data security.

Can I apply the same page order to multiple PDFs?

Currently one file at a time. Batch processing with the same page ordering pattern is under development.
