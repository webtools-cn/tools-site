📖 FAQ

How does it work?

Uses pdf-lib to load encrypted PDF (with password if needed), then copies all pages to a new unencrypted PDF. Must know the open password.

What can be removed?

Open passwords and permission restrictions (print/copy/edit). Permissions can be removed via page copying.

Is it safe?

Yes. All local. Files never uploaded. Passwords only used temporarily.

Why is file smaller?

Page copying may lose metadata (bookmarks, annotations, forms). Content is preserved.

All PDFs unlockable?

No. Supports AES-128/256. No RC4, certificate encryption, or digital signatures. Unknown passwords cannot be cracked.
