FAQ

Q1: What is the difference between word wrap and hard wrap?

Word wrap breaks at word boundaries, never splitting words, but lines may be slightly shorter than the column width. Hard wrap breaks at the exact column width, potentially splitting words mid-way, suitable for code or fixed-width formats.

Q2: When do I need text wrapping?

Common scenarios: email body formatting (78-column standard), code comment alignment, plain text file layout, terminal output formatting, Markdown long paragraph wrapping.

Q3: What is the indent feature for?

Indent adds specified spaces at the beginning of each line, useful for nested quotes, code indentation, email reply quoting, etc.
