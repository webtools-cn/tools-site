Calendar Event Guide

What is an ICS File?

ICS (iCalendar) is a universal calendar file format (RFC 5545) supported by Google Calendar, Apple Calendar (iCal), Outlook, Yahoo Calendar, and more. It contains event details like title, time, location, and description.

What is a Google Calendar Link?

A Google Calendar link is a special URL that opens Google Calendar and auto-creates an event with pre-filled details. Perfect for sharing events via websites, emails, or social media.

Use Cases

Event Registration: Provide "Add to Calendar" links after user registration to increase attendance rates for webinars, conferences, and meetups.

Meeting Invitations: Generate calendar events for team meetings and share via email or chat to ensure everyone attends on time.

Website Events: Add "Add to Calendar" buttons on product launch pages, workshop listings, and promotional event pages.

Ticket Confirmations: Include calendar links in ticket confirmation emails for concerts, movies, exhibitions, and more.

What is an ICS file?

ICS (iCalendar) is a universal calendar file format supported by Google Calendar, Apple Calendar, Outlook, and other major calendar apps.

What is a Google Calendar link?

A special URL that opens Google Calendar and auto-fills event details for quick creation.

What's the difference between ICS and Google Calendar links?

An ICS file can be downloaded and imported into any calendar app. A Google Calendar link opens directly in Google Calendar.

How to add "Add to Calendar" on my website?

Provide both options: 1) Google Calendar link as an anchor href, 2) ICS file download. Cover all users' preferences.

What timezone is used?

The event uses your local timezone. The ICS file includes timezone info. Google Calendar links also accept timezone parameters.
