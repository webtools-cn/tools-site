❓ FAQ

What's the difference between ICO and regular images?

ICO is a Windows icon format that can contain multiple sizes in one file. The OS picks the best size automatically. Regular formats like PNG/JPG are usually single-size.

What size should I choose for a favicon?

Different platforms need different sizes: browser tabs use 16x16, desktop shortcuts 32x32, iOS home screen 180x180, Android 192x192. Upload a square high-res image (512x512 ideal).

What input formats are supported?

PNG, JPG, and WebP. PNG is recommended for best results since ICO natively supports transparency. JPG gets a white background.
