❓ FAQ

What adjustments are supported?

Seven core color parameters: Brightness (-100 to +100), Contrast (-100 to +100), Saturation (-100 to +100), Hue Rotation (0°-360°), Exposure (-2 to +2), Temperature (-100 to +100), and Shadows/Highlights. Each can be independently adjusted with real-time preview.

What image formats are supported?

Common formats: JPG/JPEG, PNG, WebP, and BMP. Output is PNG format for maximum quality. Export at original resolution.

Does adjustment reduce image quality?

Each adjustment is computed from the original image data, so there's no cumulative quality loss. However, extreme adjustments (max brightness, zero saturation) may lose detail. Output is lossless PNG at original resolution.

Is my image secure?

Absolutely. All processing is done locally using the Canvas API. Your image is never uploaded to any server. You can use this tool completely offline.

Is there a size limit?

No hard limit, but very large images (over 4000×4000 px) may impact performance. The tool auto-scales preview but exports at original resolution.
