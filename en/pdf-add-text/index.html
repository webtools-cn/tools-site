❓ FAQ

Is the PDF Add Text tool free?

Yes, this tool is completely free to use. No registration required, no usage limits. All processing happens locally in your browser.

Is my PDF file secure?

Absolutely secure. This tool uses client-side processing. Your PDF file is never uploaded to any server. All operations happen in your browser and file data is automatically cleared after processing.

What text styling options are available?

You can customize text content, font size (8-120px), text color, opacity, rotation angle, and position. You can also choose to add text to specific pages or all pages.

Will adding text reduce PDF quality?

No. This tool uses the pdf-lib library to add text directly as a PDF layer, without converting to images. The original PDF quality is preserved.

Does it support non-Latin characters?

Yes. Non-Latin characters (Chinese, Japanese, Korean, etc.) are embedded as glyphs in the PDF, ensuring correct display in any PDF reader.
