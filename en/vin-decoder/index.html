What is a VIN Decoder?

Free online VIN (Vehicle Identification Number) decoder. Enter a 17-character VIN to automatically parse the manufacturer, production country, brand, model year, assembly plant, and serial number. Pure frontend processing — your VIN is never uploaded to any server.

Core Features

  • WMI Parsing: Identify manufacturer, country, and brand (first 3 characters)
  • Model Year: Decode the 10th character to production year
  • Assembly Plant: Parse the 11th character for plant code
  • Serial Number: Extract production sequence (characters 12-17)
  • Check Digit Validation: Verify VIN validity via 9th character

Technical Background

VIN is defined by ISO 3779 and ISO 3780 standards with a unified 17-character format worldwide. Since 1981, the US requires all road vehicles to have a VIN. WMI (World Manufacturer Identifier) is assigned by SAE International to ensure global uniqueness. The 9th check digit is calculated using a weighted algorithm to detect input errors.

FAQ

What is a VIN number?

VIN (Vehicle Identification Number) is a unique 17-character code assigned to every motor vehicle. It contains information about the manufacturer, model, engine type, and production year.

Where is the VIN located on a car?

Common locations: dashboard lower-left corner (visible through windshield), driver's side door jamb label, vehicle registration certificate, insurance documents.

What does the 10th digit of a VIN mean?

The 10th digit is the model year code. 2025=A, 2026=B. Digits 1-9 represent 2001-2009, letters A-Y (excluding I/O/Q/U/Z) represent 2010-2030.

Can a VIN tell me if a car has been in an accident?

VIN decoding only provides basic vehicle info. To check accident history, you need a vehicle history report service like Carfax or AutoCheck.

Why is my VIN decode incomplete?

WMI (first 3 digits) and model year (10th digit) are standardized, but VDS and VIS sections vary by manufacturer. This tool decodes standardized info; manufacturer-specific codes require official references.
