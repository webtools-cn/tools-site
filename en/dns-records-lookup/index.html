❓ FAQ

What are DNS records?

DNS (Domain Name System) records are database entries mapping domain names to IP addresses and service information. Common types: A (IPv4), AAAA (IPv6), MX (mail servers), CNAME (aliases), TXT (text), NS (nameservers).

How do I look up DNS records?

Enter a domain name, select the record type (or ALL for all types), and click Lookup. The tool uses Google's DNS-over-HTTPS API for real-time results.

What's the difference between A and CNAME records?

An A record points a domain directly to an IP address. A CNAME record points a domain to another domain name (alias). CNAMEs cannot be used for root domains (e.g., example.com), only subdomains.

Why are my query results empty?

Possible reasons: 1) Domain doesn't exist or has no records of that type; 2) DNS cache hasn't updated (new changes need TTL time); 3) DNS provider restrictions. Try again after a few minutes.
