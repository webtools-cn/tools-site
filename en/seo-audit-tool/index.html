📖 How to Use

Step 1: Get Page HTML

Open the webpage in your browser, right-click and select "View Page Source" (Ctrl+U / Cmd+U), select all (Ctrl+A) and copy (Ctrl+C) the HTML code.

Step 2: Paste and Audit

Paste the HTML source code into the input field and click "Run Audit". The tool analyzes all SEO elements automatically.

Step 3: Review Results

Each audit item shows a status (Pass/Warning/Fail), current value, and specific optimization recommendations.

🎯 Audit Checklist

  • Title Tag: Present, unique, 30-60 characters, keyword-rich
  • Meta Description: Present, compelling, 50-160 characters
  • H1 Tag: One and only one, contains primary keyword
  • Heading Hierarchy: Logical structure H1→H2→H3, no skipping
  • Image Alt Text: All images have descriptive alt attributes
  • Structured Data: JSON-LD or Microdata for rich snippets
  • Canonical Tag: Prevents duplicate content issues
  • Open Graph: Optimizes social sharing appearance
  • Viewport: Mobile-responsive meta tag
  • Page Size: Optimized for fast loading

💡 SEO Tips

Title tags are one of the most important ranking factors. Each page should have a unique, descriptive title with primary keywords near the front.

Meta descriptions don't directly affect rankings, but good descriptions improve click-through rates (CTR).

Heading structure should form a clear hierarchy. Use one H1 per page, followed by H2s for sections and H3s for subsections.

Image alt text helps search engines understand images and improves accessibility. It's also a ranking factor for image search.

Structured data (Schema.org) helps search engines understand your content better and enables rich snippets in search results.
