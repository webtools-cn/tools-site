How do I create a WiFi QR code?

Select WiFi type, enter the SSID (network name), password, and encryption type (WPA/WPA2/WEP/None). The generated QR code can be scanned by a phone camera to connect to the WiFi network instantly without manually entering the password.

What do error correction levels mean?

QR code error correction determines how much damage the code can withstand while remaining scannable. L (Low) recovers 7%, M (Medium) 15%, Q (Quartile) 25%, H (High) 30% of data. If you plan to add a logo in the center, use H level for best reliability.

How much data can a QR code store?

Maximum capacity depends on version and error correction level. At Version 40 with L-level correction: 7,089 digits, 4,296 alphanumeric, 2,953 bytes, or 1,817 Chinese characters. For URLs and short text, lower versions are typically sufficient.

What fields does vCard QR code support?

The vCard QR code supports: name, company, job title, phone, email, website, and address. When scanned, it can be saved directly to the phone's contacts — one of the most popular ways to share digital business cards.
