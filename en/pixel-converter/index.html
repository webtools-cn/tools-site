FAQ

❓ How do you convert pixels (px) to centimeters (cm)?

The formula is: cm = pixels / PPI × 2.54. PPI (Pixels Per Inch) is the key parameter. Standard screen PPI is 96, Retina is 192-326, print standard is 300 DPI. For example, at 96 PPI, 100 pixels ≈ 2.65 cm; at 300 DPI, 100 pixels ≈ 0.85 cm.

❓ What are common PPI values?

Common PPI values: Old monitors 72 PPI, standard desktop 96 PPI, HD screens 120 PPI, Mac Retina 192-226 PPI, iPhone 326-458 PPI, Android flagships 400-500+ PPI, Print 300 DPI, High-quality print 600 DPI.

❓ What's the difference between PX and PT (Points)?

PT (Point) is a print typography unit. 1pt = 1/72 inch. In CSS, 1pt ≈ 1.333px (at 96 PPI). Formula: pt = px × 72 / PPI. At 96 PPI, 1px = 0.75pt. PT is used in print styles, while PX is for screen display.

❓ What PPI should I use for UI design?

Use 96 PPI for web design (standard screen), logical resolution for mobile, and 300 DPI for print. Figma and Sketch use 1x logical resolution and export at 2x/3x. For precise print sizes, confirm the target PPI/DPI before designing.
