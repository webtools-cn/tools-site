What is a System Prompt?

A System Prompt is an instruction sent to an AI model before the conversation begins, setting the AI's role, behavior, and output format. Unlike regular user messages, System Prompts have stronger influence over AI behavior and are a core technique in Prompt Engineering.

How to Write High-Quality System Prompts

Key elements of effective System Prompts:

  • Clear Role Definition: Tell the AI who it is and its expertise
  • Specify Output Format: Request Markdown, JSON, tables, etc.
  • Set Behavioral Boundaries: Define what the AI should and shouldn't do
  • Provide Examples: Use few-shot examples to demonstrate expected output
  • Use Conditional Instructions: Give different handling for different situations
