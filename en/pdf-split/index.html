📖 FAQ

How to split a PDF?

Three modes: extract specific pages, split every N pages, or separate all pages. Upload PDF and choose mode.

How to specify pages?

Comma-separated (1,3,5), range (1-5), or mixed (1,3-5,8). Pages start at 1.

How are files downloaded?

Each split file downloads separately. Extract mode merges selected pages into one file.

Is it safe?

Yes. PDFs processed locally using pdf-lib. Nothing uploaded.

File size limits?

Up to 100MB recommended. Larger files may be slow due to browser memory.
