FAQ

What is a URL query string?

A query string is the part after the ? in a URL, used to pass parameters. Format: key1=value1&key2=value2.

Does the query string need encoding?

Yes. Special characters like spaces, &, = must be encoded with encodeURIComponent to avoid breaking the structure.

Is this tool safe?

Yes. All parsing happens locally in your browser. Input URLs are never sent to any server.

What is the difference between encoding options?

encodeURI preserves :/?#[]@ etc., while encodeURIComponent encodes more thoroughly, suitable for parameter values.

Does it support nested parameters?

This version supports flat key-value parsing. Nested JSON parameters may be supported in a future version.
