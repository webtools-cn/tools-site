FAQ

Q1: How do I separate slides?

Use three hyphens --- on a separate line to split your Markdown into different slides. Content between separators becomes an individual slide.

Q2: What Markdown syntax is supported?

Standard Markdown: headings (#, ##, ###), lists (- or 1.), bold (**text**), italic (*text*), inline code (`code`), code blocks (```), links, etc. Content is automatically centered on each slide.

Q3: How do I present in fullscreen?

Click the "Fullscreen" button. Use left/right arrow keys or click the sides of the screen to navigate. Press ESC to exit fullscreen.
