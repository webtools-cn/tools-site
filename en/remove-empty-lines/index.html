FAQ

Q1: What is an empty line?

An empty line contains no visible characters, only a newline. If "Treat whitespace-only lines as empty" is checked, lines with only spaces and tabs are also considered empty.

Q2: What is the difference between removing and collapsing empty lines?

Removing deletes all empty lines for compact text. Collapsing reduces consecutive empty lines to a single one, preserving paragraph separation. Useful for maintaining document structure while reducing excess whitespace.

Q3: Is my data safe?

Yes. All processing happens locally in your browser. No text data is uploaded to any server. Data disappears when you close the page.
