FAQ

What is a pip?

A pip is the smallest price move in forex. For most pairs, 1 pip = 0.0001; for JPY pairs, 1 pip = 0.01.

How is pip value calculated?

Pip value = contract size × pip size. A standard lot (1 Lot) = 100,000 units of base currency, mini lot = 10,000, micro lot = 1,000. The tool automatically determines the pip size based on the currency pair.

Is my data safe?

Absolutely. This tool is pure client-side — all calculations happen in your browser's local memory. Nothing is uploaded to any server.

Can I use it on mobile?

Yes. The tool is fully responsive and works on mobile devices, tablets, and desktops. The interface automatically adjusts to fit your screen.
