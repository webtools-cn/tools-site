❓ FAQ

How does the batch QR generator work?

Enter URLs or text in the input area, one per line. Click 'Generate QR Codes' and each line becomes a QR code. You can customize size, error correction level, and colors. Each QR code can be downloaded individually or all at once as a single image.

How many QR codes can I generate at once?

There's no strict limit, but we recommend no more than 50 at a time for performance. Generated QR codes are arranged in a grid for easy viewing.

Can I customize QR code colors?

Yes. Use the color pickers to customize foreground (default black) and background (default white). Ensure sufficient contrast between colors for reliable scanning.

What error correction levels are supported?

Four QR error correction levels: L (7%), M (15%), Q (25%), H (30%). Higher levels improve scan reliability when the code is dirty or damaged, but make the pattern denser. M level is recommended for general use.

Is my data secure?

Yes. All QR code generation is done locally using the qrcode.js library. Your data is never uploaded to any server.
