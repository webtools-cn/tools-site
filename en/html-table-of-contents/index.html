Paste HTML code with heading tags (h1-h6) into the input box, click "Generate TOC" to auto-create a structured table of contents.

Supports numbering (1, 1.1, 1.1.1...) and anchor links. Choose max heading level. The generated TOC HTML can be copied directly.
