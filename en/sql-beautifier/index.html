📖 How to Use

This SQL formatter supports MySQL, PostgreSQL, SQLite, SQL Server and more.

Features

  • Format (UPPERCASE)：将SQL关键词转为大写，标准缩进排版
  • Format (lowercase)：将SQL关键词转为小写，标准缩进排版
  • Minify：移除多余空白和换行，Minify为一行

Supported

  • SELECT、INSERT、UPDATE、DELETE等所有DML语句
  • JOIN（INNER/LEFT/RIGHT/FULL/CROSS）自动换行
  • 子查询嵌套
  • WHERE条件自动换行（AND/OR）
  • GROUP BY、ORDER BY、HAVING自动对齐
  • CREATE TABLE、ALTER TABLE等DDL语句

All processing happens in your browser. Your SQL never leaves your computer.
