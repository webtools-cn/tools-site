📖 FAQ

How does it work?

Uses Canvas API to read pixel data, applies color quantization to extract dominant colors. All processing is local in your browser.

What formats are supported?

JPG, PNG, GIF, BMP, WebP, SVG and any browser-renderable image format.

How to use extracted colors?

Click any swatch to copy the HEX value. Export all colors as text with HEX+RGB values for web design, UI, and branding.

Can I adjust color count?

Yes. Choose 3, 5, 7, or 10 colors.

Is my image safe?

Absolutely. Processed locally via Canvas API. Nothing is uploaded.
