Frequently Asked Questions

What can this PDF editor do?
Add text, merge PDFs, extract pages, rotate pages — all in your browser. No file upload required.
Are files uploaded to a server?
No. All processing uses pdf-lib.js in your browser. Files never leave your device.
What operations are supported?
Add text, merge PDFs, extract pages by range, rotate pages. More features coming soon.
File size limits?
Individual files should be under 50MB. Larger files may be slow due to browser memory limits.
