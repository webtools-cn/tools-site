📖 Prompt Engineering Best Practices

1. Role Setting

Give the AI a clear role, like "You are a senior frontend engineer." Role settings help AI respond from a specific perspective.

2. Output Format

Specify the format, like "Output in Markdown", "List key points", "Generate JSON". Clear format requirements make output more usable.

3. Context Information

Provide sufficient background, like "I'm developing a React project." Context helps AI understand your specific scenario.

4. Constraints

Set limits, like "No more than 500 words", "Don't use external libraries." Constraints make output more practical.

5. Examples

Provide input/output examples, like "For input x, output y." Examples are the most effective way to communicate.

6. Structured Instructions

Break complex tasks into steps, like "Step 1: ... Step 2: ...". Structured instructions help AI execute systematically.
