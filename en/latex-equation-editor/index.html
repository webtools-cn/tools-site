FAQ

What is LaTeX?

LaTeX is a document preparation system widely used in academia for scientific papers, theses, and mathematical typesetting. It uses markup tags to describe document structure and mathematical notation.

KaTeX vs MathJax?

KaTeX, developed by Khan Academy, renders 10-100x faster than MathJax, making it ideal for real-time preview. MathJax has broader coverage but is slower. This tool uses KaTeX for instant rendering.

Which LaTeX commands are supported?

All KaTeX-supported commands are available, including: fractions, square roots, integrals, sums, matrices, limits, Greek letters, cases environments, and more.

Is my data sent to a server?

No. All LaTeX rendering happens locally in your browser using KaTeX. Zero data is uploaded to any server.
