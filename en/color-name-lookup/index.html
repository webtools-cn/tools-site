What Can Color Name Lookup Do?

Free online color name lookup tool with 140+ CSS standard named colors. Search by name, find closest named color by HEX value, and use the color picker for real-time preview. All queries run locally in your browser.

Core Features

  • Name Search: Fuzzy search by color name with real-time filtering
  • HEX Lookup: Enter HEX value to find the closest CSS named color
  • Color Picker: Use system color picker to select and find color names
  • Color Difference: CIE76 algorithm for perceptually accurate matching
  • One-click Copy: Copy color name, HEX, or RGB values

How to Use

  1. Enter a color name (e.g. "coral") or HEX value (#FF7F50) in the search box
  2. Or use the color picker to select a color
  3. Click "Find Closest" to get the matching named color
  4. Click a color card to copy its HEX value

Use Cases

Case 1: Frontend Development

Use semantic color names in CSS instead of HEX values for better code readability.

Case 2: Design Communication

Quickly find named colors matching design mockups for easier team communication.

Case 3: Learning Reference

Browse and learn the CSS named color system and naming conventions.

Extended Knowledge

CSS named colors are defined by W3C, originally 16 basic colors (HTML 4.01), expanded to 140+ (CSS3). Names originate from X11 Window System color definitions like AliceBlue, CornflowerBlue. Named colors are more semantic than HEX, but precise design still requires HEX/RGB values.
