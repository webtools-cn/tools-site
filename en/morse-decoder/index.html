What Can the Morse Code Decoder Do?

The Morse Code Decoder is a free online tool that performs bidirectional conversion between Morse code and text. It supports auto-detection of input type (text or Morse code), audio playback of Morse signals, and adjustable frequency and speed. Includes a complete Morse code reference table for quick lookup. Ideal for amateur radio operators, Morse code learners, and cryptography enthusiasts. All processing happens locally in your browser—no data is ever uploaded to a server.

Core Features

  • Bidirectional Conversion: Text→Morse code and Morse code→Text with one click
  • Auto-detection: Intelligently identifies whether input is text or Morse code
  • Audio Playback: Uses Web Audio API to generate Morse code audio with adjustable frequency and speed
  • Reference Table: Built-in complete Morse code lookup for letters, numbers, and common punctuation
  • History: Automatically saves your last 5 conversions

How to Use

1. Enter text or Morse code in the input box.

2. Click "Decode" (Morse→Text) or "Encode" (Text→Morse), or enable auto-detection.

3. Click "Play" to hear the Morse code audio signal.

4. Adjust frequency and speed parameters to customize audio.

5. Click "Copy" or download the result.

Technical Background

Morse code was invented by Samuel Morse and Alfred Vail in 1836 for telegraph communication. International Morse code (ITU standard) uses dots (dit, ~0.1s) and dashes (dah, ~0.3s) to represent letters and numbers. The gap between dots/dashes is 1 unit, between letters 3 units, and between words 7 units. SOS (···---···) is the international distress signal. Morse code is still used in maritime, aviation, and amateur radio today.

❓ FAQ

What is Morse code?

An encoding system using short signals (dots) and long signals (dashes) to represent letters and numbers, invented by Samuel Morse in 1836.

Can it play Morse code audio?

Yes. It uses Web Audio API to generate Morse code audio with adjustable frequency and speed.

Does it support bidirectional conversion?

Yes. Convert text to Morse code and Morse code to text with one click.

Is my data uploaded?

No. All decoding and encoding happens locally in your browser.

How are dots and dashes distinguished?

Dots use . or ·, dashes use -. Letters separated by spaces, words by /.

Does it auto-detect input type?

Yes. Automatically detects whether input is text or Morse code and selects the appropriate direction.
