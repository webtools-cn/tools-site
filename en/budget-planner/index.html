FAQ

Why budget planning?

Budget planning is a core personal finance skill. By setting spending limits for each category, you control expenses, avoid overspending, and increase savings. Studies show budgeters save 20% more on average.

Budget vs Expense Tracking?

Expense tracking records what you've already spent (after the fact). Budgeting plans future spending (before the fact). Using both together is most effective: set budgets first, then track actuals against them.

What if I exceed my budget?

If you exceed a category budget: 1) Analyze if the overspend was necessary; 2) Adjust from other categories; 3) Reconsider the budget amount next month; 4) Check if fixed costs have increased.

Is my data safe?

All data is stored in your browser's localStorage. Nothing is uploaded to servers. Clearing cache will erase data, so backup periodically.
