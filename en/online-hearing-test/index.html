What Can the Online Hearing Test Do?

The online hearing test is a free, pure-frontend hearing assessment tool using Web Audio API to generate pure tones at different frequencies and volumes. Supports left/right ear testing, generates audiogram charts. All data processed locally in your browser.

Core Features

  • Multi-frequency pure tone testing: 125Hz to 16000Hz
  • Left/right ear testing: Test each ear separately or both
  • Audiogram chart: Real-time hearing curve visualization
  • Fine volume control: 0-80dB range
  • Result assessment: Automatic hearing level evaluation

How to Use

  1. Prepare: Put on headphones, find a quiet room
  2. Select ear and frequency: Choose left/right ear, start from 125Hz
  3. Play tones: Click play, mark Heard or Not Heard
  4. Adjust volume: Increase until you can hear, record minimum level
  5. View results: Check audiogram, normal hearing is 0-25dB

Use Cases

  • Routine self-check: Quick test when suspecting hearing decline
  • Headphone testing: Check frequency response
  • Occupational health: Regular monitoring for noise-exposed workers
  • Elderly care: Initial assessment for age-related hearing loss
  • Education: Learn about human hearing range

Background

Normal human hearing spans 20Hz-20000Hz, with high-frequency sensitivity declining with age. An audiogram plots frequency (log scale) vs hearing level (dB HL). Normal: 0-25dB; Mild: 26-40dB; Moderate: 41-55dB; Moderately severe: 56-70dB; Severe: 71-90dB; Profound: 90dB+.

FAQ

Is the online hearing test accurate?

This tool uses Web Audio API to generate precise pure tone signals for preliminary hearing assessment. Use headphones in a quiet environment. Results are for reference only.

Do I need headphones?

Strongly recommended. Headphones isolate ambient noise and allow testing each ear separately. Speaker testing is significantly less accurate.

What frequency range is covered?

Standard mode covers 125Hz to 8000Hz. Extended mode goes up to 16000Hz.

Is my data safe?

All processing runs locally in your browser. No audio data or results are uploaded to any server.

What causes hearing loss?

Common causes: aging, prolonged noise exposure, ear canal blockage, middle ear infections, certain medications.

How to read the audiogram?

Normal: 0-25dB; Mild loss: 26-40dB; Moderate: 41-55dB; Moderately severe: 56-70dB; Severe: 71-90dB; Profound: 90dB+.
