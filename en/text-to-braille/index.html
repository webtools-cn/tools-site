❓ FAQ

Is this tool free?

Yes, completely free. No registration or login required. All features run locally in your browser — no data is uploaded to any server.

What are Braille dot positions?

Braille uses a 3×2 matrix of 6 dots numbered 1-6 (left column: 1-2-3, right column: 4-5-6). Dot position notation like "1-3" means dots 1 and 3 are raised, corresponding to the Braille character ⠁. This tool shows both the Braille pattern and dot numbers.

Does it support Chinese Braille?

This tool supports English Grade 1 Braille (letter-level), including letters, numbers, and common punctuation. Chinese Braille uses a different mapping system (double pinyin scheme) and is not currently supported.

How are numbers represented in Braille?

Numbers are preceded by the number sign ⠼ (dots 3-4-5-6), followed by the corresponding letter positions: 1=⠁(a), 2=⠃(b), ..., 0=⠚(j). For example, "123" is written as ⠼⠁⠃⠉.

Is my data safe?

Yes. All processing happens locally in your browser. No data is sent to any server. The tool even works offline.

ℹ️ About Braille

Braille was invented by Louis Braille in France. It's a tactile writing system used by people who are visually impaired.

Each Braille character consists of 6 dots in a 3×2 matrix, giving 64 possible combinations (including the blank).

This tool uses Unicode Braille patterns (U+2800-U+28FF).
