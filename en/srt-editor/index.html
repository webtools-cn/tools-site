❓ FAQ

What is the SRT subtitle format?

SRT (SubRip Text) is the most common subtitle file format, consisting of sequence numbers, timestamps, and subtitle text. Almost all video players support SRT format, making it the standard for subtitle sharing.

How do I adjust subtitle timing?

Enter the offset in seconds in the "Time offset" field (negative to shift earlier, positive to shift later), then click "Adjust Timeline". For example, entering -2.5 will shift all subtitles 2.5 seconds earlier. Scale factor adjusts playback speed (1.0 = original).

What's the difference between SRT and VTT?

SRT is the traditional subtitle format with the widest compatibility across all players. VTT (WebVTT) is the web standard format designed for HTML5 video, supporting more styling options (font color, position). This tool supports editing SRT and exporting to VTT.

Is my subtitle file safe?

Completely safe. All editing is done locally in your browser. Your subtitle files are never uploaded to any server. Data is cleared when you close the page.

Can I batch edit subtitles?

Yes. You can batch adjust timeline offset and scale factor, and all subtitle entries will update simultaneously. Each subtitle can also be edited individually.
