❓ FAQ

How to use the image comparison slider?

① Click 'Before' to select the original image, 'After' to select the edited version ② Drag the circular handle on the image to reveal before/after sections ③ Default is horizontal comparison; switch to vertical for top/bottom ④ Click 'Reset' to return the slider to center ⑤ Click 'Swap' to exchange before and after images. Perfect for photo editing showcases, design comparisons, and restoration demonstrations.

What image formats are supported?

All browser-displayable formats: JPEG/JPG, PNG, GIF (first frame of animated), WebP, BMP, SVG (rasterized). There's no file size limit, but very large images (5000×5000+ pixels) may affect slider responsiveness — consider compressing first. All processing is local — no images are uploaded.

Why do my two images display at different sizes?

The tool auto-scales both images to the same display size (based on the 'After' image dimensions). If original dimensions differ, results might appear stretched. For best results, use images with identical dimensions. Use the Image Resizer tool on this site to standardize sizes first.

What are ideal use cases for the comparison slider?

① Photography — before/after editing ② Web design — UI redesign comparisons ③ Medical — treatment progress imaging ④ Beauty — makeup or skincare results ⑤ Fitness — body transformation progress ⑥ Product — before/after cleaning or restoration ⑦ Image editing — filter or adjustment comparisons.

Are my images uploaded to a server?

No. Everything runs locally in your browser. Both images are only used for on-screen display and comparison. No image data is uploaded to any server — all processing stays in your device's memory. Images are cleared immediately on page refresh.
