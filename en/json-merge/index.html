JSON Merge Tool

Free online JSON merge tool. Deep merge recursively merges nested objects, shallow merge only merges top-level keys, array concat merges arrays by concatenation. Real-time preview. Pure frontend.

Core Features

  • Deep Merge: Recursively merge nested objects, B overrides A
  • Shallow Merge: Only merge top-level properties
  • Array Concat: Concatenate arrays instead of overriding
  • Real-time Preview: Instant updates on change

FAQ

Deep vs Shallow merge?

Deep merge recursively merges nested objects. Shallow merge only merges top-level keys, nested objects are replaced by B.

What is array concat strategy?

When both values are arrays, they are concatenated instead of overridden.

Is my data safe?

Completely safe. All merging happens locally in your browser.
