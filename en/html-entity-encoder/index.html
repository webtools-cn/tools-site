📖 What Are HTML Entities?

HTML entities are encodings used to display special characters in HTML documents. Characters like <, >, and & have special meaning in HTML.

Common HTML Entities

  • &lt; - Less than <
  • &gt; - Greater than >
  • &amp; - Ampersand &
  • &quot; - Double quote "
  • &#39; - Single quote '
  • &copy; - Copyright ©

Encoding Types

  • Named entities: &lt; - Easy to remember
  • Decimal: &#38; - Unicode codepoint in decimal
  • Hexadecimal: &#x26; - Unicode codepoint in hex
