Frequently Asked Questions

What is an emboss effect?

Embossing creates a 3D relief effect by highlighting raised areas and shading recessed areas, simulating a stamped or carved metal look. It shifts and subtracts pixel values to create depth.

How does the light angle work?

The light angle determines which direction the virtual light source comes from. Different angles (0°, 45°, 90°, 135°, etc.) create different highlight and shadow patterns on the embossed surface.

Can I control how pronounced the effect is?

Yes! The "Depth" slider controls how strong the emboss effect is. Higher values create more dramatic relief, while lower values produce subtler texture.
