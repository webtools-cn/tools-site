About Binary Clock

Free online Binary Clock. Displays current time in binary format. 3 display modes: Standard BCD (6 columns), True Binary (pure binary for hours/minutes/seconds), and Hybrid mode. Each light represents a binary digit (on=1, off=0). Real-time updates, 12/24 hour toggle. Perfect for programmers, computer enthusiasts, geeks, and tech displays.

Display Guide

  • BCD Mode: 6 columns for each digit (H1 H2 M1 M2 S1 S2)
  • True Binary: Pure binary for hours, minutes, seconds
  • Hybrid: BCD for hours, binary for minutes and seconds
  • Lit = 1, Dim = 0. Reading top to bottom (highest to lowest bit)
