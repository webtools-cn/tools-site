JSON Path Finder

JSONPath is the standard expression language for querying JSON data, similar to XPath for XML. This tool lets you enter JSONPath expressions and see matching results in real-time, quickly locating and extracting specific data from JSON. All queries run locally in your browser—no data is uploaded.

JSONPath Syntax Reference

  • $ — Root node
  • . or [] — Child node
  • [] — Array index (0-based)
  • * — Wildcard, matches all elements
  • .. — Recursive descent
  • [start:end] — Array slice
  • [?()] — Filter expression
  • [(@.length-1)] — Last array element
What is JSONPath?
JSONPath is an expression language for querying JSON data, similar to XPath for XML. It uses dot and bracket notation to locate specific nodes in JSON, e.g., $.store.book[0].title.
What are common JSONPath syntax elements?
$ is the root node, . is child node, [] is array index or condition, * is wildcard, .. is recursive descent, [?()] is filter expression.
How to extract specific data from JSON?
Paste your JSON data, enter a JSONPath expression, and the tool shows matching results in real-time. You can also click nodes in the JSON tree to auto-generate paths.
Is my data uploaded to a server?
No. All queries run locally in your browser. No data is uploaded to any server.
