❓ FAQ

What is Mermaid?

Mermaid is a text-based diagramming language integrated in GitHub, GitLab, and Notion. Create diagrams using simple Markdown-like syntax.

How to use the editor?

Enter Mermaid syntax in the left panel, the diagram renders automatically on the right. Export as SVG or copy code.

Supported diagram types?

Flowchart, sequenceDiagram, gantt, classDiagram, pie, stateDiagram, erDiagram, gitGraph and more.

Can I use exported SVGs commercially?

Yes. Diagrams you create belong to you and can be used for any purpose.
