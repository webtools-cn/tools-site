❓ FAQ

How many images can I watermark at once?

Up to 50 images per batch. All processing is done locally on your device using Canvas API. For optimal performance, we recommend processing 10-20 images at a time. Maximum file size is 20MB per image. Supported input formats: JPEG, PNG, WebP. Output is always PNG (lossless).

Is this tool really private?

Yes. This is a 100% client-side tool. Your images are processed entirely in your browser using Canvas API. Nothing is uploaded to any server. This makes it suitable for watermarking confidential images, private photos, or unreleased product shots. No registration or login needed.

What's the best format for image watermark?

PNG with transparent background is ideal for logo watermarks. Create your watermark in design software (Photoshop, GIMP, Canva) as a transparent PNG, then upload it here. Recommended watermark size: 200-500px. White or light-colored marks work best. The opacity slider helps you achieve the perfect balance between visibility and subtlety.

How are output files named?

Exported files are automatically named as 'originalname_watermarked.png'. You can download all processed images at once by clicking 'Download All' — they will download sequentially. Note: your browser may prompt you to allow multiple downloads. Enable pop-ups for this page if downloads are blocked.
