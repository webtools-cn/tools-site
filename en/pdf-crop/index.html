📖 How to Use

PDF Crop Tool removes unwanted margins, white space, headers, and footers from your PDF pages. Upload a PDF, set crop values in pt (1pt ≈ 0.35mm), preview the crop area in real-time, then click Crop to process all pages.

⚡ How It Works

This tool uses PDF.js to render previews and pdf-lib to modify the MediaBox of PDF pages. Cropping adjusts the visible page boundary without re-encoding content, preserving original quality.

❓ FAQ

❓ Is PDF Crop tool free?
Yes, it's completely free to use. No registration or login required.
❓ Does PDF Crop upload my file to a server?
No. This tool runs entirely in your browser. Your PDF file never leaves your device. All processing is done locally.
❓ What PDF file size is supported?
Files up to 50MB are recommended. Larger files consume more memory and take longer to process.
❓ What unit is the crop value in?
Crop values are in pt (points). 1pt ≈ 0.35mm. You can set top, bottom, left, and right margins independently.
❓ Will cropping affect PDF quality?
No. Cropping only adjusts the visible area (MediaBox) of PDF pages. It does not re-encode PDF content, so original quality is preserved.
❓ Can I crop specific pages only?
The current version applies the same crop to all pages. For different crops on different pages, split the PDF first.
