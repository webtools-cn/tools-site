What is a Decision Matrix?

A decision matrix (also known as a weighted scoring matrix or Pugh matrix) is a structured multi-criteria decision analysis tool. It works by assigning weights to each evaluation criterion, scoring each option against those criteria, and calculating weighted totals to objectively compare different options.

How Does a Decision Matrix Work?

The core logic is: Final Score = Σ(Score × Weight). Steps:

  • Identify the options to compare (e.g., 3 job offers)
  • Define evaluation criteria (e.g., salary, commute, growth, culture)
  • Assign weights to each criterion (e.g., salary 30%, growth 30%, commute 20%, culture 20%)
  • Score each option on each criterion (1-5 scale)
  • Calculate weighted totals — the highest score is the recommended option

When to Use a Decision Matrix

A decision matrix is ideal whenever you need to choose between multiple options:

  • Career decisions: Compare job offers by salary, growth, commute, etc.
  • Product purchases: Compare products by performance, price, brand, etc.
  • Technology selection: Compare tech stacks by performance, cost, maintainability
  • Vendor evaluation: Compare suppliers by price, quality, delivery time
  • Project prioritization: Rank projects by impact, cost, risk

Advantages of a Decision Matrix

Compared to gut-feel decisions, a decision matrix offers: quantified subjective judgments for transparent, traceable decisions; weight mechanisms ensuring important factors have more influence; ability for multiple people to independently score and average, reducing individual bias; exportable results for team discussion.

What is a decision matrix?
A decision matrix (weighted scoring matrix or Pugh matrix) is a multi-criteria decision analysis tool. It assigns weights to evaluation criteria, scores each option against those criteria, and calculates weighted totals to objectively compare different options.
How do I set weights and scores?
Weights represent the importance of each criterion (should sum to 100%). Scores represent how well each option performs on that criterion (1-5 scale). Final score = sum of (score × weight) for all criteria.
When should I use a decision matrix?
Use a decision matrix when choosing between multiple options with multiple criteria: job offers, product purchases, project prioritization, technology selection, vendor evaluation, etc.
