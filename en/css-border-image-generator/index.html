📖 CSS border-image Explained

The border-image shorthand lets you use images or gradients to create custom element borders. It includes:

  • border-image-source — URL or gradient for the border
  • border-image-slice — How to slice the border image
  • border-image-width — The width of the border image
  • border-image-outset — How far the border extends
  • border-image-repeat — stretch, repeat, or round

Gradient Borders

The most common use of border-image is creating gradient borders. Use linear-gradient() as the source with proper slice values for smooth gradient borders.
