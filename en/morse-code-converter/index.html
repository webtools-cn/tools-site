💡 Did You Know?

Morse code was invented by Samuel Morse in the 1830s for telegraph communication. It uses dots (short signals) and dashes (long signals) to represent letters, numbers, and punctuation.

The SOS distress signal (... --- ...) is universally recognized and one of the most famous Morse code sequences.

❓ FAQ

Why are spaces needed between Morse characters?

Spaces separate individual characters in Morse code. Without them, "... --- ..." (SOS) would be indistinguishable from other combinations.

Can I play Morse code audio?

Yes! After converting text to Morse, click "Play Morse" to hear the code as audio tones. The pitch is set to 700Hz for clear listening.
