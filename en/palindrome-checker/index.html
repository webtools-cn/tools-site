📖 About Palindromes

Definition

A palindrome reads the same forward and backward, ignoring spaces, punctuation, and case

Examples

"racecar", "madam", "level", "radar", "civic", "kayak", "refer", "tenet"

Famous Phrases

"A man, a plan, a canal, Panama", "Never odd or even", "Was it a car or a cat I saw"

Applications

Coding interviews, string algorithms, DNA sequence analysis, data validation
