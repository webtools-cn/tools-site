📖 CSS scroll-behavior Explained

The scroll-behavior property controls the transition animation when scrolling within an element. With smooth, all anchor jumps and scrollIntoView() calls animate smoothly.

Related Properties

  • scroll-padding — defines inset padding for the scroll container
  • scroll-margin — defines individual element margin for anchor offset
  • scroll-snap-type / scroll-snap-align — scroll snapping for carousels
