❓ FAQ

How do I choose the right grid size?

8x8 for simple icons and emoji, 16x16 for classic game sprites (Minecraft-style), 32x32 for detailed characters, 64x64 for complex pixel art. Start with 16x16 if you're new to pixel art. You can always resize later, but note that changing grid size clears your current drawing.

What tools are available?

Pen: draw individual pixels with the selected color. Eraser: remove pixels (make transparent). Fill (Flood Fill): fill a contiguous area of the same color with the current color. Eyedropper: pick a color from existing pixels. The color palette offers 24 predefined colors plus a custom color picker.

Is there an undo feature?

This simple tool doesn't have a formal undo history. We recommend saving frequently by exporting PNGs at different stages. For professional pixel art with full undo support, consider dedicated tools like Aseprite or Piskel. This tool is designed for quick sketches, prototyping, and educational use.

Can I use the artwork commercially?

Yes! All pixel art you create with this tool is 100% your own intellectual property. You can use it in commercial games, websites, NFTs, merchandise, or any project with no restrictions. No attribution required. Since everything runs locally, no platform license applies to your work.
