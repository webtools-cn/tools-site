About Focus Timer

Free online Focus Timer based on Pomodoro Technique. Customizable focus and break durations (default: 25min focus / 5min short break / 15min long break). Built-in task list to track what you work on. Desktop notifications, real-time session counter, and completion history. Data saved locally in browser. Perfect for studying, working, coding.

How to Use

  • Set focus and break durations, click "Start Focus" to begin
  • Auto-switches between focus and break cycles
  • Every 4 focus sessions, you get a long break
  • Use the task list to track what you're working on
