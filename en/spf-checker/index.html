❓ FAQ

What is an SPF record?
SPF (Sender Policy Framework) record is a DNS TXT record that specifies which mail servers are authorized to send email on behalf of a domain, helping prevent email spoofing and spam.
How does SPF work?
When a receiving mail server gets an email, it queries the sender domain's SPF record to check if the sending server IP is in the allowed list. A match passes; a mismatch may result in rejection or spam marking.
What are common SPF issues?
Common issues: missing SPF record, DNS lookups exceeding the 10-query limit, using ~all instead of -all, not including all legitimate senders, having multiple SPF records.
What do SPF qualifiers mean?
+ means Pass (default), - means Fail (reject), ~ means SoftFail (mark but don't reject), ? means Neutral (no assertion). Using -all is recommended for best protection.
