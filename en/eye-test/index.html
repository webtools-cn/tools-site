How It Works

The Online Eye Test provides E-chart vision testing and color vision testing to help you check your vision at home.

E-Chart Test: Standard logarithmic vision chart with 10 levels from 4.0 to 5.2. Identify the direction of the E opening.

Color Vision Test: Based on the Ishihara plate principle, tests for red-green color vision deficiencies.

⚠️ Disclaimer: This test is for reference only and cannot replace professional eye exams. If you have vision problems, please visit an eye care professional.

FAQ

Is the online eye test accurate?

Online tests are affected by screen resolution, brightness, and viewing distance. Results are for reference only. Visit a professional for accurate results.

How far should I sit from the screen?

About 3 meters (10 feet), consistent with standard vision chart testing distance.

Can the color test detect all color vision problems?

This test mainly detects red-green color vision deficiencies. For comprehensive testing, visit a professional eye care provider.

Can children use this test?

Yes, with parental assistance. Recommended for children 3 years and older. Parents can help click the direction buttons.
