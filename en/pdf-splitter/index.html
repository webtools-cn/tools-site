How to Use

This PDF splitter is straightforward and packed with features. Here is a step-by-step guide:

Step 1: Upload Your PDF

Click the dashed box in the center of the page to select a PDF file from your computer, or drag and drop a PDF directly into the box. The file is loaded locally in your browser—it is never uploaded to any server, ensuring complete privacy for your documents.

Step 2: Choose a Splitting Mode

The tool offers four splitting modes to match your needs: Page Range lets you input specific page numbers (e.g. "1-5, 8, 10-15") to extract only those pages; Single Pages displays all pages as thumbnails so you can cherry-pick individual pages; Odd/Even Separation instantly creates two PDFs—one with odd pages and one with even pages—ideal for double-sided printing or distribution; Every N Pages automatically splits the document into chunks of a fixed size, useful for dividing a long report into equal sections.

Step 3: Split and Download

Click the "Start Splitting" button and the tool will process your PDF using pdf-lib.js entirely within your browser. Once complete, a results list appears with a download link for each new file. Click any link to save it to your computer. If multiple files were created, you can download them individually.

Important Notes

Page content such as text, images, and vector graphics is preserved. However, document-level elements like bookmarks, internal links, and form data may not be retained because they are tied to the overall document structure. Password-protected PDFs must be decrypted before splitting. For very large PDFs over 500 pages, processing may take a moment—please be patient.

Use Cases

PDF splitting is widely used in offices, schools, and by individuals. Here are some typical scenarios:

Contracts and Legal Documents

Lawyers and legal professionals often need to split a single PDF containing multiple contracts into separate files for individual archiving, client distribution, or court submission. The "Page Range" mode lets you precisely extract each contract into its own file.

Research Papers and Reports

Researchers may need to extract the experimental data section from a long paper to share with collaborators, or split a report into cover, table of contents, body, and appendix for different reviewers. The single-page selection and range modes handle these needs with precision.

Scanned Document Organization

When a scanner combines multiple documents into one PDF, the "Every N Pages" or "Odd/Even" modes can quickly separate them back into individual files. This is especially useful after batch-scanning invoices, receipts, or forms.

E-books and Course Materials

Students and educators can split an e-book by chapter using page ranges, making it easier to read on tablets or e-readers. By consulting the table of contents for page numbers, entire chapters can be extracted in seconds.

Knowledge Base

How PDF Files Work

A PDF (Portable Document Format) is built on the PostScript language and consists of four main parts: a header, a body (containing all objects such as pages, fonts, and images), a cross-reference table (for fast object lookup), and a trailer. Each page is essentially an independent content stream, making page-level extraction theoretically straightforward. However, embedded resources and document-level structures (bookmarks, forms, JavaScript) can complicate the process.

How pdf-lib.js Works

This tool uses pdf-lib.js, a powerful JavaScript library that can create and modify PDFs in both browsers and Node.js. It parses the internal object structure of a PDF, copies the selected pages along with their dependencies (fonts, images, etc.), and assembles them into a new document. Unlike server-side tools, pdf-lib.js runs entirely in your browser, so your data never leaves your device.

PDF Splitting vs PDF Compression

PDF splitting and PDF compression are two different operations. Splitting divides a PDF into multiple files based on page selection, while compression reduces the file size of a single PDF by optimizing images, removing redundancy, and merging duplicate fonts. This tool handles splitting only; for compression, use our dedicated PDF compression tool.

PDF/A and Long-Term Archiving

PDF/A is an ISO-standardized subset of PDF designed for long-term archiving. It prohibits features that are unsuitable for preservation, such as JavaScript, external references, and encryption. If your PDF is in PDF/A format, splitting it generally does not affect compliance, but you may want to verify with a validation tool afterward.

Frequently Asked Questions

Does splitting a PDF change its file size?

Split PDFs are typically smaller than the original because they contain only the selected pages. However, the actual size depends on the complexity of the page content. This tool uses pdf-lib.js for lossless splitting and does not compress or reduce page quality.

Does it support password-protected PDFs?

If a PDF has an open password (user password), it must be decrypted first before splitting. This tool does not support splitting password-protected PDFs. Please remove the password protection using another tool first.

Will bookmarks and links be preserved?

This tool uses pdf-lib.js for page-level copying, so page content (text, images, vector graphics) is preserved. However, bookmarks (outline), internal links, and form data may not be retained because these are tied to the overall document structure.

What is the maximum number of pages supported?

There is no strict page limit, but actual performance depends on your device's memory and browser capabilities. For very large PDFs over 1000 pages, consider splitting in batches or using desktop software. Most office PDFs (a few hundred pages or less) are handled smoothly.

Is my data uploaded to a server?

No. All processing is done locally in your browser. PDF files are never uploaded to any server. Your document privacy is fully under your control, making this tool suitable for handling sensitive documents.

Why does text look strange after splitting some PDFs?

This usually happens with PDFs that use font subsetting. When only some pages are extracted, the complete font information may not be present in the selected pages, causing text rendering issues. If you encounter this, consider using professional desktop software like Adobe Acrobat.

Can I process multiple PDFs at once?

The current version processes one PDF at a time. You can upload and split multiple PDFs sequentially. If you need to apply the same splitting rules across many files, consider using a batch desktop tool or our command-line version (planned for a future release).

Can I reorder pages during splitting?

The current version preserves the original page order. If you need to reorder pages, use our "PDF Merger" tool to rearrange pages in the desired sequence before or after splitting.
