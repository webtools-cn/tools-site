❓ FAQ
Is this dice roller suitable for D&D?
Yes! Supports D4, D6, D8, D10, D12, D20 and more. Customize dice count and modifiers (e.g., 2d20+5). Perfect for D&D and other TRPG games. Track your roll history for easy reference.
Are the dice rolls truly random?
Uses JavaScript crypto.getRandomValues() for cryptographically secure random numbers — more reliable than Math.random(). Each roll is independent and fair, with no pseudo-random bias.
Can I customize dice?
Yes! Beyond presets (D4/D6/D8/D10/D12/D20/D100), you can input any number of sides (up to 1000) and add modifiers (e.g., +3, -2).
Can I roll multiple dice at once?
Yes! Set the dice count from 1-100 (e.g., 3d6), showing individual values and total sum.
