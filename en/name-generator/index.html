FAQ

Are generated names unique?

Names are generated randomly by combining name fragments from curated pools. While most combinations are unique, duplicates can occur. For creative writing or D&D campaigns, we recommend generating multiple options and picking your favorite.

Can I use these names commercially?

Yes. Generated names are algorithmically created and not subject to copyright. However, avoid names that match existing brands, trademarks, or well-known public figures. When in doubt, a quick search is recommended.

What name styles are available?

15+ styles including: Fantasy Elven (Sindarin-inspired), Fantasy Dwarven, Fantasy Orcish, Fantasy Tauren, English First+Last, Japanese, Chinese, Sci-Fi, Username generators, Nicknames, Pet names, and more. Each style uses custom naming rules and name pools.

Is there a limit on generation?

You can generate 1-50 names per batch. There's no daily limit. Each name can be copied individually with one click.

How does the gender filter work?

For styles that support gender (English, Chinese, Japanese), you can select Male, Female, or Any. The algorithm uses gender-specific name pools accordingly.
