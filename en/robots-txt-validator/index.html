📖 Robots.txt Specification

• User-agent: Specifies which crawler the rules apply to (* = all crawlers)

• Disallow: Paths the crawler should not access

• Allow: Paths the crawler can access (can override Disallow)

• Sitemap: Declares the sitemap URL for crawlers

• Crawl-delay: Sets the request interval in seconds

• File location: Should be at the root of the website (/robots.txt)
