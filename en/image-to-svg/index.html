FAQ


How does image to SVG conversion work?
This tool quantizes the image to a specified number of colors, then converts each color region into SVG paths. Pixel mode converts each pixel to an SVG rectangle.


What types of images work best?
Best for logos, icons, and illustrations with few colors. For photos, use fewer colors and higher simplification.


Is my data safe?
Absolutely. All processing happens locally in your browser. No data is ever sent to any server.
