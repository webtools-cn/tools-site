Kanban Board - Drag-and-Drop Task Management

Free online Kanban Board with drag-and-drop interaction, supporting multiple columns, labels, and priorities. All data auto-saved in browser localStorage. No registration, no server upload, works offline.

FAQ

What is a Kanban board?▼
A visual task management tool using drag-and-drop to move task cards between columns, showing progress at a glance. Originated from Toyota's production system, widely used in agile project management.
Is my data saved?▼
Yes. Auto-saved in browser localStorage. Persists across reloads. Export backups regularly since clearing cache erases data.
Can I create multiple boards?▼
Currently single board, but customizable columns. Use different tags to distinguish projects.
Is data export supported?▼
Yes. Export to JSON for backup, import from JSON to restore.
Is data uploaded?▼
No. All data stored locally. Works offline.
