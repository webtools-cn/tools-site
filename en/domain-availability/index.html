❓ FAQ

How do you check if a domain is available?
We check domain availability by querying DNS NS records. If a domain has no NS records, it typically means the domain is not registered and may be available for purchase. Final availability should be confirmed with a registrar.
What TLDs are supported?
We support common TLDs including .com, .net, .org, .cn, .io, .co, .dev, .app, .me, .info, .biz, .xyz, .cc, .tv, .us, .uk, .de, .fr, .jp, .tech, .online, .site, .store and more.
Are the results 100% accurate?
DNS queries provide high-accuracy reference results but are not 100% certain. Some domains may be registered without DNS configured, or be in a grace period. Confirm with a registrar for final availability.
What if the query is slow?
DNS queries depend on public DNS server response times. Some less common TLDs may have slower DNS servers. The tool automatically switches to backup DNS servers, please be patient.
