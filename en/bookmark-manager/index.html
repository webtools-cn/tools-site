How to Use

添加书签：输入Title和URL地址，选择Category后点击添加。所有数据Save在浏览器本地存储（localStorage）中，不会上传到服务器。

管理与搜索：可按Category筛选或在搜索框中输入关键词实时搜索。点击书签Title直接打开链接，点击「Edit」修改书签信息，点击「Delete」移除书签。

Backup: Click Export to download a JSON backup. Click Import to restore from a previously exported JSON file. Imports are merged with existing data.

Privacy: All data is stored only in your browser. Data persists after closing the page. Clearing browser cache or using incognito mode may cause data loss.
