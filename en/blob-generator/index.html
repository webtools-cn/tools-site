❓ FAQ

What is an SVG Blob?
An SVG Blob is an organic, irregular shape used in web design as decorative elements, background graphics, or illustrations. Made from Bézier curves, they look natural and fluid without sharp edges.
How can I use the generated blobs?
Copy the SVG code directly into HTML, or download PNG for design software. Blobs are great for: web backgrounds, hero section graphics, card decorations, illustration elements, and logo design.
What does the complexity parameter control?
Complexity controls edge detail. Low (3-4) creates smooth, simple shapes. High (8-12) creates more undulating, detailed shapes. 5-7 is usually the sweet spot.
Can I use the generated SVG code in my project?
Yes. The output is standard SVG that can be embedded in HTML or saved as a standalone SVG file. The code is clean with a single path element and minimal file size.
Is every generated blob unique?
Yes. Each click produces a different random blob. If you find a shape you like, copy or download it immediately — the same random result can't be precisely reproduced.
