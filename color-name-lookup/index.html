📋 CSS命名颜色列表 (0种)

颜色名称查询工具能做什么？

免费在线颜色名称查询工具，包含140+种CSS标准命名颜色。支持按名称搜索、按HEX值查找最接近的命名颜色、颜色选择器实时预览。所有查询在浏览器本地完成，不上传数据。

核心功能

  • 名称搜索：输入颜色名称模糊搜索，实时过滤匹配结果
  • HEX查找：输入HEX值，自动查找最接近的CSS命名颜色
  • 颜色选择器：使用系统颜色选择器选取颜色并查找名称
  • 色差计算：使用CIE76算法计算感知色差，确保匹配准确
  • 一键复制：复制颜色名称、HEX值或RGB值

使用教程

  1. 在搜索框输入颜色名称（如"coral）或HEX值（#FF7F50）
  2. 或使用颜色选择器选取颜色
  3. 点击"查找最接近"获取匹配的命名颜色
  4. 点击颜色卡片复制颜色值

应用场景

场景1：前端开发

在CSS中使用语义化的颜色名称替代HEX值，提高代码可读性。

场景2：设计配色

快速查找设计稿中颜色对应的命名颜色，便于团队沟通。

场景3：学习参考

浏览和学习CSS命名颜色体系，了解颜色命名规律。

扩展知识

CSS命名颜色由W3C定义，最初16种基本颜色（HTML 4.01），后扩展至140+种（CSS3）。这些名称源自X11窗口系统的颜色定义，如AliceBlue、CornflowerBlue等。使用命名颜色比HEX值更语义化，但精确设计仍需HEX/RGB值。
