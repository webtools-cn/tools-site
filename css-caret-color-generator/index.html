📖 CSS caret-color 详解

caret-color 属性用于控制输入框、文本域或可编辑元素中文本插入光标（caret）的颜色。这是一个纯视觉属性，不影响功能。

使用场景

  • 品牌色统一 — 让光标颜色与品牌色一致
  • 暗色模式 — 在深色背景上使用亮色光标更容易定位
  • 错误状态 — 表单验证失败时将光标变为红色
  • 隐藏光标 — 使用 transparent 在某些 UI 模式中隐藏光标

浏览器兼容性

所有现代浏览器均已支持 caret-color 属性，无需前缀。
