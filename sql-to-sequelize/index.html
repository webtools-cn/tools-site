SQL转Sequelize模型工具能做什么？

SQL转Sequelize模型工具是一款免费在线开发者工具，能够将SQL CREATE TABLE语句一键转换为Sequelize Model定义代码。自动识别SQL数据类型并映射为Sequelize数据类型，支持MySQL、PostgreSQL、SQLite三种方言，解析主键、外键、默认值、NOT NULL等约束，生成符合Sequelize规范的Model代码。适用于Node.js开发者从现有数据库快速生成ORM模型，大幅减少手动编写Model的工作量。所有转换在浏览器本地完成，数据绝不上传服务器。

核心功能

  • 智能类型映射：自动将SQL数据类型（INT、VARCHAR、TEXT、DATETIME等）映射为Sequelize类型（INTEGER、STRING、TEXT、DATE等）
  • 多方言支持：支持MySQL、PostgreSQL、SQLite三种SQL方言，自动适配类型差异
  • 约束解析：解析PRIMARY KEY、FOREIGN KEY、NOT NULL、DEFAULT、UNIQUE、AUTO_INCREMENT等约束
  • 关联生成：外键约束自动生成belongsTo/hasMany关联代码
  • 历史记录：自动保存最近5次转换记录，方便回溯和复用

扩展知识

Sequelize是Node.js最流行的ORM框架之一，支持PostgreSQL、MySQL、SQLite和MSSQL。Sequelize Model定义了数据库表的结构映射，包括字段类型、约束、索引和关联。SQL到Sequelize的类型映射遵循一定规则：INT→INTEGER，VARCHAR(n)→STRING(n)，TEXT→TEXT，BOOLEAN→BOOLEAN，DATETIME→DATE，DECIMAL(p,s)→DECIMAL(p,s)。Sequelize的timestamps选项默认添加createdAt和updatedAt字段，paranoid选项添加deletedAt实现软删除。freezeTableName选项阻止Sequelize自动将模型名复数化作为表名。
