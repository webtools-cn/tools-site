扩展知识

CSS变形动画（Morph Animation）通过@keyframes定义关键帧，在关键帧之间平滑过渡CSS属性值。常见的变形动画利用border-radius属性改变元素形状（如圆形到方形到椭圆），结合transform的rotate、scale和translate实现旋转、缩放和位移，再配合background-color或gradient实现颜色变化。现代CSS还支持CSS Houdini的@property注册自定义属性，实现更复杂的渐变动画。本工具生成的代码使用标准CSS语法，兼容Chrome、Firefox、Safari、Edge等主流浏览器。
