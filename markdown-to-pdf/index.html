关于 Markdown 转 PDF

本工具将 Markdown 格式文本实时渲染为 HTML 预览，并支持通过浏览器原生打印功能导出为 PDF 文件。

支持的 Markdown 语法：

• 标题 (H1-H6) · 粗体/斜体/删除线 · 行内代码 / 代码块 · 有序/无序列表 · 表格 · 引用 · 链接 · 图片 · 水平线 · HTML标签

使用方法：编写或粘贴 Markdown → 实时预览 → 点击"打印为PDF" → 在弹出的打印对话框中选择"另存为PDF" → 保存。

所有处理在浏览器本地完成，文档内容不会上传到任何服务器。

如何使用Markdown转PDF

使用Markdown转PDF进行转换只需三个步骤：

  1. 输入要转换的值
    在输入区域输入或粘贴要转换的数据。Markdown转PDF支持多种输入格式——输入数据后工具会自动识别。
  2. 选择源格式和目标格式
    从可用选项中选择源格式和目标格式。工具支持双向转换（如适用）。设置会即时生效。
  3. 获取转换结果
    转换结果实时显示。点击"复制"复制到剪贴板，或点击"下载"保存。所有转换都在您的设备本地完成。

常见问题

文档内容会上传到服务器吗？

不会。所有 Markdown 到 HTML 的转换和 PDF 生成完全在您的浏览器中完成，文档内容不会离开您的设备。

为什么点击打印后保存的是网页而不是PDF？

在打印对话框的"目标"或"打印机"选项中，选择"另存为PDF"（Save as PDF）即可保存为 PDF 文件，而不是选择实体打印机。

支持图片和表格吗？

是的，支持 Markdown 标准的表格、图片、代码块、引用等所有常用语法。图片需要是网络可访问的URL或base64编码。
