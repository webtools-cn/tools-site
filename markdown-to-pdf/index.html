常见问题

文档内容会上传到服务器吗？

不会。所有 Markdown 到 HTML 的转换和 PDF 生成完全在您的浏览器中完成，文档内容不会离开您的设备。

为什么点击打印后保存的是网页而不是PDF？

在打印对话框的"目标"或"打印机"选项中，选择"另存为PDF"（Save as PDF）即可保存为 PDF 文件，而不是选择实体打印机。

支持图片和表格吗？

是的，支持 Markdown 标准的表格、图片、代码块、引用等所有常用语法。图片需要是网络可访问的URL或base64编码。
