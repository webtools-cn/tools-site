使用教程

本摩斯密码转换器操作简单且功能完善，以下是详细的使用指南：

文本转摩斯电码：在左侧输入框中输入要转换的文本内容（支持英文字母A-Z、数字0-9及常见标点符号），点击「编码 → 摩斯电码」按钮，工具会立即将文本转换为对应的摩斯电码。每个字母的编码之间用空格分隔，单词之间用斜杠（/）分隔。例如输入 "HELLO" 会得到 "···· / · / ·−·· / ·−·· / −−−"（开启「显示时长标记」后）。

摩斯电码转文本：将摩斯电码粘贴到输入框，使用点（· 或 .）表示短信号，划（− 或 -）表示长信号，字母之间用空格分隔。点击「解码 ← 摩斯电码」按钮，工具会自动识别并转换回原始文本。工具同时兼容使用句点（.）和连字符（-）的输入方式。

声音播放功能：转换完成后，点击「播放摩斯电码」按钮，工具会使用Web Audio API生成标准的摩斯电码声音。你可以通过滑块调节播放速度（WPM，每分钟字数）和音调频率（Hz）。速度范围5-40 WPM，新手建议从10-15 WPM开始练习；频率范围400-1200 Hz，700Hz是业余无线电中最常用的标准频率。

闪烁演示：播放声音的同时，页面中央的圆形指示灯会同步闪烁——黄色表示有声音信号（点或划），灰色表示静默。这模拟了实际使用手电筒或信号灯发送摩斯密码的场景，帮助学习者建立视觉与听觉的对应关系。底部的蓝色进度条也会同步显示信号时长。

快捷操作：「交换」按钮可快速互换输入和结果内容；「清空」按钮一键清除所有内容；「复制结果」将结果复制到剪贴板；「结果填入输入」则将结果移至输入框以便进一步处理。

扩展知识

摩斯密码的历史起源：摩斯密码由美国画家兼发明家塞缪尔·摩尔斯（Samuel Morse）和他的助手阿尔弗雷德·维尔（Alfred Vail）于1836年至1844年间发明。1844年5月24日，第一条摩斯电报从华盛顿发到了巴尔的摩，内容是圣经名句「What hath God wrought」。此后近一个半世纪，摩斯密码成为全球电报通信的标准语言。

国际摩斯密码标准（ITU-R M.1677）：目前通用的摩斯密码由国际电信联盟（ITU）制定标准。每个字母的编码长度经过精心设计，常用字母（如E、T、A、I、N）使用更短的编码（1-2个符号），不常用字母（如Q、Z、X）使用更长的编码（4个符号），这是一种早期的「哈夫曼编码」思想，旨在减少平均传输长度。

摩斯密码的编码规则：一个「点」的时长定义为基本时间单位（通常为0.05-0.1秒），一个「划」的时长等于3个点。点与划之间的间隔为1个点时长，同一字母内各符号间隔为1个点，字母之间间隔为3个点（即1个划时长），单词之间间隔为7个点。这些严格的时序规则确保了接收方能够正确区分字母边界。

摩斯密码与现代通信的关系：虽然数字通信已经完全取代了摩斯密码在民用领域的地位，但它在某些专业领域仍有生命力。全球业余无线电社区每年举办的「CW竞赛」吸引了数万名参赛者；美国海军直到2015年才正式停止使用闪光灯摩斯密码；一些应急信标（EPIRB）仍在使用摩斯编码发送识别信息。摩斯密码也是学习数字信号处理和通信原理的理想入门教材。

常见问题（FAQ）

摩斯密码是谁发明的？

摩斯密码由美国人塞缪尔·摩尔斯（Samuel Morse）和阿尔弗雷德·维尔（Alfred Vail）在1830年代发明，最初用于电报通信。它通过短信号（点·）和长信号（划－）的组合来表示字母、数字和标点符号。

摩斯密码现在还实用吗？

尽管现代通信技术已经非常发达，摩斯密码在特定场景下仍有实用价值：无线电通信中的呼号识别、应急求救信号（SOS）、军事和航空领域的备用通信方式、以及户外探险中的视觉/声音信号传递。此外，学习摩斯密码有助于理解通信编码的基本原理。

国际摩斯密码和美式摩斯密码有什么区别？

国际摩斯密码（ITU标准）是目前通用的标准，被大多数国家和组织采用。美式摩斯密码（又称陆地线电报码）是早期版本，部分字符编码与国际版不同。本工具使用国际摩斯密码标准（ITU-R M.1677-1），确保与全球无线电操作员的互通性。

SOS 在摩斯密码中是什么意思？

SOS 是国际通用的求救信号，在摩斯密码中表示为 ··· －－－ ···（三短、三长、三短）。需要注意的是，SOS 并不是任何单词的缩写，而是因为这个组合在摩斯密码中容易识别和发送，因此被国际无线电公约采纳为求救信号。

如何学习摩斯密码？

学习摩斯密码建议采用以下步骤：1）先熟记字母表对应的编码（可从常用字母E、T、A、N等开始）；2）使用本工具的声音播放功能训练听觉识别能力；3）每天练习15-20分钟，逐步提升解码速度；4）尝试用手电筒或敲击方式发送简单单词。持之以恒，普通人通常2-4周可以达到每分钟10-15字的收发速度。

本工具支持中文吗？

本工具支持英文字母、数字和常见标点符号的摩斯密码转换。中文由于字符数量庞大，没有标准的摩斯密码编码表，因此不支持中文直接转换。如需转换中文内容，建议先使用拼音或英文翻译后再进行摩斯编码。

什么是WPM？如何设置合适的速度？

WPM（Words Per Minute）表示每分钟传输的单词数，是衡量摩斯密码收发速度的标准单位。基于「PARIS」这个标准测试词（包含5个字母和一个空格，共50个时间单位），20 WPM意味着每分钟发送20组「PARIS」。初学者建议从5-10 WPM开始，熟练后可以达到20-30 WPM，专业无线电操作员可达40 WPM以上。
