什么是System Prompt？

System Prompt（系统提示词）是在与AI对话开始前发送的指令，用于设定AI的角色、行为方式和输出格式。与普通用户消息不同，System Prompt对AI的行为有更强的约束力，是提示词工程（Prompt Engineering）的核心技术之一。

  • 设计创意：UI设计建议、品牌命名、创意brainstorm等

如何编写高质量的System Prompt

编写高质量System Prompt的关键要素：

  • 明确角色定位：告诉AI它是谁，有什么专业背景
  • 指定输出格式：明确要求Markdown、JSON、表格等格式
  • 设定行为边界：规定AI应该做什么、不应该做什么
  • 提供示例：用few-shot示例展示期望的输出
  • 使用条件指令：根据不同情况给出不同的处理方式
