扩展知识

MCP（Model Context Protocol）是Anthropic于2024年底发布的开放标准，定义了AI模型与外部工具的通信协议。MCP服务器提供工具（Tools）、资源（Resources）和提示（Prompts）三种能力。通过标准化接口，AI应用可以连接任意MCP服务器获取能力。Claude Desktop已原生支持MCP，社区已开发数百个MCP服务器。

常见问题

什么是MCP？

MCP（Model Context Protocol）是Anthropic发布的开放标准，定义AI模型与外部工具的通信协议。类似于USB-C统一了设备连接，MCP统一了AI工具接口。

生成的代码可以直接运行吗？

生成的代码是完整的服务器骨架，包含工具注册和基本框架。安装依赖后即可运行，然后根据需要添加具体工具逻辑。

支持哪些编程语言？

目前支持TypeScript（使用@modelcontextprotocol/sdk）和Python（使用mcp包）。

如何添加自定义工具？

在工具定义区域添加工具名称、描述和参数Schema，生成器会自动在代码中注册该工具。

数据会上传服务器吗？

不会。所有代码生成完全在浏览器本地完成。
