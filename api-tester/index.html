常见问题

API测试工具支持哪些HTTP方法？

支持全部常用HTTP方法：GET、POST、PUT、PATCH、DELETE、HEAD、OPTIONS。不同方法自动显示或隐藏请求体输入区。

请求会被记录或保存吗？

不会。请求直接从你的浏览器发送到目标API，本工具不会记录、保存或转发任何请求数据。所有处理在浏览器本地完成。

为什么有些接口请求失败了？

可能是CORS限制：如果目标API没有设置允许跨域请求的CORS头（Access-Control-Allow-Origin），浏览器会阻止请求。你可以尝试使用支持CORS的API或在服务端配置代理。

支持HTTPS吗？

支持。你可以在URL中使用http://或https://协议。建议始终使用HTTPS以确保数据传输加密。

可以上传文件吗？

当前版本不支持文件上传。如果需要测试文件上传API，建议使用Postman或curl等专业工具。

技术说明

本工具使用现代Web技术在浏览器中100%本地运行。数据永远不会发送到任何服务器。
