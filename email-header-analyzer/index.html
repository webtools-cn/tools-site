📖 常见问题

什么是邮件头?

邮件头是电子邮件的元数据，包含发件人、收件人、发送路径、时间戳、认证结果等信息。每封邮件都附带完整的头部信息，可以用来追踪邮件的真实来源和传输过程。

如何获取邮件的原始头部?

Gmail: 打开邮件 → 点击右上角三个点 → 显示原始邮件 → 复制头部内容。Outlook: 打开邮件 → 文件 → 属性 → Internet邮件头。Apple Mail: 查看 → 邮件 → 原始内容。

SPF/DKIM/DMARC是什么?

SPF(发件人策略框架)验证发件服务器是否被授权；DKIM(域名密钥识别邮件)验证邮件是否被篡改；DMARC基于SPF和DKIM结果制定处理策略。三者配合防止邮件伪造和钓鱼。

邮件数据会上传到服务器吗?

不会。所有邮件头解析都在浏览器本地完成，不发送任何数据到服务器。您的邮件隐私完全受保护。

如何判断邮件是否被伪造?

检查以下指标：1)SPF是否通过(pass)；2)DKIM是否通过(pass)；3)DMARC是否通过(pass)；4)发件人域名与Reply-To是否一致；5)Received头中的IP地址是否与发件域名匹配。如果多项不通过，邮件可能被伪造。
