使用说明

CSS color-mix() 函数混合两种颜色，指定色彩空间和比例。
语法： color-mix(in <colorspace>, <color1> <percentage>, <color2> <percentage>)
色彩空间： sRGB（标准）、OKLCH（感知均匀）、HSL（色相饱和）、Lab（感知）等
提示： 颜色比例总和不必等于100%，浏览器会自动归一化。
