💡 了解CSS Reset

什么是CSS Reset？

一组CSS规则，用于消除浏览器默认样式的差异，确保跨浏览器一致性

Normalize vs Hard Reset

Normalize保留有用默认值，Hard Reset全部归零。推荐团队项目用Normalize

现代最佳实践

box-sizing:border-box + 边距重置 是现代CSS开发的标配

知名方案

Eric Meyer Reset、Normalize.css、sanitize.css 是业界知名的CSS重置方案
