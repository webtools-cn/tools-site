JSON路径提取器能做什么？

JSON路径提取器是一款纯前端在线工具，用于通过JSONPath表达式从JSON数据中快速提取指定内容。支持标准JSONPath语法，包括子节点选择、数组索引、通配符、递归下降和过滤表达式。输入JSON数据和路径表达式即可实时预览提取结果，适合API调试、数据清洗和JSON数据分析等场景。所有数据在浏览器本地处理，绝不上传服务器。

核心功能

  • 标准JSONPath语法：支持$、.、[]、[*]、..、[?()]等完整JSONPath语法
  • 实时预览：输入路径表达式即时显示匹配结果，无需手动点击
  • 过滤表达式：使用[?(@.field==value)]条件筛选，支持比较运算符
  • 快捷路径按钮：常用语法一键插入，降低学习成本
  • 自动格式化：一键美化JSON，错误时提示行号列号

使用教程

步骤1：在输入框粘贴JSON数据，或拖拽.json文件上传。点击"加载示例"快速体验。

步骤2：在路径输入框输入JSONPath表达式，如$.users[*].name提取所有用户名。

步骤3：点击"提取"或按Ctrl+Enter，查看匹配结果。使用快捷路径按钮辅助语法。

步骤4：复制或下载提取结果。Ctrl+Shift+C一键复制。

扩展知识

JSONPath是JSON的查询语言，类似XML的XPath。核心语法：$表示根节点，.表示子节点，[]表示数组索引，[*]选择所有元素，..表示递归下降所有层级，[?(expr)]为过滤表达式。过滤表达式中@表示当前节点，支持==、!=、>、<等比较运算符。JSONPath已被多个编程语言实现，如JavaScript的jsonpath-plus、Python的jsonpath-ng。

常见问题 (FAQ)

支持哪些JSONPath语法？

支持$根节点、.子节点、[]数组索引、[*]所有元素、..递归下降、[?()]过滤表达式、[start:end]切片等标准JSONPath语法。

数据会上传到服务器吗？

不会。所有JSON解析和路径查询均在浏览器本地完成，数据绝不上传。

支持多大的JSON文件？

建议不超过5MB，超大JSON文件可能导致浏览器卡顿。支持拖拽上传文件。

过滤表达式怎么写？

使用[?(@.field==value)]语法，如$.users[?(@.age>18)]筛选年龄大于18的用户。支持==、!=、>、<、>=、<=比较运算符。

能自动检测JSON格式吗？

能。工具会自动检测并格式化输入的JSON，格式错误时给出具体行号提示。
