❓ 常见问题

HTML Diff和普通文本Diff有什么不同？

HTML Diff专门针对HTML结构优化，不仅比较文本差异，还关注标签结构的变更。本工具基于行级别的LCS（最长公共子序列）算法实现差异检测，适合比较HTML代码片段、模板文件和网页结构的变更。

比较结果如何解读？

结果以并排方式显示：左侧为原始版本，右侧为新版本。绿色行（+号）表示新增内容，红色行（-号）表示删除内容，黄色行表示修改内容。每行前显示行号便于定位。

代码数据安全吗？

绝对安全。所有比较在浏览器本地完成，代码数据不上传任何服务器。你可以放心比较包含敏感信息的代码。

支持中文字符差异比较吗？

完全支持。基于Unicode的字符串比较算法可以正确处理中文、日文、韩文等多字节字符。
