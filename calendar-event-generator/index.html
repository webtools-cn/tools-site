日历事件 使用指南

什么是ICS文件？

ICS（iCalendar）是一种通用的日历文件格式（RFC 5545），被Google Calendar、Apple Calendar（iCal）、Outlook、Yahoo Calendar等主流日历应用广泛支持。ICS文件包含事件的标题、开始时间、结束时间、地点、描述等信息。

什么是Google日历链接？

Google日历链接是一个特殊的URL地址，点击后可以直接在Google Calendar中创建一个新日程，并自动填充事件详细信息。适合在网站、邮件或社交媒体中分享。

使用场景

活动报名确认：在用户报名活动后，提供「添加到日历」链接或ICS文件下载，方便用户标记日程，提高活动出席率。

会议邀请：在团队协作中生成会议事件，通过邮件或聊天工具分享，确保所有参与者在正确的时间参加。

网站事件：在产品发布、网络研讨会、促销活动等页面添加「添加到日历」按钮，让用户一键将活动加入自己的日历。

票务确认：在购票确认邮件中包含日历链接，方便用户记住演唱会、电影、展览等时间。

ICS文件格式说明

ICS文件基于iCalendar标准（RFC 5545），包含以下关键字段：

BEGIN:VEVENT — 开始事件定义

DTSTART — 事件开始时间（UTC格式：YYYYMMDDTHHMMSSZ）

DTEND — 事件结束时间

SUMMARY — 事件标题

DESCRIPTION — 事件描述

LOCATION — 事件地点

END:VEVENT — 结束事件定义

什么是ICS文件？

ICS（iCalendar）是一种通用的日历文件格式，可以被Google Calendar、Apple Calendar、Outlook等主流日历应用打开。ICS文件包含事件的标题、时间、地点、描述等信息。

什么是Google日历链接？

Google日历链接是一个特殊URL，点击后可以直接在Google Calendar中创建一个新的日程事件，并自动填充标题、时间、描述等信息。

ICS文件和Google日历链接有什么区别？

ICS文件是一个可下载的文件，可以导入到任何支持iCalendar格式的日历应用。Google日历链接则直接在浏览器中打开Google Calendar并自动填充事件信息。

如何在我的网站中添加「添加到日历」按钮？

你可以同时提供两种方式：1) 使用本工具生成Google日历链接，设置为a标签的href；2) 生成本地ICS文件，提供文件下载链接。推荐两种都用，覆盖不同用户的习惯。

事件时间使用什么时区？

本工具生成的事件使用你的本地时区。ICS文件会包含时区信息，Google日历链接也需要指定时区参数。建议在事件描述中注明时区，避免歧义。
