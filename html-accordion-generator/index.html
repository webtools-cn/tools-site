❓ 常见问题

单开和多开模式有什么区别？

单开模式（Accordion）一次只能展开一个面板，展开新面板时自动关闭其他面板。多开模式（Collapsible）允许同时展开多个面板，每个面板独立开关。单开模式适合 FAQ 列表，多开模式适合分组内容展示。

生成的代码包含 JavaScript 吗？

生成的是纯 HTML+CSS 实现，使用 CSS :target 伪类或 checkbox hack 实现手风琴效果。无需 JavaScript，兼容性好。你也可以根据需要自行添加 JS 增强交互。

如何接入现有项目？

复制 HTML 和 CSS 代码到你的项目中即可。CSS 以 .accordion-xx 前缀命名，避免样式冲突。支持自定义主题色、圆角、间距等参数。
