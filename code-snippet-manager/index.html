使用说明

  1. 点击「新建片段」创建代码片段
  2. 输入标题、选择语言、粘贴代码、添加标签
  3. 保存后可在列表中搜索和管理所有片段

功能介绍

免费在线代码片段管理器，帮助开发者收藏和管理常用代码片段。支持创建分类、添加标签、语法高亮显示，内置全文搜索快速找到你需要的内容。所有数据保存在浏览器本地存储中，支持导出为JSON文件备份。

常见问题

数据保存在哪里？安全吗？

所有代码片段数据保存在浏览器的localStorage中，不会上传到任何服务器。数据完全由你掌控，关闭页面后数据不会丢失。但清除浏览器缓存会导致数据丢失，建议定期导出备份。

最多可以保存多少代码片段？

浏览器localStorage通常有5-10MB的限制，每个代码片段平均1-5KB，因此大约可以保存1000-5000个代码片段。如果接近上限，建议导出备份后清理不常用的片段。

如何导出和导入数据？

点击「导出JSON」按钮将所有代码片段保存为JSON文件。需要恢复时，使用「导入JSON」功能选择备份文件即可。这是迁移和备份数据的主要方式。

支持哪些编程语言？

支持JavaScript、TypeScript、Python、HTML、CSS、Java、C++、C#、PHP、Ruby、Go、Rust、SQL、Swift、Kotlin、Bash、YAML、JSON、Markdown等30+种语言的语法高亮。

可以给代码片段加标签吗？

可以。每个代码片段可以添加多个标签，方便分类和搜索。例如一个「React Hooks」片段可以同时添加「react」「hooks」「frontend」等标签。标签支持中文和英文。

代码片段会在页面关闭后丢失吗？

不会。数据保存在localStorage中，关闭浏览器甚至重启电脑后数据仍然存在。但请注意：使用隐私/无痕模式浏览时，localStorage会在关闭最后一个标签页时被清除。
