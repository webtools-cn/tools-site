❓ 常见问题

什么是AVIF格式？

AVIF（AV1 Image File Format）是新一代图片格式，基于AV1视频编码。相比JPEG和WebP，AVIF可提供更高的压缩率和更好的图像质量，文件体积可减少50%以上。主流浏览器Chrome、Firefox均已支持。

如何转换AVIF图片？

上传AVIF、PNG、JPEG或WebP图片，选择目标格式，调整质量参数（1-100），点击转换即可。转换完全在浏览器本地完成，不会上传到服务器。

AVIF和WebP哪个更好？

AVIF压缩率通常优于WebP（约20-30%），支持HDR和更广色域。但WebP兼容性更好（所有主流浏览器支持），编码速度更快。建议根据目标平台选择。
