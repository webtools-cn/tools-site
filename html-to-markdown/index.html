🔍 扩展知识

什么是Markdown？

Markdown是一种轻量级标记语言，由John Gruber于2004年创建。它允许人们使用易读易写的纯文本格式编写文档，然后转换成结构化的HTML。Markdown的语法简洁直观，现在已成为技术文档、博客写作和笔记记录的流行格式。

HTML与Markdown的区别

HTML是标记语言，功能强大但标签繁多；Markdown是简化格式，用少量符号即可表达丰富的排版效果。两者可以互相转换，Markdown更适合编写和阅读，HTML更适合网页渲染和复杂布局。

支持的标签范围

  • 标题：h1-h6 → # - ######
  • 段落：p → 纯文本段落
  • 链接：a → [文本](URL)
  • 图片：img → ![alt](src)
  • 列表：ul/ol/li → - / 1.
  • 表格：table/tr/td/th → 表格语法
  • 代码：code/pre → ` / ```

❓ 常见问题

HTML转Markdown支持哪些标签？

支持常见HTML标签，包括标题(h1-h6)、段落(p)、链接(a)、图片(img)、无序列表(ul/li)、有序列表(ol/li)、引用(blockquote)、代码(code/pre)、表格(table/tr/td/th)、粗体(strong/b)、斜体(em/i)、分隔线(hr)等。

转换后的数据会上传服务器吗？

不会。本工具完全在浏览器本地运行，所有HTML内容仅在您的设备上处理，数据不会离开浏览器，保障隐私安全。

转换结果可以直接复制使用吗？

可以。转换完成后点击"复制结果"按钮即可一键复制到剪贴板，直接粘贴到Markdown编辑器、GitHub、Notion等支持Markdown的平台中使用。

为什么有些HTML样式没有转换？

Markdown是轻量级标记语言，不支持CSS样式、内联样式等复杂HTML特性。工具会将结构化的HTML内容转换为对应的Markdown语法，但不会保留颜色、字体大小等样式信息。

表格能正确转换吗？

可以。工具支持将HTML table、tr、th、td标签转换为Markdown表格语法。包括表头、对齐方式和单元格内容都会保留。

代码块如何处理？

pre和code标签会转换为Markdown的代码块（```）和行内代码（`）。pre标签内容会被包裹在代码块中，code标签内容会被包裹在行内代码标记中。

嵌套列表能正确转换吗？

可以。工具支持多级嵌套列表的转换。ul和ol标签会分别转换为无序列表和有序列表，嵌套结构通过缩进来表示，保持原有的层级关系。

如何下载转换结果？

转换完成后，点击"下载为 .md"按钮，工具会自动将Markdown内容保存为.md文件到您的设备中，方便后续编辑和使用。
