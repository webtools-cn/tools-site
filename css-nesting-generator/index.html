💡 关于CSS Nesting

CSS Nesting 是CSS原生支持的嵌套语法（CSS Nesting Module Level 1），允许在样式规则内部嵌套子规则。 使用 & 符号引用父选择器。例如：

.card {
  background: #fff;
  & .title { font-size: 1.2rem; }
  &:hover { box-shadow: 0 2px 8px rgba(0,0,0,.1); }
}
📌 注意事项：CSS Nesting 要求嵌套的选择器以 &、.、#、:、[、>、+、~ 开头，否则需要使用 :is() 包裹。
