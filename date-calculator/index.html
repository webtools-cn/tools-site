使用教程

本日期计算器提供五种核心功能，满足不同场景的日期计算需求。以下是详细的使用指南：

日期间隔计算：在"日期间隔"标签页中，选择两个日期，工具会自动计算并显示它们之间的总天数、周数、月数和年数。例如想知道从2020年1月1日到今天一共过了多少天，只需分别选择这两个日期即可。结果会精确显示总天数，并换算为周数（向下取整）、月数（约值）和年数（约值）。

日期加减运算：在"日期加减"标签页中，输入起始日期和需要加减的数值及单位（天、周、月、年），工具会自动计算出结果日期。例如想知道30天后是哪一天，只需选择今天，输入30天即可。如果选择了负值，则相当于减法运算。这对于项目管理中估算截止日期、合同续期提醒等场景非常实用。

工作日计算：在"工作日"标签页中，输入起始日期和所需工作日数量，工具会自动跳过周末，计算出满足条件的结束日期。您还可以添加自定义节假日（用逗号分隔多个日期），这些日期也会被排除在工作日之外。例如项目需要10个工作日完成，从周一开始计算，工具会自动排除周末，给出准确的交付日期。

年龄计算：在"年龄计算"标签页中，输入出生日期，工具会精确计算出您的周岁年龄（精确到年月日），以及距离下一个生日的天数。这个功能在填写表格、申请证件时特别有用，可以确保年龄信息准确无误。

倒计时：在"倒计时"标签页中，输入目标日期，工具会显示距离该日期还有多少天、几个月零几天，并标注周几。适合用于重要事件倒计时，如项目截止日、假期开始、生日派对等。

应用场景

日期计算器在日常工作和生活中有广泛的应用场景，以下是几个典型示例：

项目管理与合同管理：项目经理可以使用"日期加减"功能快速计算项目各阶段的截止日期。例如项目周期90天，从签约日开始，工具可以自动算出项目交付日期。同样，合同续期提醒也可以通过日期加减功能设置提前提醒日期。"工作日"功能则帮助排除节假日，给出准确的工作日安排。

财务与薪酬计算：财务人员经常需要计算两个日期之间的间隔，比如计算利息天数、试用期时长、工资发放周期等。工具提供的日期间隔计算可以精确到天，避免了手动计算的繁琐和出错风险。年龄计算功能在计算员工司龄、工龄时也很有用。

个人生活规划：日常生活中，我们常用"倒计时"功能来追踪重要事件。比如距离春节还有几天、宝宝出生还有多久、旅行出发日期倒数等。年龄计算则适用于填写各种需要精确年龄信息的表格和申请材料。

扩展知识

格里高利历与日期计算：本工具采用公历（格里高利历）进行计算。格里高利历是目前国际通用的历法，由教皇格里高利十三世于1582年颁布。在这种历法中，一年有365天，闰年有366天，闰年规则为：能被4整除且不能被100整除，或能被400整除的年份。在计算日期间隔时，工具会自动处理闰年的2月29日。

工作日与自然日的区别：自然日是指连续的日历天数，包括周末和节假日。工作日则是指正常工作的日子，通常不包括周六、周日和法定节假日。在不同国家和地区，工作日的定义可能略有不同。本工具默认排除周六和周日，用户可自行添加其他需要排除的节假日。国际标准工作日为周一到周五，但本工具也支持自定义。

JavaScript 日期对象精度：本工具底层使用JavaScript的Date对象进行计算，精度为毫秒级。Date对象以1970年1月1日UTC（Unix纪元）为起点，内部存储为毫秒数。这种设计使得日期运算非常高效，但需要注意时区问题——工具显示的结果均为用户本地时区的日期。

日期计算器支持哪些功能？

本工具支持五种核心功能：日期间隔计算（精确到天数）、日期加减运算（支持日/周/月/年）、工作日计算（排除周末和自定义节假日）、年龄计算（精确到年月日）以及倒计时功能。

工作日计算包含法定节假日吗？

工具默认排除周六和周日。对于法定节假日，您可以手动添加自定义节假日日期，工具会自动将其从工作日中排除。不同国家和地区节假日不同，因此采用自定义方式更加灵活。

数据会发送到服务器吗？

不会。所有计算均在浏览器本地完成，纯前端处理，您的日期数据绝不上传服务器，完全保护隐私。即使在没有网络的情况下，工具也能正常使用。

支持农历日期计算吗？

当前版本支持公历（阳历）日期计算。农历日期计算因涉及复杂的农历历法转换，暂未支持。后续版本会考虑添加农历支持。

日期加减中的"月"和"年"是如何计算的？

"月"的计算方式是直接在月份数字上加减，例如1月15日加1个月是2月15日。如果目标月份没有对应日期（如1月31日加1个月），则取该月的最后一天。"年"的计算方式类似，在年份数字上直接加减。

计算结果可以导出吗？

当前版本的结果可以直接复制到剪贴板。您也可以截取屏幕保存结果。如果需要批量计算或导出数据，建议使用Excel或编程方式处理。本工具定位为快速在线计算，重点在于即时性和便捷性。

支持的历史日期范围是多少？

JavaScript Date对象支持的时间范围约为公元1年到公元275760年。但受浏览器限制，实际可用的日期范围通常在1900年到2100年之间。对于极端年份的计算，建议验证结果。

如何快速重置所有数据？

每个功能模块都配有"清空"或"设为今天"按钮，点击即可快速重置。您也可以直接刷新页面恢复初始状态。所有数据均为临时存储在浏览器中，关闭页面后自动清除。
