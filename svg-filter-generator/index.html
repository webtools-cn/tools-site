常见问题

什么是SVG滤镜？

SVG滤镜（filter）使用feGaussianBlur、feDropShadow等滤镜原语为图形添加视觉效果。所有渲染由浏览器本地完成。

SVG滤镜支持哪些效果？

常见效果：模糊(blur)、投影(drop shadow)、颜色矩阵(color matrix)、浮雕(emboss)、位移(displacement)、边缘检测等。

SVG滤镜和CSS滤镜有什么区别？

CSS filter使用简单但功能有限。SVG filter更强大灵活，支持多滤镜组合、颜色矩阵变换、自定义通道操作等高级效果。

SVG滤镜的浏览器兼容性如何？

SVG滤镜在所有现代浏览器中都有良好支持，包括Chrome、Firefox、Safari和Edge部分滤镜需注意兼容性。
