📖 支持的哈希类型

  • MD5 — 32位十六进制，常用于文件完整性校验
  • SHA1 — 40位十六进制，Git commit hash
  • SHA256 — 64位十六进制，现代密码学标准
  • SHA512 — 128位十六进制，高安全性场景
  • CRC32 — 8位十六进制，网络传输校验
  • Bcrypt — 以$2a$开头，密码存储首选
  • Base64 — 含+/=字符，编码格式
  • MySQL — 以*开头40位十六进制
  • MD5-Crypt — 以$1$开头
  • SHA256-Crypt — 以$5$开头
  • SHA512-Crypt — 以$6$开头

使用方法

将哈希字符串粘贴到输入框，点击"识别类型"即可。工具会分析哈希值的长度、字符集和格式特征，自动匹配最可能的哈希算法。支持单个和批量识别（每行一个哈希值）。

识别原理

通过正则表达式模式匹配和长度验证进行识别。不同哈希算法有独特的输出特征：MD5固定32位十六进制、SHA1固定40位、Base64包含特殊字符等。本工具综合这些特征给出识别结果。
