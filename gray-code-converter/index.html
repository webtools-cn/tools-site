🔁 格雷码转换器

二进制与格雷码双向转换，支持十进制输入和批量处理

⚙️ 转换模式

输入只包含0和1的二进制字符串
格雷码结果
二进制 → 格雷码，每行一个二进制数

📖 格雷码知识

  • 格雷码（Gray Code）是一种反射二进制码，相邻两个数只有一位不同
  • 二进制 → 格雷码：最高位不变，其余每位 = 原位异或上一位（G[i] = B[i] XOR B[i-1]）
  • 格雷码 → 二进制：最高位不变，其余每位 = 格雷码位异或前一位二进制结果（B[i] = G[i] XOR B[i-1]）
  • 格雷码常用于旋转编码器、Karnaugh图、错误检测和模数转换
  • 示例：二进制 1011 → 格雷码 1110
