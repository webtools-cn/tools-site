❓ 常见问题

生成的Rust代码需要哪些依赖？▼

需要在Cargo.toml中添加serde和serde_json依赖：serde = { version = "1", features = ["derive"] } 和 serde_json = "1"。工具生成的#[derive(Serialize, Deserialize)]注解直接使用这两个库。

JSON数据安全吗？▼

完全安全。所有处理在浏览器本地完成，数据不上传服务器。可放心处理敏感数据。

camelCase和snake_case怎么选择？▼

建议勾选snake_case生成Rust风格的字段名（如user_name），并勾选camelCase自动添加#[serde(rename_all = "camelCase")]注解。这样Rust代码使用snake_case风格，但JSON序列化/反序列化时自动映射到JavaScript风格的camelCase字段名。

如何处理可空字段？▼

工具会自动将JSON中的null值字段映射为Option类型。您也可以勾选"所有字段都Option"强制所有字段为可空类型。建议配合#[serde(default)]使用，使缺失字段也能正常反序列化。

💡 所有处理在浏览器本地完成。生成的Rust代码可直接用于项目。
