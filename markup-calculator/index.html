加价率 vs 利润率

加价率（Markup）：加价金额相对于成本的百分比。公式：加价率 = (售价 - 成本) / 成本 × 100%

利润率（Margin）：利润相对于售价的百分比。公式：利润率 = (售价 - 成本) / 售价 × 100%

例如：成本80元，售价100元，则加价率 = 20/80 = 25%，利润率 = 20/100 = 20%。同一个20元利润，两种不同角度。

常见问题

Q：为什么要区分加价率和利润率？

加价率以成本为基准，常用于采购和批发定价。利润率以售价为基准，常用于财务报表和零售分析。混淆两者会导致定价错误。例如想获得50%的利润率，加价率需要达到100%。

Q：如何将加价率转换为利润率？

转换公式：利润率 = 加价率 / (1 + 加价率)。例如加价率25%，利润率 = 0.25/1.25 = 20%。反过来：加价率 = 利润率 / (1 - 利润率)。例如利润率20%，加价率 = 0.2/0.8 = 25%。
