📖 使用教程

将包含HTML表格的代码粘贴到输入区，选择对齐方式后点击「转换」即可快速转换为Markdown格式表格。工具自动识别table标签，提取表头（th）和单元格（td）数据，支持rowspan和colspan合并单元格。

⚡ 技术原理

本工具使用DOMParser解析HTML代码，自动遍历所有table元素，提取行列数据。根据选定的对齐方式生成符合GitHub Flavored Markdown (GFM) 标准的表格格式。支持thead/tbody/tfoot标签区分，兼容大多数主流HTML表格结构。
