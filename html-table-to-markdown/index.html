📖 使用教程

将包含HTML表格的代码粘贴到输入区，选择对齐方式后点击「转换」即可快速转换为Markdown格式表格。工具自动识别table标签，提取表头（th）和单元格（td）数据，支持rowspan和colspan合并单元格。

⚡ 技术原理

本工具使用DOMParser解析HTML代码，自动遍历所有table元素，提取行列数据。根据选定的对齐方式生成符合GitHub Flavored Markdown (GFM) 标准的表格格式。支持thead/tbody/tfoot标签区分，兼容大多数主流HTML表格结构。

❓ 常见问题

❓ HTML表格转Markdown工具免费吗？
是的，完全免费使用，无需注册登录。
❓ 支持哪些HTML表格属性？
工具自动识别表格的thead、tbody、th、td、rowspan、colspan等属性。对于合并单元格，工具会进行处理和填充。
❓ 可以批量转换多个表格吗？
可以。如果HTML代码中包含多个table标签，工具会自动识别并分别转换每个表格，以分隔线隔开。
❓ 转换后的Markdown表格能直接用吗？
是的，转换后的Markdown表格符合GitHub Flavored Markdown (GFM) 标准，可直接用于GitHub、GitLab、Typora等支持Markdown的平台。
❓ 支持自定义对齐方式吗？
可以。您可以在转换前选择默认对齐方式（左对齐、居中、右对齐）。如果HTML中td或th标签包含align属性，工具会优先使用HTML中的对齐设置。
