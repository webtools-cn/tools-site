使用教程

本JSON转Excel工具操作简单直观，以下是详细的使用指南：

输入JSON数据：在左侧输入框中直接粘贴JSON格式的数据，支持数组对象（最常用）和单个对象两种格式。如果JSON数据存储在文件中，也可以直接将.json文件拖放到指定区域或点击选择文件上传。工具会自动读取文件内容并填充到输入框中。

配置转换选项：在转换前，您可以根据需要调整选项。勾选「扁平化嵌套对象」后，工具会自动将多层嵌套的JSON结构扁平化为平面表格，嵌套键名用点号连接（如 user.address.city 变为 user.address.city 列）。勾选「包含表头」会在生成的表格第一行添加列名。还可以选择不同的分隔符（逗号、制表符或分号）。

预览与导出：点击「转换为表格」后，右侧会实时显示表格预览。您可以先确认数据是否正确转换，然后选择「下载CSV文件」将数据保存到本地，或点击「复制CSV」将结果复制到剪贴板。生成的CSV文件包含UTF-8 BOM头，确保在Excel中打开时中文不会乱码。

应用场景

JSON转Excel工具在数据处理和开发工作中应用广泛，以下是几个典型的使用场景：

API数据导出：开发者从REST API获取到JSON格式的数据后，通常需要将其导出为Excel格式供产品经理、运营人员或客户查看分析。本工具可以直接将API返回的JSON数组快速转换为结构化的表格数据，省去手动整理的繁琐步骤。支持嵌套对象扁平化，即使是复杂的API响应也能一键转换。

数据库查询结果处理：数据库查询工具（如MySQL Workbench、MongoDB Compass）导出JSON格式数据后，本工具可以帮助快速转换为Excel，方便在Excel中进行数据透视、筛选和图表分析。特别适合数据分析师处理从NoSQL数据库导出的嵌套文档数据。

配置文件批量编辑：当需要将JSON格式的配置数据（如用户列表、产品目录等）批量导入到Excel中进行编辑，修改完成后再转换回JSON时，本工具可以作为中间桥梁。相比手动复制粘贴，自动转换能显著提高效率并减少人为错误。

扩展知识

JSON与CSV格式对比：JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，适合表示层级结构和复杂对象关系。CSV（Comma Separated Values）则是一种简单的表格数据格式，每行代表一条记录，字段用分隔符隔开。JSON适合程序处理和数据传输，CSV适合表格查看和Excel分析。本工具的核心价值就是将JSON的灵活性转换为CSV的易读性。

嵌套JSON扁平化原理：当JSON对象包含嵌套结构时，直接转换为表格需要将层级关系展平。本工具采用「点号路径」方式，将嵌套对象的键用点号连接形成列名。例如 {"user":{"name":"张三","age":25}} 会转换为 user.name 和 user.age 两列。这种方式保留了原始数据的结构信息，同时兼容表格的二维特性。

Excel打开CSV乱码问题：CSV文件本质是纯文本，默认编码为UTF-8。但早期版本的Excel（尤其是Windows版）在打开CSV时不识别UTF-8编码，导致中文字符显示为乱码。解决方案是在文件开头添加BOM（Byte Order Mark，即 EF BB BF 三个字节），这样Excel就能正确识别UTF-8编码。本工具生成的CSV文件已自动包含BOM头。

常见问题解答

JSON转Excel支持嵌套数据吗？

支持。工具会自动将嵌套的JSON对象扁平化，使用点号连接嵌套键名。例如 {"user":{"name":"张三"}} 会转换为列 user.name。数组类型的值会被转换为JSON字符串。

转换后的文件能在Excel中正常打开吗？

可以。工具生成的CSV文件包含UTF-8 BOM头，确保Excel打开时中文不会乱码。下载后直接用Excel打开即可。如果数据量很大，建议先预览确认格式正确再下载。

数据会上传服务器吗？

不会。所有处理都在浏览器本地完成，数据绝不离开您的设备，完全保护隐私安全。您可以在断网状态下使用本工具。

支持多大的JSON数据？

工具在浏览器端运行，通常可处理几MB到几十MB的JSON数据，具体取决于设备内存。建议分批处理超大文件，或使用服务器端工具处理GB级别数据。

数组类型的值怎么处理？

数组类型的值会被转换为JSON字符串形式存入单元格。如需更细粒度的拆分（如将数组拆分为多行），建议先手动处理数据结构。

为什么有些字段显示为空？

如果某些记录缺少某个键，或者该键对应的值为 null / undefined，对应单元格会显示为空。这是正常的，不影响其他数据。

可以转换单个对象而不是数组吗？

可以。如果输入的是单个JSON对象（而非数组），工具会自动将其视为单条记录处理，生成一行数据的表格。

生成的CSV文件如何在Excel中保存为.xlsx格式？

在Excel中打开CSV文件后，点击「文件」→「另存为」，选择「Excel工作簿(.xlsx)」格式保存即可。这样转换后的文件保留了Excel的所有功能（如公式、样式、筛选等）。
