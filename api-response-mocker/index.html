API响应模拟器能做什么？

API响应模拟器是一款纯前端在线工具，用于快速生成REST API的模拟响应数据。支持自定义HTTP状态码、响应延迟、Content-Type和自定义响应头，内置用户列表、商品数据、分页响应、错误响应等常用模板。生成的模拟响应包含完整的HTTP状态行、响应头和响应体预览，并自动生成对应的cURL命令。所有数据在浏览器本地处理，绝不上传服务器。

核心功能

  • 快速模板：内置6种常用API响应模板，一键填充
  • 自定义状态码：支持所有标准HTTP状态码，也可自定义
  • 延迟模拟：0-30秒可调延迟，模拟真实网络环境
  • 自定义响应头：添加任意Header，支持CORS配置
  • cURL生成：自动生成对应的cURL命令，方便测试

使用教程

步骤1：选择快速模板或手动输入JSON响应体。

步骤2：配置状态码、延迟、Content-Type和自定义响应头。

步骤3：点击"生成模拟响应"，查看完整的HTTP响应预览和cURL命令。

步骤4：复制cURL命令或下载响应数据用于测试。Ctrl+Enter快速生成。

扩展知识

REST API响应由三部分组成：状态行（HTTP版本+状态码+原因短语）、响应头（Content-Type、Cache-Control等）、响应体（JSON/XML数据）。常见状态码：2xx成功、3xx重定向、4xx客户端错误、5xx服务端错误。Mock Server是开发中常用的工具，如json-server、MockJS等，本工具提供轻量级的纯前端模拟方案。
