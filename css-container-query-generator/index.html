什么是CSS容器查询？
CSS容器查询（Container Queries）允许根据父容器的尺寸而非视口大小来应用样式。这使得组件可以根据自身所在空间自适应，实现真正的组件级响应式设计。
容器查询和媒体查询有什么区别？
媒体查询基于视口大小，容器查询基于父容器大小。容器查询让组件在不同布局位置都能自适应，不受全局视口限制。
浏览器兼容性如何？

CSS容器查询生成器

CSS容器查询（Container Queries）是CSS的新特性，允许根据父容器的尺寸而非视口大小来应用样式。与媒体查询不同，容器查询让组件可以根据自身所在空间自适应，实现真正的组件级响应式设计。本工具帮助你可视化配置容器查询，实时预览效果并生成CSS代码。

容器查询 vs 媒体查询

  • 媒体查询：基于视口大小，全局影响
  • 容器查询：基于父容器大小，组件级影响
  • 容器查询让同一组件在不同布局位置都能自适应
  • 容器查询更适合组件化开发和设计系统

使用场景

  • 卡片组件在侧边栏和主内容区显示不同布局
  • 导航组件根据可用空间切换水平/垂直布局
  • 表单组件根据容器宽度调整列数
  • 任何需要根据父容器空间自适应的组件
