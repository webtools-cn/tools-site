什么是CSS容器查询？

CSS容器查询是现代CSS特性，允许您根据父容器的大小而非视口来设置元素样式。与响应视口大小的媒体查询不同，容器查询使组件能根据其布局上下文自适应——无论是在侧边栏、主内容区还是卡片布局中。

核心功能

  • 可视化构建器：添加断点并实时查看生成的CSS
  • 实时预览：调整预览容器大小查看样式变化
  • 命名容器：支持container-name进行定向查询
  • 多断点：按需添加min-width/max-width断点
  • 单位支持：px、rem和em单位

❓ 常见问题

什么是CSS容器查询？

根据父容器大小而非视口应用样式，使组件能根据布局上下文自适应。

容器查询和媒体查询的区别？

媒体查询响应视口大小，容器查询响应父容器大小，使组件更可复用。

浏览器支持？

Chrome 105+、Firefox 110+、Safari 16+、Edge 105+。覆盖全球90%以上。

数据会上传吗？

不会。所有CSS生成在浏览器本地完成。

什么是container-type？

定义包含类型：inline-size（宽度查询）、size（双轴）、normal（无包含）。

可以使用命名容器吗？

可以。使用container-name并引用：@container sidebar (min-width: 300px)。
