📖 CSS首字母下沉详解

首字母下沉（Drop Cap） 是一种经典的排版装饰技术，常用于书籍、杂志、博客文章的开篇段落，让首字母脱颖而出。

实现方式

  • initial-letter 属性 — CSS新特性，可指定首字母占据的行数和缩放比例（如 initial-letter: 3）
  • ::first-letter 伪元素 — 跨浏览器兼容的传统方式，配合 float、font-size、line-height 等属性

设计技巧

  • 深色背景 + 亮色文字的首字母能形成强烈视觉对比
  • 使用不同的字体让首字母与正文区分
  • 添加背景色和圆角打造精美的装饰效果
  • 首字母颜色与品牌色一致可以强化品牌识别

浏览器兼容性

initial-letter 属性在 Safari 9+ 中支持，Chrome 110+ 需启用实验性标志。传统 ::first-letter 方式在所有浏览器中兼容。

常见问题 (FAQ)

CSS首字母下沉是什么？

CSS首字母下沉(Drop Cap)是一种排版装饰技术，让段落的首字母以更大的尺寸、不同的字体或颜色突出显示，常用于杂志、书籍和博客文章中，增强视觉吸引力。

initial-letter属性如何使用？

initial-letter属性接受两个数字值：第一个数字控制下沉行数（如3表示首字母占据3行高度），第二个可选数字控制字体大小缩放（默认1）。例如 initial-letter: 3 2; 表示首字母占据3行高度并放大2倍。

initial-letter的浏览器兼容性如何？

initial-letter属性目前在Safari 9+和Chrome 110+（需开启实验性Web平台功能标志）中支持。不支持的情况下，可以使用float、font-size和line-height的传统方式实现首字母下沉效果。本工具同时提供两种实现方式。

如何用传统方式实现首字母下沉？

传统方式使用 ::first-letter 伪元素，配合 float: left、font-size、line-height、padding 和 margin 属性来实现跨浏览器兼容的首字母下沉效果。
