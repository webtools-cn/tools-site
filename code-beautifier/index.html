代码美化和代码压缩有什么区别？

代码美化（Beautify）是将压缩或格式混乱的代码重新排版，添加缩进和换行，使其易于阅读和维护。代码压缩（Minify）则相反，去除空格换行注释，减小文件体积以提升加载速度。两者是互逆操作。

我的代码安全吗？

完全安全。所有格式化操作都在浏览器本地完成，代码绝不会上传到任何服务器。您的源代码始终留在您的设备上，适合处理包含敏感信息的配置文件和源码。
