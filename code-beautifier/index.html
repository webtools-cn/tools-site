代码美化器支持哪些语言？

本工具支持JavaScript、HTML、CSS、JSON、XML、SQL、Python等7种常见编程语言的代码格式化和美化。每种语言都有针对性的格式化规则，确保输出代码符合该语言的常见风格规范。

代码美化和代码压缩有什么区别？

代码美化（Beautify）是将压缩或格式混乱的代码重新排版，添加缩进和换行，使其易于阅读和维护。代码压缩（Minify）则相反，去除空格换行注释，减小文件体积以提升加载速度。两者是互逆操作。

我的代码安全吗？

完全安全。所有格式化操作都在浏览器本地完成，代码绝不会上传到任何服务器。您的源代码始终留在您的设备上，适合处理包含敏感信息的配置文件和源码。

如何使用代码美化器？

选择代码语言类型，将代码粘贴到输入框中，点击美化按钮即可获得格式化后的代码。还可以调整缩进大小和缩进类型（空格/Tab）。支持一键复制和下载结果。
