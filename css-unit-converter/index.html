使用教程

本CSS单位转换器是专为前端开发者设计的在线工具，帮助您在不同CSS单位之间快速转换。以下是详细的使用指南：

基本操作

在「输入数值」区域，输入您要转换的数值并选择源单位（如16px）。然后在「基准设置」中配置相关参数：根字体大小默认16px（影响rem/em转换）、视口宽高默认1920×1080（影响vw/vh/vmin/vmax转换）、父元素字体大小默认16px（影响em转换）。点击「开始转换」按钮，所有结果会实时显示在下方结果区域。

快速预设

工具提供了常见设备视口尺寸的快捷预设，包括iPhone SE、iPhone 11、iPad、笔记本、桌面显示器、2K和4K分辨率等。点击任一预设即可快速设置视口宽高，方便您针对不同设备进行单位转换。

复制结果

转换完成后，点击任意结果卡片即可将该单位的转换结果复制到剪贴板。系统会显示「已复制」提示，方便您快速粘贴到CSS代码中。

自定义基准

如果您的设计稿使用非标准根字体大小（如设计系统使用10px或14px作为根字体大小），可以在「根字体大小」输入框中修改。同样，如果您的项目针对特定视口尺寸进行设计，也可以自定义视口宽高参数。

应用场景

CSS单位转换器在以下场景中特别有用：

响应式设计开发：当您需要将设计稿中的固定像素值转换为vw/vh等响应式单位时，本工具可以帮助您快速计算转换值。例如，设计稿中一个宽度为375px的元素，在375px宽度的视口中对应的vw值，使用本工具可以立即得到结果。

设计系统迁移：许多设计系统（如Tailwind CSS、Ant Design）使用rem作为基础单位。当您需要将现有的px单位代码迁移为rem单位时，本工具可以快速批量转换。只需设置正确的根字体大小，输入px值即可获得对应的rem值。

打印样式开发：在开发打印样式时，通常需要使用pt、cm、mm、in等物理单位。本工具可以将网页中的px单位转换为打印友好的物理单位，确保打印输出效果与设计一致。

跨设备适配：不同设备有不同的像素密度（DPR）和视口尺寸。通过本工具，您可以快速计算同一尺寸在不同设备上的等效值，确保UI元素在各种屏幕上都保持合适的比例。

扩展知识

CSS绝对单位与相对单位：CSS单位分为绝对单位和相对单位。绝对单位（px、pt、cm、mm、in、pc）与设备物理特性相关，在屏幕上表现固定。相对单位（rem、em、vw、vh、%等）则相对于其他值计算，更适合响应式设计。现代前端开发推荐使用rem和vw等相对单位，以提升网站的可访问性和适应性。

根字体大小设置技巧：在HTML根元素上设置字体大小时，推荐使用百分比而非固定像素值，如html{font-size:62.5%}（相当于10px），这样用户可以方便地通过浏览器设置调整整体字体大小，提升可访问性。此时1rem=10px，计算更加直观。

视口单位的浏览器支持：vw、vh、vmin、vmax等视口单位在现代浏览器中已得到广泛支持（IE9+支持vw/vh，IE10+支持vmin，Edge12+支持vmax）。在需要兼容旧版浏览器的场景下，可以配合JavaScript动态计算px值作为回退方案。

常见问题

CSS单位转换器支持哪些单位？

本工具支持px（像素）、rem（根元素字体大小倍数）、em（父元素字体大小倍数）、vw（视口宽度百分比）、vh（视口高度百分比）、vmin（视口最小边百分比）、vmax（视口最大边百分比）、ch（字符宽度0的宽度）、ex（字符x的高度）、pt（磅）、pc（派卡）、cm（厘米）、mm（毫米）、in（英寸）等十余种CSS单位互转。

px和rem怎么转换？

px（像素）和rem（根字体大小倍数）的转换关系取决于根元素的字体大小。默认情况下，大多数浏览器的根字体大小为16px，因此1rem = 16px。转换公式为：rem = px / 根字体大小，px = rem * 根字体大小。您可以在工具中自定义根字体大小，默认值为16px。

vw和px怎么转换？

vw（视口宽度百分比）和px的转换需要知道视口宽度。1vw等于视口宽度的1%，因此转换公式为：px = vw * 视口宽度 / 100，vw = px * 100 / 视口宽度。例如，在1920px宽度的视口中，100px = 5.21vw。您可以在工具中设置视口宽度，默认值为1920px。

em和rem有什么区别？

em是相对单位，其计算基准是当前元素的父元素的字体大小。如果父元素字体大小为14px，那么1em = 14px。rem也是相对单位，但其计算基准始终是根元素（html标签）的字体大小，不随嵌套层级变化。因此rem在多层嵌套中更加可预测，是现代前端开发中推荐使用的相对单位。

这个工具的数据会上传吗？

绝对不会。所有转换计算都在浏览器本地完成，没有任何数据上传到服务器。您可以断网使用此工具，也可以放心输入任何敏感数值，您的数据完全在本地处理，安全可靠。

为什么有时转换结果有小数？

由于不同单位之间的换算关系通常不是整数倍（例如1英寸=96像素），转换结果可能会出现小数。这是正常的数学现象。如果您需要整数结果，可以使用工具的取整功能。在生产环境中，浏览器会自动处理这些小数值，通常不会影响最终显示效果。

vmin和vmax是什么单位？

vmin表示视口宽高中较小值的百分比，vmax表示较大值的百分比。例如，在1920x1080的视口中，1vmin = 10.8px（因为1080是较小值），1vmax = 19.2px（因为1920是较大值）。这两个单位特别适合实现响应式设计，确保元素在不同屏幕尺寸下都能保持良好的比例。

ch和ex单位有什么用？

ch单位表示数字0（零）的宽度，ex单位表示小写字母x的高度。这两个单位在设计等宽字体布局和精确控制文本行高时非常有用。例如，设置max-width: 60ch可以确保每行最多显示约60个等宽字符，这是提升可读性的最佳实践。ex单位则常用于微调行高和垂直对齐。
