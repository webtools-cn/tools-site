📚 JSON Path构建器能做什么？

JSON Path构建器是一个可视化工具，帮助用户交互式构建JSONPath查询表达式。通过点击JSON树节点自动生成路径，实时预览查询结果，无需记忆JSONPath语法。

扩展知识

JSONPath由Stefan Gössner于2007年提出，灵感来自XPath。它定义了一组操作符用于查询JSON结构：$（根节点）、@（当前节点）、.或[]（子节点）、..（递归下降）、*（通配符）、?()（过滤）、[start:end]（数组切片）。JSONPath已被广泛集成到各种编程语言和工具中。

❓ 常见问题

什么是JSONPath？

JSONPath是用于查询JSON数据的表达式语言，类似XPath对XML的作用。通过JSONPath表达式可以从JSON文档中提取特定节点、过滤数据、访问嵌套结构。

支持哪些JSONPath语法？

支持标准JSONPath语法：$（根节点）、.（子节点）、[]（数组索引/条件）、*（通配符）、..（递归下降）、?()（过滤表达式）、@（当前节点）等。

如何构建JSONPath表达式？

在左侧输入JSON数据，点击JSON树中的节点自动生成路径，也可手动编辑路径表达式。右侧实时显示查询结果。

支持过滤表达式吗？

支持。可以使用?()过滤表达式，如$.store.book[?(@.price<10)]筛选价格低于10的书籍，支持比较运算符和逻辑组合。

查询结果可以导出吗？

可以。支持一键复制查询结果和JSONPath表达式，也可下载为JSON文件。

支持大型JSON数据吗？

支持。工具在浏览器本地处理，性能取决于JSON大小和浏览器内存。建议JSON数据不超过5MB以获得最佳体验。
