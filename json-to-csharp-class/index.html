关于JSON转C#类生成器

JSON转C#类生成器是一个高效的开发者工具，将JSON数据快速转换为C#实体类代码。在.NET开发中，经常需要处理JSON数据的反序列化，手动编写对应的C#类既繁琐又容易出错。本工具自动分析JSON结构并生成准确的C#类定义，大幅提高开发效率。

功能特点

  • 自动类型推断：根据JSON值类型自动选择正确的C#类型
  • 嵌套对象支持：JSON中的嵌套对象自动生成嵌套类
  • 数组处理：数组自动映射为List<T>类型
  • JsonProperty特性：可选添加，保持JSON字段命名
  • 生成代码质量高：符合C#编码规范，可直接使用

常见问题

生成的C#类代码质量如何？

生成的代码符合C#编码规范，包含：1)正确的命名空间和类定义；2)属性类型自动推断（int、double、string、bool、DateTime、List、Dictionary等）；3)JsonProperty特性标记（与Newtonsoft.Json兼容）；4)支持nullable值类型；5)嵌套类自动生成。代码可直接复制到Visual Studio中使用。

支持哪些JSON结构？

支持所有常见JSON结构：1)简单对象（flat JSON对象）；2)嵌套对象（对象中包含对象）；3)数组（包括对象数组和原始类型数组）；4)混合结构（数组中的对象包含嵌套对象）；5)可选字段（同一位置出现不同类型时自动处理）。对于复杂的深层嵌套JSON也能正确处理。

类型是如何推断的？

工具根据JSON值的实际类型自动推断C#类型：数字无小数→int/（若JSON中为浮点数→double）、有小数→double、字符串→string、布尔→bool、对象→嵌套class、数组→List、null→可空类型。如果同一字段出现多种类型，优先使用更通用的类型。

支持JsonProperty特性吗？

支持。生成器会自动为每个属性添加[JsonProperty("originalName")]特性，确保反序列化时能正确匹配JSON字段名。这样可以保留原始的JSON命名风格（如snake_case），而C#属性使用PascalCase命名规范。

生成的代码可以在.NET framework和.NET Core中使用吗？

可以。生成的代码兼容.NET Framework 4.5+和.NET Core/5/6/7/8/9等所有现代.NET版本。使用了标准System.Collections.Generic命名空间和Newtonsoft.Json特性。如果需要System.Text.Json兼容版本，可以开启'使用System.Text.Json'选项。
