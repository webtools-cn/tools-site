常见问题

什么是CSS perspective？

CSS perspective定义了用户视线与z=0平面之间的距离。值越小3D效果越强烈，值越大3D效果越平缓。

perspective-origin有什么作用？

perspective-origin决定3D空间的消失点位置。改变它相当于移动观察角度，就像你移动头部观察一个3D物体。

生成的CSS可以直接用于生产吗？

可以。所有生成的CSS属性都是标准的CSS3属性，在主流浏览器中均可使用。复制CSS代码直接应用到你的项目中即可。

perspective和transform:perspective()有什么区别？

父元素上的perspective为所有子元素创建共享3D空间。transform:perspective()只应用于单个元素，在变换链中使用。
