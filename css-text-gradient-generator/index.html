常见问题

什么是CSS渐变文字？

CSS渐变文字是利用background-clip: text属性配合background渐变，让文字显示出渐变颜色的效果。文字本身的颜色被设置为透明，背景渐变色透过文字显示出来。

浏览器兼容性如何？

所有现代浏览器（Chrome 90+、Firefox 90+、Safari 15+、Edge 90+）都支持background-clip: text。老版本浏览器会回退到fallback颜色。
