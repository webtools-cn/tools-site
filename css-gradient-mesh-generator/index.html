❓ 常见问题

什么是 Mesh Gradient（网格渐变）？

Mesh Gradient 是通过多层 radial-gradient 组合而成的一种有机、流动的渐变风格，形似网格状色彩混合。相较于传统线性/径向渐变，Mesh Gradient 色彩过渡更自然柔和，没有明显方向性，常用于现代 UI 设计中的背景和插画风格。

生成的 CSS 兼容性好吗？

本工具使用标准 CSS radial-gradient 语法，兼容所有现代浏览器（Chrome、Firefox、Safari、Edge）。纯 CSS 实现，无需 JavaScript 支持。代码可直接用于生产环境。

为什么调整颜色点位置会影响整体效果？

每个颜色点是一个独立的 radial-gradient 层，其位置（百分比）决定了渐变中心点的坐标。调整位置可以让某个颜色在背景中占据更多或更少的空间，从而改变整体色彩分布和视觉重心。

建议使用多少个颜色点？

3-5 个颜色点通常能获得最佳视觉效果——太少显得单调，太多可能导致色彩混乱。推荐从 3 个点开始，逐步增加直到满意。模糊度建议在 60-100% 之间。
