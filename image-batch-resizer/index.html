📖 使用教程

本工具是一款纯前端批量图片尺寸调整工具，无需安装任何软件，在浏览器中即可完成多张图片的统一尺寸处理。以下是详细的使用步骤：

第一步：上传图片

点击上传区域或直接将图片文件拖入页面。支持同时选择多张图片，格式涵盖 JPG、PNG、WebP、GIF 和 BMP。上传后会在下方显示所有图片的缩略图预览，您可以点击缩略图右上角的删除按钮移除不需要处理的图片。

第二步：选择目标尺寸

在「调整设置」区域，您可以选择预设的社交媒体尺寸，包括微信头像（300×300）、微信公众号封面（900×500）、Instagram 帖子（1080×1080）、Instagram 快拍（1080×1920）、Twitter/X 帖子（1200×675）、YouTube 缩略图（1280×720）、小红书（1080×1440）以及 A4 打印尺寸（2480×3508）。也可以直接输入自定义宽度和高度。

第三步：选择调整模式

工具提供三种调整模式：拉伸填充直接将图片变形拉伸至目标尺寸；等比缩放（裁剪）保持原比例，放大覆盖目标尺寸后裁掉多余部分，适合需要填满特定尺寸的场景；等比缩放（留白）保持原比例完整放入目标尺寸内，多余区域用背景色填充，适合不想丢失画面内容的场景。

第四步：设置输出格式与质量

输出格式可选择「保持原格式」，或统一转换为 JPEG、PNG、WebP。当输出为 JPEG 时，可通过滑块调整压缩质量（10%-100%），建议设置在 85%-95% 之间以获得最佳画质与文件大小平衡。等比缩放（留白）模式下还可以设置背景填充颜色。

第五步：批量处理与下载

点击「开始批量处理」按钮，所有图片将在浏览器本地逐张处理，进度条会实时显示完成进度。处理完成后，每张图片都会显示独立的下载按钮，点击即可将调整后的图片保存到本地。所有处理均在本地完成，图片不会上传到任何服务器，充分保护您的隐私。

🔍 扩展知识

理解图片尺寸调整背后的原理，可以帮助您在不同场景下做出更明智的选择：

下采样与上采样

当目标尺寸小于原图时，工具需要进行下采样（降采样），即减少像素数量。浏览器 Canvas 默认使用双线性插值算法，在缩小图片时通常表现良好，但极端缩小（如 5000px 缩到 100px）可能产生摩尔纹。当目标尺寸大于原图时，进行上采样（插值放大），图片清晰度不会真正提升，只是通过算法填充了更多像素，因此建议不要过度放大低分辨率图片。

有损压缩与无损压缩

JPEG 采用有损压缩，通过丢弃人眼不太敏感的高频信息来减小文件体积，质量设置越低，丢弃的信息越多。PNG 采用无损压缩，保留所有像素信息，适合包含文字、图标或需要透明背景的图片。WebP 同时支持有损和无损模式，通常能在同等画质下提供比 JPEG 和 PNG 更小的文件体积，但兼容性略逊于前两者。

DPI、PPI 与像素尺寸

许多用户混淆了像素尺寸和打印分辨率。DPI（每英寸点数）是打印领域的概念，表示打印机每英寸能输出多少墨点；PPI（每英寸像素数）是屏幕显示的概念。一张 1200×675 像素的图片，在 72 PPI 屏幕上显示约 16.7×9.4 英寸，而在 300 PPI 打印时仅约 4×2.25 英寸。本工具调整的是像素尺寸，DPI 信息不会改变。

Canvas API 与隐私安全

本工具基于 HTML5 Canvas API 实现所有图片处理逻辑。图片被读取为 FileReader 数据后在内存中解码为 ImageBitmap，通过 Canvas 绘制并导出，全程不经过网络传输。处理完成后，所有内存中的图片数据会在页面刷新时由浏览器垃圾回收机制自动释放。

❓ 常见问题

批量图片调整大小工具收费吗？

完全免费。本工具是纯前端应用，所有图片处理在浏览器本地完成，图片不会上传到任何服务器，因此没有任何费用，也没有数量限制。

支持哪些图片格式？

支持 JPEG、PNG、WebP、BMP、GIF 等浏览器可读取的常见图片格式。输出格式可选择保持原格式，或统一转换为 JPEG、PNG 或 WebP。

调整后的图片质量会下降吗？

如果目标尺寸小于原图，会不可避免地损失部分细节（下采样）。如果目标尺寸大于原图，则会通过插值放大，清晰度不会提升。建议设置 JPEG 质量为 85%-95% 以获得最佳文件大小与画质平衡。

等比缩放（裁剪）和等比缩放（留白）有什么区别？

裁剪模式会保持比例，将图片放大至完全覆盖目标尺寸，多余部分被裁掉，最终没有空白。留白模式也会保持比例，将图片完整放入目标尺寸内，多余区域用指定背景色填充，图片不会被裁切。

一次最多能处理多少张图片？

理论上没有上限，取决于设备内存和浏览器性能。实际测试中，普通电脑一次处理 50 张以内的高清图片比较流畅。超过 100 张建议分批处理。

社交媒体预设尺寸分别是什么？

微信头像 300×300、微信公众号封面 900×500、Instagram 帖子 1080×1080、Instagram 快拍 1080×1920、Twitter/X 帖子 1200×675、YouTube 缩略图 1280×720、小红书 1080×1440、A4 打印 2480×3508。

处理后的图片会保存到哪里？

处理后的图片直接通过浏览器下载到您的本地设备，不会上传到任何服务器或云端存储。关闭页面后所有数据立即清除，保护您的隐私。
