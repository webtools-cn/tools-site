❓ 常见问题

这个工具支持哪些SQL方言？▼

支持MySQL、PostgreSQL、SQLite、SQL Server和Oracle。工具自动识别INSERT INTO语句中的字段名和值。对于表格格式，支持常见的SELECT结果输出形式，包括MySQL命令行输出、pgAdmin输出等。

数据安全吗？▼

完全安全。所有SQL解析和JSON生成都在浏览器本地完成，没有数据上传到服务器。您可以放心处理包含敏感数据的SQL语句。

表格格式支持什么样式？▼

支持带表头的表格格式，包括MySQL命令行输出样式（带边框）、TSV（制表符分隔）、CSV（逗号分隔）、以及用空格/竖线对齐的格式化表格。第一行作为字段名，后续行作为数据。

如何处理NULL值？▼

SQL中的NULL值会自动转换为JSON的null。空字符串保持为空字符串。数字字符串（如"123"）在JSON中保持为字符串类型，不做自动类型转换，确保数据准确性。
