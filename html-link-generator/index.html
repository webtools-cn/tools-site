关于 HTML 链接属性

target="_blank" — 在新标签页或新窗口打开链接。配合 rel="noopener noreferrer" 可防止新页面通过 window.opener 访问原页面，提升安全性。

rel="nofollow" — 告诉搜索引擎不要追踪此链接，不传递权重。常用于外部链接、广告链接或用户生成内容中的链接。

rel="noopener" — 阻止新页面通过 window.opener 访问原页面，防止反向标签页劫持攻击。

rel="noreferrer" — 阻止浏览器在请求时发送 HTTP Referer 头部信息，保护用户隐私。

title — 鼠标悬停在链接上时显示的提示文本，有助于可访问性和用户体验。
