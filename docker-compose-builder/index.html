常见问题 (FAQ)

Docker Compose的version应该选哪个？

Docker Compose V2不再需要version字段，本工具默认不添加，兼容最新规范。

ports的宿主端口:容器端口是什么意思？

冒号左边是宿主机端口，右边是容器端口。8080:80表示宿主机8080映射到容器80。

volumes和bind mount有什么区别？

Volumes由Docker管理适合持久化，bind mount挂载宿主机目录适合开发同步代码。
