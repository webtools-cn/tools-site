常见问题

Q1: Docker Compose的version应该选哪个？

从Docker Compose V2开始，version字段已不再需要。本工具默认不添加version字段，兼容最新的Docker Compose规范。如需兼容旧版本可手动添加version: "3.8"。

Q2: ports的"宿主端口:容器端口"是什么意思？

冒号左边是宿主机暴露端口，右边是容器内部端口。例如8080:80表示宿主机8080映射到容器80端口，访问localhost:8080即可访问容器服务。

Q3: volumes和bind mount有什么区别？

Docker Volumes由Docker管理，存储在Docker数据目录中，适合持久化数据。Bind mount直接挂载宿主机目录，适合开发时同步代码。主机路径:容器路径为bind mount，卷名:容器路径为named volume。
