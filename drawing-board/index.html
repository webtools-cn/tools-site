📖 使用教程

在线画板是一款完全免费的网页绘图工具，无需下载安装任何软件，打开浏览器即可直接使用。下面为您介绍如何使用这款工具：

  • 选择工具：点击上方工具栏中的画笔、直线、矩形、圆形或橡皮擦按钮，选择您需要的绘图工具。
  • 设置颜色：点击颜色选择器可以更改绘图颜色，支持任意颜色选择。
  • 调节粗细：拖动粗细滑块可以调整画笔或线条的粗细，范围从1px到50px。
  • 设置背景：点击背景色选择器可以更改画布背景色，默认为深色主题。
  • 撤销重做：点击"撤销"按钮可以回退上一步操作，点击"重做"可以恢复被撤销的操作。
  • 清空画布：点击"清空"按钮可以一键清空整个画布，清空前请确认是否需要保存作品。
  • 导出图片：完成绘制后，点击"导出PNG"按钮可以将作品保存为PNG图片到本地。

画板支持鼠标和触摸屏操作，在电脑、平板和手机上都能正常使用。所有操作均在本地完成，不会上传到服务器，保障您的隐私安全。

🔍 扩展知识

HTML5 Canvas 是一种在网页上绘制图形的技术，它通过 JavaScript 在浏览器中直接渲染2D图形。Canvas 技术广泛应用于游戏开发、数据可视化、图像处理等领域。

在线画板使用 Canvas 2D 上下文进行绘图，支持路径绘制、矩形、圆形、线条等多种图形操作。相比传统的 Flash 技术，Canvas 无需插件，性能更好，兼容性更广。

画板的撤销/重做功能通过保存绘图历史记录实现，每次绘图操作后会将当前画布状态保存到历史栈中。这种方式虽然会占用一定内存，但对于一般用途的画板来说完全足够。

❓ 常见问题

在线画板的数据会上传到服务器吗？
不会。所有绘图操作都在浏览器本地完成，使用HTML5 Canvas技术，数据不会上传到任何服务器，保障您的隐私安全。
支持哪些绘图工具？
支持画笔、直线、矩形、圆形、橡皮擦五种工具。还可以调节画笔颜色、粗细、画布背景色，支持撤销/重做操作。
可以导出画作为图片吗？
可以。支持将画作导出为PNG格式的图片，您可以保存到本地设备，也可以直接复制到剪贴板。
在线画板支持手机使用吗？
支持。画板采用响应式设计，在手机和平板设备上也能正常使用。同时支持触摸操作，方便移动设备绘图。
画作不小心被清空了可以恢复吗？
可以。画板支持撤销/重做功能，即使不小心清空了画布，也可以通过撤销按钮恢复之前的绘制内容。建议定期导出保存重要作品。
画布大小可以调整吗？
画布会自动适应屏幕大小，填满可用区域。在移动设备上会自动缩放，确保最佳绘图体验。
