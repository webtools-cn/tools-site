使用说明

输入二次方程 ax² + bx + c = 0 的系数 a、b、c，点击计算即可求解。

工具会自动计算判别式 Δ = b² - 4ac，并显示实根或虚根结果。

所有计算在你的浏览器中完成，数据不会上传到任何服务器。

常见问题

❓ 常见问题

什么是一元二次方程？

形如ax²+bx+c=0(a≠0)的方程。a、b、c为系数，x为未知数。解为x=(-b±√(b²-4ac))/2a。

判别式有什么意义？

判别式Δ=b²-4ac决定方程根的情况：Δ>0有两个不等实根，Δ=0有一个重根，Δ<0无实根（有两个复数根）。

如何使用求根公式？

输入a、b、c三个系数，工具自动计算判别式和根。注意a不能为0，否则不是二次方程。

什么是虚数根？

当判别式Δ<0时，方程没有实数根，根包含虚数单位i。例如x²+1=0的根为x=±i。本工具支持复数根的计算和显示。

数据会上传服务器吗？

不会。所有计算在浏览器本地完成，无需联网即可使用。
