Common Questions

Is Free ToolBase really free?

Yes, all 1300+ tools are completely free with no signup required. We sustain the site through advertising revenue.

Is my data safe?

Absolutely. All processing happens in your browser. We never upload, store, or transmit your data to any server. Your files and text never leave your device.

Can I request a new tool?

Yes! Open a feature request on our GitHub Issues page and we'll consider adding it to our collection.

Do you offer an API?

Not currently, but we're exploring this option. Let us know if you're interested via GitHub Issues.

Response Time

We typically respond to emails within 48 hours. GitHub Issues are reviewed regularly. For urgent matters, please use email.
