❓ 常见问题

什么是 HTML Image Map？

HTML Image Map（图片热区地图）允许在一张图片上定义多个可点击区域，每个区域可以链接到不同的 URL。常用于地图导航、产品展示、交互式信息图等场景。通过 <map> 和 <area> 标签实现。

三种形状分别适合什么场景？

矩形(rect)适合按钮、卡片等规则区域。圆形(circle)适合头像、图标、坐标点等。多边形(poly)适合不规则形状，如地图上的国家、产品图中的特定零件。可根据实际需求灵活选择。

热区坐标在不同屏幕下会失效吗？

热区坐标是基于原始图片尺寸的像素值。如果图片缩放显示，坐标仍然正确（因为坐标相对于图片本身）。建议在使用时不要使用 srcset 等方式改变图片的实际渲染尺寸。
