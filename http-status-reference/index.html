常见问题

Q1: 301和302重定向有什么区别？对SEO有什么影响？

301是永久重定向，搜索引擎会将权重转移到新URL，旧URL从索引中移除。302是临时重定向，搜索引擎保留旧URL的索引，不转移权重。网站迁移或URL变更应使用301，临时维护或A/B测试使用302。错误使用302代替301会导致SEO权重无法传递。

Q2: 404页面如何优化SEO？

1）自定义404页面，提供导航链接和搜索框；2）避免返回软404（页面返回200但内容是错误信息）；3）定期检查并修复404链接；4）在Google Search Console中监控404错误；5）对已移除的页面返回410（Gone）而非404，帮助搜索引擎更快从索引中移除。

Q3: 503和500错误有什么区别？

500 Internal Server Error表示服务器遇到意外错误，通常是代码bug或配置问题。503 Service Unavailable表示服务器暂时无法处理请求（如过载或维护），但预计会恢复。503应配合Retry-After头使用，告知客户端何时重试。搜索引擎对503更宽容，会稍后重试；对500则可能降低抓取频率。

301是永久重定向，转移SEO权重；302是临时重定向，不转移权重。网站迁移用301，临时维护用302。

自定义404页面，避免软404，定期修复404链接，在GSC中监控，对已移除页面返回410。

500是意外错误（代码bug），503是暂时不可用（维护/过载）。搜索引擎对503更宽容。
