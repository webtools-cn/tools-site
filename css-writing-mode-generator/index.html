📖 CSS writing-mode 详解

writing-mode 属性定义了文本在块级元素中的排列方向。它是实现中文竖排、古籍排版、多语言文字混排的关键属性。

常用值

  • horizontal-tb — （默认）内容从左到右水平排列，行从上到下
  • vertical-rl — 内容从上到下垂直排列，行从右到左（传统中文竖排）
  • vertical-lr — 内容从上到下垂直排列，行从左到右（蒙古文等）

搭配 text-orientation

text-orientation 控制垂直书写模式下字符的方向。使用 upright 可以让拉丁字母保持直立，mixed 允许文字自然旋转。
