🔤 字符类

.任意字符（除换行符）
\\d数字 [0-9]
\\D非数字 [^0-9]
\\w单词字符 [a-zA-Z0-9_]
\\W非单词字符
\\s空白字符（空格、制表符、换行）
\\S非空白字符
[abc]a、b或c中的任意一个
[^abc]不是a、b或c
[a-z]字符范围a到z

🔢 量词

*0次或多次（贪婪）
+1次或多次（贪婪）
?0次或1次（可选）
{n}恰好n次
{n,}n次或更多
{n,m}n到m次
*?0次或多次（懒惰）
+?1次或多次（懒惰）
??0次或1次（懒惰）

📍 锚点与边界

^字符串/行开头
$字符串/行结尾
\\b单词边界
\\B非单词边界

📦 分组与引用

(abc)捕获组
(?:abc)非捕获组
\\1反向引用第1组
|或（OR）

👁️ 环视

(?=abc)正向前瞻
(?!abc)负向前瞻
(?<=abc)正向后顾
(?<!abc)负向后顾

什么是正则表达式速查表？

正则表达式速查表是一个免费在线交互式参考工具。它提供正则元字符、量词、锚点、分组、环视和标志的综合快速参考表，结合实时模式测试器。点击参考表中的任何模式即可插入到测试器中。所有处理在浏览器本地完成。

核心功能

  • 实时测试器：实时测试正则模式
  • 点击插入：点击速查表中的模式直接插入测试器
  • 常用模式：邮箱、URL、电话和IP验证的预构建模式
  • 匹配高亮：测试字符串中所有匹配项的可视化高亮
  • 分组捕获：显示每个匹配的捕获组

❓ 常见问题

什么是正则表达式？

定义搜索模式的字符序列，用于字符串匹配、验证和文本解析。

贪婪vs懒惰量词？

贪婪(*, +)尽可能多匹配。懒惰(*?, +?)尽可能少匹配。

什么是前瞻/后顾？

零宽度断言，检查后面/前面断言，检查但不消耗字符。

数据会上传吗？

不会。所有正则测试在浏览器本地完成。

支持哪种正则风格？

JavaScript ECMAScript RegExp。大多数模式与PCRE、Python和Java兼容。

如何转义特殊字符？

使用反斜杠：\\.匹配字面点号，\\\\匹配字面反斜杠。
