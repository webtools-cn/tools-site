❓ 常见问题

CSS Loading Spinner生成器能创建哪些类型的加载动画？
支持40+种加载动画样式，包括：旋转圆环(spinner)、双环旋转(double ring)、脉冲圆点(pulse dot)、波纹扩散(ripple)、弹跳点(bounce)、进度条(progress bar)、条形加载(bar loader)、菱形旋转(diamond)、心电图(eKG)、无限循环(infinity)等。每种样式均可自定义颜色、大小、旋转速度，满足不同设计风格需求。
生成的CSS代码可以直接在生产环境中使用吗？
完全可以。生成的CSS代码仅包含@keyframes关键帧动画和基础样式，无任何外部依赖。支持所有现代浏览器（Chrome/Firefox/Safari/Edge），代码体积小（通常<2KB），可直接复制粘贴到项目中使用。已包含-webkit-浏览器前缀以确保兼容性。
Loading Spinner的动画性能怎么样？
所有生成的spinner动画均使用CSS transform和opacity属性驱动，浏览器会对这些属性进行GPU加速合成，性能开销极低（通常<1% CPU）。相比GIF或JavaScript动画方案，纯CSS loading动画占用资源更少、动画更流畅，不会影响页面其他交互的响应速度。
如何将生成的Spinner集成到React/Vue项目中？
集成非常简单。在React中：将生成的CSS代码放入组件的.css文件，然后在JSX中直接使用对应的HTML结构即可。在Vue中：将CSS代码放入<style scoped>标签或单独的CSS文件，在模板中使用对应类名。本工具生成的CSS代码是纯CSS，与任何前端框架（React/Vue/Angular/Svelte）兼容。
数据会被上传到服务器吗？
绝对不会。本工具完全在浏览器本地运行，所有样式生成、预览渲染均在客户端完成。无需注册、无需登录、不上传任何数据到服务器，你的设计参数和生成的代码仅保存在你的浏览器中。
支持自定义Spinner的哪些属性？
支持自定义以下属性：主颜色（通过颜色选择器或HEX值输入）、次要颜色（用于双色渐变或多色样式）、尺寸（从24px到128px的滑块调节，适配按钮/卡片/全屏等不同场景）、动画速度（从0.5s到5s）、边框粗细（适用于环型spinner）。所有修改实时预览，所见即所得。
