扩展知识

CSS动画通过@keyframes定义关键帧，animation属性控制播放。缓动函数决定动画速度曲线：ease慢-快-慢，linear匀速，ease-in慢启动，ease-out慢结束。cubic-bezier可自定义曲线。animation-fill-mode控制动画前后样式：forwards保持结束状态，backwards应用起始状态。will-change属性可提示浏览器优化渲染性能。
