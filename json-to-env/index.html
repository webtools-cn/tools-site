💡 使用说明

本工具支持 JSON ↔ .env 双向转换，纯前端本地处理，数据不上传。

.env 文件格式：每行一个变量，格式为 KEY=VALUE，支持注释（以 # 开头）。

嵌套对象：开启"展开嵌套"后，{"db":{"host":"localhost"}} 将转为 DB__HOST=localhost。

常见用途：将Docker Compose、Kubernetes ConfigMap、CI/CD配置中的JSON转为.env文件，方便本地开发和测试。
