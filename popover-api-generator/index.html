常见问题


Popover API代码生成器是什么？
可视化创建CSS Popover API弹出层组件的在线工具，配置参数后自动生成HTML+CSS代码。


什么是Popover API？
浏览器原生弹出层API（Chrome 114+），通过HTML属性popover即可创建弹出层，无需JS库。


Popover和Tooltip的区别？
Popover是交互式弹出层，可包含任意HTML；Tooltip是信息提示，通常只有文本。


支持哪些浏览器？
Chrome 114+、Edge 114+、Safari 17+。Firefox暂不支持。


支持哪些动画？
淡入淡出、缩放、滑入、翻转等CSS @starting-style动画。
