📖 使用教程

本在线图片批量调整大小工具操作简单直观，以下是详细的使用指南：

上传图片：点击上传区域或直接将图片拖拽到页面中，支持一次选择多张图片。工具支持 JPG、PNG、WebP、GIF、BMP 等所有浏览器支持的图片格式。上传后图片列表会显示每张图片的缩略图、原始尺寸和文件大小。

设置目标尺寸：在调整设置面板中，您可以设置目标宽度或高度（单位：像素）。如果只设置宽度，高度会根据原始宽高比自动计算；反之亦然。如果您希望按固定比例缩放，可以在「缩放百分比」中输入百分比数值（如 50 表示缩小到原来的一半）。

输出格式与质量：默认保持原格式输出，您也可以手动指定为 JPEG、PNG 或 WebP。当选择 JPEG 或 WebP 时，可以通过质量滑块调节压缩级别（1%-100%），建议设置在 80%-90% 之间以获得最佳质量与文件大小的平衡。PNG 格式为无损输出，不支持质量调节。

批量处理与下载：点击「批量处理全部」按钮后，工具会使用 HTML5 Canvas 在本地逐张处理图片。处理完成后，每张图片右侧会显示新尺寸和新文件大小，并出现「下载」按钮。点击「下载全部」可以一次性保存所有处理后的图片。

🎯 应用场景

图片尺寸调整是日常工作和生活中极为常见的需求，以下是几个典型的应用场景：

电商平台商品图标准化：各大电商平台对商品主图、详情图的尺寸有严格要求（如淘宝主图 800×800px、京东 800×800px）。使用本工具可以批量将拍摄的高分辨率原图统一缩放到平台要求的尺寸，大幅提升上架效率。

网站性能优化：网页加载速度直接影响用户体验和搜索引擎排名。将网站中的大图批量调整为合适的显示尺寸（如文章配图 1200px 宽），可以显著减少文件大小、降低带宽消耗，提升 Core Web Vitals 中的 LCP 指标。

社交媒体头像与封面：不同社交平台对头像和封面的尺寸要求各不相同。使用本工具可以快速将照片裁剪调整到微信头像（建议 300×300px）、微博封面（建议 980×300px）、LinkedIn 头像（400×400px）等指定尺寸。

🔬 扩展知识

图片缩放算法：浏览器 Canvas 在缩放图片时默认使用双线性插值（Bilinear Interpolation），这是一种在速度和质量之间取得平衡的算法。对于缩小操作，效果通常很好；但对于大幅放大，可能会出现锯齿和模糊。如果需要更高质量的放大效果，可以使用双三次插值（Bicubic）或 Lanczos 算法，但这些需要借助专门的图像处理库。

像素密度与物理尺寸：图片的「尺寸」通常指像素尺寸（如 1920×1080），而非物理打印尺寸。打印尺寸由像素尺寸和 DPI（每英寸点数）共同决定。例如 3000×2000 像素的图片，在 300 DPI 下打印尺寸约为 10×6.7 英寸（25.4×17cm）。本工具调整的是像素尺寸，如需调整打印尺寸请使用专业的图像编辑软件。

有损 vs 无损格式：JPEG 和 WebP 是有损压缩格式，通过丢弃部分人眼不敏感的信息来减小文件大小，适合照片类图片。PNG 是无损格式，保留所有像素信息并支持透明通道，适合图标、截图、设计稿。选择输出格式时，建议照片用 JPEG/WebP，透明图形用 PNG。

❓ 常见问题 (FAQ)

图片调整大小后会变模糊吗？

放大图片通常会导致一定程度的质量损失，因为浏览器需要在像素之间进行插值计算。缩小图片则一般不会明显损失质量。如果需要大幅放大，建议使用专业的AI超分辨率工具。本工具在缩小图片时采用高质量双线性插值，效果优良。

可以批量处理多少张图片？

理论上没有数量上限，但实际处理能力取决于您设备的内存和性能。建议单次批量处理不超过50张图片，每张不超过10MB，以获得最佳体验。

图片会上传到服务器吗？

绝对不会。本工具是纯前端实现，所有图片处理都在您的浏览器中通过HTML5 Canvas完成，图片数据不会离开您的设备，隐私绝对安全。

支持哪些图片格式？

支持所有浏览器可读取的图片格式，包括JPEG、PNG、WebP、GIF、BMP、SVG等。导出时可选择保持原格式或转换为JPEG、PNG、WebP。

如何保持图片的宽高比？

默认开启了保持宽高比选项。当您设置宽度时，高度会自动按比例计算；设置高度时，宽度也会自动按比例计算。如需非比例缩放，可以关闭保持宽高比开关。

为什么缩小后的图片文件变小了？

这是因为像素数量减少了。例如将4000×3000像素的图片缩小到800×600，像素数量减少了96.25%，即使质量设置不变，文件大小也会大幅减小。

导出的JPEG质量设置多少合适？

一般建议80%-90%是质量和文件大小的最佳平衡点。70%以下可能会出现明显的压缩artifacts（块状失真），95%以上文件大小增加明显但画质提升有限。

GIF动图调整大小后会保留动画吗？

本工具使用Canvas处理图片，目前会将GIF转换为静态帧导出。如需保留动画，建议使用专门的GIF处理工具。导出时可以选择PNG格式以保留透明背景。
