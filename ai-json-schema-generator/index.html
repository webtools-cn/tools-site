AI JSON Schemaз”ҹжҲҗеҷЁиғҪеҒҡд»Җд№Ҳпјҹ

AI JSON Schemaз”ҹжҲҗеҷЁжҳҜдёҖж¬ҫдё“дёәLLMз»“жһ„еҢ–иҫ“еҮәи®ҫи®Ўзҡ„е…Қиҙ№еңЁзәҝе·Ҙе…·пјҢиғҪеӨҹйҖҡиҝҮеҸҜи§ҶеҢ–з•Ңйқўеҝ«йҖҹз”ҹжҲҗз¬ҰеҗҲеҗ„AIе№іеҸ°и§„иҢғзҡ„JSON SchemaгҖӮж”ҜжҢҒOpenAI Function CallingгҖҒAnthropic Tool UseгҖҒж ҮеҮҶJSON Schemaе’ҢйҖҡз”Ёе·Ҙе…·и°ғз”Ёеӣӣз§Қиҫ“еҮәж јејҸпјҢдёҖй”®еҲҮжҚўж— йңҖжүӢеҠЁдҝ®ж”№гҖӮжүҖжңүз”ҹжҲҗеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚдёҠдј гҖӮ

ж ёеҝғеҠҹиғҪ

  • еӣӣз§Қиҫ“еҮәж јејҸпјҡOpenAI Function CallingгҖҒAnthropic Tool UseгҖҒж ҮеҮҶJSON SchemaгҖҒйҖҡз”Ёе·Ҙе…·и°ғз”ЁпјҢдёҖй”®еҲҮжҚў
  • еҸҜи§ҶеҢ–еӯ—ж®өзј–иҫ‘пјҡйҖҡиҝҮиЎЁеҚ•ж·»еҠ еҸӮж•°еӯ—ж®өпјҢи®ҫзҪ®зұ»еһӢгҖҒжҸҸиҝ°гҖҒеҝ…еЎ«гҖҒжһҡдёҫеҖјгҖҒй»ҳи®ӨеҖј
  • еөҢеҘ—еҜ№иұЎж”ҜжҢҒпјҡж”ҜжҢҒд»»ж„ҸеұӮзә§зҡ„еөҢеҘ—еҜ№иұЎе’Ңж•°з»„пјҢж»Ўи¶іеӨҚжқӮж•°жҚ®з»“жһ„йңҖжұӮ
  • жһҡдёҫзәҰжқҹпјҡдёәstring/numberзұ»еһӢж·»еҠ enumжһҡдёҫеҖјпјҢйҷҗеҲ¶LLMиҫ“еҮәиҢғеӣҙ
  • е®һж—¶йў„и§Ҳпјҡзј–иҫ‘еӯ—ж®өж—¶е®һж—¶жӣҙж–°з”ҹжҲҗзҡ„Schema
  • дёҖй”®еӨҚеҲ¶/дёӢиҪҪпјҡеӨҚеҲ¶дёәJSONжҲ–дёӢиҪҪдёәж–Үд»¶пјҢзӣҙжҺҘз”ЁдәҺAPIи°ғз”Ё

дҪҝз”Ёж•ҷзЁӢ

1. иҫ“е…ҘеҮҪж•°еҗҚз§°е’ҢжҸҸиҝ°пјҢйҖүжӢ©иҫ“еҮәж јејҸгҖӮ

2. зӮ№еҮ»"ж·»еҠ еӯ—ж®ө"ж·»еҠ еҸӮж•°пјҢи®ҫзҪ®еҗҚз§°гҖҒзұ»еһӢгҖҒжҸҸиҝ°гҖҒжҳҜеҗҰеҝ…еЎ«гҖӮ

3. еҰӮйңҖжһҡдёҫеҖјпјҢеңЁenumеҲ—иҫ“е…ҘйҖ—еҸ·еҲҶйҡ”зҡ„еҖјпјҲеҰӮпјҡsunny,cloudy,rainyпјүгҖӮ

4. еҰӮйңҖеөҢеҘ—еҜ№иұЎпјҢзӮ№еҮ»"ж·»еҠ еөҢеҘ—еҜ№иұЎ"пјҢз„¶еҗҺж·»еҠ еӯҗеӯ—ж®өгҖӮ

5. зӮ№еҮ»"з”ҹжҲҗSchema"жҹҘзңӢз»“жһңпјҢеӨҚеҲ¶жҲ–дёӢиҪҪдҪҝз”ЁгҖӮ

еә”з”ЁеңәжҷҜ

еңәжҷҜ1пјҡOpenAI Function Calling

дёәGPT-4зӯүжЁЎеһӢе®ҡд№үе·Ҙе…·еҮҪж•°пјҢи®©жЁЎеһӢж №жҚ®з”ЁжҲ·иҫ“е…ҘиҮӘеҠЁйҖүжӢ©и°ғз”ЁеҗҲйҖӮзҡ„еҮҪж•°е№¶з”ҹжҲҗз»“жһ„еҢ–еҸӮж•°гҖӮ

еңәжҷҜ2пјҡAnthropic Tool Use

дёәClaudeжЁЎеһӢе®ҡд№үе·Ҙе…·пјҢе®һзҺ°з»“жһ„еҢ–зҡ„е·Ҙе…·и°ғз”Ёе’ҢеҸӮж•°з”ҹжҲҗгҖӮ

еңәжҷҜ3пјҡLLMз»“жһ„еҢ–иҫ“еҮә

йҖҡиҝҮJSON SchemaзәҰжқҹLLMзҡ„иҫ“еҮәж јејҸпјҢзЎ®дҝқз”ҹжҲҗзҡ„еҶ…е®№з¬ҰеҗҲйў„е®ҡд№үзҡ„ж•°жҚ®з»“жһ„пјҢдҫҝдәҺзЁӢеәҸеҢ–еӨ„зҗҶгҖӮ

еңәжҷҜ4пјҡAPIжҺҘеҸЈе®ҡд№ү

еҝ«йҖҹз”ҹжҲҗAPIжҺҘеҸЈзҡ„еҸӮж•°SchemaпјҢз”ЁдәҺж–ҮжЎЈз”ҹжҲҗгҖҒеҸӮж•°йӘҢиҜҒе’ҢжҺҘеҸЈжөӢиҜ•гҖӮ

жү©еұ•зҹҘиҜҶ

JSON SchemaжҳҜжҸҸиҝ°JSONж•°жҚ®з»“жһ„зҡ„IETFж ҮеҮҶпјҲdraft 2020-12пјүпјҢеңЁAIйўҶеҹҹиў«е№ҝжіӣз”ЁдәҺзәҰжқҹLLMзҡ„иҫ“еҮәж јејҸгҖӮOpenAIзҡ„Function CallingйҖҡиҝҮtoolsеҸӮж•°дј йҖ’JSON SchemaпјҢжЁЎеһӢж №жҚ®Schemaз”ҹжҲҗз¬ҰеҗҲж јејҸзҡ„еҸӮж•°гҖӮAnthropicзҡ„Tool UseйҮҮз”Ёзұ»дјјжңәеҲ¶гҖӮз»“жһ„еҢ–иҫ“еҮәзҡ„е…ій”®дјҳеҠҝпјҡ1пјүзЎ®дҝқиҫ“еҮәж јејҸеҸҜйў„жөӢпјҢдҫҝдәҺзЁӢеәҸеҢ–еӨ„зҗҶпјӣ2пјүеҮҸе°‘е№»и§үпјҢжЁЎеһӢеңЁSchemaзәҰжқҹдёӢжӣҙеҮҶзЎ®пјӣ3пјүж”ҜжҢҒеӨҚжқӮеөҢеҘ—з»“жһ„пјҢж»Ўи¶ідёҡеҠЎйңҖжұӮгҖӮжңҖдҪіе®һи·өпјҡеӯ—ж®өжҸҸиҝ°иҰҒжё…жҷ°е…·дҪ“пјҢжһҡдёҫеҖјиҰҒиҰҶзӣ–жүҖжңүеҸҜиғҪпјҢеҝ…еЎ«еӯ—ж®өиҰҒжңҖе°ҸеҢ–пјҢеөҢеҘ—еұӮзә§дёҚи¶…иҝҮ3еұӮгҖӮ

еёёи§Ғй—®йўҳ

д»Җд№ҲжҳҜJSON Schemaпјҹ

JSON SchemaжҳҜжҸҸиҝ°JSONж•°жҚ®з»“жһ„зҡ„и§„иҢғпјҢе®ҡд№үдәҶеӯ—ж®өзұ»еһӢгҖҒеҝ…еЎ«йЎ№гҖҒжһҡдёҫеҖјгҖҒй»ҳи®ӨеҖјзӯүзәҰжқҹгҖӮеңЁAIйўҶеҹҹпјҢJSON Schemaз”ЁдәҺжҢҮеҜјLLMз”ҹжҲҗз¬ҰеҗҲйў„жңҹж јејҸзҡ„з»“жһ„еҢ–иҫ“еҮәгҖӮ

ж”ҜжҢҒе“Әдәӣиҫ“еҮәж јејҸпјҹ

ж”ҜжҢҒж ҮеҮҶJSON SchemaгҖҒOpenAI Function Callingж јејҸгҖҒAnthropic Tool Useж јејҸгҖҒд»ҘеҸҠйҖҡз”Ёе·Ҙе…·и°ғз”Ёж јејҸгҖӮдёҖй”®еҲҮжҚўпјҢж— йңҖжүӢеҠЁдҝ®ж”№гҖӮ

еҰӮдҪ•ж·»еҠ еөҢеҘ—еӯ—ж®өпјҹ

еңЁеӯ—ж®өеҲ—иЎЁдёӯпјҢе°Ҷзұ»еһӢйҖүжӢ©дёәobjectпјҢз„¶еҗҺзӮ№еҮ»иҜҘеӯ—ж®өзҡ„ж·»еҠ еӯҗеӯ—ж®өжҢүй’®пјҢеҚіеҸҜж·»еҠ еөҢеҘ—еұһжҖ§гҖӮж”ҜжҢҒд»»ж„ҸеұӮзә§еөҢеҘ—гҖӮ

ж•°жҚ®дјҡдёҠдј еҲ°жңҚеҠЎеҷЁеҗ—пјҹ

дёҚдјҡгҖӮжң¬е·Ҙе…·йҮҮз”ЁзәҜеүҚз«ҜжҠҖжңҜпјҢжүҖжңүз”ҹжҲҗи®Ўз®—йғҪеңЁжөҸи§ҲеҷЁжң¬ең°е®ҢжҲҗпјҢж•°жҚ®з»қдёҚзҰ»ејҖжӮЁзҡ„и®ҫеӨҮгҖӮ

з”ҹжҲҗзҡ„SchemaеҸҜд»ҘзӣҙжҺҘз”ЁдәҺOpenAI APIеҗ—пјҹ

еҸҜд»ҘгҖӮйҖүжӢ©OpenAI Function Callingж јејҸеҗҺпјҢз”ҹжҲҗзҡ„JSONеҸҜзӣҙжҺҘдҪңдёәtoolsеҸӮж•°дј е…ҘOpenAI Chat Completions APIгҖӮ

ж”ҜжҢҒе“Әдәӣеӯ—ж®өзұ»еһӢпјҹ

ж”ҜжҢҒstringгҖҒnumberгҖҒintegerгҖҒbooleanгҖҒarrayгҖҒobjectе…ӯз§Қеҹәжң¬зұ»еһӢпјҢд»ҘеҸҠenumжһҡдёҫгҖҒdefaultй»ҳи®ӨеҖјгҖҒdescriptionжҸҸиҝ°зӯүзәҰжқҹгҖӮ
