📖 使用说明

JSON转PHP数组：在左侧输入标准的JSON数据，右侧自动生成对应的PHP数组代码。支持单层关联数组、多层嵌套数组、索引数组等所有JSON数据结构。

数据类型映射：

  • JSON字符串 → PHP单引号字符串（自动转义特殊字符）
  • JSON数字 → PHP数字（整数/浮点数）
  • JSON布尔值 → PHP true/false
  • JSON null → PHP null
  • JSON对象 → PHP关联数组
  • JSON数组 → PHP索引数组

默认使用PHP短数组语法 []（PHP 5.4+），可切换为传统 array() 语法。

❓ 常见问题

JSON和PHP数组有什么区别？

JSON是一种轻量级数据交换格式，使用文本表示结构化数据。PHP数组是PHP语言中的数据结构，可以同时作为索引数组和关联数组使用。本工具在两者之间自动转换。

生成的PHP代码可以直接使用吗？

可以。生成的PHP数组代码语法正确，可以直接粘贴到PHP项目中使用。如果开启「包含变量声明」，会生成 $data = [...] 的完整赋值语句。

支持多大的JSON数据？

纯前端处理，受浏览器内存限制。一般几百KB到几MB的JSON文件可以正常处理。超大文件建议在本地使用脚本转换。
