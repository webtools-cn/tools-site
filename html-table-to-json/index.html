可以转换多个表格吗？

本工具默认处理第一个表格。如果你的HTML中包含多个表格，可能需要分别提取每个表格的数据。

输出格式有什么不同？

JSON（数组）输出每行作为一个对象；JSON（行对象）输出每行作为一个数组；CSV输出逗号分隔的值，可用于Excel等电子表格软件。

表格数据会上传到服务器吗？

不会。所有解析和转换都在你的浏览器中本地完成，数据不会发送到任何服务器。

表头行怎么处理？

选择"首行作表头"会将第一行作为JSON的键名。选择"自动列名"会自动生成col1、col2等列名。
