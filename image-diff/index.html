常见问题 (FAQ)

图片对比工具支持哪些图片格式？

支持 PNG、JPG/JPEG、WebP、GIF、BMP 等常见图片格式。如果图片格式不同，工具会自动将其统一到 Canvas 中进行像素级对比。建议使用相同格式和尺寸的图片以获得最佳对比效果。

图片对比的精度如何？差异是如何检测的？

本工具采用像素级对比算法，逐像素比较两张图片的 RGB 值。当对应像素的颜色差异超过预设阈值（默认 10/255）时，该像素会被标记为差异区域。差异区域会以红色高亮显示，方便您快速定位变化。您也可以在设置中调整敏感度阈值。

对比结果可以保存吗？支持哪些导出方式？

支持3种导出方式：① 差异叠加图导出为 PNG 图片（点击「导出差异图」按钮）；② 并排对比截图可通过浏览器保存；③ 差异统计信息（差异像素数、差异百分比）可直接复制。所有处理均在本地完成，无任何数据上传。

上传的图片会被存储到服务器吗？

不会。本工具完全在浏览器本地运行，使用 HTML5 Canvas API 进行图片处理。您的图片仅存储在您的设备内存中，关闭页面后自动清除，不会上传至任何服务器。

图片有大小限制吗？最大支持多大尺寸？

理论上没有严格限制，但出于性能考虑，建议单张图片尺寸不超过 4096×4096 像素。超大图片（超过 5000×5000 像素）可能导致处理速度变慢或浏览器内存不足。如果您需要处理超大图片，建议先使用图片压缩工具缩小尺寸。
