渐变色生成器使用说明

渐变色生成器是一款免费在线工具，帮助设计师和开发者快速生成美观的 CSS 渐变色。支持线性渐变和径向渐变两种模式，可自由调整色标颜色和位置，实时预览效果并一键复制 CSS 代码或导出 PNG 图片。

线性渐变

线性渐变沿一条直线方向过渡颜色。通过调节角度（0°~360°）控制渐变方向，0° 为从左到右，90° 为从下到上，180° 为从右到左，270° 为从上到下。

径向渐变

径向渐变从中心点向四周扩散。可选择圆形或椭圆形，调整中心位置和扩散大小，营造从内到外的色彩过渡效果。

色标管理

每个渐变支持 2~5 个色标。点击色块选择颜色，拖动滑块调整位置（0%~100%）。色标会自动按位置排序，最少保留 2 个色标。

导出功能

生成的 CSS 代码可直接复制到样式表中使用。PNG 导出功能可以将渐变保存为图片，方便在设计软件中使用。

使用教程

生成渐变：选择渐变类型（线性/径向），然后通过色标管理区域设置颜色和位置。线性渐变可以调节角度；径向渐变可以调节中心位置、形状和大小。

使用预设：点击下方的预设模板卡片，可一键应用日落、海洋、霓虹等经典渐变配色。

复制 CSS：在「CSS 代码」区域点击「复制 CSS」按钮，即可将生成的 background 属性代码复制到剪贴板，直接粘贴到你的项目中。

导出图片：点击「导出 PNG」按钮，将当前渐变保存为高清 PNG 图片（1080×320 像素），方便在设计稿中使用。

常见问题 (FAQ)

CSS渐变有哪些类型？

CSS支持三种渐变：线性渐变（linear-gradient，沿直线方向过渡）、径向渐变（radial-gradient，从中心向外扩散）、锥形渐变（conic-gradient，围绕中心旋转）。每种都支持多色停靠点和角度调节。

如何让渐变在所有浏览器中正常显示？

现代浏览器（Chrome、Firefox、Safari、Edge）都完整支持CSS渐变。如需兼容旧浏览器，建议提供纯色作为fallback：background: #color; background: linear-gradient(...)。

渐变设计有哪些最佳实践？

1. 避免使用过多颜色停靠点（3-5个最佳）；2. 注意颜色过渡的自然性，避免突兀变化；3. 考虑文字可读性，在渐变背景上使用对比色文字；4. 使用角度和位置参数创造视觉层次感。
