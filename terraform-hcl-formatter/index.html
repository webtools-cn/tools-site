Terraform HCL格式化器能做什么？

Terraform HCL格式化器是一款免费在线工具，能够自动格式化Terraform配置文件。支持缩进对齐、等号对齐、块排序、注释保留等功能，让杂乱的HCL代码瞬间变得整洁规范。无需安装Terraform CLI，浏览器中即可完成格式化。所有处理在本地完成，代码绝不上传。

核心功能

  • 智能缩进：自动检测块层级，统一缩进风格（2空格/4空格/Tab）
  • 等号对齐：同一块内的赋值语句等号自动对齐
  • 块排序：按字母序或类型排序resource、data、module等块
  • 注释保留：格式化时保留所有#和//注释
  • 拖拽文件：支持拖拽.tf文件到输入框

使用教程

1. 粘贴HCL代码到输入框，或拖拽.tf文件。

2. 选择缩进风格、排序方式、是否对齐等号。

3. 点击"格式化"或Ctrl+Enter，立即查看格式化结果。

4. 复制或下载格式化后的代码。

应用场景

场景1：代码审查前整理

提交PR前格式化Terraform代码，确保团队代码风格统一，减少审查噪音。

场景2：遗留代码整理

扩展知识

HCL(HashiCorp Configuration Language)是HashiCorp设计的配置语言，语法特点：块结构用{}包裹，标签用字符串引号，赋值用=，支持for表达式、条件运算符和模板。Terraform官方格式化工具terraform fmt使用2空格缩进，不排序块。HCL2引入了类型系统和函数，支持dynamic块和variable验证。建议团队统一使用terraform fmt或本工具保持代码风格一致。

❓ 常见问题

什么是HCL？

HCL(HashiCorp Configuration Language)是HashiCorp开发的配置语言，用于Terraform等工具定义基础设施资源。语法介于JSON和YAML之间，支持注释、变量和表达式。

格式化会改变HCL的语义吗？

不会。格式化只调整空白、缩进和排列顺序，不改变任何逻辑或值。

支持Terraform 1.x语法吗？

支持Terraform 1.x的大部分HCL语法，包括resource、data、module、variable、output、locals等块类型。

数据安全吗？

完全安全。所有格式化在浏览器本地完成，HCL代码绝不上传服务器。

和terraform fmt有什么区别？

terraform fmt需要安装Terraform CLI，本工具在浏览器中运行无需安装。功能类似但部分高级格式化可能略有差异。

支持拖拽文件吗？

支持。将.tf文件拖拽到输入框即可自动读取内容。

如何处理大型Terraform项目？

建议逐文件格式化。对于模块化的项目，按模块分别处理效果最佳。
