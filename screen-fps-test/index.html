ℹ️ 关于FPS测试

FPS（Frames Per Second）即每秒帧数。这个工具使用浏览器的 requestAnimationFrame API 精确测量当前标签页的实际渲染帧率。

FPS参考标准

• 144+ FPS：电竞级流畅体验（需144Hz+显示器）

• 60 FPS：标准流畅体验（60Hz显示器）

• 30 FPS：基本可用，略有卡顿

• <30 FPS：明显卡顿，建议优化

❓ 常见问题

什么是FPS？

FPS（Frames Per Second）即每秒帧数，表示显示器或浏览器每秒能渲染的画面数量。60FPS为流畅标准，120FPS/144FPS为高刷新率，30FPS以下会感觉卡顿。

这个FPS测试准确吗？

使用浏览器的requestAnimationFrame API精确测量，误差极小。测试结果反映当前浏览器标签页的实际渲染帧率。

为什么我的FPS被限制在60？

浏览器默认将动画帧率同步到显示器刷新率。60Hz显示器FPS上限为60。高刷显示器（120Hz/144Hz/240Hz）可以看到更高FPS。

如何提高FPS？

关闭其他标签页、禁用浏览器扩展、使用高性能模式、确保显卡驱动更新、使用高刷新率显示器。
